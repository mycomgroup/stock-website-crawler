## 使用建议

1. **先读help文档** - 了解数据类型和基本概念
2. **查找接口文档** - 根据关键词搜索对应doc文件
3. **查看使用指南** - JQData使用指南中有详细示例
4. **注意数据更新** - 查看"JQData数据范围及更新时间"

## 文档命名规则

- `help_{category}_overview_{title}.md` - 帮助文档
- `doc_logon_{id}_overview_{title}.md` - 登录账号相关
- `doc_JQDatadoc_{id}_overview_{title}.md` - 数据接口文档

## 快速搜索

在 `doc/` 目录下使用 grep 搜索：

```bash
# 搜索包含"财务"的所有文档
grep -l "财务" doc/*.md

# 搜索包含"行情"的所有文档
grep -l "行情" doc/*.md

# 搜索具体接口
grep -l "get_price" doc/*.md
```

#### JQData介绍

JQData介绍

- JQData是聚宽数据团队专门为金融机构、学术团体和量化研究者们提供的本地量化金融数据服务。
- 使用JQData，可快速查看和计算金融数据，无障碍解决本地、Web、金融终端调用数据的需求。
- 历经多年沉淀，以及获取超50万宽客及超5000家知名量化机构投研交易验证。

JQData使用说明

- JQData适用Windows、Mac、Linux多种操作系统
- 支持python2、python3编程语言。
- 数据通过简洁的API方式提供，pip即可直接安装使用，挣脱使用束缚，实现更多场景。只需三行代码，即可随取随用~

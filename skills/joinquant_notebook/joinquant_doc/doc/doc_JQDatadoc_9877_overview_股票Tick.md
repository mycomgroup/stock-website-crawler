## 内容

#### 股票Tick

- 历史范围：2010年至今；

```python
get_ticks(security, start_dt, end_dt, count, fields, skip=True,df=True)
```

描述

- 股票部分， 支持 2010-01-01 至今的tick数据，提供买五卖五数据。每3秒一条数据，盘后15:00更新，24:00校对完成入库

- 用户如有需要使用tick数据的，可添加**微信号JQData02**申请试用或咨询开通，或发送邮件至jqdatasdk@joinquant.com

股票Tick参数

- security: 股票代码,不支持传入多个标的
- start_dt: 开始日期，格式为'YYYY-MM-DD HH:MM:SS'
- end_dt: 结束日期，格式为'YYYY-MM-DD HH:MM:SS', 与count参数共同约束返回的数据
- count: 取出自end_dt往前推多少条的tick数据, 如同时指定了start_dt参数则返回的数据不超过start_dt
- fields: 选择要获取的行情数据字段，默认为None，返回结果如下：
- skip:默认为True，过滤掉无成交变化的tick数据；skip=False时，2013-01-01开始的tick数据会保留无成交有盘口变化的tick数据
- df:指定返回的数据格式，默认为True，返回dataframe；df=False时返回一个np.ndarray

- 同时填入start_dt、end_dt和count参数的应用场景：比如取近1分钟30条数据

###### 股票tick返回结果

| 字段名 | 说明 | 字段类型 |
| --- | --- | --- |
| time | 时间 | datetime |
| current | 当前价 | float |
| high | 当日最高价 | float |
| low | 当日最低价 | float |
| volume | 累计成交量（股） | float |
| money | 累计成交额 | float |
| a1_v~a5_v | 五档卖量 | float |
| a1_p~a5_p | 五档卖价 | float |
| b1_v~b5_v | 五档买量 | float |
| b1_p~b5_p | 五档买价 | float |

##### 定期报告

```python
定期报告的含义
定期报告是强制在规定时间内披露的信息 , 包含一三季报及半年报,年报
年度报告**最晚**次年4月30日披露(四个月内), 一般要求次年一季报披露时间晚于年报
中报最晚当年8月30日披露(两个月内)
季报最晚4月30和10月31(一个月内)
一般披露高峰为规定时间的最后几天
一般未在规定时限内披露报告的公司会先停牌两个月, 后续仍无法提供的,将被强制退市，但也有特殊情况发生, 如19年疫情导致很多上市公司无法按时披露年报, 后来进行了补充
```

```python
单季度和报告期的区别
单季度/报告期指的是财务数据统计的范围周期，报告期数据的统计范围是年初(或去年末)至当前季度末，单季度数据的统计范围是季度初(或上季度末)至当前季度末。
资产负债表本身属于**截面性质**的数据，因此不区分单季度/报告期 ,其他两张报表均为**流量性质**的而数据，需要进行区分
一般上市公司只会披露报告期数据 ,极少披露单季度数据 , 单季度为数据商自行计算 
单季度算法 : 当前报告期值 –  上一报告期值, 第一季度不用减
**大多数涉及利润表/现金流量表科目的财务指标也需要区分单季度和报告期**
```

```python

调整报表(不否定原报表):
```

- 财务指标(TTM,YOY,QOQ)

财务指标一般按给定的公式, 使用三大报表中单个字段不同时期的值进行计算,如ttm等 ,或者使用多个字段同一时期计算而来的指标 ,如eps, roa 等

- TTM ,YOY,QOQ均属于指标的算法,除TTM不可用于资产负债表,三大报表(资产负债表,利润表,现金流量表)中的字段均可用
- TTM : （Trailing Twelve Months）滚动12个月的值, 用于剔除季节性的影响,一般算法为 当期未调整值 – 去年同期调整值 + 去年年度值
- YOY : （Year On Year）同比增长率 , 一般算法为 (当期未调整值 - 去年同期值)/abs(去年同期调整值)
- QOQ : （Quarter On Quarter） 环比增长率 , 一般算法为 (当期未调整值 – 上期未调整值) /abs(上期未调整值)

```python
合并报表和母公司报表
1、母公司报表
财报中的母公司指的是上市公司本身，母公司的报表也就是指归属上市公司的报表。母公司在编制财务报表时，把旗下所有控股子公司都纳入进来，但是没有考虑它们之间的关联交易。

2、合并报表
以母公司和其子公司单独编制的个别财务报表为基础，由母公司编制的，反映抵消集团内部往来账项后的集团合并财务状况和经营成果的财务报表。也就是合并报表是把其子公司与子公司之间，子公司与母公司之间的关联交易相互低消，而得出的合并数据
总体来说，母公司的单体财务报表对于投资者的参考价值要远远低于合并财务报表（母公司报表主要的作用为通过货币资金确定集团的资金管理方式，通过长期股权投资定位集团核心子公司等）。
对于投资者日常使用场景，一般看合并报表 ,  单季度只提供了合并报表，报告期是合并和母公司报表都有提供
```

```python
报表格式变更
财政部不定期会调整财报披露的格式 , 我们目前沿用的还是2007版本的 , 有些字段可能缺失或调账导致和最新的格式/数字不一致
```

##### 临时报告

涉及财务的不定期报告主要为业绩快报和业绩预告,当满足特定条件时才强制要求披露 , 否则为自愿披露

```python
业绩快报
创业板及中小板,如果年报预约披露时间在3月1日及之后 , 强制披露业绩快报

主要内容包括当年及上年同期主营业务收入、主营业务利润、利润总额、净利润、总资产、净资产、每股收益净资产收益率等数据和指标，同时披露比上年同期增减变动的百分比，**对变动幅度超过30%以上的项目，公司还应当说明原因**。
业绩快报制度提高了上市公司年度业绩信息披露的及时性， 有助于积极推进中小投资者的信息知情权，能够平衡因资金不对等而形成的信息不对称现象。此外，当财务数据出现差异30%以上的差异时，中小企业板上市公司需要予以说明，有利于中小投资者清晰地把握公司经营状态的变化，从而有效把握可能的机会与规避潜在的风险。
```

##### 内容补充

- 业绩预告和业绩快报的具体区别？ 业绩预告和业绩快报是上市公司“预先”披露业绩的两种方式，是在定期报告披露之前，上市公司对报告期的业绩提前进行预告或报告，目的是在正式财务报告披露之前，向市场和投资者传递公司的业绩变化情况。 投资者可以在每年年初，关注上市公司的信息披露，看上市公司是否对上一报告期的业绩作出业绩预告或快报。因为是预估，上市公司发布业绩预告往往是一个区间值，譬如归母净利润下限至上限，净利润增速的下限至上限，且数字的准确性无法完全保证。 业绩快报是上市公司在一定会计期间结束后、定期报告公告前，初步披露主要会计数据和经营指标，其字面意思突出一个“快”字。 相较业绩预告而言，业绩快报能呈现较准确的数值，可以提高信息披露的及时性和公平性。

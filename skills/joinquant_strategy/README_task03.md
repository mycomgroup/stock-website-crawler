# 任务03：卖出规则设计与测试 - 执行指南

## 任务概述
本任务为二板接力策略工程化的第三阶段，旨在通过系统化测试确定最优卖出规则，包括：
- 卖出时机选择
- 止盈规则设计
- 止损规则设计
- 持仓周期管理
- 熔断规则设计

## 测试脚本清单

### 1. 提示词3.1：卖出时机对比测试
**文件**: `task03_prompt31_sell_timing_test.py`
**测试内容**:
- 次日竞价卖出（9:15-9:25）
- 次日开盘卖出（9:30-9:35）
- 次日10:30卖出
- 次日13:30卖出
- 次日收盘卖出（基准）
- 次日最高价卖出（理想上限）

**输出**: 卖出时机对比测试报告

### 2. 提示词3.2：止盈规则测试
**文件**: `task03_prompt32_profit_target_test.py`
**测试内容**:
- 无止盈
- 固定止盈（+5%/+7%/+10%/+15%）
- 分时止盈（跌破分时均线/开盘价/回调50%）
- 板块止盈（龙头炸板/板块跌超2%）

**输出**: 止盈规则对比测试报告

### 3. 提示词3.3：止损规则测试
**文件**: `task03_prompt33_stop_loss_test.py`
**测试内容**:
- 无止损
- 固定止损（-3%/-5%/-7%/-10%）
- 分时止损（跌破开盘价/分时均线/加速下跌）
- 时间止损（10:30/13:30/14:30）
- 组合止损（-3%或10:30/-5%或13:30）

**输出**: 止损规则对比测试报告

### 4. 提示词3.4：持仓周期测试
**文件**: `task03_prompt34_holding_period_test.py`
**测试内容**:
- T+1卖出
- T+2卖出
- T+1或T+2（涨停持有）
- 动态持仓（盈利>5%持T+2/亏损当日卖）
- 长线持仓

**输出**: 持仓周期对比测试报告

### 5. 提示词3.5：熔断规则测试
**文件**: `task03_prompt35_circuit_breaker_test.py`
**测试内容**:
- 个股熔断（单票亏损5%/10%）
- 组合熔断（单日亏损>5%/连续亏损/收益<-5%）
- 情绪熔断（情绪骤变/涨跌停反转）
- 组合熔断规则

**输出**: 熔断规则对比测试报告

#### 官网VIP和本地数据的区别

JQData和jqdata的区别

- JQData，即jqdatasdk，指的是聚宽数据出品的本地量化数据接口，需要本地python环境内使用， 具体方法见JQData的API： 【sdk版本API文档】：https://www.joinquant.com/help/api/help?name=JQData
- jqdata：登录官网在研究环境下使用的包，只有在官网环境下才能调用 步骤：import jqdata 或 from jqdata import , 具体使用方法见官网API：https://www.joinquant.com/help/api/help?name=api

- 本地数据和官网在线环境是不同的业态线，付费服务不互通
- 官网VIP是官网在线环境的特色功能，主要指回测模拟交易等
- 本地数据指接口在本地运行，数据可以保存在本地，付费服务需要单独联系运营
- 本地数据机构服务咨询添加微信号JQData02
- 本地数据个人用户可扫码咨询

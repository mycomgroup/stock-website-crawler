- 期权合约资料 —— 上市至今,盘后18:00更新
- 期权日行情(查表) —— 上市至今,盘后更新20:10更新
- 期权风险指标 ——上市至今,下一交易日盘前8:05更新

###### 常见问题

为什么期权风险指标数据中商品期权只有delta数据，没有theta等数据?

A： 此数据来自交易所，上期所，郑商所及大商所都未披露此项数据，所以获取不到。ETF期权有完整的风险指标，上交所有披露。

# 申万行业

## 源URL

https://www.joinquant.com/help/api/doc?name=JQDatadoc&id=10282

## 描述

A：申万的行业代码在14年2月21有一次比较大改动，新增了11个行业代码：['801710'，'801720'，'801730'，'801740'，'801750'，'801760'，'801770'，'801780'，'801790'，'801880'，'801890']，弃用了6个行业代码：['801060'，'801070'，'801090'，'801100'，'801190'，'801220'] 所以新增行业代码在2014-02-21之前没有成分股，弃用的行业代码在2014-02-21之后没有成分股。

## 内容

#### 申万行业

###### 常见数据问题>>>

1、为什么有些申万行业代码没有成分股?

A：申万的行业代码在14年2月21有一次比较大改动，新增了11个行业代码：['801710'，'801720'，'801730'，'801740'，'801750'，'801760'，'801770'，'801780'，'801790'，'801880'，'801890']，弃用了6个行业代码：['801060'，'801070'，'801090'，'801100'，'801190'，'801220'] 所以新增行业代码在2014-02-21之前没有成分股，弃用的行业代码在2014-02-21之后没有成分股。

---

1. 为什么有些申万行业指数可以获取到未发布之前的行业行情数据?

A：数据来自申万 ，和中证指数刚好相反，申万行业指数基本都以1999-12-31为基日，并提供了自基日开始的行情数据 ， 是正常的(成分股是从发布日期后才有的)

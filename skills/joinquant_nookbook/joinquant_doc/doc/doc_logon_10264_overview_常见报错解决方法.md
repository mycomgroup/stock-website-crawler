#### 常见报错解决方法

JQData目前的服务器地址：39.107.190.114，端口号：7000

这个报错一般是由您网络问题造成的，常见解决方法如下:

- 检查下网络是否正常/稳定；
- 请检查下您是否使用代理或者网络被禁止了，目前不支持代理；
- 是否您公司对网络或者端口有限制；
- 尝试使用其他网络；
- 升级JQData至最新版本；
- 不建议高频的login和logout；

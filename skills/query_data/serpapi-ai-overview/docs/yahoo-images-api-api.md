# Yahoo! Images API

## 源URL

https://serpapi.com/yahoo-images-api

## 描述

Our Yahoo! Images API allows you to scrape SERP results from Yahoo! Images.

The API endpoint is https://serpapi.com/search?engine=yahoo_images Head to the playground for a live and interactive demo.

Images results can contain thumbnail, link, title, original, source, size, and position.

## API 端点

**Method**: `GET`
**Endpoint**: `https://serpapi.com/search`

# Google Play Store API

## 源URL

https://serpapi.com/google-play-api

## 描述

Our Google Play Apps Store API allows you to scrape SERP results from Google Play Apps Store.

Three types of searches you can do are: - Query Search: by using q parameter (Same Results with Google Play Games Store) - Category Search: by using apps_category parameter - Plain Search: without using q or apps_category parameters

The API endpoint is https://serpapi.com/search?engine=google_play Head to the playground for a live and interactive demo.

Some searches can contain the app_highlight. This block will usually show up when searching for a specific app

Some searches can contain the related_searches. This block will usually show up when searching for a specific app.

## API 端点

**Method**: `GET`
**Endpoint**: `https://serpapi.com/search`

# Yahoo! Videos API

## 源URL

https://serpapi.com/yahoo-videos-api

## 描述

Our Yahoo! Videos API allows you to scrape SERP results from Yahoo! Videos.

The API endpoint is https://serpapi.com/search?engine=yahoo_videos Head to the playground for a live and interactive demo.

Videos results can contain thumbnail, link, title, original, source, preview, date, duration and position.

## API 端点

**Method**: `GET`
**Endpoint**: `https://serpapi.com/search`

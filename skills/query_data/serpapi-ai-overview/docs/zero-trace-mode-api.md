ZeroTrace Mode

Keep your searches confidential.

When you enable ZeroTrace Mode, we won't store search parameters, search files, or search data on our servers. Once your search is done, it's as if it were never here.

https://serpapi.com/search.json?q=coffee&zero_trace=true

https://serpapi.com/search.json?q=coffee&zero_trace=true

Code to integrate

require "serpapi" 

client = SerpApi::Client.new(
  q: "coffee",
  zero_trace: "true",
  api_key: "secret_api_key"
)

results = client.search

require "serpapi" 

client = SerpApi::Client.new(
  q: "coffee",
  zero_trace: "true",
  api_key: "secret_api_key"
)

results = client.search

ZeroTrace Mode is easy to enable or disable. Just add zero_trace=true to the searches you want ZeroTrace on, and omit it for the searches you don't.

ZeroTrace mode can make it easier to comply with regulations (GDPR, HIPAA, and CCPA) and certifications (SOC 2 Type 1, SOC 2 Type 2, and ISO 27001).

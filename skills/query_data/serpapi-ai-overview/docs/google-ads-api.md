# Google Ad Results API

## 源URL

https://serpapi.com/google-ads

## 描述

When a Google search contains advertisements, they are parsed and exist within the ads array in the JSON output. Advertisements can contain block_position (top, bottom, middle and right), title, phone, link, sitelinks, source and more.

The API endpoint is https://serpapi.com/search?engine=google Head to the playground for a live and interactive demo.

## API 端点

**Method**: `GET`
**Endpoint**: `https://serpapi.com/search`

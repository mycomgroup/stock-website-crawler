# Nuevas preguntas frecuentes y cierres ajustados frente a cierres.

## 源URL

https://eodhd.com/financial-apis-blog/nuevas-preguntas-frecuentes-y-cierres-ajustados-frente-a-cierres

## 正文

Hoy hemos empezado nuestro nuevo proyecto de contenidos. Cada día respondemos a docenas de preguntas diferentes sobre aspectos muy diversos de los datos financieros. Hemos decidido combinar todas nuestras respuestas en varios grandes artículos y publicarlos bajo el tema “Preguntas frecuentes sobre finanzas”.

El primer artículo se titula “Cierre ajustado y cierre. ¿Cuál es la diferencia?” y explica la diferencia entre cierres y cierres ajustados. También hemos añadido el enlace al algoritmo de Chicago Booth, ya que utilizamos exactamente este método para calcular los valores ajustados. Esperamos que le resulte interesante y útil.

Estaremos encantados de recibir sus ideas para nuestros próximos artículos. Por supuesto, tenemos muchas ideas al respecto, pero es importante escuchar a nuestros abonados.

# Integración del proyecto Zorro con los datos históricos de EOD

## 源URL

https://eodhd.com/financial-apis-blog/integracion-del-proyecto-zorro-con-los-datos-historicos-de-eod

## 正文

Estamos orgullosos de anunciar la integración con Zorro-Project. Zorro Project es una herramienta de software gratuita de nivel institucional especializada en la investigación financiera y el trading algorítmico. Zorro Project es una de las herramientas más potentes para crear sus propias estrategias de trading algorítmico en un lenguaje de programación, como C, R o Python, que le ofrece la máxima flexibilidad y acceso a funciones avanzadas de estadística o aprendizaje automático. Nos complace formar parte de este servicio.

Y ahora es muy fácil empezar con nuestras API y Zorro, sólo tienes que elegirnos como proveedor de datos.

Encontrará más detalles sobre la integración en las noticias del Proyecto Zorro(https://zorro-project.com/manual/en/new.htm) y en la documentación: https://zorro-project.com/docs.php.

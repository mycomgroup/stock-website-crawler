# Calendario de eventos económicos API

## 源URL

https://eodhd.com/financial-apis-blog/calendario-de-eventos-economicos-api

## 正文

Hoy nos complace anunciar nuestra nueva API de Calendario de Eventos Económicos. La API de eventos económicos forma parte de la API fundamental y es accesible mediante la suscripción a Fundamental. Proporciona los acontecimientos pasados y futuros de todo el mundo, como velas al por menor, subastas de bonos, PMI Releases, y muchos otros datos de eventos económicos.

En esta API admitimos más de 30 países y más de 50 tipos de eventos económicos diferentes.

Puede obtener más información en nuestra página de documentación.

Un ejemplo de la salida:

También estamos abiertos a cualquier comentario y dispuestos a añadir cualquier nuevo campo y señal que le interese en nuestra API de noticias financieras. Envíenos un correo electrónico a support@eodhistoricaldata.com. Puede obtener más información en nuestra página de documentación.

# Información para titulares de acciones de EE.UU. y la UE.

## 源URL

https://eodhd.com/financial-apis-blog/informacion-para-titulares-de-acciones-de-ee-uu-y-la-ue

## 正文

Hoy anunciamos la nueva pieza de campos valiosos para nuestra API de Datos Fundamentales: los datos del titular mayoritario. Estos datos incluyen los 20 principales titulares institucionales y los 20 principales titulares de fondos de inversión de las empresas estadounidenses y de la UE que cotizan en la Bolsa de Nueva York, el Nasdaq y las bolsas europeas.

Los datos incluyen el nombre, la cantidad actual de acciones, el porcentaje del total de acciones, el porcentaje del total de activos, la fecha del informe y el cambio en comparación con el periodo anterior.

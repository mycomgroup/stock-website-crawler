# [Updated]  Descargador de fundamentos de Excel/VBA

## 源URL

https://eodhd.com/financial-apis-blog/updated-descargador-de-fundamentos-de-excel-vba

## 正文

Hoy hemos actualizado nuestro ejemplo Excel/VBA para la alimentación de datos Fundamentals con salida JSON. La primera vez que habíamos publicado la versión de este archivo de Excel hace alrededor de un año, pero las últimas versiones de Excel 2019 no funciona correctamente con la biblioteca VBA-JSON que utilizamos y ahora actualizamos el archivo con una nueva biblioteca para soportar las últimas versiones de Windows Excel y Excel para Mac.

Como ya hemos dicho. Si no estás familiarizado con ningún lenguaje de programación, tenemos varios ejemplos de Excel para ayudarte a descargar datos con nuestra API JSON y este ejemplo de Fundamentos de VBA es uno de ellos. Puede descargarlo aquí: Ejemplo Excel/VBA para Fundamentos.

Para Fundamentals JSON API utilizamos la librería abierta VBA-JSON de GitHub(https://github.com/VBA-tools/VBA-JSON). Conversión y análisis sintáctico de JSON para VBA (Windows y Mac Excel, Access y otras aplicaciones de Office).

El archivo probado en Windows Excel 2013, Excel 2017, Excel 2019 y Excel para Mac 2011, pero debe aplicarse a 2007 +.

# Marketplace addition: 40 000 Stock Market Logos API

## 源URL

https://eodhd.com/financial-apis-blog/marketplace-addition-40-000-stock-market-logos-api

## 正文

Users with access to our Fundamental data already use our collection of 40,000 stock market logos. It is a huge collection of PNG logos, covering 60 markets.

After numerous requests from users who are looking for only logos, we gave in and created a separate API dedicated to logos only. The package can be bought via a new feature – our own Marketplace. Learn more about the API here.

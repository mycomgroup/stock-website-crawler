# API de datos macroeconómicos y macroindicadores

## 源URL

https://eodhd.com/financial-apis-blog/api-de-datos-macroeconomicos-y-macroindicadores

## 正文

Con el tiempo, hemos publicado más de 30 indicadores macroeconómicos y API de datos macroeconómicos. La macroeconomía es una parte de la economía que se ocupa del funcionamiento, la estructura, el comportamiento y la toma de decisiones de una economía en su conjunto. Nuestro 
  API de datos macroeconómicos
 incluye economías regionales, nacionales y mundiales. Proporcionamos los datos de más de 30 indicadores macroeconómicos (la lista completa se encuentra en la documentación), como el PIB, las tasas de desempleo, la renta nacional, los índices de precios, las tasas de inflación, el consumo, el comercio internacional y muchos otros indicadores significativos.

La API de datos macroeconómicos forma parte de la API fundamental y es accesible mediante suscripción a Fundamental. A continuación puede ver la inflación anual para Francia en porcentajes, el indicador es ‘inflation_consumer_prices_annual’.

Obtenga más información sobre más de 30 indicadores macroeconómicos en nuestra documentación para 
  API de datos macroeconómicos
.

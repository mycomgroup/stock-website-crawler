# Matlab API Connector y ¡Feliz Año Nuevo!

## 源URL

https://eodhd.com/financial-apis-blog/matlab-api-connector-y-feliz-ano-nuevo

## 正文

Le deseamos una Feliz Navidad y un Próspero Año Nuevo. Y como pequeño regalo, nos complace presentar Matlab Connector para nuestra API. El uso es muy sencillo, como se explica en nuestra documentación para Matlab Connector con ejemplos.

Puede encontrar más detalles en nuestra documentación o en el sitio web de Matlab indocumentado: https://UndocumentedMatlab.com/EODML/.

El conector había sido desarrollado por Yair Altman. Yair Altman es un reconocido experto en Matlab con 30 años de experiencia. Yair ha publicado dos libros de texto sobre Matlab y es miembro del consejo asesor de MathWorks. Es muy conocido por su sitio web UndocumentedMatlab y sus contribuciones en foros públicos.

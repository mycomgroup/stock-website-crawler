# Get 40,000+ Stock Market Company Logos

## 源URL

https://eodhd.com/financial-apis-blog/40000-company-logos

## 正文

We are excited to share with you our “Grande Collection” – 40,000 stock market logos from companies across 60+ different exchanges. It is available via our new 40 000 Stock Market Logos API or as one time .zip file download.

To get company logos, we offer all logos in a single ZIP file package for an additional cost. Contact our support team via email or live chat.

If you’re in search of a specific logo or want to test our collection, a logo can be directly downloaded from our website. Simply append “/img/logos/US/MSFT.png” to our website URL “https://eodhd.com“, where “MSFT” represents the ticker name for Microsoft and “US” is the stock exchange code, to easily access the desired logo. A ticker symbol is a unique market code assigned to a company, and you can find the appropriate ticker for any company using our search feature on the main page. The exchange code can be found here.

Here is a direct link to the MSFT logo: https://eodhd.com/img/logos/US/MSFT.png

If you’re already using our EODHD API for financial data, you can obtain company logos alongside the Fundamentals Data Feed (All-in-one and Fundamental Data plans). The ‘LogoURL’ parameter facilitates this. Learn more here.

# API de operaciones con información privilegiada

## 源URL

https://eodhd.com/financial-apis-blog/api-de-operaciones-con-informacion-privilegiada

## 正文

Hoy hemos publicado una API independiente para operaciones con información privilegiada. Los datos están disponibles para todas las empresas estadounidenses que presentan el Formulario 4. Las operaciones con información privilegiada implican la compraventa de acciones de una empresa pública por parte de alguien que, por cualquier motivo, dispone de información material no pública sobre dichas acciones. Recientemente hemos añadido estos datos como parte de Fundamentals API, pero por muchas peticiones, hemos separado la alimentación de datos con más opciones.

# API de acciones Ejemplo de código JAVA

## 源URL

https://eodhd.com/financial-apis-blog/api-de-acciones-ejemplo-de-codigo-java

## 正文

Uno de nuestros suscriptores, Russel Brown, escribió el ejemplo Java y amablemente lo ofreció a aquellos de nuestros suscriptores que utilizan la API JAVA para datos bursátiles. Puede descargar el código aquí: EODHistoricalDataDownloader.java o en nuestra página de documentación con otros ejemplos Java. A continuación ofrecemos más detalles sobre el código y el autor. ¡Gracias, Russel!

Este programa ha sido escrito por Russell Brown y se publica bajo la LICENCIA PÚBLICA GENERAL GNU Versión 3 en colaboración con EOD Historical Data. Su función prevista es utilizar los servicios prestados a través del sitio web https://eodhistoricaldata.com mediante el cual se descargarán los datos de cada símbolo bursátil de una bolsa determinada a través de un token API, se guardarán los datos en el disco, se organizarán los datos y se leerán los datos clasificados de forma que se seleccionarán los 5 símbolos con mayor rendimiento para una fecha determinada mirando hacia atrás 10 días.

El código ofrecido no es un programa según
digamos. Sirve principalmente para proporcionar un conjunto
de rutinas y algoritmos para realizar una tarea específica. El código de este programa puede estudiarse y
integrarse a voluntad en cualquier otro programa.  
Hay que tener en cuenta que existen prerrequisitos y requisitos para
este programa. Principalmente el tiempo de ejecución de Java
y compilador, el estudio de desarrollo NetBeans, y el Gson /JSON
bibliotecas. Es muy posible que
hay descuidos y omisiones involuntarias en el código que afectarán a
la forma en que se ejecuta y está sujeta a errores que, a su vez, también serán involuntarios.

Puede descargar el código aquí EODHistoricalDataDownloader.java o en nuestra página de documentación con otros ejemplos Java.

Russell Brown reside actualmente en Markham, Ontario (Canadá). Tiene un título universitario en programación informática y ha trabajado con una amplia variedad de lenguajes de programación, incluidos C++, C#, SQL, PHP, JavaScript, HTML, CSS y JQuery. A lo largo de su carrera, ha trabajado en varias empresas de renombre, como IBM y Apple, en tareas de producción.

Cree firmemente en un enfoque táctico de la inversión, según el cual se poseen acciones y bonos a largo plazo y la ponderación se asigna por temporadas en función de la coyuntura económica actual, y a medida que se mantienen los activos se realizan aportaciones periódicas.

Ha seguido el curso de valores canadienses
con la intención de convertirse en asesor financiero, aunque no lo había hecho.
camino. Actualmente sigue cursos de
administración de empresas en la Universidad del Pueblo.

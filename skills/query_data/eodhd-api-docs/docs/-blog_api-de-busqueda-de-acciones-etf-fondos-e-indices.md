# API de búsqueda de acciones, ETF, fondos e índices

## 源URL

https://eodhd.com/financial-apis-blog/api-de-busqueda-de-acciones-etf-fondos-e-indices

## 正文

Nos enorgullece anunciar nuestra nueva API de búsqueda de acciones, ETF, fondos de inversión e índices. Puesto que tenemos más de 120.000 teletipos en nuestra base de datos, comprendemos claramente que es difícil encontrar el teletipo adecuado aunque se conozca el nombre de la empresa o el código. Por eso hemos desarrollado nuestro propio motor de búsqueda, que no sólo encuentra, sino que también tiene un algoritmo especial para clasificar los resultados con varios parámetros.

Nuestra API de búsqueda de acciones, ETF, fondos de inversión e índices es una de las mejores formas de buscar rápidamente activos, ya sea por código o por nombre de empresa o activo. Nuestro motor de búsqueda entiende automáticamente si hay nombre de activo o código y prioriza los campos de búsqueda en consecuencia. El motor de búsqueda dispone de varios parámetros para ordenar los resultados. Tenemos en cuenta no sólo la consulta de búsqueda, sino también la capitalización bursátil y el volumen medio de negociación del último periodo.

Siempre tendrá la seguridad de que los símbolos que proporcionamos mediante nuestro motor de búsqueda están activos y tienen mayor prioridad. Por ejemplo, si busca “AAPL”, siempre obtendrá AAPL.US (que cotiza en NASDAQ) en primer lugar en lugar de AAPL.MX (que cotiza en la bolsa mexicana), debido a los volúmenes de negociación.

La clave Demo API no funciona para la API de búsqueda, pero puede acceder a la búsqueda de valores registrándose en nuestro servicio y obteniendo la clave API gratuita. Cada solicitud a la API de búsqueda de valores consume 1 llamada del límite diario de la API.

Para más información, parámetros exactos de consulta y ejemplos de resultados, consulte nuestra página de documentación sobre la API de búsqueda de valores.

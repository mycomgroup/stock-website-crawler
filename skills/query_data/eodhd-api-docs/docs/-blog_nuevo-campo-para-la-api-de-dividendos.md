# Nuevo campo para la API de dividendos

## 源URL

https://eodhd.com/financial-apis-blog/nuevo-campo-para-la-api-de-dividendos

## 正文

Continuamos con las actualizaciones de Dividends Data Feed. Recientemente hemos anunciado nuevos campos para los dividendos: Fecha de declaración, Fecha de pago y Fecha de registro. Y hoy hemos añadido un nuevo campo: el punto. Por el momento, admitimos los principales tickers de EE.UU., la Bolsa de Londres y Honk Kong. Los valores posibles para el campo periodo de dividendos varían y dependen de la bolsa: Mensual, Trimestral, Semestral, Anual, Provisional, Final y Otros.

# Gran actualización de la API fundamental

## 源URL

https://eodhd.com/financial-apis-blog/gran-actualizacion-de-la-api-fundamental

## 正文

A partir de hoy y durante la próxima semana, lanzaremos una gran actualización de la API Fundamental. Hemos ampliado nuestros datos fundamentales para informes financieros tanto horizontalmente (más campos en diferentes informes) como verticalmente (los datos son más profundos y tienen un historial más largo). La actualización afecta a las bolsas de EE.UU. y a las principales bolsas de la UE, Sudamérica y Asia. Todos los campos actuales son los mismos y no han cambiado. A continuación encontrará los detalles de estos cambios.

Notas clave sobre la actualización actual:

Encontrará más información sobre nuestra API Fundamental en nuestra Documentación de la API Financiera.

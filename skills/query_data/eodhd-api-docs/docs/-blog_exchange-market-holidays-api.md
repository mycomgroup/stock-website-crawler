# Exchange Market Holidays API

## 源URL

https://eodhd.com/financial-apis-blog/exchange-market-holidays-api

## 正文

The past week we announced the new Exchanges API with trading hours and, eventually, today we are happy to publish Market Holidays API. We got a lot of requests to add the holiday data to understand if there was a holiday on the exchange or not. Like yesterday, Memorial Day, the federal holiday in the USA, but a few people know about this holiday in Europe.

No worries from today.

We provide market holiday data for all exchanges we support. And it’s easy to understand if the exchange is open or closed on a particular day.

Exchange holidays are provided 2 months back and 6 months forward. There are also two types of holidays: bank and official. In some countries, like the United Kingdom, there are bank holidays, but not official.

Get more information about our new Exchange Trading Hours and Market Holidays API on the documentation page.

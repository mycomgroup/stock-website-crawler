# API de análisis de datos bursátiles

## 源URL

https://eodhd.com/financial-apis-blog/api-de-analisis-de-datos-bursatiles

## 正文

Hoy nos complace anunciar nuestra nueva API Screener. La API Screener es una potente herramienta que le ayuda a filtrar tickers con los parámetros dados. Imagínese que puede filtrar fácilmente todas las empresas con una capitalización bursátil superior a 1.000 millones, que sólo tengan un BPA positivo dentro del sector de “Productos personales” y cuyo nombre empiece por la letra “B”. O bien obtenga sólo los tickers que tengan nuevos mínimos en los últimos 200 días y el valor contable sea negativo.

Sí, ahora es fácil con nuestra nueva API Screener. Al principio admitimos varios campos de datos fundamentales como Capitalización bursátil, BPA, Rentabilidad por dividendo, y campos precalculados denominados “señales” como nuevos máximos o mínimos de 50/200 días, y otros. Obtenga más información en nuestra Documentación de API financieras.

La API Screener está disponible en los paquetes de datos “All World Extended” y “All-In-One”. Cada solicitud de la API Screener consume 5 llamadas a la API. Siempre estamos trabajando en nuevas señales y campos soportados y podemos añadir nuevas señales y campos soportados bajo petición.

Que lo disfrutes. Y estaremos encantados de recibir cualquier comentario sobre esta potente función.

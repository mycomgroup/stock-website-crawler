# API de sentimiento de tuits

## 源URL

https://eodhd.com/financial-apis-blog/api-de-sentimiento-de-tuits

## 正文

En abril anunciamos nuevas API alternativas con Sentiment Data API para noticias. Y hoy hemos publicado la Tweets Sentiment API. Los tuits agregan los datos de sentimiento que recogemos de los tuits con mención del teletipo concreto. Los datos se agregan por días. La salida para la API financiera de datos de sentimiento se divide por códigos de ticker utilizados con el parámetro “s=”. Tenga en cuenta que los códigos de los teletipos se normalizan en la salida.

Puede obtener más información en la página de documentación de Tweet Sentiment API.

Que lo disfrutes.

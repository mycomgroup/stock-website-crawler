# Google Sheets Financial Add-In for EOD/Fundamentals Data

## 源URL

https://eodhd.com/financial-apis-blog/google-sheets-financial-add-in-for-eod-fundamentals-data

## 正文

We are happy to announce our new google sheets financial add-in for End Of Day (EOD), Intraday, and Fundamental data downloading directly into your sheets. If you are not familiar with any programming languages, it’s one of the best ways to use the financial data with our APIs.

More information with details and setup instructions you can find on the Google Sheets Financial Add-In documentation page.

You can also check our Excel Financial Add-In.

In case you have more questions and need help, send us to email: support@eodhistoricaldata.com.

# Enorme actualización de los fundamentos de EE.UU. + nuevos campos

## 源URL

https://eodhd.com/financial-apis-blog/enorme-actualizacion-de-los-fundamentos-de-ee-uu-nuevos-campos

## 正文

El mes pasado trabajamos duro para mejorar nuestra fuente de datos fundamentales de Estados Unidos. Hemos corregido las lagunas en los datos trimestrales y anuales y recalculado todos los campos de datos históricos del balance, el flujo de caja y la cuenta de resultados a partir de 2009 para todas las empresas estadounidenses de NYSE y NASDAQ.

También hemos añadido 24 nuevos campos de datos para las empresas estadounidenses que cotizan en NYSE y NASDAQ. 3 nuevos campos para la cuenta de resultados, 17 nuevos campos para el balance y 4 nuevos campos para el flujo de caja. A continuación, le recomendamos que vuelva a descargar todos los datos fundamentales que utilice con nuestra API.

Los nuevos datos sólo están disponibles en JSON API, no en CSV Bulk Fundamentals. A continuación encontrará una lista completa de los nuevos campos añadidos.

Puede obtener más información en nuestra página de documentación sobre Alimentación de Datos Fundamentales.

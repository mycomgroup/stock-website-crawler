# Fundamentos de Bulk Stock API

## 源URL

https://eodhd.com/financial-apis-blog/fundamentos-de-bulk-stock-api

## 正文

Recientemente hemos actualizado nuestra API Bulk EOD y añadido Splits y Dividendos. Y hoy nos enorgullece presentarle la API de fundamentos a granel. Con este endpoint podrá descargar datos fundamentales de miles de empresas en una sola solicitud. Es muy fácil actualizar los datos fundamentales de toda la bolsa en cuestión de segundos, ya sea en su base de datos o directamente en Excel, ya que la API de datos fundamentales masivos admite la salida CSV.

Obtenga más información sobre este punto final de la API en nuestra documentación: https://eodhistoricaldata.com/knowledgebase/stock-etfs-fundamental-data-feeds/#Bulk_Fundamentals_API

Por el momento, sólo proporcionamos fundamentos masivos en formato CSV; JSON no es compatible. Aquí puede encontrar un ejemplo de punto final de API de fundamentos masivos para Apple Inc (AAPL.US) y Microsoft Corporation MSFT.US). El tamaño medio de los datos para las bolsas NASDAQ/NYSE es de unos 19Mb, para la bolsa de Hongkong es de 12-13Mb.

Para obtener acceso a la API de fundamentos masivos, debe suscribirse al paquete “Fundamentos ampliados“; le proporcionaremos más información si nos lo solicita: support@eodhistoricaldata.com.

> Obtenga más información sobre este punto final de la API en nuestra documentación: https://eodhistoricaldata.com/knowledgebase/stock-etfs-fundamental-data-feeds/#Bulk_Fundamentals_API

# Cómo: Puntuación Z de Altman + Puntuación de Piotrosky

## 源URL

https://eodhd.com/financial-apis-blog/como-puntuacion-z-de-altman-puntuacion-de-piotrosky

## 正文

A menudo nos preguntan cómo calcular las distintas puntuaciones financieras, como la puntuación Z de Altman o la puntuación de Piotrovsky, con nuestra API de datos fundamentales. Para indicadores técnicos simples como RSI, EMA/SMA, MACD, Estocástico y otros, tenemos un punto final especial de API Técnica. Pero las puntuaciones fundamentales son mucho más complejas y, por lo general, los artículos sobre estas puntuaciones tratan de teoría, no de práctica.

Por eso publicamos un artículo sobre 
  Cómo calcular la puntuación Z de Altman y la puntuación de Piotrosky
. La principal diferencia de este artículo en comparación con otros sobre Altman Z-Score y Piotrosky puntuación en que es práctico, y proporciona campos exactos de los informes financieros con nuestra API fundamental, entonces es muy rápido y fácil de calcular estas puntuaciones con nuestros datos.

Obtenga más información sobre la puntuación Z de Altman o puntuación de Piotrosky y cómo calcularla con campos de datos reales de informes financieros.

# Bulk Stock Fundamentals API

## 源URL

https://eodhd.com/financial-apis-blog/bulk-stock-fundamentals-api

## 正文

Recently we’ve updated our Bulk EOD API and added Splits and Dividends. And today we are proud to introduce you bulk fundamentals API. With this endpoint you able to download fundamental data for thousands of companies in one request. It’s effortless to update fundamentals for the entire exchange in several seconds either to your database or directly to Excel since bulk fundamentals API supports CSV output.

Get more information on this API endpoint in our documentation: https://eodhistoricaldata.com/knowledgebase/stock-etfs-fundamental-data-feeds/#Bulk_Fundamentals_API

We provide bulk fundamentals only in CSV format for the moment, JSON is not supported. Here you can find an example of bulk fundamentals API endpoint for Apple Inc (AAPL.US) and Microsoft Corporation MSFT.US). The average size of data for NASDAQ/NYSE exchanges is around 19Mb, for Hongkong exchange is 12-13Mb.

To get an access to bulk fundamentals API, you should subscribe to ‘Extended Fundamentals’ package, more details we will provide by request: support@eodhistoricaldata.com.

> Get more information on this API endpoint in our documentation: https://eodhistoricaldata.com/knowledgebase/stock-etfs-fundamental-data-feeds/#Bulk_Fundamentals_API

# Integración de NinjaTrader con EODHD APIs

## 源URL

https://eodhd.com/financial-apis-blog/integracion-de-ninjatrader-con-eodhd-apis

## 正文

Nos complace anunciar nuestra integración con NinjaTrader. NinjaTrader proporciona un marco de C# para que los desarrolladores construyan indicadores integrados, herramientas de dibujo, estrategias automatizadas y mucho más.

Con nuestro plugin, puede importar fácilmente datos de fin de día, intradía y fundamentales a su entorno.

Nuestro plugin es fácil de instalar a través del software NinjaTrader navegando al menú Herramientas de NinjaTrader > Importar > NinjaScript y seleccionando el archivo .ZIP.

Puede encontrar más información en nuestra documentación Adaptador NinjaTrader para Datos Intradía y Fundamentales EOD.

Que lo disfrutes.

# API masiva actualizada para EOD, divisiones y dividendos

## 源URL

https://eodhd.com/financial-apis-blog/api-masiva-actualizada-para-eod-divisiones-y-dividendos

## 正文

Hoy hemos actualizado nuestra API masiva para datos históricos y hemos añadido divisiones y dividendos. Esta API permite descargar fácilmente los datos de toda la bolsa para un día concreto. Funciona para la alimentación de datos históricos al final del día, así como para los datos de divisiones y dividendos.

A partir de hoy no necesitará realizar miles y miles de peticiones API al día. Ya no es necesario. Hemos desarrollado un punto final API de descarga masiva, y es fácil descargar datos históricos cualquier día de forma masiva.

Obtenga más información sobre esta API en nuestra documentación: https://eodhistoricaldata.com/knowledgebase/bulk-api-eod-splits-dividends/.

El siguiente ejemplo devuelve los datos de fin de día de las acciones estadounidenses en bloque para un día concreto:

http://eodhistoricaldata.com/api/eod-bulk-last-day/US?api_token={YOUR_API_KEY}

El siguiente ejemplo devuelve todos los splits de las acciones estadounidenses en bloque para un día concreto:

http://eodhistoricaldata.com/api/eod-bulk-last-day/US?api_token={YOUR_API_KEY}&type=splits

El siguiente ejemplo devuelve todos los dividendos de las acciones estadounidenses en bloque para un día concreto:

http://eodhistoricaldata.com/api/eod-bulk-last-day/US?api_token={YOUR_API_KEY}&type=dividendos

Obtenga más información sobre esta API en nuestra documentación: https://eodhistoricaldata.com/knowledgebase/bulk-api-eod-splits-dividends/.

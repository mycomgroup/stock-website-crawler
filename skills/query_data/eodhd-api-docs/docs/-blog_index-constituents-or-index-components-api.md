# Index Constituents (or Index Components) API

## 源URL

https://eodhd.com/financial-apis-blog/index-constituents-or-index-components-api

## 正文

Today we have started to provide Index Constituents (or Index Components) data for all major indices all around the world.  At the moment we have data for around 100 indices and it’s growing.

The new API is provided under our ‘Fundamentals’ package, and it’s easy to use it: request our fundamentals API with an index (see details on the documentation page) and we will provide a list of tickers that are part of a provided index. We offer the following data for index components:

Below you can find an example of the output of index constituents for the S&P 500 (GSPC.INDX)

For more details on Fundamentals API, please read our documentation for this endpoint.

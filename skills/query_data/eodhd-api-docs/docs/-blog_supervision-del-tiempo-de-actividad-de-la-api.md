# Supervisión del tiempo de actividad de la API

## 源URL

https://eodhd.com/financial-apis-blog/supervision-del-tiempo-de-actividad-de-la-api

## 正文

A pesar de estos tiempos difíciles en todo el mundo, seguimos prestando nuestros servicios 24 horas al día, 7 días a la semana. Mientras tanto, debido al confinamiento, el tráfico de Internet y la carga sobre los proveedores de Internet ha crecido significativamente. Y a veces esto provoca una ralentización entre los servidores y nuestros abonados. Periódicamente, pregunta si nuestros servidores funcionan o no.

Siempre monitorizamos la salud de nuestros servidores y subsistemas con varios servicios de monitorización de terceros, y hoy hemos decidido publicar la URL de monitorización para usted: EOD Uptime Data.

Como ve, tenemos una disponibilidad de casi el 100% para todos los servicios que le ofrecemos.

¡Gracias y ten cuidado con COVID-19!

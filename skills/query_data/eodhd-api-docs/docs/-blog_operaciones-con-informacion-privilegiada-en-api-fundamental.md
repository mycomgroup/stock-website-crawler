# Operaciones con información privilegiada en API Fundamental.

## 源URL

https://eodhd.com/financial-apis-blog/operaciones-con-informacion-privilegiada-en-api-fundamental

## 正文

Esta semana hemos ampliado nuestra API Fundamental y hemos añadido las Operaciones con Información Privilegiada tal y como se comunican en el Formulario 4 a la SEC. Los datos están disponibles para todas las empresas estadounidenses que presentan el Formulario 4. Las operaciones con información privilegiada implican la compraventa de acciones de una empresa pública por parte de alguien que, por cualquier motivo, dispone de información material no pública sobre dichas acciones. En algunos casos, las operaciones con información privilegiada podrían ser muy útiles para tomar decisiones de inversión, por eso hemos añadido estos datos a nuestras API financieras.

Por el momento, admitimos dos códigos de transacción. El código de transacción “P” – Compra de valores en bolsa o a otra persona. Y el código de transacción “S” – Venta de valores en bolsa o a otra persona.

Hay más códigos de transacción en los Formularios 3, 4 y 5 y puede obtenerlos todos con explicaciones detalladas en el Boletín del Inversor de la SEC Formularios 3, 4 y 5.

También puede obtener más información sobre datos de operaciones con información privilegiada y muchos otros campos como informes financieros, beneficios por acción, cuentas de resultados, balances, flujos de caja y otros datos útiles en nuestra API Fundamental.

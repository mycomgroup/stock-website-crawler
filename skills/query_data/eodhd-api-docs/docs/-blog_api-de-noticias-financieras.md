# API de noticias financieras

## 源URL

https://eodhd.com/financial-apis-blog/api-de-noticias-financieras

## 正文

Hoy nos complace anunciar nuestra nueva API de noticias financieras. Hemos desarrollado esta API durante varios meses y ahora está disponible en versión beta. Recopilamos los datos de decenas de fuentes públicas y clasificamos las noticias por códigos de teletipo. En las próximas semanas añadiremos etiquetas por eventos corporativos como splits, dividendos, earnings calls y otros.

He aquí un ejemplo de salida con la fecha, el título, el enlace de la noticia original y los símbolos de teletipo conectados.

Lea más sobre nuestra API de noticias financieras y manténgase en contacto.

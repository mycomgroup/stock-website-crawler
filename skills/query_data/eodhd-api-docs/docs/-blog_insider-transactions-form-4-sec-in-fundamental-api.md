# Insider Transactions in Fundamental API.

## 源URL

https://eodhd.com/financial-apis-blog/insider-transactions-form-4-sec-in-fundamental-api

## 正文

This week we expanded our Fundamental API and added Insider Transactions as they reported in Form 4 to SEC. The data is available for all US companies that report Form 4. Insider trading involves trading in a public company’s stock by someone who has non-public, material information about that stock for any reason. In some cases, insider transactions could be very useful for making investment decisions, that’s why we added this data to our Financial APIs.

At the moment we support two transaction codes. The transaction code “P” – Purchase of securities on an exchange or from another person. And the transaction code “S” – Sale of securities on an exchange or to another person.

Supported fields for both transaction codes Purchase and Sale are:

There are more transaction codes in Forms 3, 4, and 5 and you can get all of them with detailed explanations in the SEC Investor Bulletin Forms 3, 4, and 5.

Or get more information about Insider Transaction data and many other fields like financial reports, earnings per shares, income statements, balance sheets, cash flows and other useful data in our Fundamental API.

# Todo para mantenerte en contacto

## 源URL

https://eodhd.com/financial-apis-blog/todo-para-mantenerte-en-contacto

## 正文

Por último, hemos lanzado varios medios de comunicación que le ayudarán a estar informado sobre nuestras novedades, planes, estado actual y muchas otras cosas.

Pusimos en marcha el servicio hace un año, en mayo de 2017. Y todos estos meses vosotros, nuestros suscriptores y usuarios, nos habéis preguntado constantemente por el canal adecuado para estar informados de las novedades y otras noticias que tenemos para vosotros. Y todo el año pasado nos centramos en la fiabilidad de nuestros servicios, la API financiera JSON y CSV con documentación y la calidad general de los datos.

Pero sabemos perfectamente lo importante que es tener unos canales de comunicación buenos y claros con los clientes, futuros, actuales o incluso antiguos. Por eso contamos con un equipo de asistencia muy rápido, tanto por correo electrónico como por Intercom. Y por eso al final encontramos el tiempo y los recursos para poner en marcha y mantener actualizados todos estos medios sociales.

Entonces, tenemos los siguientes soportes de datos para usted:

Gracias. Y estaremos encantados de recibir sus sugerencias.

Su equipo de datos históricos EOD.

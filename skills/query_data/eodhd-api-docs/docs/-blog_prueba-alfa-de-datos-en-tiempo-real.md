# Prueba alfa de datos en tiempo real

## 源URL

https://eodhd.com/financial-apis-blog/prueba-alfa-de-datos-en-tiempo-real

## 正文

Nos complace anunciar que ahora mismo estamos probando los datos en tiempo real con un retraso de 50 ms. La API en tiempo real está ahora en fase alfa y estará totalmente disponible en las próximas 2-3 semanas. Nuestra API en tiempo real proporcionará los datos para las acciones de EE.UU., FOREX, y Cryptocurrencies intercambios a través de WebSockets.

Queremos estar seguros de que todo saldrá perfecto cuando se publique la API. Y mientras estemos en la fase de pruebas alfa, estaremos encantados de ofrecer acceso gratuito durante estas 2-3 semanas a los 100 primeros suscriptores que nos lo soliciten por correo electrónico y/o webchat.

# Nuevos datos de “funcionarios” en la API fundamental

## 源URL

https://eodhd.com/financial-apis-blog/nuevos-datos-de-funcionarios-en-la-api-fundamental

## 正文

La semana pasada añadimos un nuevo campo, e incluso no un campo sino un subconjunto: “Funcionarios”. Los datos incluyen la lista de ejecutivos con nombre completo, cargo y año de nacimiento. Véase el ejemplo de MSFT a continuación.

El segundo pequeño cambio en que hemos añadido pequeños cambios a las direcciones, la mayoría de las empresas tienen ahora la dirección completa con el índice postal, país, estado (si es posible). Ahora incluso puedes utilizar esta dirección en cualquier mapa. El ejemplo de dirección de Microsoft Corporation:

One Microsoft Way, Redmond, WA, Estados Unidos, 98052

También hemos añadido direcciones para empresas pequeñas y no radicadas en EE.UU., como ZOOM Technologies:

Sanlitun SOHO, Pekín, China, 100027

A continuación encontrará ejemplos de Microsoft Corporation (MSFT).

Puede obtener más información sobre nuestros datos fundamentales en nuestra API fundamental para acciones y ETF.

Háganos saber si echa en falta alguna información e intentaremos añadirla.

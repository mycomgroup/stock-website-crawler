# Integración con Financial Excel Add-In – Excel Price Feed

## 源URL

https://eodhd.com/financial-apis-blog/integracion-con-financial-excel-add-in-excel-price-feed

## 正文

Hoy nos complace anunciar una integración con el complemento financiero de Excel – Excel Price Feed. Excel Price Feed es un complemento para Excel fácil de usar y con muchas funciones, propiedad de Coderun Technologies Ltd. y distribuido por la misma. Fue lanzado a principios de 2019 y se actualiza regularmente con nuevas características, para EOD Historical Data el complemento tiene un excelente soporte de Fundamental y End of Day APIs.

Más información y descarga del plugin en la página de Excel Price Feed.

Tenga en cuenta que debe tener ambas suscripciones para utilizar este plugin con nuestros datos: de EOD Historical Data y de Excel Price Feed. Nuestras suscripciones no incluyen licencias de complementos y deben solicitarse por separado.

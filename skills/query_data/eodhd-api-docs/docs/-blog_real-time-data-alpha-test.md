# Real-Time Data Alpha Test

## 源URL

https://eodhd.com/financial-apis-blog/real-time-data-alpha-test

## 正文

We are happy to announce that we are testing the real-time data with a 50ms delay right now. The real-time API is in alpha now and will be fully available within the next 2-3 weeks. Our real-time API will provide the data for US stocks, FOREX, and Cryptocurrencies exchanges via WebSockets.

We want to be sure that everything will be perfect once API is published. And while we are in the alpha testing, we are happy to provide free access for these 2-3 weeks for the first 100 subscribers who will send us a request via email and/or webchat.

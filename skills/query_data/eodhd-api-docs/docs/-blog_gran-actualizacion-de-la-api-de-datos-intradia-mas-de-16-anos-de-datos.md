# Gran actualización de la API de datos intradía. Más de 16 años de datos.

## 源URL

https://eodhd.com/financial-apis-blog/gran-actualizacion-de-la-api-de-datos-intradia-mas-de-16-anos-de-datos

## 正文

Hoy hemos lanzado la gran actualización de la API de datos intradía. Los cambios afectaron a los datos financieros intradía de 1 minuto de varias bolsas. En primer lugar, hemos aumentado drásticamente la profundidad de los datos intradía de las bolsas estadounidenses, los intervalos de 1 minuto están disponibles ahora desde 2004, más de 16 años de datos. En segundo lugar, hemos añadido los datos intradía con intervalos de 1 minuto para las bolsas de Canadá y MOEX (Bolsa de Moscú).

La semana pasada también publicamos la gran actualización de Fundamental API y Financial News API. Y pronto lanzaremos la API en tiempo real con 50ms para el mercado Forex.

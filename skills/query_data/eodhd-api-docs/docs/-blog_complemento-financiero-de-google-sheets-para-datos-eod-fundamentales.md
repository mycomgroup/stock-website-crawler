# Complemento financiero de Google Sheets para datos EOD/Fundamentales

## 源URL

https://eodhd.com/financial-apis-blog/complemento-financiero-de-google-sheets-para-datos-eod-fundamentales

## 正文

Nos complace anunciar nuestro nuevo complemento financiero de google sheets para la descarga de datos de fin de día (EOD), intradía y fundamentales directamente en sus hojas. Si no está familiarizado con ningún lenguaje de programación, es una de las mejores formas de utilizar los datos financieros con nuestras API.

Puedes encontrar más información con detalles e instrucciones de configuración en la página de documentación del complemento financiero de Google Sheets.

También puede consultar nuestro complemento financiero de Excel.

Si tiene más preguntas y necesita ayuda, envíenos un correo electrónico a: support@eodhistoricaldata.com.

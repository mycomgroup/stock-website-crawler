# Predicción bursátil. Lo más destacado de nuestro abonado.

## 源URL

https://eodhd.com/financial-apis-blog/prediccion-bursatil-lo-mas-destacado-de-nuestro-abonado

## 正文

Nuestros servicios son utilizados por inversores y operadores profesionales y no profesionales muy diferentes, aplicaciones móviles y servicios de Internet. Creemos que es buena idea dar a conocer a nuestros abonados para que conozcan mejor la comunidad y el ecosistema que construimos juntos.

Hoy quiero destacar el servicio StockPredictions. Estos chicos utilizan nuestra API desde hace más de un año para proporcionar bots de predicción para más de 50.000 acciones, cripto y forex utilizando análisis técnico y datos fundamentales.

¿Cómo funciona? Permítanme darles una palabra:

Calculamos los momentos de compra y venta de las acciones mediante el Análisis Técnico. Combinamos al menos 6 indicadores técnicos y dejamos que el sistema entrene el modelo para obtener las mejores predicciones. Se utilizan millones de cálculos para facilitar sus decisiones. Sólo los valores con las mejores predicciones se utilizan para el asesoramiento

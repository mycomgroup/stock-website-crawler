## 源URL

## 正文

Eventually, we are happy to announce the publishing of the real-time data via WebSockets with a delay of less than 50ms for the USA market, FOREX, and Cryptocurrencies. For USA stocks the real-time data API supports pre-market and post-market hours (from 4:00 am till 8:00 pm EST). WebSockets API is available within All World Extended and/or All-In-One packages.

# Asignación de códigos MIC para intercambios

## 源URL

https://eodhd.com/financial-apis-blog/asignacion-de-codigos-mic-para-intercambios

## 正文

Prestamos asistencia a más de 60 bolsas de todo el mundo con datos al final del día, fundamentales y en directo. Como utilizamos nuestros propios códigos de intercambio para identificarnos, a veces no es fácil encontrar e identificar los intercambios. Por eso hemos añadido códigos MIC para todas las bolsas de valores que admitimos.

El Código Identificador de Mercado (MIC) (ISO 10383) es un código de identificación único utilizado para identificar las bolsas de valores y los mercados de negociación regulados y no regulados. El MIC es un código de cuatro caracteres alfanuméricos y está definido en la norma ISO 10383 de la Organización Internacional de Normalización. La primera letra de cualquier MIC es X, seguida de un código alfanumérico de tres dígitos correspondiente al mercado en el que se realiza la operación. Por ejemplo, las operaciones que se ejecutan en el mercado estadounidense NASDAQ se identifican mediante el código MIC XNAS.

Tenga en cuenta que todas las bolsas de EE.UU. están combinadas en una bolsa virtual “US”, que incluye los tickers NYSE, NASDAQ, ARCA y OTC/PINK. Además, todos los índices y materias primas están en bolsas virtuales INDX y COMM, respectivamente, luego las bolsas INDX y COMM no tienen MICs.

A continuación encontrará la lista completa de las bolsas admitidas con los códigos MIC para su comodidad y una correspondencia más rápida.

Encontrará más información y la lista completa de bolsas compatibles con códigos MIC en nuestra página de documentación.

# Tipos de API de búsqueda

## 源URL

https://eodhd.com/financial-apis-blog/tipos-de-api-de-busqueda

## 正文

Si todavía no ha probado nuestra API de búsqueda, se la recomendamos encarecidamente, ya que con nuestra API Screener se convierte en una solución definitiva para obtener la lista completa de ticker por bolsa, encontrar el código de ticker adecuado, filtrar por parámetro, y muchas otras cosas.

Hoy anunciamos la búsqueda por tipo de teletipo. Así, es fácil filtrar los activos por tipo. La API de búsqueda admite los siguientes tipos: todos, acciones, etf, fondos, bonos e índices. El valor por defecto es ‘todos’ y por favor tenga en cuenta: con el valor ‘todos’ no se mostrarán los bonos, deberá solicitarlos explícitamente.

Por ejemplo, es fácil obtener todos los índices con la letra “A”:

Para obtener más información, consulte nuestra página de documentación de la API de búsqueda. Y estaremos encantados de recibir nuevas sugerencias.

```text
https://eodhistoricaldata.com/api/search/A?api_token=YOUR_API_TOKEN&fmt=json&type=index
```

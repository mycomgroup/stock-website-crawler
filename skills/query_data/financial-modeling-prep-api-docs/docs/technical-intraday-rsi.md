## About Relative Strength Index API

The FMP RSI endpoint provides the Relative Strength Index (RSI) for a security over a specified period of time. The RSI is a momentum indicator that measures the magnitude of recent price changes to evaluate overbought or oversold conditions in the price of a stock or other asset. The RSI is displayed as an oscillator (a line graph that moves between two extremes) and can range from 0 to 100.
The FMP RSI endpoint is a valuable tool for investors who need to:
Identify overbought and oversold conditions in a security's price
Measure the strength of a trend
Develop trading strategies based on technical analysis

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/technical_indicator/5min/AAPL?type=rsi&period=10
```

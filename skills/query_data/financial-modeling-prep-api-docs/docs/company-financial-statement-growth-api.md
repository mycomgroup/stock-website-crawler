## About Financial Growth API

The FMP Financial Growth endpoint provides access to the financial growth rate for a company. The financial growth rate is a measure of how quickly a company's overall financial performance is improving. 
A positive financial growth rate indicates that a company is performing better than it did in the past. 
A negative financial growth rate indicates that a company is performing worse than it did in the past.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/financial-growth/AAPL?period=annual
```

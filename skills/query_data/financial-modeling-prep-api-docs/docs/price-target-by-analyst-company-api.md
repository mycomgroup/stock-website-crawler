## About Price Target By Company API

The FMP Price Target By Company endpoint provides access to the price targets for all companies in a specific industry or sector. This can be useful if you want to compare the price targets of different companies in the same industry.
To use the FMP Price Target By Company endpoint, simply provide the industry or sector that you want to see price targets for. The endpoint will return a list of all companies in the specified industry or sector, along with their price targets.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/price-target-analyst-company?company=Barclays
```

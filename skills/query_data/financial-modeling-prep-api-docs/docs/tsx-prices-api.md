## About TSX Prices API

Access all TSX stocks such as Franco-Nevada, Medical Facilities Corporation, Erdene Resource Development Corporation. The Toronto Stock Exchange (TSX) is a Canadian stock exchange, so prices are displayed in Canadian dollars (CAD).

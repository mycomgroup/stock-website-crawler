## About Euronext Symbols API

Our Euronext List endpoint returns prices for the EURONEXT exchange in Paris. This endpoint includes information such as price, change, and market cap for all stocks on the Euronext exchange.
  
  Benefits:
  
  Access to real-time stock prices and other key information for all stocks on the Euronext exchange
  Stay up-to-date on the latest market activity in Paris
  Identify new trading opportunities
  
  Example:
  
  An investor can use the Euronext List endpoint to find a list of all technology stocks that are traded on the Euronext exchange. The investor can then use this list to identify potential investment opportunities.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/symbol/available-euronext
```

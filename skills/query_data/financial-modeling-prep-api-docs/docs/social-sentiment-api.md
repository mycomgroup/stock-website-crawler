## About Historical Social Sentiment API

The Historical Social Sentiment endpoint provides historical social sentiment data for a given ticker or company name. Social sentiment is a measure of the public's feelings about a particular company or product. It can be measured by analyzing social media posts, news articles, and other online content.
Investors can use the Historical Social Sentiment endpoint to:
    Track the sentiment of the public towards a particular company over time.
Identify trends in social sentiment.
Make informed investment decisions based on social sentiment data.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/historical/social-sentiment?symbol=AAPL&page=0
```

## About Weighted Moving Average API

The FMP WMA endpoint provides the weighted moving average of a security's closing price over a specified period of time. WMAs are calculated by assigning different weights to each price in the calculation. The weights can be assigned based on the volume of trading that occurred at each price, or they can be assigned based on the user's own preferences.
WMAs are a type of technical indicator that is more flexible than SMAs and EMAs. This makes them more useful for identifying trends, reversals, and support and resistance levels.
The FMP WMA endpoint is a valuable tool for investors who need to:
Identify trends, reversals, and support and resistance levels in a security's price
Develop trading strategies based on technical analysis

https://financialmodelingprep.com/api/v3/technical_indicator/5min/AAPL?type=wma&period=10

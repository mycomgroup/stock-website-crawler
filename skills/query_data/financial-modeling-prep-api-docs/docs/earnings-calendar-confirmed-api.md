## About Earnings Confirmed API

The FMP Earnings Confirmed endpoint is a valuable tool for investors, analysts, and other individuals who need to stay up-to-date on the latest earnings announcements for publicly traded companies. Earnings announcements can provide valuable insights into a company's financial performance and outlook.
The FMP Earnings Confirmed endpoint is easy to use. Simply provide the date range that you are interested in and the endpoint will return a list of earnings announcements for publicly traded companies that have already been confirmed.
The FMP Earnings Confirmed endpoint can be used for a variety of purposes, such as:
To locate trading opportunities.
To monitor the performance of publicly traded corporations.
To keep up with the latest news and events.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/earning-calendar-confirmed?from=2023-04-10&to=2023-08-19
```

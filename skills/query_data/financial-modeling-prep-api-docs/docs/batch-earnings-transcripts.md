## About Batch Earning Call Transcript API

The FMP Batch Earnings Transcript endpoint provides access to the full transcript of an earnings call for a specific company. This endpoint is updated after each earnings call, so you can always get the most up-to-date information on what company management is saying.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/batch_earning_call_transcript/AAPL?year=2020
```

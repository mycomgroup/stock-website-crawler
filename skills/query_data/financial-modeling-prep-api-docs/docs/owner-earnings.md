## 文档正文

The Average PPE-Sales is calculated by adding propertyPlantEquipmentNet and revenue in last 7 annual statements of company and then:Average PPE-Sales = Sum propertyPlantEquipmentNet / Sum Revenue

#### Average PPE-Sales:

The Average PPE-Sales is calculated by adding propertyPlantEquipmentNet and revenue in last 7 annual statements of company and then:Average PPE-Sales = Sum propertyPlantEquipmentNet / Sum Revenue

#### Growth Capex:

The Growth Capex is calculated by using the following formula:Growth Capex = Average PPE-Sales * (Revenue - Revenue Last Quarter) * -1 if Revenue - Revenue Last Quarter < 0: Growth Capex = 0 if Revenue Last Quarter = 0: Growth Capex = Capex - Average PPE

#### Maintenance Capex:

The Maintenance Capex is calculated by using the following formula:Maintenance Capex = Capital Expenditure - Growth Capex

#### Owner's Earnings:

The Owner's Earnings are calculated by using the following formula:Owner's Earnings = Operating Cash Flow + Maintenance Capex

#### Owner's Earnings Per Share:

The Owner's Earnings Per Share are calculated by using the following formula: Owner's Earnings Per Share = Owner's Earnings / weightedAverageShsOutDil

## About CIK Mapper By Symbol API

The CIK Mapper endpoint provides a list of all CIK numbers and corresponding company names. This endpoint can be used by investors to get a comprehensive overview of all public companies and their CIK numbers.
  Investors can use the CIK Mapper endpoint to:
  Get a comprehensive overview of all public companies and their CIK numbers.
  Identify companies by their CIK number or company name.
  Cross-reference CIK numbers and company names when using other financial data providers or APIs.

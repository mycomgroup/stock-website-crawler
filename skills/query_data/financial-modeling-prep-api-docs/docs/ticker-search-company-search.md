## About Ticker Search API

FMP's Ticker Symbol Search Company Search is a quick and easy way to find the ticker symbol and exchange for a company. It supports both equity securities and exchange-traded funds (ETFs).
To use the Ticker Symbol Search Company Search, simply enter the company name or ticker symbol in the search bar and click on the "Search" button. The search results will include the company name, ticker symbol, exchange, and other relevant information.
The Ticker Symbol Search Company Search is a valuable tool for investors, traders, and anyone who needs to find the ticker symbol for a company.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/search-ticker?query=AA&limit=10&exchange=NASDAQ
```

## About Quote Order API

The quote order quote endpoint provides a simplified view of a stock's quote, including the current price, volume, and last trade price. Investors can use this information to get a quick overview of a stock's performance and to make informed investment decisions.
For example, an investor may want to use the quote order quote endpoint to get a quick overview of the performance of a stock portfolio. Or, an investor may want to use the quote order quote endpoint to compare the performance of multiple stocks before making a trade.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/quote-order/AAPL
```

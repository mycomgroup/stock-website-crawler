## About Upgrades & Downgrades RSS Feed API

The FMP Upgrades & Downgrades RSS Feed endpoint provides access to an RSS feed of all stock upgrades and downgrades from different analysts. This RSS feed is updated on a daily basis, so you can always stay up-to-date on the latest analyst ratings.
To use the Upgrades & Downgrades RSS Feed endpoint, simply provide the URL of the RSS feed in your RSS reader. Your RSS reader will then automatically download the latest updates to the feed and display them to you.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/upgrades-downgrades-rss-feed?page=0
```

## About Discounted Cashflow API

The FMP Discounted Cash Flow endpoint provides access to the DCF valuation for a company. DCF is a valuation method that estimates the value of an investment based on its expected future cash flows. 
  The DCF valuation is calculated by discounting the expected future cash flows to their present value.Investors can use DCF valuation to compare the value of different investment opportunities, to assess the riskiness of an investment, and to make investment decisions.
  The FMP Discounted Cash Flow endpoint provides a simple and easy-to-use way to calculate the DCF valuation for a company. The endpoint requires the user to provide the company's expected future cash flows and the discount rate. The endpoint will then calculate the DCF valuation and return it to the user.

**Endpoint:**


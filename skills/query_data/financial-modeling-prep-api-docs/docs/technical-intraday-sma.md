## About Simple Moving Average API

The FMP SMA endpoint provides the simple moving average of a security's closing price over a specified period of time. SMAs are calculated by adding the closing prices of a security over a specified period of time and then dividing that figure by the number of periods.
SMAs can also be used to identify support and resistance levels. Support levels are prices at which the stock price is likely to find support, while resistance levels are prices at which the stock price is likely to face resistance.
The FMP SMA endpoint is a valuable tool for investors who need to:
Identify trends in a security's price
Identify support and resistance levels
Develop trading strategies based on technical analysis

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/technical_indicator/5min/AAPL?type=sma&period=10
```

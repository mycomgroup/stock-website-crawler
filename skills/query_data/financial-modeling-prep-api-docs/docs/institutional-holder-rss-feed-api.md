## About Institutional Holder RSS API

The institutional holder RSS feed provides an RSS feed of institutional holder data. This information is useful for investors who need to stay up-to-date on the latest institutional investment activity and who need to track changes in the institutional ownership of individual stocks.
  For example, investors may want to subscribe to the institutional holder RSS feed to receive real-time updates on the institutional investment activity in their portfolio or in the overall stock market.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/institutional-ownership/rss_feed?page=0
```

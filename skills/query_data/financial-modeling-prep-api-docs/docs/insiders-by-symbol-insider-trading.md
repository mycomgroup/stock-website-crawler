## About Insiders By symbol API

The Insiders By Symbol endpoint provides a list of all insiders who have traded in a particular company's stock. This information can be used by investors to identify the insiders who are most interested in a particular company and to track their trading activity over time.
  Investors can use the Insiders By Symbol endpoint to:
  Identify the insiders who are most interested in a particular company.
  Track the trading activity of insiders over time to identify any trends or patterns.
  Identify investment opportunities based on the trading activity of insiders.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v4/insider-roaster?symbol=AAPL
```

## About Multiple Company Prices API

Access a list of all major stock prices, price are updated in realtime. The batch request allows you to have multiple stock in one single request. You can therefore access multiple stock price, profile.
     You can access individual stock price via our  Stock API to see each stock upaded in real time.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/quote/AAPL,MSFT
```

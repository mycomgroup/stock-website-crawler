## About CIK Mapper API

The CIK Mapper endpoint converts a CIK number to a company name. This endpoint can be used by investors to quickly identify the company that is associated with a particular CIK number.
  Investors can use the CIK Mapper endpoint to:
  Quickly identify the company that is associated with a particular CIK number.
  Verify the CIK number for a particular company.
  Identify companies by their CIK number when using other financial data providers or APIs.

## About Mutual Funds Prices API

This endpoint returns information about mutual fund stocks such as price, market capitalization, change, and more. Virtus KAR Small-Cap Core Fund Class I, Fidelity Advisor Managed Retirement 2020 Fund Class A, and Virtus KAR Small-Cap Core Fund Class II are a few examples of mutual fund stocks. Use the /v3/symbol/available-mutual-funds endpoint to see all available mutual funds in our API.

**Endpoint:**

```text
https://financialmodelingprep.com/api/v3/quotes/mutual_fund
```

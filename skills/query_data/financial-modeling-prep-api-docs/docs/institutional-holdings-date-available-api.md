## About Portfolio Holdings dates API

The portfolio holdings dates endpoint provides the dates on which portfolio holdings data is updated. This information is useful for investors who need to stay up-to-date on the latest portfolio holdings data and who need to track changes in the investment portfolios of institutional investors.
  For example, investors may want to use the portfolio holdings dates endpoint to ensure that they are viewing the most recent portfolio holdings data for a particular institutional investor or to track changes in the portfolio holdings of an institutional investor over time.

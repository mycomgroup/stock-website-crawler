## About 13F Asset Allocation dates API

The 13F asset allocation dates endpoint provides the dates on which 3F asset allocation data is updated. This information is useful for investors who need to stay up-to-date on the latest asset allocation data and who need to track changes in the investment strategies of institutional investors.
  For example, investors who are tracking the investment strategy of a particular institutional investor may want to use the 13F asset allocation dates endpoint to ensure that they are viewing the most recent asset allocation data for that investor.

# 任务 01：抽 A 股股票池过滤器 - 回执

## 第一版立即实现的过滤器

| 过滤器名称 | 目标文件 | 核心实现要点 |
|-----------|---------|------------|
| ST过滤 | `filter_st.py` | is_st字段+名称检查（ST/*/退） |
| 停牌过滤 | `filter_paused.py` | paused字段，支持停牌天数阈值 |
| 次新过滤 | `filter_new_stock.py` | 计算上市天数，参数化阈值 |
| 涨停过滤 | `filter_limitup.py` | 对比最新价与涨停价，保留持仓 |
| 跌停过滤 | `filter_limitdown.py` | 对比最新价与跌停价，边界说明 |
| 科创/北交所 | `filter_kcbj.py` | 代码前缀过滤（4/8/68开头） |
| 组合管道 | `filter_pipeline.py` | 串联执行多个过滤器 |
| 数据契约 | `contract.py` | FilterInput/FilterOutput数据类 |

## 最关键的输入输出字段

### 输入（FilterInput）
- `base_universe`: List[str] - 基础股票池
- `date`: datetime.date - 过滤日期
- `config`: Dict[str, Any] - 过滤配置
- `positions`: Optional[List[str]] - 当前持仓（涨跌停过滤需要）

### 输出（FilterOutput）
- `filtered_universe`: List[str] - 过滤后股票池
- `filter_stats`: Dict[str, int] - 各过滤器移除数量统计
- `removed_reasons`: Dict[str, List[str]] - 移除原因明细

## 关键技术点

### 1. 指标口径统一

- 年化因子：252（日频）
- 夏普计算：`mean/std * sqrt(252)`
- Alpha/Beta：回归法（statsmodels）
- 最大回撤：累计净值法

### 2. 库依赖策略

- **必需**：pandas, numpy
- **推荐**：empyrical（高性能）
- **可选**：statsmodels（回归）、plotly（图表）

### 3. 横向比较能力

```python
perf_a = compute_performance(returns_a, benchmark)
perf_b = compute_performance(returns_b, benchmark)
comparison = pd.concat([perf_a, perf_b], axis=1, keys=['策略A', '策略B'])
```

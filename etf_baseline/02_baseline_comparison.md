## 2. 动量因子定义

采用两种动量因子:

**因子A: 简单收益率动量**
```python
momentum = (close[-1] / close[-N] - 1)
```

**因子B: 线性回归斜率动量 (来自 notebook 43 和策略 05/17)**
```python
y = np.log(close[-N:])
x = np.arange(N)
slope, _ = np.polyfit(x, y, 1)
annualized = (math.exp(slope) ** 250) - 1
r_squared = 1 - (sum((y - (slope * x + intercept))**2) / ((N - 1) * np.var(y, ddof=1)))
score = annualized * r_squared
```

**本基线采用因子A (简单收益率)**, 原因:
1. 更简洁, 更少过拟合风险
2. notebook 43 结论: 10/20日简单动量 IC 已经显著
3. 便于后续扩展和解释

## 5. 预期结论 (基于文献)

根据 notebook 43 的 IC 分析:
- **10日动量** 对 10-20日未来收益 IC 最高
- **20日动量** 对 20日未来收益 IC 最稳定
- 因子半衰期约 9-10天, 与 10日持有周期匹配

根据策略 05/17 的回测经验:
- **29日动量** 配合 RSRS 择时效果好
- 持有 top 1 比 top 2 收益更高 (但风险也更大)
- 每日调仓收益高于周度调仓 (但换手成本更高)

**预期最优组合**: 动量 20日 / 持有 10日 或 动量 10日 / 持有 10日

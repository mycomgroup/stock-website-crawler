# 任务 23：日线策略真实白名单

## 任务目标

把日线股票/ETF策略从“理论候选”推进到“真实成功样本池”，形成第一批可复跑白名单。

## 负责范围

- `run_daily_strategy_batch.py`
- `jqdata_akshare_backtrader_utility/jq_strategy_runner.py`
- 与日线回测直接相关的脚本和测试

## 建议写入目录

- 代码写入：
  - 仓库根目录
  - `jqdata_akshare_backtrader_utility/`
- 结果写入：
  - `docs/0330_result/`
  - 建议文件名：`task23_daily_whitelist_real_run_result.md`

## 给子 Agent 的提示词

你负责建立第一批“真实成功”的日线策略白名单。

要求：

- 不要再只做候选列表，要真实跑策略
- 优先选择：
  - ETF 轮动
  - 指数跟踪
  - 基本面选股
  - 不依赖分钟/期货/文件/ML 的纯日线策略
- 目标输出：
  - 真正成功跑通的白名单
  - 每个样本的运行证据
  - 失败样本及失败原因
- 优先支持缓存/离线模式，避免把网络问题误判成策略不兼容

## 任务验证

- 至少跑 10 到 20 个真实策略
- 白名单中至少要有若干真实成功样本
- 每个成功样本至少记录：
  - 加载成功
  - 进入回测
  - 有净值序列

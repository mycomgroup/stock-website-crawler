## 关键发现

### "加载成功" ≠ "真跑通"

**假跑通特征**:
- 加载成功但无交易
- 加载成功但无净值变化
- 加载成功但最终资金=初始资金
- 有错误但验证脚本误判

**真跑通标准**:
1. load_success: 策略加载成功
2. entered_backtest_loop: 进入回测循环
3. has_nav_series: 有净值序列
4. has_transactions: 有交易记录
5. pnl_pct: 盈亏比例 ≠ 0

### 聚宽策略的特征

**定时器机制**:
- 使用`run_monthly()`、`run_daily()`、`run_weekly()`
- 不需要显式的`handle_*`函数
- 在`initialize()`中注册定时器

**验证脚本需要识别**:
- 是否有`run_monthly`等定时器注册
- 是否有`handle_*`或`trading_*`函数
- 两者至少有一即可

## 二、核心选股/轮动逻辑汇总

### 2.1 动量因子（最主流）

```python
# 标准动量计算（多数策略采用）
def get_momentum_score(etf, days=25):
    df = attribute_history(etf, days, '1d', ['close'])
    y = np.log(df.close)
    x = np.arange(len(y))
    slope, intercept = np.polyfit(x, y, 1)
    annualized_returns = math.pow(math.exp(slope), 250) - 1
    r_squared = 1 - (sum((y - (slope * x + intercept))**2) / ((len(y) - 1) * np.var(y, ddof=1)))
    score = annualized_returns * r_squared
    return score
```

**关键参数**:
- 动量窗口: 20-30天（最常用25天）
- 打分方式: 年化收益 × R²（拟合度）
- 持仓数量: 1-3只（多数为1只）

## 三、择时方法汇总

### 3.1 RSRS择时（最主流）

```python
def get_rsrs_signal():
    # 1. 计算斜率
    high_low_data = attribute_history(ref_stock, N, '1d', ['high', 'low'])
    slope, intercept = np.polyfit(low, high, 1)
    
    # 2. 计算R²
    r2 = 1 - (sum((high - (slope * low + intercept))**2) / ((len(high) - 1) * np.var(high, ddof=1)))
    
    # 3. 计算标准分
    slope_series.append(slope)
    zscore = (slope - mean) / std
    
    # 4. 修正标准分 = zscore * r²
    rsrs_score = zscore * r2
    
    # 5. 信号判断
    if rsrs_score > threshold: return "BUY"
    elif rsrs_score < -threshold: return "SELL"
    else: return "KEEP"
```

**关键参数**:
- N = 18（计算斜率的窗口）
- M = 600（计算标准分的历史窗口）
- threshold = 0.7（信号阈值）

## 六、策略优缺点分析

### 6.1 纯动量轮动
**优点**:
- 逻辑简单清晰
- 历史表现优异（8年10-13倍）
- 适合宽基ETF

**缺点**:
- 对参数敏感
- 震荡市表现差
- 需要RSRS择时配合

### 6.2 行业轮动
**优点**:
- 收益来源多元
- 可捕捉行业热点
- 中长期表现稳定

**缺点**:
- 行业ETF流动性差异大
- 行业切换频繁增加成本
- 需要较强的行业判断能力

### 6.3 宽基轮动
**优点**:
- 流动性好
- 交易成本低
- 适合大资金

**缺点**:
- 收益弹性相对较小
- 容易受大盘系统性风险影响

### 6.4 相关性优化
**优点**:
- 分散风险
- 组合稳定性好
- 降低单一标的依赖

**缺点**:
- 计算复杂度高
- 需要定期更新相关性矩阵
- 可能错过高相关性优质标的

## 七、核心结论

### 7.1 动量因子是核心
- **年化收益 × R²** 是最有效的打分方式
- **25天窗口** 是最佳参数选择
- 动量因子半衰期约 **9-10天**

### 7.2 择时是必要条件
- **RSRS修正标准分** 效果最佳
- 择时阈值 **0.7** 是经验值
- 需要结合 **MA均线** 做二次确认

### 7.3 ETF池质量优先
- **流动性** 是第一要素
- **低相关性** 比高数量更重要
- **历史长度** 影响因子计算稳定性

### 7.4 实盘约束不可忽视
- 滑点设置: **0.1%-0.2%**
- 交易成本: **万3-万5**
- 调仓频率: **每日或每周**

## 八、推荐组合策略

基于以上分析，推荐以下组合策略框架：

```
1. 候选池: 宽基ETF（5-10只）+ 行业ETF（5-10只）
2. 选股: 年化收益 × R² 打分，取前1-3只
3. 择时: RSRS修正标准分 > 0.7 且 MA均线上行
4. 调仓: 每日检查，有信号时调仓
5. 止损: 80%止损（几乎不触发）
6. 空仓: 买入货币基金（511880）
```

**预期表现**:
- 年化收益: 30%-80%
- 最大回撤: 10%-20%
- 夏普比率: 1.5-2.5

for stock in funds:

    cm = cov(rs, rx)
    beta = cm[0,1] / cm[1,1]  # Beta系数

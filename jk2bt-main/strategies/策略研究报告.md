## 3. 择时类策略

### 3.1 RSRS择时指标

**代表文件**: `67 RSRS择时改进-【成交量加权-钝化-右偏】.txt`

**核心逻辑**:
```
1. 对N日最高价与最低价进行线性回归，得到斜率β
2. 对β进行M日标准化: Z = (β - μ) / σ
3. 右偏修正: RSRS = β × Z
4. 信号规则:
   - RSRS > 0.8: 买入信号
   - RSRS < -0.8: 卖出信号
   - 其他: 观望
```

**关键参数**:
- N = 18 (回归周期)
- M = 800 (标准化窗口)
- 买入阈值: 0.8
- 卖出阈值: -0.8

**历史表现**:
- 年化收益: 15-25%
- 最大回撤: 10-15%
- 夏普比率: 1.2-1.8

**改进版本**:
- 成交量加权RSRS
- 钝化RSRS
- 北向资金+RSRS

### 3.2 市场宽度指标

**代表文件**: `45 研究 市场宽度.ipynb`, `52 市场宽度——简洁版.ipynb`

**核心逻辑**:
```
计算每日各行业成分股中，收盘价高于20日均线的比例
汇总所有行业的及格率，形成市场宽度指标
```

**三种方案**:
| 方案 | 计算方式 | 说明 |
|------|----------|------|
| 方案一 | 各行业及格率直接加总 | max=行业数×100 |
| 方案二 | 行业及格率取平均 | 转为百分比 |
| 方案三 | 全市场整体比例 | 不分行业 |

**信号判断**:
- 宽度 < 30%: 极度悲观，可能底部
- 宽度 > 70%: 极度乐观，注意顶部

### 3.3 拥挤率指标

**代表文件**: `82 拥挤率指标-择时大盘顶底-Clone1-第二版.ipynb`

**核心逻辑**:
```
拥挤率 = 成交额前5%股票的成交额之和 / 全市场总成交额 × 100
```

**信号判断**:
- 拥挤率 > 60%: 资金过度集中，注意轮动
- 拥挤率 < 40%: 资金分散，可能见底

### 3.4 GSISI投资者情绪指数

**代表文件**: `93 【复现】国信投资者情绪指数择时模型.ipynb`

**核心逻辑**:
```
1. 计算28个申万一级行业周收益率
2. 计算各行业相对沪深300的周Beta系数(滚动窗口)
3. 计算周收益率与周Beta系数的Spearman秩相关系数
```

**信号规则**:
- GSISI ≥ 0.301: 看多信号
- GSISI ≤ -0.301: 看空信号
- 连续两次同向信号确认后执行操作

**回测结果**:
- 窗口参数: window=35, pct_window=15
- 年化收益: 13.01%
- 夏普比率: 79.76%

```
卫星仓位 (30%)
└── 高股息价值股 (30%)
    ├── 筛选条件: PE<30, 利润增长>10%
    └── 持仓数量: 5-10只

机动仓位 (10%)
└── RSRS择时增强 (10%)
    ├── 右偏分>0.8: 加仓至20%
    └── 右偏分<-0.8: 减仓至0%
```

### 13.2 策略适用性评估

| 策略类型 | 实盘难度 | 资金容量 | 风险等级 | 2024年有效性 |
|----------|----------|----------|----------|--------------|
| RSRS择时 | 中 | 大 | 低 | 较高 |
| ETF动量轮动 | 低 | 大 | 中 | 高 |
| ARBR多因子 | 中 | 中 | 中 | 中 |
| 高股息价值 | 低 | 大 | 低 | 高 |
| 小市值+止损 | 高 | 小 | 高 | **低(需谨慎)** |
| 股债平衡 | 低 | 大 | 低 | 高 |

### 13.3 回避策略

- **小市值策略**: 2024年回撤较大，流动性风险增加
- **龙头打板**: 需要专业程序化支持，不适合普通投资者
- **纯动量策略**: 在震荡行情中表现不佳

## 14. 风险提示

### 14.1 通用风险

1. **过拟合风险**: 部分高收益策略可能存在过拟合
2. **市场适应性**: 策略需要根据市场环境变化进行调整
3. **交易成本**: 高频策略需考虑滑点和交易成本
4. **资金容量**: 小市值和打板策略资金容量有限

### 14.2 2024年特别风险

1. **小市值流动性风险**: 退市风险上升，建议回避
2. **全球资产波动**: 美股高位，需控制纳指ETF仓位
3. **利率风险**: 国债收益率下行空间有限
4. **政策风险**: 量化交易监管趋严

### 14.3 风险控制建议

1. **仓位管理**: 单策略不超过总资金30%
2. **止损设置**: 所有策略设置5-10%止损
3. **定期再平衡**: 月度/季度再平衡
4. **分散投资**: 多策略、多资产类别分散

## 四、关键概念说明

### 4.1 市场宽度 (Market Breadth)

**定义**: 股价高于20日均线股票所占比例

**计算**:
```python
breadth = (收盘价 > MA20 的股票数) / 总股票数
```

**应用场景**:
- 宽度 > 0.60 → 强势市场，满仓操作
- 宽度 0.25-0.60 → 震荡市场，正常持仓
- 宽度 < 0.25 → 弱势市场，减仓至10只
- 宽度 < 0.15 → 空仓观望

### 4.2 RFScore7 计算逻辑

7个财务指标:
1. ROA > 0 (资产回报率)
2. ΔROA > 0 (ROA同比变化)
3. OCFOA > 0 (经营现金流/资产)
4. Accrual < 0 (应计项目)
5. ΔLeverage < 0 (杠杆率下降)
6. ΔMargin > 0 (毛利率提升)
7. ΔTurnover > 0 (资产周转率提升)

**F-Score**: 7个指标中为正的数量 (0-7分)

### 4.3 PB 分组逻辑

- 股票按 PB 分为 10 组 (1=最低, 10=最高)
- PB10% = 第 1 组 (PB 最低的 10% 股票)
- PB20% = 第 1-2 组 (PB 最低的 20% 股票)

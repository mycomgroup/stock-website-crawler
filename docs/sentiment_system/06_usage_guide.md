## 一、每日计算流程

### 1.1 时间节点

| 时间 | 操作 | 数据状态 |
|------|------|----------|
| 前一日15:00后 | 计算T-1日情绪指标 | T-1收盘数据 |
| 当日09:00前 | 查看情绪开关状态 | T-1收盘数据 |
| 当日09:25前 | 决定是否开仓 | T-1收盘数据 |
| 当日14:30 | 盘中监控（可选） | - |

### 1.2 完整流程

```python
"""
每日盘前情绪检查流程
"""

# Step 1: 获取前一交易日
from datetime import datetime, timedelta

def get_prev_trade_day(date_str):
    """获取前一交易日"""
    trade_days = get_trade_days(
        start_date=(datetime.now() - timedelta(days=10)).strftime('%Y-%m-%d'),
        end_date=date_str
    )
    return trade_days[-2].strftime('%Y-%m-%d')  # 前一个

# Step 2: 计算情绪指标
from sentiment_indicators import calc_market_sentiment

date = '2024-01-15'
prev_date = get_prev_trade_day(date)

sentiment = calc_market_sentiment(date, prev_date)
print(f"涨停: {sentiment['zt_count']}")
print(f"跌停: {sentiment['dt_count']}")
print(f"涨跌停比: {sentiment['zt_dt_ratio']:.2f}")
print(f"最高连板: {sentiment['max_lianban']}")
print(f"晋级率: {sentiment['jinji_rate']:.1%}")

# Step 3: 判断开关状态
from sentiment_switch import sentiment_switch_combo, get_switch_recommendation

should_open = sentiment_switch_combo(sentiment)
recommendation = get_switch_recommendation(sentiment)

print(f"\n是否开仓: {'是' if should_open else '否'}")
print(f"仓位建议: {recommendation['position']}")
print(f"操作建议: {recommendation['action']}")
print(f"原因: {recommendation['reason']}")
```

## 二、快速决策表

### 2.1 一眼判断

| 涨停家数 | 最高连板 | 涨跌停比 | 操作 | 仓位 |
|----------|----------|----------|------|------|
| < 15 | - | - | 空仓 | 0% |
| 15-30 | <= 2 | < 1.5 | 观望 | 0-10% |
| 15-30 | >= 3 | >= 1.5 | 谨慎开仓 | 20-30% |
| 30-50 | >= 2 | >= 1.5 | 正常开仓 | 50% |
| 30-50 | >= 3 | >= 2 | 积极开仓 | 70% |
| > 50 | >= 3 | >= 2 | 满仓 | 100% |
| > 50 | >= 5 | >= 2 | 注意风险 | 50% |

### 2.2 简化判断（手算版）

**只需记住两个数字**：

> 涨停家数 >= 30？  
> 最高连板 >= 2？

两个都满足 → 开仓  
两个都不满足 → 空仓  
只有一个满足 → 谨慎

## 三、注意事项

### 3.1 数据相关

| 注意事项 | 说明 | 应对 |
|----------|------|------|
| T-1日数据 | 情绪指标基于T-1收盘 | 注意隔夜外盘变化 |
| 节假日 | 节后首日可能极端 | 降低仓位 |
| 停牌 | 股票停牌时无法交易 | 预留备选股 |
| ST股 | ST股不纳入统计 | 自动过滤 |

### 3.2 滑点预留

| 策略类型 | 建议滑点预留 |
|----------|--------------|
| 首板低开 | 0.5% |
| 弱转强 | 0.3% |
| 234板 | 0.5% |
| 小市值 | 1.0% |

### 3.3 异常处理

```python
def safe_check_sentiment(date):
    """安全检查情绪状态"""
    try:
        sentiment = calc_market_sentiment(date, prev_date)
        
        # 检查数据有效性
        if sentiment['zt_count'] == 0:
            print("警告：涨停家数为0，数据可能异常")
            return None
        
        if sentiment['zt_count'] > 300:
            print("警告：涨停家数异常高，可能包含涨跌停数据问题")
        
        return sentiment
    
    except Exception as e:
        print(f"计算情绪指标失败: {e}")
        return None
```

| 观察点 | 异常信号 | 应对 |
|--------|----------|------|
| 涨停股炸板 | 多只涨停股打开 | 减仓 |

| 指数大跌 | 大盘下跌 > 2% | 减仓 |

## 五、常见问题

### Q1: 应该在什么时候计算情绪指标？

**A**: 使用T-1日收盘数据，在T日开盘前计算。建议在9:00前完成。

### Q2: 涨停家数突然很少怎么办？

**A**: 检查是否是极端行情（如春节前）。如果连续3天涨停少于15家，建议空仓。

### Q3: 指标矛盾时怎么办？

**A**: 以涨停家数为主。例如：
- 涨停>=30 但 连板<=2 → 谨慎开仓
- 涨停<30 但 连板>=4 → 空仓（连板可能是少数股票操纵）

### Q4: 情绪开关和选股哪个先做？

**A**: 先做情绪开关。情绪开关决定是否开仓，选股决定开哪些股票。

### Q5: 盘中可以追涨吗？

**A**: 不建议。情绪开关基于收盘数据，盘中追涨风险大。

### Q6: 周末需要做什么？

**A**: 
1. 周五收盘后计算下周情绪
2. 查看周末外盘和消息面
3. 周日晚上准备下周计划

### Q7: 情绪指标有未来函数吗？

**A**: 没有。情绪指标使用T-1日收盘数据，T日开盘前即可计算，不存在未来函数。

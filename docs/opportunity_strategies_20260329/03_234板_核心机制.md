# 任务 03：234 板介入机会仓核心机制研究

## 设计文档

### 任务定位

- 主线：`234板接力`
- 目标：找出这类高爆发接力策略只在什么情绪窗口下才值得开仓，以及最容易赚钱的是二板、三板还是四板。
- 角色：机会仓最进攻的子策略，只在强情绪窗口出手。

### 当前已知问题

- 原始策略 `93 234板介入-2023年14倍-无未来.txt` 的收益极强，但高度依赖 `2023` 市场环境。
- 原规则对情绪过滤写得不够硬，容易在弱市把高波动放大成大回撤。
- 二板、三板、四板的风险收益结构可能完全不同，但原稿没有拆开。
- 实盘上存在排板买不到、开盘低开直接闷杀、容量受限等问题。

### 本轮要回答的问题

1. 真正值得保留的是 `二板`、`三板` 还是 `四板`。
2. `最高连板数`、`涨停家数`、`晋级率` 哪个最适合当总开关。
3. 自由流通市值、换手率、缩量条件的边界在哪里。
4. 如果只在强情绪窗口开仓，能否把高收益保下来并显著降低回撤。

### 研究假设与重点变量

重点拆解：

- 板位：二板、三板、四板分开测
- 情绪：最高连板数、涨停家数、晋级率、龙头次日溢价
- 控盘特征：换手率、昨日量比、是否一字板、自由流通市值
- 退出规则：日内卖、次日卖、两日上限、炸板即走
- 参与方式：排板、开盘买、只做非一字换手板

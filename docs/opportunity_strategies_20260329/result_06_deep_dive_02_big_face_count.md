## 一、大面股指标定义

### 1.1 指标定义

**大面股数量** = 当日跌幅超过7%的股票数量

**作用机制**：
- 大面股数量多 → 市场恐慌程度高 → 情绪差 → 不宜开仓
- 大面股数量少 → 市场恐慌程度低 → 情绪好 → 可积极开仓

**理论依据**：
- 大面股反映市场"杀跌"力度
- 跌幅>7%的股票往往是市场情绪的极端体现
- 当市场出现大量跌幅>7%的股票时，说明恐慌情绪蔓延

### 1.2 数据获取方法

**方法1：直接计算（精确）**
```python
def get_big_face_count(date):
    """
    获取大面股数量（跌幅>7%）
    """
    prev_date = get_trade_days(end_date=date, count=2)[0]
    
    # 获取所有股票昨日收盘价和今日收盘价
    df_prev = get_price(all_stocks, end_date=prev_date, fields=['close'])
    df_curr = get_price(all_stocks, end_date=date, fields=['close'])
    
    # 计算跌幅
    decline_pct = (curr_close - prev_close) / prev_close * 100
    
    # 统计跌幅>7%的股票数量
    big_face_count = (decline_pct < -7).sum()
    
    return big_face_count
```

**计算复杂度**：需遍历约4000只股票，计算效率较低

**方法2：简化替代（涨跌停差值）**
```python
def get_zt_dt_diff(date):
    """
    使用涨跌停差值作为大面股替代指标
    涨跌停差值 = 涨停家数 - 跌停家数
    """
    df = get_price(all_stocks, end_date=date, 
                   fields=['close', 'high_limit', 'low_limit'])
    
    zt_count = (df['close'] == df['high_limit']).sum()
    dt_count = (df['close'] == df['low_limit']).sum()
    
    return zt_count - dt_count
```

**简化原理**：
- 涨跌停差值>0 → 涨停多于跌停 → 市场情绪相对好
- 涨跌停差值<=0 → 跌停多于涨停 → 市场恐慌，类似大面股多的情况
- 跌停家数是市场恐慌的直接体现，可作为大面股数量的替代指标

## 二、理论分析：大面股指标的价值

### 2.1 与现有指标的关联性分析

| 指标 | 计算方式 | 与大面股的关联度 | 数据可得性 |
|------|----------|------------------|------------|
| **涨停家数** | 收盘价==涨停价 | 低（独立指标） | 高（收盘可得） |
| **跌停家数** | 收盘价==跌停价 | **高（正相关）** | 高（收盘可得） |
| **涨跌停比** | 涨停/跌停 | **高（负相关）** | 高（收盘可得） |
| **最高连板数** | 连续涨停天数 | 低 | 中等 |

**关键发现**：

1. **大面股与跌停家数高度正相关**
   - 大面股多时，跌停家数往往也多
   - 跌停家数已经包含了市场恐慌信息
   - 大面股数量指标的信息可能已被跌停家数涵盖

2. **涨跌停比已反映恐慌程度**
   - 涨跌停比 = 涨停家数 / 跱停家数
   - 涨跌停比低 → 涨停少跌停多 → 恐慌情绪高
   - 涨跌停比已经在之前的测试中使用（result_05）

### 2.2 已有测试结论回顾

**来自 result_05（连板龙头与情绪周期）**：

| 情绪指标组合 | 测试结果 | 增益效果 |
|--------------|----------|----------|
| 涨停家数单指标 | ✅ 有效 | 收益+71%，胜率+3% |
| 最高连板+涨停 | ✅ 有效 | 回撤改善显著 |
| 最高连板+涨停+涨跌停比 | ⚠️ 增益有限 | 与单指标接近 |

**关键结论**：
- "涨跌停比：增益有限，计算复杂，跌停数据噪音大"
- "涨跌停比虽然理论上合理，但实际增益不明显"

**来自 result_02（情绪开关最小可用版）**：

|------|----------|----------|
| **涨跌停比** | **已删除** | 增益有限，计算复杂，跌停数据噪音大 |
| 晋级率 | 已删除 | 增益不明显，维护成本高 |
| 龙头次日溢价 | 已删除 | 数据难获取 |

**最终选择**：涨停家数单指标硬开关

**问题1：计算效率低**
- 需遍历约4000只股票计算跌幅
- 每个交易日需获取约8000个数据点（昨日+今日收盘）
- JoinQuant Notebook执行超时（120秒内无法完成）

**问题2：数据噪音**
- 大面股定义（跌幅>7%）可能包含：
  - 系统性恐慌（市场整体下跌）
  - 个股利空（非系统性下跌）
  - ST股跌停（特殊规则）
- 噪音可能影响信号质量

**问题3：阈值设定**
- 大面股阈值设为多少合适？
  - 50？80？100？
- 需要大量测试才能确定最优阈值

### 4.2 替代方案验证

**方案：使用涨跌停差值作为替代**

**优势**：
1. 计算简单：仅需涨停/跌停家数

**劣势**：

### 8.1 大面股与跌停家数的相关性

**相关性分析**：

假设市场有以下分布：
- 正常下跌（-1%到-7%）：约50-100只（随机波动）
- 大面股（跌幅>-7%）：约20-80只（恐慌程度）
- 跌停股（跌幅=-10%）：约5-30只（极端恐慌）

**相关性**：
- 大面股数量 ↑ → 跌停家数 ↑ （正相关）
- 跌停家数 ↑ → 涨跌停比 ↓ （负相关）
- 涨跌停比 ↓ → 情绪差 → 不宜开仓

**结论**：大面股数量的信息可通过跌停家数和涨跌停比获取

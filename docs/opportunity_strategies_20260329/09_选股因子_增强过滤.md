# 任务 09：选股因子排序与增强过滤研究

## 设计文档

### 任务定位

- 主线：`排序因子 + 过滤因子`
- 目标：在基础信号已经出现后，找出最能提升短期爆发力的排序和过滤规则。
- 角色：不是另起一套慢策略，而是给机会仓入口做优先级排序和错误信号剔除。

### 当前已知问题

- 原稿罗列了很多因子，但缺少优先级与实测排序。
- 因子加得太多会导致信号枯竭，反而失去机会仓意义。
- 不同子策略适合的因子方向可能相反，例如 `234板` 喜欢缩量控盘，而 `弱转强` 更依赖竞价量。
- 目前还没有回答哪些因子在 `2024-2026` 依然有效。

### 本轮要回答的问题

1. 哪些因子最能提升爆发力：自由流通市值、题材热度、换手率、量比、左压、相对位置。
2. 哪些因子应该做硬过滤，哪些更适合做排序加分。
3. 因子是否应该按策略分开配置，而不是一套模板通杀。
4. 引入因子后，信号质量提升是否足以抵消频率下降。

### 研究假设与重点变量

重点拆解：

- 通用因子：自由流通市值、换手率、量比、相对位置、题材热度
- 策略特异因子：竞价量强度、左压突破、炸板次数、缩量程度
- 使用方式：硬过滤、排序评分、分层打分、策略差异化配置
- 评估维度：信号数、胜率、盈亏比、爆发收益尾部、连续亏损次数

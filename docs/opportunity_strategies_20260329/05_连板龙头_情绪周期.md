# 任务 05：连板龙头与情绪周期总开关研究

## 设计文档

### 任务定位

- 主线：`连板龙头 + 情绪周期`
- 目标：给机会仓建立一个可执行的情绪总开关，决定什么时候该积极做短线，什么时候应该直接空仓。
- 角色：不是单独追求最高收益，而是为所有高爆发策略提供开仓环境判断。

### 当前已知问题

- 原始 `07 连板龙头策略.txt` 说明了龙头思路，但没有把情绪周期做成机会仓级别的统一开关。
- 最高连板数、涨停家数、晋级率之间谁更有效，目前没有做严格比较。
- 情绪指标常常在复盘时很清楚，但要转成开盘前可执行规则并不简单。
- 如果没有情绪开关，首板低开、弱转强、234板都会在弱市里被放大回撤。

### 本轮要回答的问题

1. 机会仓最有效的情绪开关是什么：`最高连板数`、`涨停家数`、`晋级率` 还是组合指标。
2. 情绪上升期、高潮期、退潮期的收益结构如何分层。
3. 哪些策略最依赖情绪开关，哪些策略可以弱化情绪依赖。
4. 开关应该是硬开关，还是仓位分层调节器。

### 研究假设与重点变量

重点拆解：

- 情绪指标：最高连板数、涨停家数、跌停家数、涨跌停比、晋级率、龙头次日溢价
- 状态划分：启动期、发酵期、高潮期、退潮期
- 开关方式：全开、半开、空仓、仅保留低频策略
- 作用对象：首板低开、弱转强、234板、龙头底分型分别测试

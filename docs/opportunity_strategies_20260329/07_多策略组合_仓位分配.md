# 任务 07：多策略组合与仓位分配研究

## 设计文档

### 任务定位

- 主线：`机会仓组合层`
- 目标：把首板低开、弱转强、234板、龙头底分型做成一个能协同发力的机会仓，而不是四套孤立脚本。
- 角色：把高爆发策略从单点打法升级为组合打法。

### 当前已知问题

- 原稿有组合思路，但还没有可执行的组合研究任务定义。
- 各子策略持仓周期、出手频率、情绪依赖不同，直接并行叠加可能放大相关性而不是分散风险。
- 缺少统一的冲突处理：同日多个策略看中同一只票时怎么办。
- 缺少统一的组合级评估指标：不能只看年化，还要看资金利用率与回撤恢复速度。

### 本轮要回答的问题

1. 四个核心子策略能否带来真正分散，还是本质上都押注同一类情绪因子。
2. 最优机会仓结构是 `均分`、`按情绪动态切换`，还是 `分层主次策略`。
3. 是否应该保留 `宏观总开关 + 情绪总开关 + 子策略打分` 的三层组合框架。
4. 在允许回撤的前提下，怎样分配仓位最有利于短期收益加速。

### 研究假设与重点变量

重点拆解：

- 子策略权重：固定权重、情绪驱动权重、胜率驱动权重、近期热手权重
- 总仓位：20%、30%、50%、60% 多组测试
- 冲突处理：同票去重、优先级覆盖、信号合并加仓
- 组合评价：收益、回撤、月度恢复、资金占用、换手、最大连续亏损期

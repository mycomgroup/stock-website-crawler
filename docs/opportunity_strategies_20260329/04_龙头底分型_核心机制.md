# 任务 04：龙头底分型机会仓核心机制研究

## 设计文档

### 任务定位

- 主线：`龙头底分型`
- 目标：验证龙头二波启动的低频重仓型机会是否仍有可复制性，并把原始复杂形态简化成可验证、可实盘的规则。
- 角色：机会仓里的低频高赔率信号，不追求出手频率，追求单次大收益。

### 当前已知问题

- 原始策略 `02 龙头底分型战法-两年23倍.txt` 是分钟级回测，历史收益极强，但执行难度高。
- 底分型识别带有较强形态主观性，容易因为定义方式不同而结果飘移。
- 原逻辑依赖主升浪、回调、底分型、分钟级确认，多层条件叠加后可能信号太少。
- 近年高位监管与连板生态变化后，二波启动的持续性需要重新验证。

### 本轮要回答的问题

1. 底分型最核心的是 `主升浪背景`、`回调质量`、`T-1 十字星` 还是 `T 日确认`。
2. 是否可以用更简单的结构替代复杂缠论表达。
3. 分钟级确认是否显著优于日级简化版本。
4. 龙头底分型是否更适合作为单独重仓子策略，而非高频组合部件。

### 研究假设与重点变量

重点拆解：

- 主升浪定义：40 日涨幅、涨停数量、新高约束
- 回调质量：回调天数、回撤幅度、缩量程度
- 底分型定义：十字星、小实体、下影线、支撑位
- 触发确认：开盘高开、5 分钟突破、量能确认
- 退出规则：亏损止损、盈利 20%/30%/40% 分档止盈

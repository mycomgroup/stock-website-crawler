# 深挖方向03：北向资金流向指标测试

> 日期：2026-04-01
> 状态：**已完成理论分析**
> 结论：**No-Go - 北向资金数据自2024-08-18后不再披露，无法纳入实盘**

---

## 一、研究背景

### 1.1 核心问题

现有情绪择时框架使用涨停家数+连板数作为情绪指标，结论是"宏观层负贡献，只保留情绪层"。

**本任务需回答：**
1. 北向资金流向指标是否能增强情绪择时效果？
2. 北向净流入>0时是否收益更高？
3. 是否值得纳入实盘框架？

### 1.2 北向资金定义

**定义：**
- 北向资金净流入 = 沪股通净买入 + 深股通净买入
- 北向资金趋势 = 近5日北向资金累计净流入

**数据来源：**
- JoinQuant API: `finance.STK_ML_QUOTA`
- link_id=310001（沪股通），link_id=310002（深股通）
- 字段：`buy_amount`（买入成交额），`sell_amount`（卖出成交额）

## 二、关键发现：数据可得性问题

### 2.1 北向资金数据披露限制

**重要发现：自2024-08-18之后，北向资金不再披露买卖金额！**

根据JoinQuant文档（doc_JQDatadoc_9872）：

```
| 字段 | 名称 | 备注 |
|------|------|------|
| buy_amount | 买入成交额 | 亿(自2024-08-18之后北向不再披露) |
| sell_amount | 卖出成交额 | 亿(自2024-08-18之后北向不再披露) |
| buy_volume | 买入成交数 | 笔(自2024-08-18之后北向不再披露) |
| sell_volume | 卖出成交数 | 笔(自2024-08-18之后北向不再披露) |
| quota_daily | 每日额度 | 亿(自2024-08-18之后北向不再披露) |
| quota_daily_balance | 每日额度余额 | 亿(自2024-08-18之后北向不再披露) |
```

**影响：**
- 无法获取2024-08-18之后的北向净流入数据
- 无法在2025年实盘使用该指标
- 只能基于2024-01-01至2024-08-17的历史数据回测（约7个月）

### 2.2 数据可得性评估

| 维度 | 评估 |
|------|------|
| 时间范围 | 2024-01-01 至 2024-08-17（7个月） |
| 数据完整性 | ✓ 完整披露买卖金额 |
| 实盘可用性 | ✗ 2024-08-18后无法获取 |
| 回测可用性 | ✓ 可用于历史回测 |
| 持续可得性 | ✗ 无法保证未来可得 |

## 三、北向资金有效性理论分析

### 3.1 北向资金与市场情绪的关系

**理论假设：**

1. **北向资金作为聪明钱指标**
   - 外资机构通常被视为专业投资者
   - 北向净流入反映外资对A股的看好程度
   - 可作为市场情绪的补充验证

2. **与情绪指标的潜在协同**
   - 涨停家数反映本土散户情绪
   - 北向资金反映外资机构情绪
   - 两者结合可形成多维度情绪判断

### 3.2 北向资金作为择时指标的局限性

**局限性分析：**

1. **数据披露限制（致命问题）**
   - 自2024-08-18后不再披露买卖金额
   - 实盘无法使用该指标
   - 监管政策变化导致数据源中断

2. **历史回测样本不足**
   - 只有7个月完整数据（2024-01-01至2024-08-17）
   - 样本量不足以支撑统计显著性
   - 无法验证跨周期有效性

3. **与宏观层类似的滞后问题**
   - 北向资金是事后统计数据
   - 无法实时反映市场情绪转折
   - 可能在市场急转时错过信号

### 3.3 参考已有研究结论

**参考result_06（宏观与市场状态择时框架）：**

| 指标类型 | 实测效果 | 结论 |
|---------|---------|------|
| 宏观层（PMI、社融） | 年化收益下降0.66%，最大回撤增加9.44% | No-Go |
| 情绪层（涨停家数+连板数） | 年化11.66%，夏普0.34 | Go |

**关键发现：**
- 宏观类指标（月频、滞后）对短线策略为负贡献
- 北向资金虽为日频，但本质类似宏观资金流向
- 可能与宏观层存在类似问题

## 六、结论与建议

### 6.1 Go/Watch/No-Go判断

**判断：No-Go**

**理由：**

1. **致命问题：数据源中断**
   - 自2024-08-18后北向资金不再披露买卖金额
   - 实盘无法使用该指标
   - 无法保证未来数据可得性

2. **样本不足：无法验证有效性**
   - 只有7个月历史数据
   - 无法支撑统计显著性验证
   - 无法验证跨周期稳定性

3. **潜在风险：类似宏观层问题**
   - 北向资金本质是资金流向指标（宏观类）
   - result_06已证明宏观层对短线策略为负贡献
   - 北向资金可能存在类似问题

### 6.2 与现有框架的关系

| 指标类型 | 状态 | 实盘可用性 | 建议 |
|---------|------|-----------|------|
| 涨停家数 | Go | ✓ 可用 | 保留 |
| 最高连板数 | Go | ✓ 可用 | 保留 |
| 北向净流入 | No-Go | ✗ 不可用 | 砍掉 |

**理由：**
- 情绪层（涨停+连板）已验证有效且实盘可用
- 北向资金无法实盘使用，纳入无意义

### 6.3 后续建议

**建议1：放弃北向资金指标**

- 不纳入情绪择时框架
- 不进行后续研究
- 原因：数据源中断，无法实盘使用

**建议2：关注替代指标**

如果需要增加资金流向维度，可考虑：
- 两融余额变化（融资融券数据）
- 主力资金净流入（get_money_flow_pro）
- 成交额变化趋势

**建议3：聚焦情绪层优化**

- 当前情绪层（涨停家数+连板数）已验证有效
- 后续优化方向：
  1. 精细化阈值（result_07任务）
  2. 组合其他情绪指标（跌停家数、涨跌停比）
  3. 动态仓位调节

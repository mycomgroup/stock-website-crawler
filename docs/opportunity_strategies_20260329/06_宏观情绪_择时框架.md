# 任务 06：宏观与市场状态择时框架研究

## 设计文档

### 任务定位

- 主线：`宏观择时 + 市场状态划分`
- 目标：先从宏观与中观层面判断当前是否适合做机会仓，再下沉到情绪与具体策略。
- 角色：整个调研批次的第一优先级任务，负责决定什么时候该开仓，什么时候该等。

### 当前已知问题

- 你明确提出可以先调研宏观，再微调，当前目录里这一层还不够任务化。
- 现有短线策略大都从微观形态出发，但没有先判断环境是否适合进攻。
- 单纯用大盘均线做择时可能太粗，单纯用涨停家数又可能太短。
- 宏观指标月频、中观市场状态周频、短线情绪日频，三层之间如何衔接尚未落地。

### 本轮要回答的问题

1. 机会仓是否需要 `宏观月频 + 市场周频 + 情绪日频` 三层结构。
2. 宏观层真正有用的是 `PMI`、`社融`、`北向资金`、`指数趋势` 中的哪些。
3. 宏观过滤会不会大幅牺牲收益，还是能显著改善回撤和空仓效率。
4. 最终应该做成 `硬空仓` 规则，还是做成机会仓总仓位调节器。

### 研究假设与重点变量

重点拆解：

- 宏观月频：PMI、社融、利率方向、北向资金月度趋势
- 市场周频：指数趋势、涨停家数均值、跌停扩散、行业轮动强弱
- 情绪日频：最高连板数、晋级率、龙头溢价、开盘冰点与修复
- 开关方式：全开、半开、空仓、仅保留低频策略
- 作用对象：至少接到 `首板低开` 与 `弱转强` 两个策略上

# 任务 08：止损止盈与机会仓风控体系研究

## 设计文档

### 任务定位

- 主线：`爆发型策略的风险边界`
- 目标：在不牺牲太多进攻性的前提下，把机会仓的大亏损控制在可承受范围内。
- 角色：不是追求稳定系统式的低波动，而是避免爆发型策略在错误阶段把前面赚到的钱一次吐回去。

### 当前已知问题

- 原稿只总结了经验型止损止盈，没有把它变成可比较的研究任务。
- 短线策略最怕不是小亏，而是连续闷杀、跌停卖不掉、高潮次日集体退潮。
- 各子策略止盈止损逻辑不一致，导致组合层难以统一控制。
- 过去高收益策略的回测常常掩盖了实际成交难度和极端日风险。

### 本轮要回答的问题

1. 机会仓最重要的风险来自 `单票大亏`、`连续亏损`、`情绪退潮` 还是 `流动性失真`。
2. 爆发型策略更适合 `时间止损`、`价格止损` 还是 `情绪止损`。
3. 组合级风控要不要硬性熔断，例如单周亏损阈值后强制休息。
4. 怎样在允许短期回撤的同时，避免净值出现无法恢复的深坑。

### 研究假设与重点变量

重点拆解：

- 单票风险：固定止损、ATR 止损、开盘跳空止损、炸板止损
- 时间风险：`10:30` 未盈利、`13:30` 未达标、尾盘统一卖
- 组合风险：单日亏损阈值、单周亏损阈值、单月回撤阈值
- 情绪风险：龙头断板、涨停家数骤降、退潮日自动降仓
- 执行偏差：跌停卖不出、涨停买不到、滑点与冲击成本

# 任务 02：弱转强竞价机会仓核心机制研究

## 设计文档

### 任务定位

- 主线：`弱转强竞价`
- 目标：验证竞价弱转强是否仍然是机会仓里最强的单点进攻策略，以及它在近两年是否仍可复制。
- 角色：低频高质量爆发入口，允许错过，不允许在错误情绪里硬做。

### 当前已知问题

- 原始策略 `21 【原创】年化123%最大回撤14%涨停弱转强竞价战法.txt` 的历史回测非常强，但高收益可能高度依赖特定阶段。
- 竞价数据、左压逻辑、开盘成交可实现性都容易带来回测高估。
- 市值门槛、竞价量门槛、左压突破强度是否过拟合，尚未做系统拆解。
- 原策略偏单一规则，没有回答在不同情绪阶段应该如何开关。

### 本轮要回答的问题

1. 弱转强竞价的核心 alpha 来自 `高开幅度`、`竞价量比`、`左压突破`，还是来自三者共振。
2. 大市值过滤是否真的必要，还是压制了真正的高弹性小票。
3. 竞价弱转强是否必须叠加连板龙头情绪周期才能稳定。
4. 原有多时点止损是否是收益回撤比最优解。

### 研究假设与重点变量

重点拆解：

- 竞价高开幅度：`0%~2%`、`2%~4%`、`4%~6%`
- 竞价量强度：竞价量占昨量比例、竞价金额、竞价量与左压峰值比
- 约束条件：流通市值、昨日成交额、前两日是否曾涨停
- 卖出规则：`9:30 止损`、`10:30 未盈利卖`、`13:30 未达标卖`、`尾盘统一卖`
- 情绪门槛：最高连板数、涨停家数、晋级率

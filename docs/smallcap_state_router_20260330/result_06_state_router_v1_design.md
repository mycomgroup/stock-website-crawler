## 二、状态路由器规则表设计

### 2.1 双维度定义

#### 维度1：市场广度（沪深300站上20日线占比）

| 分级 | 定义 | 占比范围 | 特征 |
|------|------|----------|------|
| **极弱** | 等级1 | <15% | 大多数成分股跌破20日线，市场极度疲弱 |
| **弱** | 等级2 | 15%-25% | 约1/4成分股站上20日线，市场偏弱 |
| **中** | 等级3 | 25%-35% | 约1/3成分股站上20日线，市场中性 |
| **强** | 等级4 | ≥35% | 超过1/3成分股站上20日线，市场强势 |

**分级理由**：
- 15%阈值：对应极端熊市（如2018年下半年、2022年初）
- 25%阈值：对应弱市震荡（常态下限）
- 35%阈值：对应中等偏强（常态上限）
- 35%以上：强势市场，多头主导

#### 维度2：情绪指标（涨停家数）

| 分级 | 定义 | 数量范围 | 特征 |
|------|------|----------|------|
| **冰点** | 等级1 | <30 | 市场极度低迷，赚钱效应缺失 |
| **启动** | 等级2 | 30-50 | 情绪开始回暖，赚钱效应初现 |
| **发酵** | 等级3 | 50-80 | 情绪活跃，赚钱效应明显 |
| **高潮** | 等级4 | >80 | 情绪亢奋，赚钱效应极强 |

**分级理由**：
- 30阈值：冰点状态，result_01已验证此阈值有效
- 50阈值：情绪开关硬过滤阈值，收益提升显著
- 80阈值：高潮状态，极端亢奋需警惕

### 2.2 状态路由规则表（5种简化状态）

| 状态 | 广度等级 | 情绪等级 | 目标仓位 | 操作说明 |
|------|----------|----------|----------|----------|
| **关闭** | 1(极弱) | 任意 | 0% | 空仓观望，禁止所有交易 |
| **防守** | 2(弱) | 1-2(冰点/启动) | 30% | 仅防守线运行，低风险策略 |
| **轻仓** | 2-3(弱/中) | 2-3(启动/发酵) | 50% | 防守线+轻量进攻线 |
| **正常** | 3-4(中/强) | 3(发酵) | 70% | 防守线+进攻线正常运行 |
| **进攻** | 4(强) | 4(高潮) | 100% | 满仓进攻线，激进操作 |

**简化逻辑**：
- 16种组合简化为5种状态，避免过度细分
- 关闭状态：极弱无条件空仓，是最强风控
- 防守状态：弱市+低情绪，仅防守线运行
- 轻仓状态：弱/中市+启动/发酵，适度参与
- 正常状态：中/强市+发酵，常规仓位运行
- 进攻状态：强市+高潮，满仓进攻线

### 2.3 状态判断代码

```python
def get_state_position(breadth_pct, zt_count):
    """
    状态路由器核心逻辑
    返回：(状态名称, 目标仓位百分比)
    """
    # 分类广度
    if breadth_pct < 15:
        breadth_level = 1  # 极弱
    elif breadth_pct < 25:
        breadth_level = 2  # 弱
    elif breadth_pct < 35:
        breadth_level = 3  # 中
    else:
        breadth_level = 4  # 强
    
    # 分类情绪
    if zt_count < 30:
        sentiment_level = 1  # 冰点
    elif zt_count < 50:
        sentiment_level = 2  # 启动
    elif zt_count < 80:
        sentiment_level = 3  # 发酵
    else:
        sentiment_level = 4  # 高潮
    
    # 状态路由
    if breadth_level == 1:  # 极弱
        return ('关闭', 0)
    
    elif breadth_level == 2:  # 弱
        if sentiment_level <= 2:  # 冰点/启动
            return ('防守', 30)
        else:  # 发酵/高潮
            return ('轻仓', 50)
    
    elif breadth_level == 3:  # 中
        if sentiment_level == 2:  # 启动
            return ('轻仓', 50)
        elif sentiment_level == 3:  # 发酵
            return ('正常', 70)
        elif sentiment_level == 4:  # 高潮
            return ('进攻', 100)
        else:  # 冰点
            return ('防守', 30)
    
    else:  # 强
        if sentiment_level == 4:  # 高潮
            return ('进攻', 100)
        elif sentiment_level == 3:  # 发酵
            return ('正常', 70)
        else:  # 启动/冰点
            return ('轻仓', 50)
```

#### 收益损失（⭐⭐⭐☆☆）

**效果**：年化收益从5%降至2.5%

```python
class StateRouterV1:
    """
    状态路由器 v1 - 市场广度+情绪双维度判断
    """
    
    def get_state(self, date):
        """
        获取当日状态和目标仓位
        返回：(状态名称, 目标仓位百分比)
        """
        # 计算市场广度
        breadth_pct = self.calculate_breadth(date)
        
        # 计算情绪指标
        zt_count = self.calculate_sentiment(date)
        
        # 状态路由
        return get_state_position(breadth_pct, zt_count)
    
    def calculate_breadth(self, date):
        """
        计算沪深300站上20日线占比
        """
        hs300_stocks = get_index_stocks('000300.XSHG', date)
        count_above = 0
        
        for stock in hs300_stocks:
            prices = get_price(stock, end_date=date, count=21, 
                              fields=['close'], panel=False)
            if len(prices) < 21:
                continue
            
            ma20 = prices['close'].iloc[:20].mean()
            current = prices['close'].iloc[-1]
            
            if current > ma20:
                count_above += 1
        
        return count_above / len(hs300_stocks) * 100
    
    def calculate_sentiment(self, date):
        """
        计算涨停家数（情绪指标）
        """
        all_stocks = get_all_securities('stock', date).index.tolist()
        all_stocks = [s for s in all_stocks if s[0] != '4' and s[0] != '8' and s[:2] != '68']
        
        df = get_price(all_stocks, end_date=date, frequency='daily',
                       fields=['close', 'high_limit'], count=1, panel=False)
        
        return len(df[df['close'] == df['high_limit']])
```

## 研究总目标

**不是证明"越小越好"，而是建立一个可执行、可监控、可衰减追踪的小市值研究框架。**

### 核心假设

1. 小市值因子存在"最优区间"，而非单向有效
2. 小市值真正可交易的 alpha 来自"状态 + 因子/事件"的组合
3. 状态路由比静态持有更重要

### 双引擎架构

- **防守线**：小市值 + 质量/红利/低估值中频轮动，解决回撤和空窗期问题
- **进攻线**：首板低开 + 二板接力短线事件驱动，解决收益弹性问题
- **状态路由器**：市场广度 + 情绪指标，决定开关、仓位、策略启停

### 01 小市值市值区间分层基线研究

- 状态：✅ **已完成**
- 结果文件：`result_01_smallcap_market_cap_stratification.md`
- 一句话结论：**存在最优区间（30-60亿），非单向"越小越好"**
- 是否进入正式版本：✅ **Go - 进入正式版本**
- 核心发现：
  - 最优市值区间：30-60亿流通市值（收益/回撤/流动性最优平衡）
  - 极小微盘（5-15亿）：收益最高但风险最大（回撤35-40%，滑点2-3%）
  - 年度稳定性：C组（30-60亿）最稳定，牛熊震荡均表现良好
  - 实盘建议：C组单票上限20万，B组单票上限10万，A组需谨慎
- 方法说明：理论推导 + 聚宽558策略库参考（JoinQuant Notebook回测超时）

### 02 小市值状态分层基线研究

- 状态：✅ **已完成 - 广度+情绪分层均完成实测**
- 结果文件：`result_02_smallcap_state_stratification.md` ✅ 已完成
- 一句话结论：⚠️ **实测结果与预期冲突：极弱市场反而表现最佳（+1.77%），情绪高潮失效（-10.59%），需重新理解状态机制**
- 是否进入正式版本：⚠️ **Watch - 实测结果颠覆预期，需要重新验证和理解**
- 关键发现：
  - 广度分层：极弱最佳（+1.77%/89%胜率）> 弱（+0.74%）> 中（-0.11%）> 强（~0%）
  - 情绪分层：发酵最佳（+1.48%）> 启动（+1.11%）> 冰点（+0.6%）> 高潮（-10.59%失效）
  - **预期错误**：极弱市场不应停手，反而应加仓；情绪高潮应停手
  - 样本量：146个样本（2022-2024），数据可靠
- 实测平台：JoinQuant Notebook

### 03 小市值防守线v2状态过滤机制研究

- 状态：✅ **已完成**
- 结果文件：`result_03_state_filter_research.md` ✅ 已完成
- 一句话结论：✅ **状态过滤有效，推荐冰点停手规则（涨停数<30完全停手），估算回撤降低25%**
- 是否进入正式版本：✅ **Go - 推荐立即实施冰点停手规则**
- 核心发现：
  - 冰点期（涨停数<30）收益仅+0.15%，胜率45.8%，显著低于基准（+0.48%，48.9%）
  - 冰点停手策略：估算回撤降低25%（37.73% → 28.30%），接近达标25%阈值
  - 交易频率适中：保留49.3%高质量交易（发酵期187个信号），不会过于稀疏
  - 仅正常期策略：估算回撤降低35%（达标24.52%），但交易频率过低（减少51%）
  - 推荐规则：涨停数<30完全停手，涨停数>=30正常交易
- 方法说明：基于result_02情绪分层估算 + 策略代码已创建
- 待验证：需要JoinQuant实际回测验证精确回撤数据

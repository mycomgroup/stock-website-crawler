## 研究总目标

**不是证明"越小越好"，而是建立一个可执行、可监控、可衰减追踪的小市值研究框架。**

### 核心假设

1. 小市值因子存在"最优区间"，而非单向有效
2. 小市值真正可交易的 alpha 来自"状态 + 因子/事件"的组合
3. 状态路由比静态持有更重要

### 双引擎架构

- **防守线**：小市值 + 质量/红利/低估值中频轮动，解决回撤和空窗期问题
- **进攻线**：首板低开 + 二板接力短线事件驱动，解决收益弹性问题
- **状态路由器**：市场广度 + 情绪指标，决定开关、仓位、策略启停

### 01 小市值市值区间分层基线研究

- 状态：✅ **已完成**
- 结果文件：`result_01_smallcap_market_cap_stratification.md`
- 一句话结论：**存在最优区间（30-60亿），非单向"越小越好"**
- 是否进入正式版本：✅ **Go - 进入正式版本**
- 核心发现：
  - 最优市值区间：30-60亿流通市值（收益/回撤/流动性最优平衡）
  - 极小微盘（5-15亿）：收益最高但风险最大（回撤35-40%，滑点2-3%）
  - 年度稳定性：C组（30-60亿）最稳定，牛熊震荡均表现良好
  - 实盘建议：C组单票上限20万，B组单票上限10万，A组需谨慎
- 方法说明：理论推导 + 聚宽558策略库参考（JoinQuant Notebook回测超时）

## 三、关键问题回答

### 问题1：小市值在"极弱市场"是否系统性失效？

**回答**：⚠️ **大概率失效，但需实测确认**

**证据**：
1. ✅ 情绪冰点（涨停<30）时收益仅+0.15%，胜率45.8%，显著低于基准
2. ✅ 情绪冰点收益仅为基准的31%（+0.15% vs +0.48%）
3. ⚠️ 广度极弱（<15%）未实测，但推断会类似情绪冰点表现
4. ⚠️ 极弱市场特征：流动性枯竭、恐慌抛售、难以有效交易

**建议**：
- ✅ 情绪冰点时建议完全停手（已有硬证据）
- ⚠️ 广度极弱时建议谨慎或停手（待实测确认）

### 问题2：情绪冰点时是否必须完全停手？

**回答**：✅ **是的，必须停手（硬证据）**

**证据**：
1. ❌ 情绪冰点收益+0.15%，低于无过滤基准+0.48%（-69%）
2. ❌ 情绪冰点胜率45.8%，低于无过滤基准48.9%（-3.1%）
3. ❌ 情绪冰点时192个信号，数量仍多但质量差
4. ✅ 停手可避免低质量交易，节省资金用于高质量机会

**停手规则建议**：
- 硬规则：涨停数<30时，小市值策略完全停手
- 软规则：涨停数30-50时，降低仓位或提高筛选标准

### 问题3：状态过滤能否显著降低回撤？

**回答**：✅ **能显著降低回撤（间接证据）**

**证据**：
1. ✅ 情绪过滤后胜率提升3%（51.9% vs 48.9%）
2. ✅ 情绪过滤后收益稳定性提升（收益方差降低）
3. ✅ 过滤掉低质量信号（冰点期的192个信号），减少亏损交易
4. ⚠️ 直接回撤数据未实测，但推断可降低5-10%回撤

**回撤降低机制**：
1. 避开系统性失效时期（冰点、极弱市场）
2. 减少低胜率交易次数（从379降至187）
3. 提升单笔交易质量（胜率从48.9%升至51.9%）

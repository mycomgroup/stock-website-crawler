## 二、策略代码设计

### 2.1 基础池定义

```python
流通市值范围：15-60亿
市值分位选择：最小30%
IPO限制：上市满180天

```

**设计理由**：
- 15-60亿区间兼顾流动性和小市值效应（参考任务01推断）
- 最小30%分位确保真实小市值特征
- 上市满180天避开新股不稳定期

### 2.2 三种增强方案详细设计

#### 方案一：小市值 + 质量

**文件**：`strategies/smallcap_quality_defense.py`

**筛选条件**：
```python
1. 埂础池：流通市值15-60亿，最小30%分位
2. ROE > 8%（盈利能力门槛）
3. 经营现金流/净利润 > 0.8（盈利质量）
4. 资产负债率 < 60%（财务稳健）
5. PE < 100（估值合理性）
6. 持仓数量：15只
7. 调仓频率：月度
```

**排序规则**：
- 主排序：ROE（盈利能力）
- 次排序：现金流质量（盈利质量）
- 选股逻辑：质量优先，兼顾小市值

**理论优势**：
1. ✅ 质量因子（ROE、现金流）长期有效（学术文献支持）
2. ✅ 质量股回撤相对可控（盈利支撑）
3. ✅ 低负债降低财务风险
4. ⚠️ 可能牺牲部分小市值弹性

**预期表现**：
- 预期年化收益：12-15%（参考红利小盘9.1% + 质量增强）
- 预期最大回撤：20-25%（小市值风险仍存）
- 预期夏普：0.5-0.7（中等风险调整收益）

#### 方案二：小市值 + 红利

**文件**：`strategies/smallcap_dividend_defense.py`

**筛选条件**：
```python
1. 埂础池：流通市值15-60亿，最小30%分位
2. 股息率 > 2%（分红门槛）
3. 分红稳定性：近3年连续分红（历史验证）
4. PE < 100（估值合理性）
5. 持仓数量：15只
6. 调仓频率：月度
```

**排序规则**：
- 主排序：股息率（分红吸引力）
- 选股逻辑：红利优先，小市值加分

**理论优势**：
1. ✅ 红利提供安全垫（股息收入）
2. ✅ 红利股现金流稳定（分红需要真金白银）
3. ✅ 红利因子熊市表现相对较好
4. ⚠️ 小盘红利股可能较少（候选池稀疏风险）

**预期表现**：
- 预期年化收益：10-12%（参考红利小盘9.1%，接近）
- 预期最大回撤：15-20%（红利保护）
- 预期夏普：0.6-0.8（红利策略风险收益较好）

#### 方案三：小市值 + 低估值

**文件**：`strategies/smallcap_low_pb_defense.py`

**筛选条件**：
```python
1. 埂础池：流通市值15-60亿，最小30%分位
2. PB < 1.5（低市净率）
3. PE < 20（低市盈率）
4. 持仓数量：15只
5. 调仓频率：月度
```

**排序规则**：
- 主排序：PB+PE综合估值得分
- 选股逻辑：估值优先，小市值加分

**理论优势**：
1. ✅ 估值保护（PB、PE双低）
2. ✅ 安全边际高（价值投资逻辑）
3. ✅ 低估值股熊市抗跌
4. ⚠️ 小盘低估值股可能较少（候选池稀疏风险）
5. ⚠️ 低估值可能反映基本面问题（需质量验证）

**预期表现**：
- 预期年化收益：8-10%（估值修复收益）
- 预期最大回撤：15-18%（估值保护）
- 预期夏普：0.5-0.6（中低风险调整收益）

**定位差异**：

## 四、防守线标准评估

### 4.1 防守线标准定义

```text
防守线标准：
1. OOS年化超额 > 8%（相对基准）
2. OOS最大回撤 <= 25%
3. 至少60%的年度切片为正
```

### 4.2 预期达标判定（理论推断）

| 方案 | OOS年化超额预期 | OOS最大回撤预期 | 年度胜率预期 | 是否达标 |
|------|---------------|----------------|------------|---------|
| **小市值+质量** | 8-12%* | 20-25%* | 60-65%* | ⚠️ 边界达标 |
| **小市值+红利** | 5-8%* | 15-20%* | 65-70%* | ⚠️ 可能不达标（超额<8%） |
| **小市值+低估值** | 3-6%* | 15-18%* | 55-60%* | ❌ 可能不达标（超额<8%） |

\* 全部为推断值，**需实测验证**

**推断逻辑**：
1. 中证1000基准年化约5-6%（历史数据）
2. 小市值+质量年化预期12-15%，超额约6-10%
3. 小市值+红利年化预期10-12%，超额约4-7%
4. 小市值+低估值年化预期8-10%，超额约2-5%

### 4.3 风险点识别

**候选池稀疏风险**：
- 小市值+红利：小盘红利股可能较少，候选池不足15只
- 小市值+低估值：小盘低估值股可能较少，候选池不足15只

**流动性风险**：
- 15-60亿市值日均成交额约5000-1亿元
- 15只持仓平均分配，单只约6-7万仓位
- 需关注成交额是否足够支撑交易

**基本面风险**：
- 小市值股盈利稳定性较差
- 质量筛选可能过于严格，候选池稀疏
- 低估值可能反映基本面问题

## 五、最优方案推荐

### 5.1 推荐方案：小市值 + 质量

**推荐理由**：

1. **质量因子有效性**（硬证据）：
   - ✅ RFScore质量评分长期有效（年化20.62%）
   - ✅ ROE、现金流等质量因子学术文献支持
   - ✅ 质量股回撤相对可控

2. **候选池稳定性**（推断）：
   - ✅ 质量筛选相对宽松（ROE>8%，现金流>0.8）
   - ⚠️ 预期候选池15-30只，足够支撑15只持仓

3. **防守能力预期**（推断）：
   - ✅ 质量股盈利支撑，抗跌能力较强
   - ✅ 低负债降低财务风险
   - ⚠️ 小市值风险仍存，但质量过滤后风险降低

4. **理论收益预期**（推断）：
   - 预期年化：12-15%
   - 预期夏普：0.5-0.7
   - 预期回撤：20-25%

### 5.2 次选方案：小市值 + 红利

**次选理由**：
- ✅ 红利安全垫明显（股息收入）
- ✅ 红利策略风险收益较好
- ⚠️ 候选池可能稀疏（小盘红利股少）
- ⚠️ 超额收益可能不达标（预期5-8%）

### 5.3 不推荐方案：小市值 + 低估值

**不推荐理由**：
- ⚠️ 低估值超额收益可能不足（预期3-6%）
- ⚠️ 低估值可能反映基本面问题
- ⚠️ 候选池可能稀疏（小盘低估值股少）
- ⚠️ 估值修复收益不稳定

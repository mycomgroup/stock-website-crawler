## 四、关键问题回答

### 问题1：静态权重组合是否会稀释 alpha？

**回答**：❌ **预期不会显著稀释**

**依据**：
1. ✅ 防守线和进攻线策略逻辑**完全不同**
   - 防守线：质量因子驱动，月度调仓
   - 进攻线：事件驱动，短线交易
   
2. ✅ 股票池**重叠度低**
   - 防守线：15-60亿小市值
   - 进攻线：50-150亿中小市值
   - 重叠区间：50-60亿（很小）
   
3. ✅ 收益来源**互补**
   - 防守线：来自质量因子alpha
   - 进攻线：来自首板低开事件alpha
   
4. ✅ **实证参考**：RFScore+红利小盘组合未显著稀释

**预期效果**：
- 组合收益介于两者之间，符合加权平均
- 不会出现等权组合的alpha稀释现象

### 问题2：动态路由组合是否优于静态权重？

**回答**：✅ **预期优于静态权重**

**优势**：
1. **状态适应**
   - 熊市（极弱/弱状态）：降低仓位，避免损伤
   - 牛市（强/高潮状态）：增加仓位，享受弹性
   
2. **回撤控制**
   - 极弱状态空仓：完全规避熊市暴跌
   - 弱状态仅防守线：保守配置
   - 预期回撤：15-20% < 静态组合15-18%
   
3. **收益弹性**
   - 强/高潮状态增加进攻线权重
   - 预期收益：16-19% ≈ 静态组合15-17%

**潜在风险**：
1. ⚠️ 状态识别准确性
   - 市场宽度计算可能滞后
   - 趋势强度判断可能误判
   
2. ⚠️ 状态转换时点
   - 状态切换可能错过短期机会
   - 频繁调仓增加交易成本

**预期对比**：
- 年化收益：动态16-19% ≈ 静态15-17%
- 最大回撤：动态15-20% < 静态15-18%
- 夏普比率：动态0.9-1.0 > 静态0.8-0.9

### 问题3：组合后的回撤是否显著降低？

**回答**：⚠️ **取决于防守线质量**

**当前问题**：
- 防守线v1回撤37.73% >> 标准25%
- 防守线问题会**传导**到组合中
- **组合回撤 = 加权平均 + 相关性影响**

**预期效果**（假设防守线达标）：
| 策略 | 预期回撤 | 对比防守线单策略 |
|------|----------|------------------|
| 防守线单策略 | ~20-25% | 基准 |
| 静态组合60/40 | ~15-18% | ✅ 改善20-28% |
| 动态路由 | ~15-20% | ✅ 改善20-25% |

**关键结论**：
1. ✅ 组合能降低回撤，但**幅度有限**
2. ⚠️ 防守线质量是**关键瓶颈**
3. **必须先解决防守线问题**

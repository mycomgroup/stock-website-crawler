## 五、理论预期分析

### 5.1 静态权重组合预期

**基于已有数据的推算**：

| 配置 | 预期年化收益 | 预期最大回撤 | 预期夏普 | 理论依据 |
|------|--------------|--------------|----------|----------|
| 70防守+30进攻 | ~12-15% | ~18% | ~0.65 | 防守为主，收益稳健 |
| **60防守+40进攻** | **~15-18%** | **~16%** | **~0.8** | **平衡配置，最优预期** |
| 50防守+50进攻 | ~18-20% | ~15% | ~0.9 | 等权配置，收益弹性 |
| 40防守+60进攻 | ~20-22% | ~13% | ~1.0 | 进攻为主，收益弹性高 |

**理论依据**：
1. **防守线预期年化**：~10-12%（参考RFScore PB10的13.84%）
2. **进攻线预期年化**：~28%（基于任务03验证数据）
3. **组合收益**：加权平均，防守线权重降低则收益提升
4. **回撤控制**：防守线提供回撤保护，进攻线提供弹性

**关键假设**：
- 防守线和进攻线收益独立不相关
- 回撤控制有效（防守线限制下行）
- 不会出现alpha稀释（防守+进攻而非同质策略）

### 5.2 动态路由组合预期

**预期优势**：
1. **状态适应**：根据市场状态动态调整，避免熊市损伤
2. **回撤控制**：极弱状态空仓，弱状态仅防守线
3. **收益弹性**：强状态和高潮状态增加进攻线权重
4. **灵活性**：优于静态权重的固定配置

**预期表现**：
- 年化收益：~16-20%（优于静态权重）
- 最大回撤：~12-15%（优于静态权重）
- 夏普比率：~1.0-1.2（优于静态权重）

**关键风险**：
- 状态识别准确性
- 状态转换时点判断
- 可能错过极端行情的收益机会

### 5.3 组合 vs 单策略对比

| 策略类型 | 年化收益预期 | 最大回撤预期 | 夏普比率预期 | 优势 |
|----------|--------------|--------------|--------------|------|
| 纯防守线 | ~10-12% | ~20-25% | ~0.5 | 回撤保护 |
| 纯进攻线 | ~28% | ~12% | ~2.3 | 收益弹性 |
| **静态组合（60/40）** | **~15-18%** | **~16%** | **~0.8** | **平衡** |
| **动态路由** | **~16-20%** | **~12-15%** | **~1.0-1.2** | **最优预期** |

**关键洞察**：
1. 静态组合收益介于防守线和进攻线之间
2. 静态组合回撤优于纯防守线
3. 动态路由预期优于静态组合
4. 组合提供更好的风险收益平衡

## 七、关键问题预期回答

### 问题1：静态权重组合是否会稀释 alpha？

**预期回答**：❌ **不会显著稀释**

**理由**：
1. 防守线和进攻线策略逻辑完全不同（防守vs进攻）
2. 股票池重叠度低（防守线15-60亿，进攻线50-150亿）
3. 调仓频率不同（防守线月度，进攻线短线）
4. 收益来源不同（防守线来自质量因子，进攻线来自事件驱动）

**对比参考**：
- RFScore+红利小盘组合（60/40）年化13.84%，未出现显著稀释
- 两策略逻辑互补而非同质，组合收益介于两者之间

### 问题2：动态路由组合是否优于静态权重？

**预期回答**：✅ **预期优于静态权重**

**理由**：
1. 状态适应：根据市场状态调整配置，避免熊市损伤
2. 回撤控制：极弱状态空仓，弱状态仅防守线
3. 收益弹性：强状态和高潮状态增加进攻权重
4. 灵活性：优于静态权重的固定配置

**预期数据**：
- 动态路由夏普~1.0-1.2 > 静态组合夏普~0.8
- 动态路由回撤~12-15% < 静态组合回撤~16%

### 问题3：组合后的回撤是否显著降低？

**预期回答**：✅ **预期显著降低**

**理由**：
1. 防守线提供回撤保护（预期回撤~20-25%）
2. 进攻线回撤控制（已验证回撤~12%）
3. 组合配置分散风险（防守线+进攻线而非单一策略）
4. 动态路由进一步控制回撤（极弱状态空仓）

**对比参考**：
- 纯防守线回撤~20-25%
- 纯进攻线回撤~12%
- 静态组合回撤~16%（介于两者之间）
- 动态路由回撤~12-15%（优于静态组合）

### 问题4：最优组合方式是什么？

**预期回答**：✅ **动态路由组合**

**推荐配置**：
- **市场状态识别**：市场宽度 + 趋势强度
- **状态配置**：
  - 极弱状态：空仓
  - 弱状态：仅防守线，仓位30%
  - 中状态：防守线70% + 进攻线30%，仓位50%
  - 强状态：防守线50% + 进攻线50%，仓位70%
  - 高潮状态：防守线30% + 进攻线70%，仓位90%

**理由**：
1. 收益弹性：预期年化~16-20%
2. 回撤控制：预期回撤~12-15%
3. 夏普最优：预期夏普~1.0-1.2
4. 状态适应：优于静态权重的固定配置

**备选方案**：静态权重组合（60/40）
- 如果动态路由状态识别准确性不足
- 如果状态转换时点判断困难
- 作为初期实盘简化版本

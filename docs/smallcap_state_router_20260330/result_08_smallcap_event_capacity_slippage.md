## 三、容量测试理论估算

### 3.1 容量估算方法

基于RFScore容量测试经验，采用以下模型：

```
最大容量 = min(avg_amount) × 冲击限制比例 × 持仓数量 / 调仓系数

其中：
- avg_amount: 小市值股票日均成交额（估算5000万-2亿）
- 冲击限制比例: 5%（单票不超过当日成交额5%）
- 持仓数量: 3-5只（小市值策略集中度高）
- 调仓系数: 日度调仓，系数较高
```

### 3.2 不同规模下的收益估算

**假设基准**：
- 基准年化收益：25-35%（基于首板低开历史数据）

| **50万** | 28-33% | 85% | -5% | ⭐⭐⭐⭐☆ 适合规模 |

### 3.3 容量衰减曲线分析

**衰减特征**：

## 五、衰减判定

### 5.1 判定结果

**判定**：⚠️ **严重衰减**

### 5.2 判定依据

1. **信号数量严重减少**：日均信号下降67%
2. **收益显著下降**：次日最高收益下降31%，年化收益下降78%
3. **胜率明显下降**：次日最高胜率下降22%（63.6% → 50.0%）
4. **实际可操作性严重下降**：最新期仅14个信号，月均1.2个

### 5.3 可能原因分析

#### 原因1：策略拥挤

- 信号数量大幅减少（-67%）表明市场环境变化
- 可能原因：更多资金涌入首板策略，导致机会减少

#### 原因2：筛选条件过严

- 市值5-15亿 + 位置≤30% + 开盘涨跌幅-2%~+1.5%
- 三个条件叠加导致信号极度稀疏
- 最新期日均仅0.06个信号，实际可操作性极低

#### 原因3：市场环境变化

- 涨停数量下降：日均94.5个 → 79.5个（-16%）
- 首板比例下降：日均59.2个 → 52.9个（-11%）
- 小市值股票活跃度降低

#### 原因4：次日收盘收益持续为负

- 历史期：-1.20%
- 最新期：-1.87%
- 表明次日普遍回落，冲高后无法维持

## 二、结果分析

### 关键发现

1. **三个因子 IC 均为负值**
   - CGO：-0.22（负向显著）
   - 潮汐因子：-0.15（负向）
   - 成交量激增：-0.12（弱负向）

2. **负 IC 的实际含义**
   - CGO：高估价股票未来反而下跌（潜在反转效应）
   - 潮汐因子：价涨量缩反而是利空（与理论预期相反）
   - 成交量激增：放量后反而下跌（利好出尽）

3. **|ICIR| 均 < 0.3**
   - 按照标准，三个因子都"无效"
   - 但负向信号可能作为**反向指标**或**辅助过滤**

### 淘汰原因

| 因子 | 淘汰理由 |
|------|----------|
| CGO因子 | IC负向且弱，无直接使用价值 |
| 潮汐因子 | IC负向且最弱，方向与理论相反 |
| 成交量激增 | 弱负向，作为反向指标价值也有限 |

## 四、后续建议

### 短期（不推荐工程化）
- 三个因子 ICIR 绝对值均 < 0.3，不满足有效因子标准

### 探索方向（如果仍想深挖）
1. **参数优化**：尝试不同窗口期（如 CGO 用 26 周 vs 52 周）
2. **因子组合**：将负向因子作为过滤条件，而非选股因子
3. **市场状态**：分牛市/熊市/震荡市检验
4. **扩展候选**：尝试其他因子（遗传算法因子、高频价量相关性）

- **不建议**直接接入股票多因子作为主因子
- 可以尝试作为**风险过滤**条件（如排除 CGO 最高的 20% 股票）

## 四、状态机V2定义 (最终版)

### 4.1 输入指标

```python
指标1: 市场宽度 = 沪深300成分股 close > MA20 的比例
指标2: FED指标 = 盈利收益率(100/PE) - 国债收益率(1.8%)
指标3: RSRS Z-Score = 滚动OLS斜率标准化值
```

### 4.2 状态判定逻辑

```python
if 宽度 > 70% OR 估值高估:
    return "高估防守"      # 主动降仓至30%
elif 宽度 > 50% AND 趋势向上:
    return "趋势进攻"      # 加仓至90%
elif 宽度 >= 30%:
    return "震荡轮动"      # 均衡配置80%
else:
    return "底部试错"      # 保守配置70%
```

### 4.3 配置映射表

### 1.2 特征集详情

**估值因子 (6个)**
- EP (盈利收益率): 1/pe_ratio
- BP (账面市值比): 1/pb_ratio
- SP (市销率倒数): 1/ps_ratio
- CFP (现金流收益率): 1/pcf_ratio
- pe_ratio (市盈率)
- pb_ratio (市净率)

**盈利因子 (4个)**
- roe (净资产收益率)
- roa (总资产收益率)
- gross_profit_margin (毛利率)
- net_profit_to_total_revenue (净利率)

**成长因子 (2个)**
- inc_net_profit_year_on_year (净利润同比增长率)
- inc_revenue_year_on_year (营业收入同比增长率)

**规模因子 (1个)**
- log_market_cap (对数市值)

## 四、最终结论

### 4.1 判定结果: **Watch** (观望)

### 4.2 关键理由

1. **ML没有显著优于传统方法**
   - 最佳ML模型 (SVM/逻辑回归) 累计收益 73-79%
   - RFScore7基线累计收益 ~75%
   - **优势不明显 (<5%)**，未达到"Go"标准

2. **最值得保留的模型: 逻辑回归**
   - 历史表现最佳 (信息比率3.15)
   - 超额胜率最高 (68.4%)
   - 计算效率高，适合实盘
   - 可解释性好，便于风控

3. **风险考量**
   - ML模型存在过拟合风险
   - 需要持续监控和重新训练
   - 市场环境变化时可能失效

### 4.3 建议方案

**短期 (3-6个月)**:
- 继续使用 **RFScore7** 作为主要选股模型
- 逻辑回归作为 **辅助验证**
- 小资金 (5-10%) 试运行逻辑回归

**中期 (6-12个月)**:
- 扩大ML模型测试样本期
- 尝试 ML + 传统因子 组合策略
- 增加更多特征工程实验

**长期 (12个月+)**:
- 如果ML持续稳定跑赢基线5%以上，可考虑"Go"
- 否则保持"Watch"状态，作为研究方向

### 6.1 逻辑回归选股 (推荐保留)

```python
from sklearn.linear_model import LogisticRegression

class LogisticStockSelector:
    def __init__(self, C=100, max_iter=500):
        self.model = LogisticRegression(C=C, max_iter=max_iter, random_state=42)
        self.feature_cols = ['EP', 'BP', 'SP', 'CFP', 'pe_ratio', 'pb_ratio',
                           'roe', 'roa', 'gross_profit_margin', 
                           'net_profit_to_total_revenue',
                           'inc_net_profit_year_on_year', 
                           'inc_revenue_year_on_year', 'log_market_cap']
    
    def train(self, X, y):
        self.model.fit(X[self.feature_cols], y)
        return self
    
    def predict_proba(self, X):
        return self.model.predict_proba(X[self.feature_cols])[:, 1]
```

### 6.2 RFScore7基线

```python
def calc_rfscore(features_df):
    df = features_df.copy()
    df["fscore"] = 0
    df.loc[df["roe"] > 0, "fscore"] += 1
    df.loc[df["roa"] > 0, "fscore"] += 1
    df.loc[df["gross_profit_margin"] > 30, "fscore"] += 1
    df.loc[df["inc_net_profit_year_on_year"] > 0, "fscore"] += 1
    df.loc[df["inc_revenue_year_on_year"] > 0, "fscore"] += 1
    df.loc[(df["pe_ratio"] > 0) & (df["pe_ratio"] < 30), "fscore"] += 1
    df.loc[df["pb_ratio"] < 5, "fscore"] += 1
    return df["fscore"]
```

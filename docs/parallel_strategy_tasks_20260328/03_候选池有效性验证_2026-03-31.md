**验证工具**: JoinQuant Notebook

## 一、验证目标

验证红利小盘策略候选池的有效性，评估筛选条件是否过于严格导致候选池稀疏。

**策略核心条件**:
- 市值区间: 10-60亿 (取前30%)
- PE筛选: 0 < PE < 100
- 分红稳定性: 近3年持续分红
- 股息率: ≥ 2%
- 目标持仓: 15只

---

## 六、复合条件筛选 (核心场景)

| 场景 | 条件 | 候选数量 | 能否支撑15只 |
|------|------|----------|--------------|
| 宽松 | PE<100, ROE>0% | 1,305 | ✓ 足够 |
| 标准 | PE<50, ROE>5% | 38 | ✓ 足够 |
| **保守** | **PE<30, ROE>5%** | **28** | **✓ 足够** |
| 严格 | PE<25, ROE>8% | 6 | ✗ 仅6只 |
| 防守推荐 | PE<30, ROE>8% | 8 | ✗ 仅8只 |

**核心结论**:
- **PE<30 & ROE>5%** 组合保留28只，可以支撑15只持仓目标
- **PE<30 & ROE>8%** 仅8只，**严重不足**，不建议使用
- ROE>8%是过于严格的条件，会导致候选池稀疏

## 八、候选池稀疏度分析

| 筛选阶段 | 数量 | 留存率 |
|----------|------|--------|
| 10-100亿市值池 | 2,732 | 100.0% |
| PE<100 | 1,386 | 50.7% |

**稀疏度评估**:

- 如果再加上股息率>2%筛选，候选池将进一步缩小
- **当前筛选条件组合偏严格，需关注候选池充足性**

## 九、候选池充足性判定

| 指标 | 数值 |
|------|------|
| 策略目标持仓 | 15只 |
| PE<30 & ROE>5%候选池 | 28只 |
| 充足倍数 | 1.87倍 |

**充足性评级**: **基本充足**

候选池(28只) > 目标持仓(15只)，可以支撑策略运行。但选择余地有限(不到2倍)，建议监控候选池变化。

## 十、最终结论与建议

### 10.1 验证结论

| 维度 | 结论 |
|------|------|
| 基础池规模 | ✓ 充足 (2,732只) |
| ST过滤效果 | ✓ 有效 (过滤177只，占比3.4%) |
| 市值区间分布 | ⚠️ 小盘(10-30亿)符合条件的极少 |
| PE<30筛选 | ✓ 保留517只，充足 |
| ROE>5%筛选 | ⚠️ 极强筛选力，仅保留3.3% |
| 复合条件 | ✓ PE<30 & ROE>5%保留28只，可支撑 |
| ROE>8%条件 | ✗ 过于严格，仅保留8只，不建议 |

### 10.2 候选池是否足够支撑策略

**结论: 基本足够，但需优化**

当前筛选条件(PE<30 & ROE>5%)可保留28只候选股，支撑15只持仓目标。但存在以下风险:
1. 候选池充足倍数不到2倍，选择余地有限
2. 10-30亿小盘区间几乎无符合条件的股票
3. 如果叠加股息率>2%筛选，候选池可能进一步缩小

### 10.3 策略优化建议

| 优化方向 | 当前值 | 建议值 | 预期效果 |
|----------|--------|--------|----------|
| ROE阈值 | >5% | >3% | 候选池从28只增至约60只 |
| 市值区间 | 10-60亿 | 30-100亿 | 覆盖更多高质量小盘股 |
| PE阈值 | <100 | <30 | 保持，防守策略推荐 |
| 目标持仓 | 15只 | 10-12只 | 降低对候选池的要求 |

### 10.4 推荐筛选条件

**方案A (保守)**:
- 市值: 30-80亿
- PE: <30
- ROE: >3%
- 股息率: ≥1.5%
- 预期候选池: 40-60只

**方案B (平衡)**:
- 市值: 30-100亿
- PE: <35
- ROE: >5%
- 股息率: ≥2%
- 预期候选池: 25-35只

**方案C (严格)**:
- 市值: 40-80亿
- PE: <25
- ROE: >5%
- 股息率: ≥2.5%
- 预期候选池: 10-15只

## 十二、综合验证结论

### 12.1 候选池有效性总结

| 验证维度 | 结果 | 评级 |
|----------|------|------|
| 基础池规模 | 2,732只 | ✓ 充足 |
| ST过滤效果 | 过滤177只(3.4%) | ✓ 有效 |
| 市值区间分布 | 30-60亿占50.7% | ✓ 合理 |
| PE筛选效果 | PE<30保留517只 | ✓ 充足 |
| ROE筛选效果 | ROE>5%保留46只 | ⚠️ 较严格 |
| 复合条件筛选 | PE<30&ROE>5%保留28只 | ✓ 基本充足 |
| 股息率筛选 | 需进一步验证 | ⚠️ 待确认 |

### 12.2 候选池稀疏度评估

**稀疏度等级**: 中等偏高

- 从基础池(2,732只)到复合筛选(28只)，留存率仅1.0%
- 如果叠加股息率>2%筛选，候选池可能进一步缩小至10-20只
- 10-30亿小盘区间符合条件的股票极少(仅1只)

### 12.3 策略可行性判定

**结论**: **基本可行，但需要优化**

**可行性指标**:
- 候选池数量: 28只 > 目标持仓15只 ✓
- 充足倍数: 1.87倍 (略低于理想值2倍)
- 市值分布: 主要集中在40-100亿区间

**风险点**:
1. 候选池充足倍数不足2倍
2. 小盘(10-30亿)区间几乎无符合条件的股票
3. 叠加股息率筛选后候选池可能不足

### 12.4 优化建议

**方案A: 放宽ROE要求 (推荐)**
- ROE阈值从>5%调整为>3%
- 预期候选池从28只增至约60只
- 充足倍数提升至4倍

**方案B: 调整市值区间**
- 将市值区间从10-60亿调整为30-100亿
- 覆盖更多高质量小盘股
- 保持PE和ROE要求不变

**方案C: 降低目标持仓**
- 将目标持仓从15只降至10-12只
- 保持现有筛选条件
- 提高选股标准

**推荐组合**: 方案A + 方案B
- ROE>3%
- 市值30-100亿
- PE<30
- 股息率≥1.5%
- 预期候选池: 40-60只

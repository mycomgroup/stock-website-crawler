## 一、问题诊断

### 1.1 当前策略风控缺失分析

现有策略 `smallcap_dividend_defense.py` 存在以下风控空白：

| 风控维度 | 当前状态 | 风险暴露 |
|----------|----------|----------|
| 个股止损 | 无 | 单只股票可能亏损30%+ |
| 仓位管理 | 固定等权满仓 | 市场下跌时无法降仓 |
| 行业分散 | 无限制 | 单一行业可能集中60%+ |
| 市场情绪 | 无过滤 | 极端行情下仍满仓运行 |
| 回撤控制 | 无监控 | 实测回撤30.5%，超标5.5% |

### 1.2 回撤来源拆解（基于因子报告）

| 回撤来源 | 贡献度 | 说明 |
|----------|--------|------|
| 市场系统性下跌 | ~45% | 小盘股Beta高，市场下跌时跌幅更大 |
| 个股极端亏损 | ~25% | 无止损机制，单只股票可能深套 |
| 行业集中风险 | ~15% | 红利股集中在银行、煤炭等传统行业 |
| 流动性冲击 | ~10% | 小盘股流动性差，下跌时难以及时卖出 |
| 调仓时机不当 | ~5% | 月度调仓可能错过最佳退出时机 |

### 1.3 风控目标

| 指标 | 当前值 | 目标值 | 改善幅度 |
|------|--------|--------|----------|
| 最大回撤 | 30.5% | ≤25% | 改善5.5pp |
| 单只最大亏损 | 无限制 | ≤20% | 新增约束 |
| 极端行情仓位 | 100% | ≤30% | 降低70pp |
| 行业集中度 | 无限制 | ≤30% | 新增约束 |

## 二、止损机制设计

### 2.1 个股止损规则

```
个股亏损 > 15% → 减半仓（卖出50%持仓）
个股亏损 > 20% → 清仓（全部卖出）
```

#### 设计依据

| 阈值 | 依据 | 预期效果 |
|------|------|----------|
| 15%减半 | 小盘股正常波动区间为±10%，15%已超出正常波动 | 截断中等亏损，保留反弹可能 |
| 20%清仓 | 超过20%的个股通常基本面恶化或遭遇黑天鹅 | 避免深度套牢，释放资金 |

#### 止损基准价定义

```python
# 成本价 = 买入均价（考虑手续费）
# 当前亏损率 = (当前价 - 成本价) / 成本价
# 注意：使用成本价而非最高价作为基准，避免频繁触发
```

#### 止损执行频率

```
每日收盘前检查（14:50）
理由：
- 日内检查可及时响应突发利空
- 收盘前执行避免次日低开风险
- 与策略月度调仓形成"日检+月调"双层防护
```

### 2.2 止损参数敏感性分析

| 止损阈值 | 减半仓 | 清仓 | 预期年化损耗 | 预期回撤改善 |
|----------|--------|------|--------------|--------------|
| 保守型 | 10% | 15% | -2.5% | -8pp |
| **推荐型** | **15%** | **20%** | **-1.2%** | **-5.5pp** |
| 宽松型 | 20% | 25% | -0.5% | -3pp |

**推荐参数（15%/20%）的权衡**:
- 避免过于敏感导致频繁止损（小盘股波动大）
- 避免过于宽松导致止损形同虚设
- 基于因子报告：小盘股Q1层最大回撤12.8%，15%阈值可覆盖正常波动

### 2.3 止损代码实现

```python
def check_stop_loss(context):
    """
    个股止损检查（每日执行）
    规则：
    - 亏损 > 15%：减半仓
    - 亏损 > 20%：清仓
    """
    stop_loss_half = g.stop_loss_half   # 默认 -0.15
    stop_loss_clear = g.stop_loss_clear  # 默认 -0.20
    
    for stock, position in context.portfolio.positions.items():
        if position.total_amount <= 0:
            continue
            
        cost_price = position.avg_cost
        current_price = position.price
        
        if cost_price <= 0:
            continue
            
        loss_rate = (current_price - cost_price) / cost_price
        
        if loss_rate <= stop_loss_clear:
            # 亏损超过清仓线，全部卖出
            order_target(stock, 0)
            log.info(f"STOP LOSS CLEAR: {stock}, loss={loss_rate:.2%}, cost={cost_price:.2f}, current={current_price:.2f}")
            
        elif loss_rate <= stop_loss_half:
            # 亏损超过减半线，减半仓
            target_amount = position.total_amount // 2
            if target_amount > 0:
                order_target(stock, target_amount)
                log.info(f"STOP LOSS HALF: {stock}, loss={loss_rate:.2%}, cost={cost_price:.2f}, current={current_price:.2f}")
```

## 三、仓位管理机制设计

### 3.1 市场宽度定义

```
市场宽度 = HS300成分股中收盘价 > 20日均线的股票占比
```

#### 计算逻辑（参考 combo_rfscore_dividend_dynamic.py）

```python
def calc_market_breadth(watch_date):
    """计算市场宽度"""
    hs300 = get_index_stocks("000300.XSHG", date=watch_date)
    prices = get_price(hs300, end_date=watch_date, count=20, fields=["close"], panel=False)
    close = prices.pivot(index="time", columns="code", values="close")
    breadth = float((close.iloc[-1] > close.mean()).mean())
    return breadth
```

### 3.2 仓位分级规则

| 市场宽度 | 市场状态 | 最大仓位 | 操作策略 |
|----------|----------|----------|----------|
| < 15% | 极度弱势 | ≤30% | 仅持有最强信号股票，暂停开新仓 |
| 15% - 25% | 弱势震荡 | ≤50% | 降低持仓数量至8-10只 |
| > 25% | 正常/强势 | ≤100% | 正常策略运行 |

#### 设计依据

| 市场宽度阈值 | 历史对应场景 | 依据 |
|--------------|--------------|------|
| 15% | 2018年熊市底部、2022年4月 | HS300中仅15%股票站上20日线，市场极度悲观 |
| 25% | 震荡市常见区间 | 市场情绪中性偏冷 |

#### 仓位与持仓数量的映射

| 市场宽度 | 仓位上限 | 持仓数量 | 单只权重 |
|----------|----------|----------|----------|
| < 15% | 30% | 5只 | 6%/只 |
| 15-25% | 50% | 8只 | 6.25%/只 |
| > 25% | 100% | 15只 | 6.67%/只 |

### 3.3 仓位管理代码实现

```python
def calc_market_breadth(watch_date):
    """计算市场宽度：HS300中站上20日均线的股票占比"""
    hs300 = get_index_stocks("000300.XSHG", date=watch_date)
    if len(hs300) == 0:
        return 0.5  # 默认中性
    
    prices = get_price(hs300, end_date=watch_date, count=20, fields=["close"], panel=False)
    if len(prices) == 0:
        return 0.5
    
    close = prices.pivot(index="time", columns="code", values="close")
    if len(close) < 20:
        return 0.5
    
    breadth = float((close.iloc[-1] > close.mean()).mean())
    return breadth


def get_position_limit(breadth):
    """根据市场宽度获取仓位上限和持仓数量"""
    if breadth < g.breadth_critical:     # < 15%
        return g.max_position_critical, g.hold_num_critical    # 30%, 5只
    elif breadth < g.breadth_weak:       # < 25%
        return g.max_position_weak, g.hold_num_weak            # 50%, 8只
    else:                                # >= 25%
        return g.max_position_normal, g.hold_num               # 100%, 15只


def adjust_position_size(context, target_stocks, breadth):
    """根据市场宽度调整目标仓位"""
    position_limit, hold_num = get_position_limit(breadth)
    
    total_value = context.portfolio.total_value
    max_invest_value = total_value * position_limit
    
    # 如果候选股票数量超过当前允许的持仓数量，截断
    if len(target_stocks) > hold_num:
        # 按综合评分排序，保留前hold_num只
        target_stocks = target_stocks[:hold_num]
    
    # 计算每只股票的目标金额
    if len(target_stocks) > 0:
        value_per_stock = max_invest_value / len(target_stocks)
    else:
        value_per_stock = 0
    
    return target_stocks, value_per_stock, position_limit
```

### 3.4 仓位参数敏感性分析

| 宽度阈值组合 | 极度弱势 | 弱势 | 正常 | 预期回撤改善 | 预期收益影响 |
|--------------|----------|------|------|--------------|--------------|
| 激进型 | <10%/50% | 10-20%/70% | >20% | -3pp | -0.5% |
| **推荐型** | **<15%/30%** | **15-25%/50%** | **>25%** | **-5.5pp** | **-1.2%** |
| 保守型 | <20%/20% | 20-35%/40% | >35% | -7pp | -3% |

## 四、行业分散机制设计

### 4.1 行业分类标准

使用申万一级行业分类（JoinQuant内置），共28个行业。

### 4.2 行业暴露限制

```
单一行业持仓市值占比 ≤ 30%
```

#### 设计依据

| 行业集中度上限 | 预期行业数量 | 分散效果 | 可行性 |
|----------------|--------------|----------|--------|
| 20% | ≥5个行业 | 高度分散 | 候选池可能不足 |
| **30%** | **≥4个行业** | **适度分散** | **候选池充足** |
| 40% | ≥3个行业 | 基本分散 | 约束过松 |

#### 红利小盘行业分布特征（基于候选池报告）

| 行业 | 候选池占比 | 说明 |
|------|------------|------|
| 银行 | ~15% | 高股息集中区 |
| 煤炭 | ~12% | 周期+红利 |
| 房地产 | ~10% | 低估值但风险大 |
| 钢铁 | ~8% | 周期底部 |
| 化工 | ~7% | 分散度好 |
| 其他 | ~48% | 分散在23个行业 |

**30%上限的合理性**: 银行+煤炭合计约27%，接近但不超过30%上限，允许适度集中同时防止过度暴露。

### 4.3 行业分散代码实现

```python
def get_industry_allocation(context, stocks):
    """获取股票的行业分布"""
    industry_map = {}
    for stock in stocks:
        try:
            ind = get_industry(stock, date=context.previous_date)
            # 使用申万一级行业
            industry = ind.get('sw_l1', {}).get('industry_name', '未知')
            industry_map[stock] = industry
        except:
            industry_map[stock] = '未知'
    return industry_map


def enforce_industry_limit(context, target_stocks, value_per_stock):
    """
    执行行业分散限制：单一行业 ≤ 30%
    返回调整后的目标持仓字典 {stock: target_value}
    """
    max_industry_weight = g.max_industry_weight  # 默认 0.30
    total_value = context.portfolio.total_value
    max_industry_value = total_value * max_industry_weight
    
    # 获取行业映射
    industry_map = get_industry_allocation(context, target_stocks)
    
    # 按行业分组
    industry_stocks = {}
    for stock in target_stocks:
        ind = industry_map.get(stock, '未知')
        if ind not in industry_stocks:
            industry_stocks[ind] = []
        industry_stocks[ind].append(stock)
    
    # 检查并调整
    target_dict = {}
    for stock in target_stocks:
        ind = industry_map.get(stock, '未知')
        industry_total = len(industry_stocks[ind]) * value_per_stock
        
        if industry_total > max_industry_value:
            # 该行业超标，按比例缩减
            allowed_num = int(max_industry_value / value_per_stock)
            if stock in industry_stocks[ind][:allowed_num]:
                target_dict[stock] = value_per_stock
            else:
                target_dict[stock] = 0  # 超出部分不配置
        else:
            target_dict[stock] = value_per_stock
    
    # 移除零配置的股票
    target_dict = {k: v for k, v in target_dict.items() if v > 0}
    
    return target_dict
```

## 五、市场情绪过滤机制设计

### 5.1 情绪指标定义

```
涨停家数 = 当日涨停（或接近涨停）的股票数量
阈值：涨停数 < 30 → 暂停开仓
```

#### 计算逻辑

```python
def count_limit_up(context):
    """统计当日涨停股票数量"""
    all_stocks = get_all_securities(types=['stock'], date=context.previous_date)
    stocks = all_stocks.index.tolist()
    
    # 排除ST、科创板、新股
    stocks = [s for s in stocks if not s.startswith('688') and not s.startswith('8')]
    
    current_data = get_current_data()
    limit_count = 0
    
    for stock in stocks:
        if current_data[stock].paused:
            continue
        if current_data[stock].is_st:
            continue
        
        last_price = current_data[stock].last_price
        high_limit = current_data[stock].high_limit
        
        if last_price >= high_limit * 0.995:  # 接近涨停
            limit_count += 1
    
    return limit_count
```

### 5.2 情绪过滤规则

| 涨停数 | 市场情绪 | 操作 |
|--------|----------|------|
| < 30 | 极度低迷 | 暂停开新仓，持有现金 |
| 30-50 | 偏弱 | 减半仓位 |
| > 50 | 正常 | 正常操作 |

#### 设计依据

| 涨停数阈值 | 对应市场状态 | 历史场景 |
|------------|--------------|----------|
| 30 | 冰点 | 2024年1月微盘股危机、2018年底 |
| 50 | 偏冷 | 震荡市常见 |

### 5.3 情绪过滤代码实现

```python
def check_market_sentiment(context):
    """
    市场情绪过滤
    返回: sentiment_factor (0.0-1.0)，用于调整仓位
    """
    limit_count = count_limit_up(context)
    
    if limit_count < g.sentiment_limit_threshold:  # < 30
        log.warning(f"市场情绪极度低迷，涨停数={limit_count}，暂停开仓")
        return 0.0  # 暂停开仓
    elif limit_count < g.sentiment_weak_threshold:  # < 50
        log.info(f"市场情绪偏弱，涨停数={limit_count}，减半仓位")
        return 0.5  # 减半仓位
    else:
        return 1.0  # 正常
```

## 六、综合风控框架

### 6.1 风控执行时序

```
每日执行（14:50）:
├── 1. 止损检查（个股级别）
│   ├── 亏损>20% → 清仓
│   └── 亏损>15% → 减半仓
│
├── 2. 市场情绪检查
│   └── 涨停数<30 → 标记暂停开仓
│
└── 3. 记录市场宽度（为月度调仓做准备）

每月执行（1日 9:35）:
├── 1. 计算市场宽度
├── 2. 确定仓位上限
├── 3. 选股
├── 4. 行业分散检查
├── 5. 情绪过滤（如暂停开仓则跳过买入）
└── 6. 执行调仓
```

### 6.2 风控优先级

当多个风控条件同时触发时，按以下优先级执行：

```
优先级1（最高）: 个股止损 → 立即执行，不可覆盖
优先级2: 市场情绪 → 暂停开仓，但不强制平仓已有持仓
优先级3: 仓位管理 → 调整新仓位规模
优先级4（最低）: 行业分散 → 选股阶段约束
```

        return 0.0
    elif limit_count < g.sentiment_weak_threshold:
        log.info(f"市场情绪偏弱，涨停数={limit_count}，减半仓位")
        return 0.5
    else:
        return 1.0


# ===================== 仓位管理模块 =====================

def calc_market_breadth(watch_date):
    """计算市场宽度：HS300中站上20日均线的股票占比"""
    hs300 = get_index_stocks("000300.XSHG", date=watch_date)
    if len(hs300) == 0:
        return 0.5
    
    prices = get_price(hs300, end_date=watch_date, count=20, fields=["close"], panel=False)
    if len(prices) == 0:
        return 0.5
    
    close = prices.pivot(index="time", columns="code", values="close")
    if len(close) < 20:
        return 0.5
    
    breadth = float((close.iloc[-1] > close.mean()).mean())
    return breadth


def get_position_config(breadth):
    """根据市场宽度获取仓位配置"""
    if breadth < g.breadth_critical:
        return g.max_position_critical, g.hold_num_critical, "CRITICAL"
    elif breadth < g.breadth_weak:
        return g.max_position_weak, g.hold_num_weak, "WEAK"
    else:
        return g.max_position_normal, g.hold_num, "NORMAL"


# ===================== 行业分散模块 =====================

def get_industry_map(context, stocks):
    """获取股票的行业映射"""
    industry_map = {}
    watch_date = context.previous_date
    for stock in stocks:
        try:
            ind = get_industry(stock, date=watch_date)
            industry = ind.get('sw_l1', {}).get('industry_name', '未知')
            industry_map[stock] = industry
        except:
            industry_map[stock] = '未知'
    return industry_map


def enforce_industry_limit(context, stocks, value_per_stock):
    """
    执行行业分散限制：单一行业 ≤ 30%
    返回: {stock: target_value}
    """
    max_industry_weight = g.max_industry_weight
    total_value = context.portfolio.total_value
    max_industry_value = total_value * max_industry_weight
    
    industry_map = get_industry_map(context, stocks)
    
    # 按行业分组
    industry_stocks = {}
    for stock in stocks:
        ind = industry_map.get(stock, '未知')
        if ind not in industry_stocks:
            industry_stocks[ind] = []
        industry_stocks[ind].append(stock)
    
    # 检查行业暴露并调整
    target_dict = {}
    for stock in stocks:
        ind = industry_map.get(stock, '未知')
        industry_total = len(industry_stocks[ind]) * value_per_stock
        
        if industry_total > max_industry_value:
            allowed_num = max(1, int(max_industry_value / value_per_stock))
            if stock in industry_stocks[ind][:allowed_num]:
                target_dict[stock] = value_per_stock
            else:
                target_dict[stock] = 0
        else:
            target_dict[stock] = value_per_stock
    
    return {k: v for k, v in target_dict.items() if v > 0}

def get_smallcap_universe(watch_date):
    all_stocks = get_all_securities(types=["stock"], date=watch_date)
    all_stocks = all_stocks[
        all_stocks["start_date"] <= watch_date - pd.Timedelta(days=g.ipo_days)
    ]
    stocks = all_stocks.index.tolist()

    is_st = get_extras("is_st", stocks, end_date=watch_date, count=1).iloc[-1]
    stocks = is_st[is_st == False].index.tolist()

    paused = get_price(
        stocks, end_date=watch_date, count=1, fields="paused", panel=False
    )
    paused = paused.pivot(index="time", columns="code", values="paused").iloc[-1]
    stocks = paused[paused == 0].index.tolist()

    stocks = [s for s in stocks if not s.startswith("688")]

    q = query(valuation.code, valuation.market_cap).filter(
        valuation.code.in_(stocks),
        valuation.market_cap >= g.min_cap,
        valuation.market_cap <= g.max_cap,
    )

    df = get_fundamentals(q, date=watch_date)
    if len(df) == 0:
        return []

    df["cap_rank"] = df["market_cap"].rank(pct=True)
    small_stocks = df[df["cap_rank"] <= 0.3]["code"].tolist()

    return small_stocks


def check_dividend_stability(code, watch_date):
    try:
        years = 3
        end_date = watch_date
        start_date = end_date - pd.Timedelta(days=365 * years)

        q = query(
            finance.STK_DIVIDEND_INFO.code,
            finance.STK_DIVIDEND_INFO.pub_date,
            finance.STK_DIVIDEND_INFO.dividend_ratio,
        ).filter(
            finance.STK_DIVIDEND_INFO.code == code,
            finance.STK_DIVIDEND_INFO.pub_date >= start_date,
            finance.STK_DIVIDEND_INFO.pub_date <= end_date,
            finance.STK_DIVIDEND_INFO.dividend_ratio > 0,
        )

        df = finance.run_query(q)

        if len(df) >= years:
            return True
        return False
    except:
        return False


def select_stocks(watch_date, hold_num):
    stocks = get_smallcap_universe(watch_date)
    if len(stocks) < 5:
        return []

    q = query(
        valuation.code,
        valuation.market_cap,
        valuation.pe_ratio,
        valuation.pb_ratio,
        indicator.roe,
    ).filter(
        valuation.code.in_(stocks),
        valuation.pe_ratio > 0,
        valuation.pe_ratio < 100,
    )

    df = get_fundamentals(q, date=watch_date)
    if len(df) == 0:
        return []

    df = df.drop_duplicates("code")

    dividend_data = {}
    for code in df["code"].tolist()[:50]:
        dividend_data[code] = check_dividend_stability(code, watch_date)

    df["dividend_stable"] = df["code"].map(dividend_data)
    df = df[df["dividend_stable"] == True]

    if len(df) == 0:
        return []

    try:
        q2 = query(
            valuation.code, valuation.market_cap, indicator.dividend_yield_ratio
        ).filter(valuation.code.in_(df["code"].tolist()))

        df2 = get_fundamentals(q2, date=watch_date)
        df2 = df2.drop_duplicates("code")

        df = df.merge(df2[["code", "dividend_yield_ratio"]], on="code", how="left")
        df = df[df["dividend_yield_ratio"] >= g.min_dividend_yield]

        if len(df) == 0:
            return []

        df = df.sort_values("dividend_yield_ratio", ascending=False)

        return df["code"].tolist()[:hold_num]
    except:
        df = df.sort_values(["roe", "pe_ratio"], ascending=[False, True])
        return df["code"].tolist()[:hold_num]


def filter_buyable(context, stocks):
    current_data = get_current_data()
    buyable = []
    for stock in stocks:
        if current_data[stock].paused or current_data[stock].is_st:
            continue
        if "ST" in current_data[stock].name or "*" in current_data[stock].name:
            continue
        last_price = current_data[stock].last_price
        if last_price >= current_data[stock].high_limit * 0.995:
            continue
        buyable.append(stock)
    return buyable


# ===================== 主调仓逻辑 =====================

def rebalance(context):
    watch_date = context.previous_date
    
    # === 风控步骤1: 计算市场宽度，确定仓位 ===
    breadth = calc_market_breadth(watch_date)
    g.current_breadth = breadth
    position_limit, hold_num, regime = get_position_config(breadth)
    
    log.info(f"Market State: breadth={breadth:.3f}, regime={regime}, position_limit={position_limit:.0%}, hold_num={hold_num}")
    
    # === 风控步骤2: 市场情绪检查 ===
    sentiment = check_market_sentiment(context)
    
    # 综合仓位系数 = 仓位限制 × 情绪系数
    effective_position_limit = position_limit * sentiment
    
    if effective_position_limit <= 0:
        log.warning("风控触发：暂停所有开仓操作，清仓离场")
        # 清仓所有持仓
        for stock in list(context.portfolio.positions.keys()):
            order_target(stock, 0)
        return
    
    # === 风控步骤3: 选股 ===
    stocks = select_stocks(watch_date, hold_num)
    stocks = filter_buyable(context, stocks)
    
    if len(stocks) == 0:
        log.info("No candidates available")
        return
    
    # === 风控步骤4: 计算目标仓位 ===
    total_value = context.portfolio.total_value
    max_invest_value = total_value * effective_position_limit
    value_per_stock = max_invest_value / len(stocks)
    
    # === 风控步骤5: 行业分散检查 ===
    target_dict = enforce_industry_limit(context, stocks, value_per_stock)
    
    if len(target_dict) == 0:
        log.warning("行业分散约束后无可用标的")
        return
    
    # === 风控步骤6: 执行调仓 ===
    current_positions = context.portfolio.positions
    
    # 先卖出不在目标列表中的股票
    for stock in list(current_positions.keys()):
        if stock not in target_dict:
            order_target(stock, 0)
    
    # 再买入目标股票
    for stock, target_value in target_dict.items():
        order_target_value(stock, target_value)
    
    log.info(f"Rebalance Complete: regime={regime}, breadth={breadth:.3f}, sentiment={sentiment}, "
             f"position_limit={effective_position_limit:.0%}, holdings={len(target_dict)}")


def record_market_state(context):
    """记录市场状态（可选）"""
    watch_date = context.previous_date
    breadth = calc_market_breadth(watch_date)
    record(breadth=breadth)

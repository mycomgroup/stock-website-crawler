## 二、候选策略分析

### 2.1 纯国债ETF (基准策略)

| 指标 | 说明 |
|------|------|
| 标的 | 511010.XSHG (国债ETF) |
| 策略类型 | 被动持有 |
| 预期年化 | 2-3% (债息收益) |
| 预期最大回撤 | 1-2% |
| 调仓频率 | 无 |
| 复杂度 | 最低 |

**优点**: 波动最低，流动性好，无操作成本
**缺点**: 收益率有限，在利率上行周期可能亏损

---

### 2.2 国债固收+ (当前配置)

| 指标 | 数值/说明 |
|------|----------|
| 标的配置 | 511010(70%) + 510880/红利(8%) + 513100/纳指(8%) + 518880/黄金(14%) |
| 基准 | 511010.XSHG |
| 调仓规则 | 偏离超15%再平衡 |
| 原回测周期 | 2014-01-01 到 2024-11-11 |
| 预期年化 | 4-6% |
| 预期最大回撤 | 3-5% |

**策略逻辑**:
- 70%国债作为稳定器
- 30%增强项(红利+纳指+黄金)提供超额收益
- 偏离15%触发再平衡，控制交易成本

**优点**:
- 结构简单，易于理解和执行
- 国债占比高，防守属性强
- 增强项提供多元化收益来源

**缺点**:
- 固定权重，不适应市场波动变化
- 纳指ETF在当前FED指标下可能高估
- 黄金近期跌幅较大(-13.16%)

### 2.3 桥水全天候策略

| 指标 | 数值/说明 |
|------|----------|
| 标的 | 沪深300ETF + 黄金ETF + 国债ETF + 纳指ETF |
| 策略类型 | 风险平价 (ES) |
| 基准 | 511010.XSHG |
| 调仓规则 | 偏离超15%或每30天 |
| 原回测表现 | 7年最大回撤仅3.5% |
| 预期年化 | 5-8% |

**策略逻辑**:
- 使用期望短缺(ES)衡量风险
- 风险高的资产配低权重，风险低的配高权重
- 四类资产：股票、商品、债券、海外

**优点**:
- 动态调整，适应市场波动
- 风险平价理论成熟，桥水验证
- 回撤控制优秀

**缺点**:
- 策略复杂，依赖量化计算
- 需要定期计算ES指标
- 对纳指依赖度可能较高

### 2.4 股债波动平衡

| 指标 | 数值/说明 |
|------|----------|
| 标的 | 富国天惠+兴全轻资+国债+纳指+标普+黄金+消费+医药 (8只) |
| 策略类型 | 波动率倒数加权 |
| 基准 | 000300.XSHG |
| 调仓规则 | 偏离超5%或每周五 |
| 权重 | 场外基金为主 |

**策略逻辑**:
- 使用40日波动率的倒数作为权重
- 波动率低的资产配高权重
- 每周五检查，偏离5%触发再平衡

**优点**:
- 波动率加权直观
- 包含场外基金，标的更丰富

**缺点**:
- 含场外基金，流动性受限
- 调仓频率高(每周)，交易成本大
- 消费+医药在当前市场环境表现不佳

### 2.5 别人恐惧我贪婪 (择时策略)

| 指标 | 数值/说明 |
|------|----------|
| 标的 | 中证1000ETF + 国债ETF |
| 策略类型 | 择时切换 |
| 触发条件 | 上证跌幅>1.5%且当日跌幅>1.4% |
| 持仓周期 | 买入后持有20天 |
| 预期年化 | 15-20% (高波动) |

**策略逻辑**:
- 大跌时从国债切换到中证1000ETF
- 持有20天后切换回国债
- 等待下一次大跌机会

**优点**:
- 在极端大跌时能抓住反弹机会
- 年化收益潜力高

**缺点**:
- 不适合作为防守底仓，波动太大
- 依赖择时，可能长期空仓等待
- 与防守目标相悖

## 六、当前30%防守仓最优放法

### 6.1 结论

**30%防守仓应全部配置到"简化版国债固收+"策略**

配置方案:
```
511010.XSHG (国债ETF): 75% × 30% = 22.5% 总资金
518880.XSHG (黄金ETF): 10% × 30% = 3.0%  总资金
510880.XSHG (红利ETF):  8% × 30% = 2.4%  总资金
513100.XSHG (纳指ETF):  4% × 30% = 1.2%  总资金
现金/货基:              3% × 30% = 0.9%  总资金
```

### 6.2 选择理由

1. **优于纯国债**: 固收+提供4-6%年化，比纯国债(2-3%)高出约2%
2. **优于桥水全天候**: 桥水依赖纳指较多，当前海外环境不确定
3. **优于股债波动平衡**: 后者含场外基金，流动性差且调仓频繁
4. **优于择时策略**: 别人恐惧我贪婪波动太大，不适合作为防守底仓

### 6.3 与原方案对比

| 项目 | 原方案 | 新方案 | 变化 |
|------|--------|--------|------|
| 国债权重 | 70% | 75% | +5% |
| 黄金权重 | 14% | 10% | -4% |
| 红利权重 | 8% | 8% | 不变 |
| 纳指权重 | 8% | 4% | -4% |
| 现金权重 | 0% | 3% | +3% |

**调整原因**:
- 黄金近期大跌(-13.16%)，降低暴露
- 纳指在FED指标偏弱时风险较大，减半配置
- 增加现金缓冲，提高流动性

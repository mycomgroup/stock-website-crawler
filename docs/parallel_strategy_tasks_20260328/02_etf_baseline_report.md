## 3. 推荐基线: Mom20d_Hold10d

### 3.1 核心参数

| 参数 | 值 | 理由 |
|------|-----|------|
| 动量窗口 | 20日 | IC最稳定, 夏普最高 |
| 持有周期 | 10日 | 匹配因子半衰期(9-10天) |
| 持仓数量 | Top 3 | 平衡集中度与分散 |
| 交易成本 | 单边0.1% | 万一手续费+滑点 |

### 3.2 为什么推荐这个基线

1. **夏普最高 (0.62)**: 风险调整后收益最优
2. **回撤可控 (-21.23%)**: 满足<25%要求
3. **年化收益最高 (9.22%)**: 接近10%目标
4. **换手适中**: 10日持有, 年化约25次调仓

### 3.3 关键发现

- **动量窗口**: 20日 > 30日 > 10日 (按夏普排序)
- **持有周期**: 10日 > 20日 (10日持有回撤更小)
- **短窗口+短持有**: Mom10d_Hold5d 回撤过大(-44.91%)
- **长窗口+长持有**: Mom20d_Hold20d 回撤也偏大(-41.33%)

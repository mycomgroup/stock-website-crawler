## 二、核心思想提取

### 2.1 板块相关性与一起涨跌

**关键发现：板块一轮都是一起涨的和跌的，有很强相关性**

来源：`51 行业反转效应`、`92 行业宽度轮动研究`

核心逻辑：
- 同一行业内的股票具有高度联动性
- 行业指数可以代表行业内多数股票的走势
- 行业宽度（MA20以上个股占比）可以反映行业整体强弱

代码示例（来自92）：
```python
# 计算行业内MA20以上个股占比
df_bias = (df_close > df_ma20)  # 个股是否在MA20之上
df_ratio = (df_bias.groupby('industry_code').sum() * 100.0) / df_bias.groupby('industry_code').count()
# 选择宽度最高的行业
top_values = df_ratio.nlargest(g.num)
```

### 2.2 行业轮动的几种模式

#### 模式A：动量轮动（追涨）
来源：`21 行业ETF轮动+择时`、`28 从相关系数角度探讨etf轮动高评分`

- 用BBI/动量指标判断行业强弱
- 选择动量最强的行业ETF持有
- 配合大盘择时过滤

#### 模式B：反转轮动（抄底）
来源：`51 行业反转效应`、`92 行业宽度轮动研究`

- 行业存在反转效应
- 过去表现差的行业未来可能反弹
- 用`nsmallest`选择涨幅最小的行业

#### 模式C：三维评分轮动
来源：`29 趋势-拥挤-景气度模型`、`56 基于趋势、拥挤、景气的行业轮动`

- 趋势：行业动量/IR
- 拥挤：换手率/波动率/成交量异常
- 景气：基本面增长
- 综合得分 = 0.51*趋势 - 0.23*拥挤 + 1.13*景气

### 2.3 板块热度追踪

来源：`02 研究 板块轮动打分与热度追踪`、`64 A股最强板块动量趋势`

# 板块热度得分 = 超阈值股票占比 × 平均涨幅

应用场景：

- 追踪热点持续性
- 辅助行业选择

### 2.4 市场宽度与拥挤度

来源：`42 大盘拥挤率极速版`、`45/52/97 市场宽度`、`82 拥挤率指标`

市场宽度：
- 全市场/行业MA20以上个股占比
- 反映市场/行业整体强弱
- 适合作为择时过滤器

拥挤度：
- 成交额前5%股票占比（市场拥挤）
- 换手率异常（行业拥挤）
- 用于识别过热风险

### 2.5 净新高占比(NH-NL)%

来源：`89 市场情绪指标专题：行业指数顶部和底部信号`

核心逻辑：
- 行业中创年度新高个股数 - 创年度新低个股数 / 行业总个股数
- 阈值：±20%（乐观/悲观）、±30%（贪婪/恐惧）
- 基于锚定效应的行为金融学解释

应用场景：
- 科技板块和周期板块表现较好
- 适合做反转策略
- 资金占用时间少，适合增强型策略

## 四、总结

### 关键洞察

1. **板块确实一起涨跌**：行业内股票高度联动，行业指标比单ETF指标更有代表性

2. **行业轮动有多种模式**：
   - 动量模式（追涨）：适合趋势行情
   - 反转模式（抄底）：适合震荡行情
   - 三维模式（综合）：更稳健但参数敏感

3. **宽度和拥挤度是关键辅助指标**：
   - 宽度反映行业整体强弱
   - 拥挤度预警过热风险

4. **净新高占比适合判断顶底**：
   - 在科技和周期板块表现好
   - 适合做反转策略

### 对行业增强任务的建议

1. **短期**：使用行业宽度替代单ETF动量作为行业筛选依据
2. **中期**：加入拥挤度过滤和反转信号
3. **长期**：构建完整的三维行业评分体系（宽度+拥挤+景气）

# Alpha191因子筛选报告

## 1. Alpha191因子概述

### 1.1 因子特点
- **来源**: WorldQuant Alpha101的扩展版本，由聚宽平台扩展至191个
- **数量**: 191个纯量化公式因子
- **特性**: 纯价量公式、无基本面依赖
- **优势**: 计算快速、逻辑透明、实盘可用
- **API**: `get_all_alpha_191(date, code, alpha)`

### 1.2 因子分类
Alpha191因子根据其计算逻辑主要分为以下几类：
1. **动量反转因子**: 基于价格动量和反转效应
2. **波动率因子**: 基于价格波动特征
3. **成交量因子**: 基于成交量变化规律
4. **量价组合因子**: 结合价格和成交量的复杂模式
5. **技术形态因子**: 基于K线形态和技术指标

### 1.3 筛选方法
- **测试期间**: 2023-2025年（近3年数据）
- **样本股票**: 20-200亿市值股票（剔除ST、停牌、次新股）
- **预测周期**: 5日收益
- **评价指标**: IC均值、ICIR、IC>0比例
- **筛选标准**: 
  - 有效：ICIR > 0.3
  - 弱有效：ICIR 0.15-0.3
  - 无效：ICIR < 0.15

### 4.3 增量价值分析

### 5.2 组合构建

#### 方法1：等权组合
```python
from jqdatasdk.alpha191 import get_all_alpha_191

# 获取有效因子值
alpha_df = get_all_alpha_191(
    date='2025-01-01',
    code=stock_list,
    alpha=effective_alpha_list
)

# 等权组合
composite_alpha = alpha_df.mean(axis=1)
```

#### 方法2：ICIR加权组合
```python
# 根据历史ICIR分配权重
icir_weights = {
    'alpha_006': 0.35,
    'alpha_021': 0.38,
    'alpha_042': 0.36,
    # ... 其他因子权重
}

# 加权组合
composite_alpha = sum(
    alpha_df[col] * icir_weights.get(col, 0) 
    for col in alpha_df.columns
) / sum(icir_weights.values())
```

#### 方法3：与RFScore7组合
```python
# Alpha191组合得分
alpha191_score = composite_alpha

# RFScore7得分
rfscore7 = get_rfscore7(stock_list, date)

# 组合得分（ICIR加权）
final_score = 0.15 * alpha191_score + 0.85 * rfscore7
```

### 5.3 调仓策略
1. **调仓频率**: 建议5-10日调仓一次
2. **调仓阈值**: IC变化超过0.01时重新筛选因子
3. **动态调整**: 每季度重新测试因子有效性

### 5.4 风险控制
1. **单因子权重**: 单个因子权重不超过15%
2. **因子数量**: 有效因子不少于10个
3. **相关性控制**: 因子间相关性不超过0.6
4. **IC衰减监控**: IC低于0.02时及时剔除因子

### 5.5 注意事项

#### 过拟合风险
- **问题**: 191个因子中，部分可能过拟合历史数据
- **对策**: 
  - 使用样本外数据验证
  - 选择逻辑清晰的因子
  - 避免过于复杂的组合

#### 衰减风险
- **问题**: 价量因子衰减较快，有效期通常6-12个月
- **对策**: 
  - 每季度重新测试因子有效性
  - 动态调整因子权重
  - 建立因子监控机制

#### 数据质量
- **问题**: 价量数据可能存在异常值、停牌等情况
- **对策**: 
  - 数据清洗和预处理
  - 剔除异常数据
  - 处理停牌股票

#### 交易成本
- **问题**: 高频因子调仓频繁，增加交易成本
- **对策**: 
  - 选择换手率适中的因子
  - 优化调仓策略
  - 考虑滑点影响

### 附录C：因子IC计算
```python
import pandas as pd
import numpy as np
from scipy.stats import spearmanr

def calculate_ic(factor_values, forward_returns):
    """
    计算因子IC
    
    参数:
    - factor_values: 因子值（Series）
    - forward_returns: 未来收益（Series）
    
    返回:
    - IC值
    """
    # 去除NaN
    valid_data = pd.DataFrame({
        'factor': factor_values,
        'return': forward_returns
    }).dropna()
    
    if len(valid_data) < 30:
        return np.nan
    
    # 计算秩相关系数
    ic, _ = spearmanr(valid_data['factor'], valid_data['return'])
    return ic

def calculate_icir(ic_series):
    """
    计算ICIR
    
    参数:
    - ic_series: IC序列
    
    返回:
    - ICIR值
    """
    if len(ic_series) == 0:
        return np.nan
    
    ic_mean = np.mean(ic_series)
    ic_std = np.std(ic_series)
    
    if ic_std == 0:
        return np.nan
    
    return ic_mean / ic_std
```

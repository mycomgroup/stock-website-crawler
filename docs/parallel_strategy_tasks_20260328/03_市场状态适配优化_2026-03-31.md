## 一、问题诊断

### 1.1 核心问题

| 版本 | 年化收益 | 说明 |

| 静态60/40组合 | **7.3%** | RFScore60% + 红利小盘40% |

### 1.2 动态版本失败原因分析

通过审查 `combo_rfscore_dividend_dynamic.py`，发现以下问题:

#### 问题1: 市场状态指标过于粗糙

```python
# 当前实现 - 仅用2个指标
breadth = float((close.iloc[-1] > close.mean()).mean())  # 单一宽度
trend_on = idx_close > idx_ma20                           # 二元趋势
```

- 宽度只有单一数值，没有分级阈值
- 趋势只用MA20，没有多周期确认
- **完全没有情绪指标**（涨停数、换手率）

#### 问题2: 切换逻辑过于激进

```python
# 当前动态权重逻辑
if breadth < 0.15 and not trend_on:
    rfscore=0.0, dividend=1.0    # 极端切换，0/100
elif breadth < 0.25 and not trend_on:
    rfscore=0.4, dividend=0.6    # 频繁切换
elif trend_on:
    rfscore=0.7, dividend=0.3    # 与目标60/40偏离
```

- 极端状态直接0/100，交易成本极高
- 没有滞后机制，容易在临界点反复切换
- 牛市70/30反而偏离了最优静态60/40

#### 问题3: 切换时点无优化

- 每月1日固定调仓，可能落在市场状态转换的中间点
- 没有状态持续时间确认，单次信号就触发切换
- 没有考虑切换成本对收益的侵蚀

## 二、市场状态识别指标设计

### 2.1 总体架构

```
市场状态识别
├── 宽度指标 (Breadth) - 三档阈值
├── 趋势指标 (Trend) - 多周期均线
├── 情绪指标 (Sentiment) - 涨停数+换手率
└── 状态综合判定 - 加权投票
```

### 2.2 宽度指标 (Breadth)

**计算标的**: 中证1000成分股（更贴合小盘策略）

**计算方法**:
```python
# 成分股收盘价 > MA20 的比例
stocks = get_index_stocks("000852.XSHG", date=watch_date)
prices = get_price(stocks, end_date=watch_date, count=20, fields=["close"], panel=False)
close = prices.pivot(index="time", columns="code", values="close")
breadth = (close.iloc[-1] > close.rolling(20).mean().iloc[-1]).mean()
```

**三档阈值设计**:

| 档位 | 阈值 | 状态标签 | 含义 | 历史分布 |
|------|------|----------|------|----------|
| 窄 | < 15% | 极度弱势 | 市场恐慌底部 | ~8% 时间 |
| 中 | 15% - 25% | 弱势震荡 | 筑底/调整期 | ~22% 时间 |
| 宽 | > 40% | 强势 | 趋势明确 | ~25% 时间 |
| 中间 | 25% - 40% | 中性 | 过渡状态 | ~45% 时间 |

**阈值依据**:
- 15%: 对应宏观路由器验证中"底部"状态(<30%)的收紧版，聚焦极度恐慌
- 25%: 对应原breadth_reduce参数，保留有效阈值
- 40%: 新增强势阈值，确认趋势强度

### 2.3 趋势指标 (Trend)

**多周期均线系统**:

| 均线 | 周期 | 用途 |
|------|------|------|
| MA20 | 20日 | 短期趋势 |
| MA60 | 60日 | 中期趋势 |

**趋势判定逻辑**:
```python
# 获取沪深300指数价格
idx = get_price("000300.XSHG", end_date=watch_date, count=60, fields=["close"])
close = idx["close"].iloc[-1]
ma20 = idx["close"].iloc[-20:].mean()
ma60 = idx["close"].mean()

# 趋势状态
if close > ma20 and ma20 > ma60:
    trend = "UP"        # 多头排列
elif close < ma20 and ma20 < ma60:
    trend = "DOWN"      # 空头排列
else:
    trend = "NEUTRAL"   # 交叉/震荡
```

**三态设计**:

| 状态 | 条件 | 标签 |
|------|------|------|
| UP | close > MA20 > MA60 | 多头趋势 |
| DOWN | close < MA20 < MA60 | 空头趋势 |
| NEUTRAL | 其他 | 震荡/交叉 |

| 状态 | 宽度 | 趋势 | 情绪 | 权重配置 |

## 三、策略切换逻辑设计

### 3.1 权重配置矩阵

| 市场状态 | 红利小盘 | RFScore | 逻辑说明 |
|----------|----------|---------|----------|
| 底部 | **60%** | **40%** | 防守为主，红利小盘抗跌性强 |
| 趋势 | **40%** | **60%** | 均衡配置，RFScore质量因子发力 |
| 牛市 | **30%** | **70%** | 进攻为主，RFScore弹性更大 |
| 震荡 | **50%** | **50%** | 平衡配置，降低切换频率 |

### 3.2 切换滞后机制（防抖动）

**问题**: 原策略在临界点反复切换，产生大量交易成本。

**解决方案**: 状态持续时间确认 + 切换冷却期

```python
# 状态确认: 连续N天同一状态才触发切换
g.state_confirm_days = 3  # 连续3天确认

# 切换冷却: 上次切换后M天内不重复切换
g.cooldown_days = 10  # 10天冷却期
```

**实现逻辑**:
```python
# 记录状态历史
g.state_history = []  # 最近N天的状态列表

# 状态确认
if len(g.state_history) >= g.state_confirm_days:
    recent_states = g.state_history[-g.state_confirm_days:]
    if all(s == "BOTTOM" for s in recent_states):
        confirmed_state = "BOTTOM"
    # ... 其他状态同理

# 冷却检查
if context.current_dt - g.last_rebalance_date < timedelta(days=g.cooldown_days):
    # 跳过本次切换
    return
```

### 3.3 切换时点优化

**原方案**: 每月1日固定调仓

**优化方案**: 状态触发 + 月度上限

```python
# 每日检查市场状态
run_daily(check_market_state, time="14:50")

# 状态变化时触发调仓（受冷却期限制）
# 同时保留月度强制调仓作为兜底
run_monthly(rebalance, 1, time="9:35")
```

**调仓触发条件**:
1. 市场状态发生变化 且 通过状态确认 且 不在冷却期
2. 每月1日强制调仓（兜底机制）

### 3.4 切换成本控制

**交易成本估算**:

| 场景 | 换手率 | 单次成本 | 年化成本 |
|------|--------|----------|----------|
| 原动态策略 | ~180%/年 | 0.18% | ~3.2% |
| 优化后(含冷却) | ~120%/年 | 0.12% | ~2.2% |
| 静态60/40 | ~60%/年 | 0.06% | ~1.1% |

**成本控制措施**:
1. 冷却期减少无效切换
2. 状态确认避免假信号
3. 权重渐进调整（每次最大变动20%）

## 五、完整状态切换代码

### 5.1 优化版策略文件

```python
# combo_rfscore_dividend_state_adaptive.py
# 市场状态自适应版 - 红利小盘 + RFScore 组合策略
# 优化点: 三维度状态识别 + 滞后确认 + 冷却机制

from jqdata import *
from jqfactor import Factor, calc_factors
import pandas as pd
import numpy as np
import datetime
from datetime import timedelta


def sign(ser):
    return ser.apply(lambda x: np.where(x > 0, 1, 0))


class RFScore(Factor):
    name = "RFScore"
    max_window = 1
    dependencies = [
        "roa",
        "roa_4",
        "net_operate_cash_flow",
        "net_operate_cash_flow_1",
        "net_operate_cash_flow_2",
        "net_operate_cash_flow_3",
        "total_assets",
        "total_assets_1",
        "total_assets_2",
        "total_assets_3",
        "total_assets_4",
        "total_assets_5",
        "total_non_current_liability",
        "total_non_current_liability_1",
        "gross_profit_margin",
        "gross_profit_margin_4",
        "operating_revenue",
        "operating_revenue_4",
    ]

    def calc(self, data):
        roa = data["roa"]
        delta_roa = roa / data["roa_4"] - 1

        cfo_sum = (
            data["net_operate_cash_flow"]
            + data["net_operate_cash_flow_1"]
            + data["net_operate_cash_flow_2"]
            + data["net_operate_cash_flow_3"]
        )
        ta_ttm = (
            data["total_assets"]
            + data["total_assets_1"]
            + data["total_assets_2"]
            + data["total_assets_3"]
        ) / 4
        ocfoa = cfo_sum / ta_ttm
        accrual = ocfoa - roa * 0.01

        leveler = data["total_non_current_liability"] / data["total_assets"]
        leveler1 = data["total_non_current_liability_1"] / data["total_assets_1"]
        delta_leveler = -(leveler / leveler1 - 1)

        delta_margin = data["gross_profit_margin"] / data["gross_profit_margin_4"] - 1

        turnover = (
            data["operating_revenue"]
            / (data["total_assets"] + data["total_assets_1"]).mean()
        )
        turnover_1 = (
            data["operating_revenue_4"]
            / (data["total_assets_4"] + data["total_assets_5"]).mean()
        )
        delta_turn = turnover / turnover_1 - 1

        indicator_tuple = (
            roa,
            delta_roa,
            ocfoa,
            accrual,
            delta_leveler,
            delta_margin,
            delta_turn,
        )
        self.basic = pd.concat(indicator_tuple).T.replace([-np.inf, np.inf], np.nan)
        self.basic.columns = [
            "ROA",
            "DELTA_ROA",
            "OCFOA",
            "ACCRUAL",
            "DELTA_LEVELER",
            "DELTA_MARGIN",
            "DELTA_TURN",
        ]
        self.fscore = self.basic.apply(sign).sum(axis=1)

def calc_breadth(watch_date):
    """计算市场宽度: 中证1000成分股中 close > MA20 的比例"""
    stocks = get_index_stocks("000852.XSHG", date=watch_date)
    if len(stocks) < 100:
        return 0.5  # 降级处理

    prices = get_price(
        stocks, end_date=watch_date, count=20, fields=["close"], panel=False
    )
    close = prices.pivot(index="time", columns="code", values="close")
    ma20 = close.rolling(20).mean().iloc[-1]
    breadth = float((close.iloc[-1] > ma20).mean())
    return breadth


def calc_trend(watch_date):
    """计算趋势状态: 基于MA20和MA60的多周期均线系统"""
    idx = get_price("000300.XSHG", end_date=watch_date, count=60, fields=["close"])
    if len(idx) < 60:
        return "NEUTRAL"

    close = float(idx["close"].iloc[-1])
    ma20 = float(idx["close"].iloc[-20:].mean())
    ma60 = float(idx["close"].mean())

    if close > ma20 and ma20 > ma60:
        return "UP"
    elif close < ma20 and ma20 < ma60:
        return "DOWN"
    else:
        return "NEUTRAL"


def calc_sentiment(watch_date):
    """计算情绪指标: 涨停数 + 换手率"""
    # 涨停数
    all_stocks = get_all_securities(types=["stock"], date=watch_date).index.tolist()
    if len(all_stocks) == 0:
        return {"limit_up": 0, "turnover": 0, "score": 0.5}

    # 采样计算（性能优化）
    sample_stocks = all_stocks[:1000] if len(all_stocks) > 1000 else all_stocks

    try:
        current_data = get_current_data()
        limit_up_count = 0
        for s in sample_stocks:
            if s not in current_data:
                continue
            cd = current_data[s]
            if cd.paused or cd.is_st:
                continue
            if cd.last_price >= cd.high_limit * 0.995:
                limit_up_count += 1

        # 按比例推算全市场
        scale_factor = len(all_stocks) / len(sample_stocks)
        limit_up_count = int(limit_up_count * scale_factor)
    except:
        limit_up_count = 0

    # 换手率 (近5日全A平均)
    try:
        turn_idx = get_index_stocks("000906.XSHG", date=watch_date)[:300]
        turnover_df = get_price(
            turn_idx, end_date=watch_date, count=5, fields=["turn"], panel=False
        )
        avg_turnover = float(turnover_df["turn"].mean())
    except:
        avg_turnover = 2.0  # 默认值

    # 情绪评分
    sentiment_score = 0.0
    if limit_up_count > 80:
        sentiment_score += 0.5
    elif limit_up_count > 30:
        sentiment_score += 0.25

    if avg_turnover > 3.0:
        sentiment_score += 0.5
    elif avg_turnover > 1.5:
        sentiment_score += 0.25

    return {
        "limit_up": limit_up_count,
        "turnover": round(avg_turnover, 2),
        "score": sentiment_score,
    }


def classify_market_state(breadth, trend, sentiment):
    """综合判定市场状态"""
    # 牛市: 宽度>40% AND 趋势UP
    if breadth > 0.40 and trend == "UP":
        return "BULL"

    # 底部: 宽度<15% AND 趋势DOWN
    if breadth < 0.15 and trend == "DOWN":
        return "BOTTOM"

    # 趋势: 趋势UP AND 宽度>=15%
    if trend == "UP" and breadth >= 0.15:
        return "TREND"

    # 默认震荡
    return "NEUTRAL"


def calc_market_state_full(watch_date):
    """完整市场状态计算"""
    breadth = calc_breadth(watch_date)
    trend = calc_trend(watch_date)
    sentiment = calc_sentiment(watch_date)
    state = classify_market_state(breadth, trend, sentiment["score"])

    return {
        "breadth": breadth,
        "trend": trend,
        "sentiment": sentiment,
        "state": state,
    }

    paused = paused.pivot(index="time", columns="code", values="paused").iloc[-1]
    stocks = paused[paused == 0].index.tolist()

    return stocks


def select_rfscore_stocks(watch_date, hold_num):
    stocks = get_rfscore_universe(watch_date)
    if len(stocks) < 10:
        return []

    factor = RFScore()
    calc_factors(stocks, [factor], start_date=watch_date, end_date=watch_date)

    df = factor.basic.copy()
    df["RFScore"] = factor.fscore

    val = get_valuation(
        stocks, end_date=watch_date, fields=["pb_ratio", "pe_ratio"], count=1
    )
    val = val.drop_duplicates("code").set_index("code")[["pb_ratio", "pe_ratio"]]
    df = df.join(val, how="left")

    df = df.replace([np.inf, -np.inf], np.nan)
    df = df.dropna(subset=["RFScore", "pb_ratio"])
    df["pb_group"] = (
        pd.qcut(
            df["pb_ratio"].rank(method="first"), 10, labels=False, duplicates="drop"
        )
        + 1
    )

    primary = df[(df["RFScore"] == 7) & (df["pb_group"] <= g.rfscore_pb_group)].copy()
    primary = primary.sort_values(
        ["RFScore", "ROA", "OCFOA", "pb_ratio"], ascending=[False, False, False, True]
    )
    picks = primary.index.tolist()

    if len(picks) < hold_num:
        secondary = df[(df["RFScore"] >= 6) & (df["pb_group"] <= 2)].copy()
        secondary = secondary.sort_values(
            ["RFScore", "ROA", "pb_ratio"], ascending=[False, False, True]
        )
        for code in secondary.index.tolist():
            if code not in picks:
                picks.append(code)
            if len(picks) >= hold_num:
                break

    return picks[:hold_num]


def select_dividend_smallcap_stocks(watch_date, hold_num):
    all_stocks = get_all_securities(types=["stock"], date=watch_date)
    all_stocks = all_stocks[
        all_stocks["start_date"] <= watch_date - pd.Timedelta(days=g.ipo_days)
    ]
    stocks = all_stocks.index.tolist()

    is_st = get_extras("is_st", stocks, end_date=watch_date, count=1).iloc[-1]
    stocks = is_st[is_st == False].index.tolist()

    paused = get_price(
        stocks, end_date=watch_date, count=1, fields="paused", panel=False
    )
    paused = paused.pivot(index="time", columns="code", values="paused").iloc[-1]
    stocks = paused[paused == 0].index.tolist()

    stocks = [s for s in stocks if not s.startswith("688")]

    q = (
        query(
            valuation.code,
            valuation.market_cap,
            valuation.pe_ratio,
            valuation.pb_ratio,
            indicator.inc_net_profit_year_on_year,
            indicator.roe,
        )
        .filter(
            valuation.code.in_(stocks),
            valuation.market_cap >= g.dividend_min_cap,
            valuation.market_cap <= g.dividend_max_cap,
            valuation.pe_ratio > 0,
            valuation.pe_ratio < g.dividend_max_pe,
            indicator.inc_net_profit_year_on_year > 5,
            indicator.roe > 5,
        )
        .order_by(valuation.pe_ratio.asc())
    )

    df = get_fundamentals(q, date=watch_date)
    if len(df) == 0:
        return []

    df = df.sort_values("pe_ratio")
    return df["code"].tolist()[:hold_num]


def filter_buyable(context, stocks):
    current_data = get_current_data()
    buyable = []
    for stock in stocks:
        if stock not in current_data:
            continue
        if current_data[stock].paused or current_data[stock].is_st:
            continue
        if (
            "ST" in current_data[stock].name
            or "*" in current_data[stock].name
            or "退" in current_data[stock].name
        ):
            continue
        last_price = current_data[stock].last_price
        if last_price >= current_data[stock].high_limit * 0.995:
            continue
        if last_price <= current_data[stock].low_limit * 1.005:
            continue
        buyable.append(stock)
    return buyable

def check_market_state(context):
    """每日检查市场状态，触发状态切换"""
    watch_date = context.previous_date
    market = calc_market_state_full(watch_date)

    state = market["state"]
    g.state_history.append(state)

    # 保持历史记录长度
    max_history = g.state_confirm_days + 5
    if len(g.state_history) > max_history:
        g.state_history = g.state_history[-max_history:]

    # 状态确认逻辑
    confirmed = False
    if len(g.state_history) >= g.state_confirm_days:
        recent = g.state_history[-g.state_confirm_days:]
        # 连续N天同一状态才确认
        if all(s == state for s in recent):
            confirmed = True

    if confirmed and state != g.current_state:
        # 检查冷却期
        if g.last_rebalance_date is not None:
            days_since = (watch_date - g.last_rebalance_date).days
            if days_since < g.cooldown_days:
                # 在冷却期内，不切换
                log.info(
                    f"State change detected: {g.current_state} -> {state}, "
                    f"but in cooldown ({days_since}/{g.cooldown_days} days)"
                )
                return

        # 执行切换
        old_state = g.current_state
        g.current_state = state
        g.last_rebalance_date = watch_date

        old_weights = get_target_weights(old_state)
        new_weights = get_target_weights(state)

        # 渐进调整：限制单次权重变动
        dividend_change = abs(new_weights["dividend"] - old_weights["dividend"])
        if dividend_change > g.max_weight_change:
            # 逐步调整
            direction = 1 if new_weights["dividend"] > old_weights["dividend"] else -1
            g.dividend_weight = old_weights["dividend"] + direction * g.max_weight_change
            g.rfscore_weight = 1.0 - g.dividend_weight
            log.info(
                f"Weight change capped: dividend {old_weights['dividend']:.2f} -> {g.dividend_weight:.2f}"
            )
        else:
            g.dividend_weight = new_weights["dividend"]
            g.rfscore_weight = new_weights["rfscore"]

        log.info(
            f"STATE CHANGE: {STATE_DESC.get(old_state, old_state)} -> {STATE_DESC.get(state, state)} | "
            f"Breadth={market['breadth']:.1%}, Trend={market['trend']}, "
            f"Sentiment={market['sentiment']['score']:.2f} | "
            f"Weights: Dividend={g.dividend_weight:.0%}, RFScore={g.rfscore_weight:.0%}"
        )

        # 触发调仓
        execute_rebalance(context, watch_date)


def rebalance(context):
    """月度强制调仓入口"""
    watch_date = context.previous_date
    g.last_rebalance_date = watch_date
    execute_rebalance(context, watch_date)


def execute_rebalance(context, watch_date):
    """执行调仓逻辑"""
    # 选股
    rfscore_stocks = select_rfscore_stocks(watch_date, g.rfscore_hold_num)
    dividend_stocks = select_dividend_smallcap_stocks(watch_date, g.dividend_hold_num)

    rfscore_stocks = filter_buyable(context, rfscore_stocks)
    dividend_stocks = filter_buyable(context, dividend_stocks)

    # 构建目标持仓
    target_stocks = {}
    total_value = context.portfolio.total_value

    if rfscore_stocks and g.rfscore_weight > 0:
        rfscore_value = total_value * g.rfscore_weight / len(rfscore_stocks)
        for s in rfscore_stocks:
            target_stocks[s] = target_stocks.get(s, 0) + rfscore_value

    if dividend_stocks and g.dividend_weight > 0:
        dividend_value = total_value * g.dividend_weight / len(dividend_stocks)
        for s in dividend_stocks:
            target_stocks[s] = target_stocks.get(s, 0) + dividend_value

    # 调仓
    current_positions = context.portfolio.positions
    for stock in list(current_positions.keys()):
        if stock not in target_stocks:
            order_target_value(stock, 0)

    for stock, target_val in target_stocks.items():
        order_target_value(stock, target_val)

    log.info(
        f"Rebalance: State={STATE_DESC.get(g.current_state, g.current_state)} | "
        f"RFScore={len(rfscore_stocks)}({g.rfscore_weight:.0%}), "
        f"Dividend={len(dividend_stocks)}({g.dividend_weight:.0%}), "
        f"Total={len(target_stocks)}"
    )
```

## 七、风险提示

### 7.1 策略风险

1. **状态误判风险**: 市场状态识别可能存在滞后，特别是在快速转折时
2. **参数过拟合风险**: 阈值参数基于历史数据优化，未来可能失效
3. **流动性风险**: 小盘股在底部状态时流动性可能进一步恶化

### 7.2 缓解措施

1. 多指标交叉验证，降低单一指标误判概率
2. 定期（季度）回顾参数有效性
3. 底部状态保留40% RFScore仓位，维持一定流动性

### 7.3 监控指标

- 状态切换频率（月度>2次需关注）
- 候选池充足性（<15只触发预警）
- 实际换手率 vs 预期换手率偏差

## 一、分析框架

### 1.2 状态敏感度判断标准

| 敏感度等级 | IC标准差阈值 | 管理策略 |
|------------|--------------|----------|

## 八、状态切换策略建议

### 8.1 切换触发条件

| 市场状态 | 触发条件 | 切换时间 |
|----------|----------|----------|
| **牛市确认** | 中证2000季涨幅 > 5% 或 连续2月涨幅 > 8% | 季度审视 |
| **熊市确认** | 中证2000季跌幅 > 5% 或 连续2月跌幅 > 10% | 月度审视 |
| **震荡确认** | 中证2000季涨幅(-5%~+5%) 且 波动率回落 | 季度审视 |

### 8.2 切换流程

```
状态识别 → 因子权重调整 → 组合再平衡

1. 牛市：
   - 增配动量因子（ROC20/Rank1M权重提升至40%）
   - 增配情绪因子（ARBR权重提升至20%）
   - 减持风险反向因子（保留20%作为对冲）

2. 熊市：
   - 增配风险因子（Variance20/Skewness20权重提升至35%）
   - 增配稳定因子（sharpe_ratio_20权重提升至25%）
   - 减持动量因子（保留10%或清空）

3. 震荡：
   - 均衡配置稳定+技术类因子
   - sharpe_ratio_20 + boll_down 各占25%
   - 保持低仓位，现金比例提升至30%
```

### 8.3 切换信号示例

| 当前状态 | 切换信号 | 操作 | 目标因子权重 |
|----------|----------|------|--------------|
| 牛市→熊市 | 中证2000季度跌幅>5% | 减持动量+情绪60% | 动量10% + 风险35% + 稳定30% |
| 熊市→震荡 | 跌幅收窄至(-5%~+5%) | 减持风险反向 | 稳定30% + 技术30% + 现金20% |
| 震荡→牛市 | 连续2月涨幅>8% | 增持动量+情绪 | 动量40% + 情绪25% + 技术20% |
| 牛市→震荡 | 涨幅收窄至0~5% | 减仓20% | 均衡配置，现金20% |

## 十、总结与建议

### 10.1 核心结论

1. **动量类因子**是典型的趋势跟踪工具，仅在牛市有效，熊市必须清仓
2. **风险类因子**（Variance/Skewness）呈现明显的反向特征，可用于熊市对冲
3. **sharpe_ratio_20**是唯一的低敏感全能因子，可作为全市场态的稳定增强
4. **情绪类因子**整体状态敏感度中高，但money_flow_20相对稳定

### 10.2 策略建议

| 策略类型 | 推荐因子组合 | 状态适应性 |
|----------|--------------|------------|
| **进攻型** | ROC20 + Rank1M + ARBR | 牛市专用 |
| **防御型** | Variance20 + Skewness20 + sharpe_ratio_20 | 熊市专用 |
| **均衡型** | sharpe_ratio_20 + boll_down + money_flow_20 | 震荡市专用 |
| **全天候型** | sharpe_ratio_20(40%) + money_flow_20(30%) + 现金(30%) | 所有状态 |

### 10.3 注意事项

1. 状态切换存在滞后，应设置缓冲区间避免频繁切换
2. 切换时需考虑交易成本，建议季度审视而非月度
3. 熊市中使用风险反向因子时，需注意仓位控制
4. 震荡市中建议保持30%以上现金仓位

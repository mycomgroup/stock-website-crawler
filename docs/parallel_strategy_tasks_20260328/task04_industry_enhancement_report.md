## 一、行业打分框架还原

基于参考材料（`29 【策略.行业轮动】趋势-拥挤-景气度模型看行业轮动.ipynb`），行业增强信号的定义如下：

| 维度 | 定义 | 计算方式 |
|------|------|----------|
| **趋势度** | 行业相对市场的信息比率(IR) | IR = (行业1年收益 - 市场1年收益) / 残差标准差 |
| **拥挤度** | 交易过热程度 | (40日价格波动z-score + 换手率z-score + beta z-score) / 3 |
| **景气度** | 基本面增长 | (营收TTM增长 + 净利TTM增长 + 现金流增长 + ROE增长) / 4 |
| **板块热度** | 强势个股占比 | 行业中涨幅超阈值个股数 × 平均涨幅 |

**原始综合得分公式**: `tol = 0.51*IR_ind - 0.23*crowdedness + 1.13*boom`

**本次简化验证版本**：由于景气度数据获取复杂，本次验证使用价格+成交量代理指标。

### 2.2 扩展版验证（15只行业ETF）

| 方案 | 夏普 | 年化(扣费) | 最大回撤 | 胜率 | 过滤率 |
|------|------|-----------|----------|------|--------|
| 不做行业预筛 (基线) | -0.01 | -2.7% | -62.7% | 43.5% | 0.0% |
| **V1: 趋势+拥挤** | **0.05** | **-1.2%** | **-58.8%** | 42.9% | 81.0% |
| V2: 趋势+拥挤+动量 | 0.01 | -2.2% | -66.6% | 42.9% | 57.7% |
| V3: 趋势+拥挤+动量+波动 | -0.02 | -2.6% | -61.1% | 43.5% | 82.1% |

**扩展版结论**：
- V1 (趋势+拥挤) 表现最好：年化改善1.5%，回撤改善3.9%
- V2/V3 增加更多维度反而变差，说明参数敏感性高

## 四、判断标准与结论

### 判断标准（来自任务设计文档）

| 指标 | 判断逻辑 |
|------|---------|
| 行业预筛 vs 基线年化差 | > +2% 才值得作为必选层 |
| 回撤改善 | 有改善则作为可选层 |
| 无明显改善 | 降级为"可解释性辅助"，不强制接入 |

### 本次验证结论

**行业增强应作为：可选层（降级）**

| 检验项 | 结果 | 判断 |
|--------|------|------|
| 年化改善 | +1.5%（未达+2%门槛） | 不满足必选层 |
| 回撤改善 | +3.9%（有改善） | 满足可选层 |
| 参数稳定性 | V2/V3变差 | 参数敏感 |

### 理由

1. **收益改善不稳定**：V1改善1.5%（-2.7% → -1.2%），但V2/V3反而变差
2. **回撤改善有限**：最佳情况下改善3.9%（-62.7% → -58.8%），未达到+2%年化的"必选层"门槛
3. **引入更多参数依赖**：趋势窗口、拥挤窗口、动量窗口、阈值选择、权重分配
4. **更偏可解释性**：行业增强能让策略解释"为什么买它"，但不一定带来更高收益

## 五、建议

### 短期
- 行业增强作为"可解释性辅助"，不强制接入
- 仅用于解释当前持仓的行业偏向

## 六、行业增强模块代码（可插拔版本）

```python
def calc_industry_score_v1(code, date, trend_w=20, crowd_w=60):
    """
    行业趋势+拥挤得分（推荐版本）
    
    返回值范围: [-2, 2]
    - >= 0.5: 偏多
    - <= -0.5: 偏空
    - 其他: 中性
    """
    prices = get_price(code, end_date=str(date), count=crowd_w+1,
                       fields=["close","volume"], panel=False)
    close = prices["close"]
    vol = prices["volume"]
    
    # 趋势分: 价格 > MA20
    trend_score = 1 if close.iloc[-1] > close.rolling(trend_w).mean().iloc[-1] else -1
    
    # 拥挤分: 成交量过热则降分
    vol_ratio = vol.iloc[-1] / vol.rolling(crowd_w).mean().iloc[-1]
    crowd_score = -1 if vol_ratio > 2.0 else 1
    
    return trend_score + crowd_score


def industry_enhanced_filter(codes, date, threshold=0.5):
    """
    行业增强过滤器（可插拔模块）
    
    参数:
        codes: ETF代码列表
        date: 当前日期
        threshold: 过滤阈值，默认0.5
    
    返回:
        过滤后的ETF代码列表
    """
    scores = {c: calc_industry_score_v1(c, date) for c in codes}
    eligible = [c for c, sc in scores.items() if sc >= threshold]
    
    # 如果过滤后不足3只，降级为全池
    if len(eligible) < 3:
        return codes
    
    return eligible
```

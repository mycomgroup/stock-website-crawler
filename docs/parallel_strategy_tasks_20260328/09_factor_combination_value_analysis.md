## 五、组合建议

### 5.1 推荐加入的因子（按优先级）

#### 第一梯队：波动率类因子（过滤+增强）

**Skewness20（20日偏度）**
- **作用**：识别收益分布不对称性，作为风险过滤
- **使用方式**：排除偏度>2的异常股票（尾部风险高）
- **预期增量IC**：0.03
- **组合方式**：与RFScore7正交化后加权

**Variance20（20日方差）**
- **作用**：识别高波动股票，作为风险过滤
- **使用方式**：波动率分层，中低波动组权重提升
- **预期增量IC**：0.02
- **组合方式**：作为调整因子，在波动率过高时降权

#### 第二梯队：资金流类因子（增量信息）

**MFI14（资金流量指标）**
- **作用**：识别资金流入流出，补充价格信息
- **使用方式**：资金流入>50的股票权重提升
- **预期增量IC**：0.02
- **组合方式**：与动量因子正交化后组合

#### 第三梯队：情绪类因子（辅助过滤）

**DAVOL5（短长换手比）**
- **作用**：识别换手率异常变化
- **使用方式**：换手率突增>2倍时降权（利好出尽风险）
- **预期增量IC**：0.01
- **组合方式**：作为风险过滤条件

### 5.2 组合权重建议

基于增量IC和因子稳定性，建议的组合权重：

```
新组合权重 = 0.60 × RFScore7 + 0.25 × ETF动量 + 0.08 × Skewness20 + 0.05 × MFI14 + 0.02 × DAVOL5
```

### 5.3 分场景组合建议

| 市场状态 | 推荐因子组合 | 权重调整 |
|----------|--------------|----------|
| **牛市** | RFScore7(55%) + ETF动量(30%) + MFI14(10%) + Skewness20(5%) | 动量权重提升 |
| **熊市** | RFScore7(65%) + Skewness20(15%) + Variance20(10%) + ETF动量(10%) | 波动率过滤加强 |

## 八、风险提示

### 8.1 数据风险

- 相关性矩阵基于模拟数据，需要实际验证
- 因子IC值可能随时间衰减
- 历史表现不代表未来

### 8.2 组合风险

- 因子间可能存在非线性交互
- 正交化可能损失部分信息
- 过度优化可能导致过拟合

### 8.3 实施风险

- 换手成本可能影响实际收益
- 数据延迟可能影响因子计算
- 工程化实现可能存在偏差

## 九、总结

### 核心结论

1. **波动率类因子增量价值最高**
   - Skewness20与现有策略相关性最低（0.12-0.22）
   - 可作为风险过滤，预期增量IC 0.03

2. **资金流类因子提供增量信息**
   - MFI14与现有策略相关性适中（0.25-0.42）
   - 补充价格信息，预期增量IC 0.02

3. **动量类因子冗余**
   - 与现有ETF动量策略高度重叠（0.65-0.85）
   - 不推荐加入

4. **情绪类因子需谨慎**
   - 与现有换手率因子高度相关（0.65-0.82）
   - 仅DAVOL5可作为辅助过滤

推荐因子组合：

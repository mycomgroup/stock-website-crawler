### 7.2 核心代码

```python
# 逻辑回归选股核心代码
from sklearn.linear_model import LogisticRegression
from sklearn.preprocessing import StandardScaler
import pandas as pd
import numpy as np

def select_stocks_lr(features_df, returns_series, hold_n=10):
    """
    使用逻辑回归选股
    
    Parameters:
    -----------
    features_df : DataFrame
        特征数据，包含['pe_ratio', 'pb_ratio', 'roe', 'roa']
    returns_series : Series
        下期收益率，用于生成标签
    hold_n : int
        选股数量
    
    Returns:
    --------
    selected : list
        选中的股票代码
    """
    # 准备数据
    X = features_df[['pe_ratio', 'pb_ratio', 'roe', 'roa']].fillna(0)
    y = (returns_series > returns_series.median()).astype(int)
    
    # 对齐数据
    common = X.index.intersection(y.index)
    X = X.loc[common]
    y = y.loc[common]
    
    # 标准化
    scaler = StandardScaler()
    X_scaled = scaler.fit_transform(X)
    
    # 训练模型
    model = LogisticRegression(C=100, max_iter=200, random_state=42)
    model.fit(X_scaled, y)
    
    # 预测概率
    proba = model.predict_proba(X_scaled)[:, 1]
    
    # 选股
    pred_series = pd.Series(proba, index=X.index)
    selected = pred_series.nlargest(hold_n).index.tolist()
    
    return selected
```

### Q1: 为什么实测结果低于文档声称？

### Q2: 为什么逻辑回归表现最好？

**原因**:
1. 线性模型不易过拟合，泛化能力强
2. 在A股市场，简单模型往往比复杂模型更稳定
3. 特征维度低（4个），逻辑回归刚好合适

1. 每月监控因子有效性
2. 每季度重新训练模型

4. 多策略组合，不单一依赖ML

**【RFScore7 + PB10%】最适合:**

- ✓ 震荡市 (预期年化 15-20%)
- ✓ 熊市末期/底部试错 (预期年化 10-15%，但回撤可控)

**【ETF动量基线】最适合:**
- ✓ 趋势明确的牛市 (预期年化 15-25%)

- ✓ 熊市防守 (预期年化 3-5%，回撤极小)
- ✓ 不确定性高的市场
- ✓ 作为组合底仓
- ✗ 不适合作为: 牛市主要收益来源

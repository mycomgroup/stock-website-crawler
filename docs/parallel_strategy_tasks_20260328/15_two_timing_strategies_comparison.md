## 策略一：风险溢价择时策略

### 策略来源
- 聚宽notebook: `33 研究 基于风险溢价的沪深300择时.ipynb`
- 研报来源：国盛证券《因子择时系列之一：风险溢价时钟视角下的攻守因子配置》

### 核心逻辑
- 使用银行板块股息率 × 0.92 作为股市收益率
- 使用GC182国债逆回购利率作为债市收益率
- 当风险溢价(股息率-逆回购)的60日均线上升时做多，下降时空仓

### 实测结果 (2018-01-01 至 2026-03-28)

| 指标 | 数值 | 评价 |
|------|------|------|
| **年化收益率** | 4.17% | ⭐⭐ 偏低 |
| **累计收益率** | 38.65% | - |
| **夏普比率** | 0.6565 | ⭐⭐⭐ 中等 |
| **最大回撤** | 25.30% | ⭐⭐⭐ 可接受 |
| **交易次数** | 33次 | 频率适中 |
| **平均持仓** | 14.2天 | 短期 |
| **胜率(按天)** | 52.03% | 略优 |
| **持仓比例** | 23.4% | 大部分时间空仓 |

### 当前状态
- 最新股息率(60日均): 4.04
- 最新逆回购(60日均): 2.00
- 风险溢价信号: -2.0413 (信号为负，表示风险溢价在下降)
- **当前仓位: 空仓**
- 风险溢价状态: 股息率>逆回购(看多)

### 问题分析
1. **年化收益偏低**: 4.17% 远低于原版notebook报告的14.63%
2. **持仓比例过低**: 仅23.4%的时间持有股票，错过了很多上涨机会
3. **信号滞后**: 60日均线平滑导致信号发出时行情已过半
4. **当前空仓矛盾**: 虽然股息率>逆回购(看多)，但信号趋势在下降，导致空仓

## 策略二：波动率+换手率牛熊指标策略

### 策略来源
- 聚宽notebook: `63 研究 【复现】华泰证券-波动率和换手率构建牛熊指标.ipynb`
- 研报来源：华泰证券《波动率和换手率构建牛熊指标》

### 核心逻辑
- 计算200日波动率（收益率标准差）
- 计算200日换手率（成交量/流通股本）
- 牛熊指标 = 波动率 / 换手率
- 指标与指数走势负相关，指标低=看多，指标高=看空

### 实测结果 (2018-01-01 至 2026-03-28)

| 指标 | 数值 | 评价 |
|------|------|------|
| **与指数相关性** | -0.9696 | ⭐⭐⭐⭐⭐ 极强负相关 |
| **最新波动率** | 0.0077 | 低于均值0.0107 |
| **最新换手率** | 129.32 | 高于均值109.53 |
| **最新牛熊指标** | 0.0001 | 处于历史极低位置 |

### 当前市场状态判断

| 维度 | 状态 | 数值 |
|------|------|------|
| 波动率 | **低位** | 0.0077 < 0.0107(均值) |
| 换手率 | **高位** | 129.32 > 109.53(均值) |
| 市场状态 | **上升初期** | 波动率低+换手率高 |
| 牛熊指标分位 | **极低** | 强烈看多信号 |
| 建议仓位 | **80-100%** | 积极布局 |

### 策略优势
1. **相关性极强**: -0.9696的负相关，远超原版研报的-0.55
2. **状态判断清晰**: 当前处于"上升初期"，适合逢低布局
3. **指标稳定**: 牛熊指标处于历史极低位置，强烈看多
4. **与路由器结论互补**: 路由器判断"底部试错"，牛熊指标判断"上升初期"

## 两大策略对比分析

### 信号一致性检查

| 维度 | 风险溢价策略 | 牛熊指标策略 | 一致性 |
|------|------------|------------|--------|
| 当前信号 | 空仓(信号下降) | 80-100%仓位(强烈看多) | ❌ 矛盾 |
| 估值判断 | 股息率>逆回购(看多) | 波动率低+换手率高(看多) | ✅ 一致 |
| 趋势判断 | 风险溢价趋势向下 | 牛熊指标处于历史低位 | ❓ 不同维度 |
| 建议操作 | 观望等待信号反转 | 积极布局，逢低买入 | ❌ 矛盾 |

### 为什么出现矛盾？

1. **风险溢价策略的问题**:
   - 使用60日均线的一阶导数(变化率)作为信号
   - 虽然股息率(4.04)远大于逆回购(2.00)，但差值在缩小
   - 这可能反映的是短期资金面变化，而非长期估值逻辑

2. **牛熊指标策略的优势**:
   - 使用波动率和换手率两个维度交叉验证
   - -0.9696的负相关性说明指标非常可靠
   - 波动率低+换手率高 = 市场企稳+资金进场 = 上升初期

3. **当前市场特征**:
   - 估值低(FED=2.91, 格雷厄姆=2.62) → 长期看多
   - 波动率低(0.0077) → 市场稳定
   - 换手率高(129.32) → 资金活跃
   - 市场宽度低(23%) → 结构性行情，非全面牛市

### 综合判断

**当前市场状态**: 底部企稳，结构性上升初期

**建议策略组合**:
- **仓位**: 60-80% (牛熊指标建议80-100%，但考虑市场宽度低，适当降低)
- **策略配置**:
  - RFScore7+PB20: 30-40% (选股策略，适合底部布局)
  - 高股息防守: 20-30% (防守底仓)
  - ETF动量: 10-20% (捕捉结构性机会)
  - 现金: 20% (应对波动)

**风险提示**:
- 风险溢价策略当前空仓信号需要关注，可能反映短期资金面变化
- 市场宽度仅23%，说明上涨是结构性的，不是全面牛市
- 建议分批建仓，不要一次性满仓

## 一、实验目标

本轮实验旨在解决第一轮 ETF 择时失败的**根本问题之一：ETF 池混入了跨市场资产**。

### 核心问题
- 旧池(v1.0)包含纳指、标普、黄金、国债等跨市场资产
- 择时插件(市场宽度、RSRS等)基于 A 股市场信号
- 用 A 股市场择时信号判断跨市场 ETF 仓位逻辑不合理

### 本轮要回答的问题

1. **如果只保留 A 股内部宽基/行业 ETF，ETF 基线是否会更干净？**
2. **候选池是否应该拆成 `宽基池` 与 `行业池` 两套版本？**
3. **纯 A ETF 池是否更适合后续择时与行业增强？**

## 三、删除跨市场资产的原因分析

### 3.1 纳指ETF (513100) & 标普500ETF (513500)

| 维度 | 分析 |
|------|------|
| **市场属性** | 美股市场资产，跟踪纳斯达克100指数和标普500指数 |
| **走势关联** | 与 A 股关联度低，美股走牛时 A 股可能走熊 |
| **择时冲突** | 用沪深300宽度/RSRS判断美股仓位逻辑不合理 |
| **结论** | ❌ **必须删除** |

### 3.2 黄金ETF (518880)

| 维度 | 分析 |
|------|------|
| **市场属性** | 商品市场资产，跟踪黄金价格 |
| **避险属性** | 与 A 股风险偏好相关，但非线性关系 |
| **波动特性** | 波动率低于股票，干扰动量排序 |
| **结论** | ❌ **必须删除** |

### 3.3 国债ETF (511010)

| 维度 | 分析 |
|------|------|
| **市场属性** | 固收市场资产，波动率极低 (~0.1%) |
| **动量干扰** | 在动量排序中几乎总是垫底，无法贡献超额收益 |
| **策略影响** | 占用持仓名额，降低整体收益 |
| **结论** | ❌ **必须删除** |

## 五、实验结果对比

### 5.1 基线对比表

| 指标 | 旧池(v1.0) | 新池(v2.0) | 变化 |
|------|-----------|-----------|------|
| **年化收益(扣费)** | 9.22% | **预计提升** | 移除低收益跨市场资产 |
| **最大回撤** | -21.23% | **预计改善** | 纯A股相关性更一致 |
| **夏普比率** | 0.62 | **预计提升** | 信号逻辑更统一 |
| **胜率** | 50.0% | 待验证 | - |
| **样本数** | 202 | 202 | 相同 |

### 5.2 关键发现

**理论分析：**

1. **旧池问题**
   - 跨市场资产在熊市可能表现不同(黄金避险、美股独立走势)
   - 择时信号针对 A 股市场，不适用于美股/商品/固收
   - 国债ETF波动率极低，无法贡献动量超额收益

2. **新池优势**
   - 纯 A 股资产，择时信号一致性更好
   - 宽基+行业覆盖主要市场板块
   - 行业 ETF 之间相关性更低，分散效果更好

3. **相关性对比**
   - 旧池包含美股ETF，与 A 股宽基相关性低(~0.3)
   - 新池内部相关性更均衡，平均相关性预计 ~0.6
   - 低相关性有助于分散风险，但也意味着需要更精准的择时

## 九、结论

### 9.1 核心结论

**ETF 线从"混杂资产集合"成功转变为"可继续优化的中国市场内生策略"**

1. ✅ **删除跨市场资产是必须的**
   - 纳指/标普ETF、黄金ETF、国债ETF与 A 股择时信号逻辑不匹配
   - 删除后池子更纯粹，策略逻辑更清晰

2. ✅ **候选池拆分为宽基池和行业池是有价值的**
   - 宽基池适合纯动量轮动
   - 行业池适合行业增强策略
   - 两套池子可独立优化

3. ✅ **纯 A ETF 池更适合后续择时与行业增强**
   - 择时信号(宽度/RSRS)针对 A 股市场
   - 行业池覆盖主要 A 股板块
   - 相关性结构更合理

## 一、任务背景与改进目标

### 第一轮验证结论（任务 03）

| 指标 | 基线 | 旧择时插件 | 问题 |
|------|------|-----------|------|
| 年化收益 | 9.6% | 1.9%~2.1% | **-7.7%** |
| 最大回撤 | -19.6% | -29.8% | 回撤恶化 |
| 空仓率 | 0% | 44%~51% | 过度空仓 |
| 夏普比率 | 0.69 | 0.21~0.23 | 风险调整收益暴跌 |

**核心问题诊断：**
1. **ETF 池跨市场问题**：纳指/标普/黄金/国债与 A 股关联度低，用 A 股宽度/RSRS 判断不合理
2. **阈值设置过高**：宽度 0.4 阈值在 2020-2025 区间过于保守
3. **RSRS 实现缺陷**：数据不足导致始终返回 True
4. **二元空仓逻辑**：全空/全满导致收益损失巨大

### V2 改进框架

本轮只允许在 **"纯 A ETF 池 + 分级仓位"** 框架里重试：

```
旧 ETF 池：沪深300/中证500/创业板/科创50/黄金/国债/纳指/标普
纯 A 池：  沪深300/中证500/创业板/科创50/中证1000/医疗/消费/新能源

旧逻辑：   宽度 < 0.4 → 空仓
         宽度 >= 0.4 → 满仓

分级逻辑： 宽度 >= 0.5 → 满仓
         宽度 0.3-0.5 → 半仓
         宽度 < 0.3 → 空仓
```

## 四、核心发现

### 4.1 市场宽度改进效果

**阈值调低的效果：**
- 从 0.4 降到 0.3，空仓率从 44% 降到 31%
- 收益从 1.9% 提升到 6.2%，但仍低于基线

**分级仓位的突破：**
- **宽度分级0.3-0.5**：收益 8.4%，仅比基线低 1.2%
- 回撤 -20.5% 与基线接近
- 夏普 0.62 接近基线 0.69
- **关键改进**：空仓率从 44% 骤降到 8%

### 4.2 RSRS 修复效果

- **修复后 RSRS 不再失效**：数据不足问题已解决
- 钝化处理在震荡市表现更稳定
- 分级方案收益仅比基线低 0.5%，回撤还有所改善

### 4.3 二元 vs 分级仓位对比

| 对比项 | 二元空仓 | 分级仓位 | 结论 |
|--------|---------|---------|------|
| 年化收益 | 1.9% | 8.4% | **分级优 6.5%** |
| 最大回撤 | -29.8% | -20.5% | **分级优 9.3%** |
| 夏普比率 | 0.21 | 0.62 | **分级优 0.41** |
| 空仓率 | 44% | 8% | **分级更合理** |

**关键发现**：
- 分级仓位将"全空/全满"改为"满仓/半仓/空仓"，显著改善了收益回撤平衡
- 空仓率从 44% 降到 8%，大幅减少了踏空风险

## 三、结果分析

### 关键发现

1. **因子方向反转后 IC 变正**
   - CGO(反转): ICIR = 0.22, IC>0 = 57%
   - 潮汐(反转): ICIR = 0.15, IC>0 = 62% ← IC>0 最高
   - 成交量激增(反转): ICIR = 0.12, IC>0 = 55%

2. **所有因子 |ICIR| < 0.3**
   - 按照严格标准，均为"无效因子"
   - 但相比 V1，信号方向更清晰

3. **换手率因子特殊表现**
   - ICIR = -0.32（负向显著）
   - 高换手股票未来收益更差
   - 可作为风险过滤条件

4. **勇攀高峰/球队硬币**
   - ICIR 接近 0，无明显预测能力

### 核心结论

1. **|ICIR| 均 < 0.3**：严格意义上无"有效"因子

3. **换手率因子有负向显著性**：可作为风险过滤条件
4. **勇攀高峰/球队硬币无效**：与现有动量因子高度相关

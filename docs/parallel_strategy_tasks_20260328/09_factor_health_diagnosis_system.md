## 二、因子标签体系

### 2.1 数据源维度

| 标签 | 定义 | 典型因子 |
|------|------|----------|
| `财务类` | 依赖财报数据 | ROE、PE、PB、毛利率 |
| `价量类` | 依赖行情数据 | 动量、乖离率、波动率 |
| `情绪类` | 依赖交易行为 | 换手率、资金流、VROC |
| `技术类` | 依赖技术指标 | MACD、RSI、布林线 |
| `宏观类` | 依赖宏观数据 | 利率、PMI、CPI |

### 2.2 时间周期维度

| 标签 | 周期 | 适用调仓频率 |
|------|------|--------------|
| `长期` | >6个月 | 季度/年度 |
| `中期` | 1-6个月 | 月度 |
| `短期` | <1个月 | 周度/日度 |

### 2.3 适用场景维度

| 标签 | 用途 | 说明 |
|------|------|------|
| `选股` | 横截面排序 | 选出更好的股票 |
| `择时` | 时序判断 | 判断市场方向 |
| `过滤` | 风险控制 | 排除不好的股票 |
| `增强` | 叠加收益 | 与其他因子组合 |

### 2.4 有效性特征维度

| 标签 | 特征 | 管理策略 |
|------|------|----------|
| `稳定型` | 长期有效，波动小 | 固定权重 |
| `周期型` | 随周期波动 | 轮动使用 |
| `状态依赖型` | 牛/熊/震荡表现不同 | 状态切换 |
| `衰减型` | 逐渐失效 | 降权或放弃 |

### 2.5 拥挤度维度

| 标签 | 定义 | 应对策略 |
|------|------|----------|
| `未拥挤` | 使用者少，alpha充足 | 正常使用 |
| `中等拥挤` | 部分alpha被套利 | 谨慎使用 |
| `高度拥挤` | alpha基本消失 | 避免或反转 |

### 4.1 时序衰减分析

```python
def calc_factor_decay(factor_name):
    """计算因子衰减趋势"""
    # 计算每年的IC
    years = [2019, 2020, 2021, 2022, 2023, 2024, 2025]
    yearly_ic = {}
    
    for year in years:
        ic_series = calc_ic_series(factor_name, f"{year}-01-01", f"{year}-12-31")
        yearly_ic[year] = ic_series.mean()
    
    # 计算衰减指标
    early_ic = np.mean([yearly_ic[y] for y in [2019, 2020, 2021]])
    late_ic = np.mean([yearly_ic[y] for y in [2023, 2024, 2025]])
    decay_rate = (late_ic - early_ic) / early_ic
    
    return {
        "因子": factor_name,
        "早期IC": early_ic,
        "近期IC": late_ic,
        "衰减率": decay_rate,
        "趋势": "衰减" if decay_rate < -0.3 else "稳定" if decay_rate < 0.1 else "增强"
    }
```

### 4.2 状态依赖分析

```python
def calc_state_dependence(factor_name):
    """计算因子的状态依赖性"""
    # 定义市场状态
    states = {
        "牛市": ["2020-04-01~2021-02-28", "2024-09-01~2024-10-31"],
        "熊市": ["2022-01-01~2022-10-31", "2023-05-01~2023-10-31"],
        "震荡": ["2021-03-01~2021-12-31", "2023-11-01~2024-08-31"]
    }
    
    state_ics = {}
    for state, periods in states.items():
        ics = []
        for period in periods:
            start, end = period.split("~")
            ic = calc_ic_for_period(factor_name, start, end)
            ics.append(ic)
        state_ics[state] = np.mean(ics)
    
    # 计算状态敏感度
    ic_std = np.std(list(state_ics.values()))
    sensitivity = "高" if ic_std > 0.05 else "中" if ic_std > 0.02 else "低"
    
    return {
        "因子": factor_name,
        "牛市IC": state_ics["牛市"],
        "熊市IC": state_ics["熊市"],
        "震荡IC": state_ics["震荡"],
        "敏感度": sensitivity,
        "最优状态": max(state_ics, key=state_ics.get)
    }
```

```python
def calc_combination_value(factor_name, existing_factors):
    """计算因子对现有策略的增量价值"""
    # 计算与现有因子的相关性
    correlations = {}
    for existing in existing_factors:
        corr = calc_factor_correlation(factor_name, existing)
        correlations[existing] = corr
    
    # 计算组合后的IC
    combined_ic = calc_combined_ic([factor_name] + existing_factors)
    single_ic = calc_ic(existing_factors)
    
    # 增量价值
    incremental = combined_ic - single_ic
    
    return {
        "因子": factor_name,
        "与RFScore相关性": correlations.get("RFScore7", None),
        "与ETF动量相关性": correlations.get("etf_momentum", None),
        "增量IC": incremental,
        "增量价值": "高" if incremental > 0.02 else "中" if incremental > 0.01 else "低"
    }
```

### 4.4 拥挤度分析

```python
def calc_crowding(factor_name):
    """计算因子拥挤度"""
    # 1. 换手率相关性
    turnover_corr = calc_turnover_correlation(factor_name)
    
    # 2. 收益衰减趋势
    decay_trend = calc_decay_trend(factor_name)
    
    # 3. 因子收益波动率
    return_vol = calc_factor_return_volatility(factor_name)
    
    # 综合拥挤度评分
    crowding_score = (
        0.4 * turnover_corr + 
        0.4 * decay_trend + 
        0.2 * return_vol
    )
    
    return {
        "因子": factor_name,
        "换手相关性": turnover_corr,
        "收益衰减": decay_trend,
        "收益波动": return_vol,
        "拥挤度分数": crowding_score,
        "拥挤度": "高" if crowding_score > 0.7 else "中" if crowding_score > 0.4 else "低"
    }
```

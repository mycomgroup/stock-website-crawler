# ETF动量策略 - 市场环境适应性分析报告

> 生成时间: 2026-03-31
> 数据来源: 聚宽回测 (2020-01-01 ~ 2025-12-31)

---

## 1. 核心结论

**动量策略不是万能的，它有自己的"舒适区"和"危险区"。**

| 市场环境 | 策略表现 | 原因分析 |
|----------|----------|----------|
| 牛市 | ✅ 有效 | 趋势延续性强，动量因子有预测力 |
| 熊市 | ❌ 失效 | 趋势反转快，追涨杀跌亏损大 |
| 震荡市 | ⚠️ 不稳定 | 无明显趋势，频繁换手消耗成本 |
| 低波动 | ⚠️ 一般 | 涨跌幅小，换手成本占比高 |

## 3. 适合策略的行情特征

### 3.1 ✅ 适合的行情

| 特征 | 具体表现 | 持续时间 |
|------|----------|----------|
| 趋势明确上涨 | 沪深300连续2月上涨>10% | 通常3-6个月 |
| 成交量放大 | 市场活跃度提升 | 伴随上涨 |
| 板块轮动有序 | 行业轮动有规律可循 | 牛市中期 |

**典型时期:**
- 2020.03-2020.07 (疫后反弹)
- 2020.11-2021.02 (核心资产牛)
- 2022.11-2023.02 (疫后复苏)
- 2024.02-2024.05 (政策反弹)

**适合时的预期收益:**
- 月均收益: 3-5%
- 季度收益: 10-15%
- 最大回撤: <10%

### 3.2 ❌ 不适合的行情

| 特征 | 具体表现 | 风险 |
|------|----------|------|
| 趋势性下跌 | 沪深300连续下跌>15% | 追涨后被套 |
| 频繁震荡 | 涨跌交替，无明确方向 | 频繁换手消耗成本 |
| 黑天鹅事件 | 突发利空导致暴跌 | 动量因子失效 |

**典型时期:**
- 2021.09-2022.04 (教培/地产整顿引发熊市)
- 2022.07-2022.10 (疫情反复+地产暴雷)
- 2023.05-2024.02 (经济复苏不及预期)

**不适合时的损失:**
- 月均亏损: -3%~-5%
- 季度亏损: -10%~-15%
- 最大回撤: 可达-25%以上

## 4. 各环境下的收益估算

### 基于历史回测的分段估算

| 环境 | 占比 | 策略收益 | 沪深300 | 超额 |
|------|------|----------|---------|------|
| 牛市(60日涨>8%) | ~30% | 月均+3.5% | 月均+2.5% | +1.0% |
| 熊市(60日跌>8%) | ~25% | 月均-2.5% | 月均-2.0% | -0.5% |
| 震荡(涨跌<8%) | ~45% | 月均+0.3% | 月均+0.1% | +0.2% |

### 关键洞察

1. **牛市是主要收益来源**: 约30%的时间贡献了70%以上的收益
2. **熊市是主要亏损来源**: 约25%的时间贡献了大部分回撤
3. **震荡市基本不赚钱**: 换手成本吃掉微薄收益

## 6. 实操建议

### 6.1 什么时候开仓

```
开仓条件 (需同时满足):
1. 沪深300 > MA20 (趋势向上)
2. 市场宽度 > 60% (多数股票在涨)
3. 或: RSRS > 0.7 (趋势确认)
```

### 6.2 什么时候空仓

```
空仓条件 (满足任一):
1. 沪深300 < MA20 连续5日
2. 市场宽度 < 40% (多数股票在跌)
3. 或: RSRS < -0.7 (下跌趋势确认)
```

### 6.3 适合时期的收益预期

| 条件 | 预期年化 | 预期回撤 |
|------|----------|----------|
| 牛市运行 | 20-30% | <10% |
| 震荡运行 | 0-5% | 5-10% |
| 全年平均(含择时) | 12-18% | <15% |

## 7. 总结

| 问题 | 回答 |
|------|------|
| 什么时间适合? | 牛市(沪深300月涨>5%, 市场宽度>60%) |
| 适合时能跑多少? | 月均3-5%, 季度10-15% |
| 不适合时亏多少? | 月均-2%~-5%, 季度-10%~-15% |
| 怎么判断适合不适合? | 看沪深300趋势+市场宽度+RSRS |
| 建议怎么做? | 牛市满仓动量轮动, 熊市空仓或持有国债ETF |

**核心观点**: 动量策略本质上是"趋势跟踪"，趋势存在时有效，趋势反转时失效。加上择时过滤是必要的。

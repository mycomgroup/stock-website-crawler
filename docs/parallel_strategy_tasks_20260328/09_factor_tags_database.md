# 聚宽因子库标签数据库

## 因子分类概览

基于聚宽API因子库，共包含以下类别因子：
- 质量因子：57个
- 基础因子：37个  
- 情绪因子：36个
- 成长因子：9个
- 风险因子：12个
- 每股因子：15个
- 动量因子：37个
- 技术因子：16个
- 风格因子：10个
- 行业因子：按分类

## 因子标签维度说明

### 数据源标签
- **财务类**：基于财务报表数据（ROE、PE、PB、毛利率等）
- **价量类**：基于价格和成交量数据（动量、乖离率、波动率等）
- **情绪类**：基于市场情绪指标（换手率、资金流、VROC等）
- **技术类**：基于技术分析指标（MACD、RSI、布林线等）

### 时间周期标签
- **长期**：>6个月（季度/年度数据）
- **中期**：1-6个月
- **短期**：<1个月

### 场景标签
- **选股**：横截面排序
- **择时**：时序判断
- **过滤**：风险控制
- **增强**：叠加收益

### 有效性特征标签
- **稳定型**：长期有效，波动小
- **周期型**：随周期波动
- **状态依赖型**：牛/熊/震荡表现不同
- **衰减型**：逐渐失效

### 拥挤度标签
- **未拥挤**：使用者少
- **中等拥挤**：部分alpha被套利
- **高度拥挤**：alpha基本消失

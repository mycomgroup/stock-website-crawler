## 五、代码实现要点

### 情绪开关正确实现

```python
# ✓ 正确：使用昨日涨停家数
def get_emotion_score(context):
    # 获取昨日数据
    df = get_price(all_stocks, end_date=context.previous_date, 
                   count=1, fields=['close', 'high_limit'])
    zt_count = len(df[df['close'] >= df['high_limit'] * 0.99])
    return zt_count

# 使用
if get_emotion_score(context) < 30:
    return  # 不开仓
```

### 停手机制实现

```python
class PauseManager:
    def __init__(self):
        self.loss_count = 0
        self.pause_days = 0
    
    def record_trade(self, pnl):
        if pnl < 0:
            self.loss_count += 1
        else:
            self.loss_count = 0
        
        if self.loss_count >= 3:
            self.pause_days = 3
    
    def can_trade(self):
        return self.pause_days == 0
```

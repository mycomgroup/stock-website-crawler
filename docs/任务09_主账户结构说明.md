## 各层功能

**进攻层**：获取超额Alpha。满配50%时持约15只股票，0%时清仓。Beta 0.40，Alpha 4.04%，专为跑赢沪深300设计。

**防守层**：30%固定仓内动态调配，职责是**降低组合整体回撤**，而非追求收益。

| 成分 | 角色 | 默认权重 |
|------|------|----------|
| 511010 国债ETF | 压舱石，利率下行时贡献资本利得 | 75%（防守层） |
| 518880 黄金ETF | 避险对冲，通胀/恐慌时表现 | 10% |
| 510880 红利ETF | 股息增强，下跌韧性 | 8% |
| 513100 纳指ETF | 海外分散，风险资产卫星 | 4% |

**现金缓冲**：最低20%，进攻层收缩时自动扩大，为极端行情提供流动性。

## 市场状态映射

```
市场状态判断（5信号评分，score 0-5）：
  score ≥ 4  → 进攻层 50%，防守层内部维持默认（国债75%）
  score = 3  → 进攻层 30%，防守层偏国债80%（情绪<40 或 广度<0.3）
  score = 2  → 进攻层 20%，防守层偏国债70%+黄金15%（连续下跌≥3天）
  score ≤ 1  → 进攻层 0%，防守层国债80%+黄金8%+红利8%+纳指4%
  北向流入>100亿 → 防守层内部纳指、红利各提至10%，表示情绪乐观
```

**调仓规则**：单标的偏离目标>15%触发再平衡；黄金单月跌幅>10%减至5%；FED指标<-1清盘纳指。

## 主仓可行性

**结论：可以作为主仓优先方案，但需要接受一个核心矛盾。**

回测3.2年（2022–2025）数据：
- 进攻50%+防守30%+现金20%组合 → 年化3.56%，最大回撤11.83%，超额收益40.78%
- 纯防守层（国债固收+）→ 年化6.15%，最大回撤1.64%，夏普0.85

**问题在于**：进攻层在熊市年份（2022–2024）几乎不贡献收益，甚至拖累组合。防守层单独运作反而比组合更好。这说明进攻层的RFScore信号在熊市期间大部分时间评分偏低（进攻层长期处于0%–20%档位），导致30%进攻仓位实际上没有发挥预期效果。

**更适合作为主仓的结构**：
- 方案A（保守）：纯国债固收+ 30%作为底仓，剩余70%自主管理——回撤1.64%，年化6.15%
- 方案B（平衡）：RFScore进攻层 40%，防守层 30%，现金 30%——用更高进攻换来更高潜在收益，但回撤会上移到12%–15%

## 风险提示

1. **进攻层Beta虽低（0.40），但满仓时最大回撤仍达23.81%**，不适合追求安稳的持有体验
2. **国债ETF暴露利率风险**：若央行加息，国债价格下跌，防守层同样受损
3. **纳指4%权重**：虽已大幅压缩，但美股剧震仍会传导
4. **黄金单月>10%跌幅**历史上出现过，动态规则（降至5%）可缓解但无法消除
5. **北向资金信号**依赖香港结算数据，极端外资撤离情形下可能失真

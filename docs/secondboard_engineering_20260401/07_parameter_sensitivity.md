### 提示词 7.4：过拟合风险评估
```
任务：评估二板接力策略的过拟合风险

评估方案：

1. 参数自由度分析
   - 参数数量：X个
   - 每个参数取值数：Y个
   - 总组合数：Z个
   - 样本数：N个
   - 自由度比率：N/Z（建议>30）

2. 样本内外差异分析
   - 对每个参数组合，计算：
     - IS年化收益 - OOS年化收益
     - IS胜率 - OOS胜率
     - IS夏普 - OOS夏普
   - 如果差异过大，可能过拟合

3. 参数稳定性分析
   - 如果参数轻微变化，表现剧烈变化，可能过拟合
   - 如果参数有单调性，过拟合风险较低
   - 如果参数最优值在边界，过拟合风险较高

4. 统计显著性检验
   - 零假设：策略收益 = 0
   - 检验方法：t检验或Bootstrap
   - 如果p值>0.05，可能不显著

5. 过拟合风险等级
   - 低风险：样本内外差异<20%，参数稳定，统计显著
   - 中风险：样本内外差异20-50%，参数较稳定
   - 高风险：样本内外差异>50%，参数不稳定

输出格式：
| 指标 | 值 | 风险等级 |
|------|-----|---------|
| 参数自由度比率 | N/Z | 低/中/高 |
| IS-OOS收益差异 | X% | 低/中/高 |
| 参数稳定性 | Y% | 低/中/高 |
| 统计显著性 | p值 | 显著/不显著 |

给出：
1. 过拟合风险评估结果
2. 风险来源分析
3. 降低过拟合风险建议
```

# 任务03：二板接力卖出规则设计与测试

## 任务定位
**阶段**：设计阶段  
**优先级**：P0  
**依赖**：任务01、任务02  
**产出**：卖出规则文档 V1.0

## 背景
买入规则已定义，卖出规则同样重要。短线策略的盈利关键在于"何时退出"。需要明确：
1. 止盈规则（什么时候获利了结）
2. 止损规则（什么时候认输）
3. 时间止损（持仓多久）
4. 熔断规则（什么情况下立即退出）

## 已知信息

### 当前结论（待验证）
- 当前测试使用"次日收盘卖出"
- 胜率87.95%
- 年化394%
- 但这可能是次优解

### 已知有效规则
- 时间止损有效（10:30/13:30时间窗口止损）
- 时间止损优于价格止损（爆发型策略）

## 核心任务

### 1. 止盈规则设计
需要明确：
- [ ] 固定点数止盈（如+5%、+10%）
- [ ] 分时止盈（如冲高回落跌破分时均线）
- [ ] 板块止盈（如板块走弱、龙头炸板）
- [ ] 情绪止盈（如情绪降温、涨停数下降）

### 2. 止损规则设计
需要明确：
- [ ] 固定点数止损（如-3%、-5%）
- [ ] 分时止损（如跌破开盘价、跌破分时均线）
- [ ] 时间止损（如10:30、13:30、收盘）
- [ ] 形态止损（如分时走弱、量价背离）

### 3. 持仓周期测试
需要明确：
- [ ] 次日卖出（T+1）
- [ ] 隔日卖出（T+2）
- [ ] 持仓不超过X天
- [ ] 是否考虑盘中卖出

### 4. 熔断规则设计
需要明确：
- [ ] 个股熔断（单票亏损超过X%）
- [ ] 组合熔断（总仓位亏损超过X%）
- [ ] 情绪熔断（市场情绪骤变）
- [ ] 连续亏损熔断（连续X笔亏损）

### 提示词 3.4：持仓周期测试
```
任务：测试不同持仓周期的表现

测试以下持仓周期：
1. T+1卖出（次日固定时机）

4. 动态持仓（根据表现决定）

4. 持有涨停股的额外收益

| 持仓周期 | 年化收益 | 胜率 | 最大回撤 | 夏普 | 年换手率 | 平均持仓天数 |
|---------|---------|------|---------|------|---------|-------------|

给出：
1. 推荐持仓周期
```

### 提示词 3.5：熔断规则设计
```
任务：设计二板接力策略的熔断规则

分析以下熔断场景：

1. 个股熔断
   - 单票最大亏损阈值
   - 单票连续跌停止损
   - 单票异常波动（如盘中闪崩）

2. 组合熔断
   - 组合当日亏损超过X%
   - 组合连续X日亏损
   - 组合回撤超过X%

3. 情绪熔断
   - 市场情绪骤变（涨停数从>50降至<30）
   - 涨跌停比反转（涨停<跌停）
   - 大盘指数熔断（如沪深300跌超5%）

4. 连续亏损熔断
   - 连续X笔交易亏损
   - 近Y日胜率低于Z%
   - 近Y日收益为负

请基于历史数据测试以下熔断规则：
- 组合单日亏损 > 5%：暂停交易1日
- 连续5笔交易亏损：暂停交易1日
- 近10日胜率 < 50%：降低仓位至50%
- 近20日收益 < -5%：暂停交易，重新评估

测试方案：
- 数据：2021-2024年
- 模拟熔断触发后的表现
- 对比有/无熔断的回撤表现

输出格式：
1. 熔断规则清单
2. 熔断触发频率统计
3. 熔断对回撤的改善效果
4. 熔断对收益的影响
```

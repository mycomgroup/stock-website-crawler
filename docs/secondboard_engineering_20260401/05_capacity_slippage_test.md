# 任务05：二板接力容量与滑点真实测试

## 任务定位
**阶段**：验证阶段  
**优先级**：P1  
**依赖**：任务01、任务02、任务03  
**产出**：容量与滑点测试报告 V1.0

## 背景
回测往往高估收益，因为忽略了：
1. 容量限制（资金量大了买不到）
2. 滑点成本（真实成交价与理论价偏差）
3. 冲击成本（大单买卖影响价格）

这些因素可能导致回测收益大幅缩水。

## 已知信息

### 当前估算
- 建议容量：300-500万
- 预估滑点：20-30bps
- 但缺乏实测验证

### 已知风险
- 小市值股票成交额有限
- 涨跌停可能无法成交
- 大单可能造成冲击

### 提示词 5.3：买入滑点分析
```
任务：分析二板接力策略的买入滑点

分析步骤：

1. 定义理想买入价格
   - 如果是收盘买入：理想价格 = 收盘价
   - 如果是开盘买入：理想价格 = 开盘价
   - 如果是竞价买入：理想价格 = 竞价成交价

2. 估算真实买入价格
   - 方法1：开盘后X分钟均价
   - 方法2：VWAP（成交量加权平均价）
   - 方法3：保守估计（开盘价 + X%滑点）

3. 计算滑点
   - 滑点 = (真实买入价 - 理想买入价) / 理想买入价
   - 统计：滑点的均值、中位数、分位数

4. 分场景分析
   - 高开场景：开盘涨幅 > 3%
   - 平开场景：开盘涨幅 -1% ~ 3%
   - 低开场景：开盘涨幅 < -1%

5. 大单冲击分析
   - 假设买入金额占成交额的X%
   - 测试：X = 1%, 2%, 3%, 5%
   - 计算：对应的冲击成本

输出格式：
| 买入时机 | 理想价格 | 真实价格估计 | 平均滑点 | 中位数滑点 | 90分位滑点 |
|---------|---------|-------------|---------|-----------|-----------|
| 开盘买入 | 开盘价 | 开盘后5分钟均价 | 0.15% | 0.10% | 0.30% |

给出：
1. 不同买入时机的滑点对比
2. 滑点与成交额占比的关系
3. 保守滑点估算建议
```

### 提示词 5.4：卖出滑点分析
```
任务：分析二板接力策略的卖出滑点

分析步骤：

1. 定义理想卖出价格
   - 次日收盘卖出：理想价格 = 收盘价
   - 次日最高卖出：理想价格 = 最高价（不可实现，作为上限）
   - 次日开盘卖出：理想价格 = 开盘价

2. 估算真实卖出价格
   - 方法1：开盘后X分钟均价
   - 方法2：VWAP
   - 方法3：保守估计（理想价格 - X%滑点）

3. 计算滑点
   - 滑点 = (理想卖出价 - 真实卖出价) / 理想卖出价
   - 统计：滑点的均值、中位数、分位数

4. 分场景分析
   - 次日高开场景：开盘涨幅 > 3%
   - 次日平开场景：开盘涨幅 -1% ~ 3%
   - 次日低开场景：开盘涨幅 < -1%
   - 次日跌停场景：无法卖出

5. 涨跌停影响分析
   - 统计：次日涨停/跌停的比例
   - 分析：涨停时是否能卖出
   - 分析：跌停时无法卖出的影响

输出格式：
| 卖出时机 | 理想价格 | 真实价格估计 | 平均滑点 | 中位数滑点 | 90分位滑点 |
|---------|---------|-------------|---------|-----------|-----------|
| 次日收盘 | 收盘价 | 收盘前5分钟均价 | 0.10% | 0.05% | 0.20% |

给出：
1. 不同卖出时机的滑点对比
2. 涨跌停对滑点的影响
3. 保守滑点估算建议
```

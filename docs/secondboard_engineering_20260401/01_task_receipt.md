### 2. 二板定义对比表

**核心对比维度**：
- 聚宽平台标准定义
- 本项目实测定义
- 聚宽连板龙头策略
- 首板低开策略

**推荐定义**：
```
二板 = 连续2日涨停 + 非一字板 + 缩量 + 换手率<30% + 市值排序
```

**理由**：
1. 实测验证有效（胜率87.95%，年化407.65%）
2. 缩量条件提升承接质量
3. 换手率过滤避免流动性风险
4. 市值排序优选小盘股
5. 排除一字板保证可成交

### 3. 盈利逻辑分析文档

**Alpha来源分解**：
| Alpha来源 | 占比 | 理论框架 | 数据证据 |
|----------|------|---------|---------|
| 情绪溢价 | 60% | 板块热度持续发酵 | 情绪开关有效 |
| 资金接力 | 30% | 首板分歧后承接转强 | 缩量条件验证 |
| 技术形态 | 10% | 连续涨停突破 | 市值排序弹性 |

**二板优势**：
- 确认性更强（胜率87.95% vs 首板50-60%）
- 情绪稳定（情绪开关有效）
- 资金集中（缩量承接）
- 盈利幅度（平均4.77%）

**二板vs三板、四板**：
- 二板最优（样本充足、胜率最高、回撤可控）
- 三板样本不足（20次交易）
- 四板回撤大（0.92%），逻辑复杂

### 4. 失败模式统计报告

**失败率统计**：
- 总体失败率：12.05%（2024实测）
- 情绪过滤后：失败率降低（涨停≥10）

**失败模式分布**：
| 失败模式 | 占比估算 | 特征 |
|---------|---------|------|
| 情绪低迷 | 40% | 涨停家数<10 |
| 深度低开 | 20% | 开盘跌幅>-4% |
| 高开低走 | 15% | 冲高回落 |
| 冲高回落 | 15% | 未及时止损 |
| 其他 | 10% | 流动性、停牌 |

**失败预警信号清单**：
- 竞价阶段：情绪低迷、板块轮动
- 开盘后：深度低开、大幅高开、一字涨停
- 持仓阶段：高位回落、未冲高、板块降温

**规避建议**：
- 情绪预警规避（涨停<10空仓）
- 开盘预警规避（深度低开不买入）
- 持仓预警规避（高位回落止损）
- 连续亏损规避（连亏3停3）

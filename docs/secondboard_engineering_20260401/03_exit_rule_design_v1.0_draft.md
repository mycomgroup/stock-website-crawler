## 2. 止盈规则

### 2.1 固定止盈
**推荐方案：无固定止盈**

**理由**：
- 二板接力为爆发型策略，过早止盈可能错过后续涨幅
- 次日卖出机制已自然止盈
- 如需止盈，建议+10%以上

| +15% | [待填入]% | [待填入]% | [待填入]% | 触发率低 |

### 2.2 分时止盈
**推荐方案：暂不采用**

**理由**：
- 实现复杂度高，需要实时监控分时数据
- 爆发型策略分时波动大，易误触发
- 收益提升不明显

**备选方案**：
- 冲高回落跌破分时均线
- 冲高回落跌破开盘价
- 冲高后回调超过涨幅50%

### 2.3 板块止盈
**推荐方案：龙头炸板时考虑止盈**

**触发条件**：
- 板块龙头股炸板（涨停打开）
- 板块整体跌幅超过2%
- 板块涨停家数下降50%

**执行方式**：
- 触发后立即卖出
- 需要实时板块数据支持

### 3.2 时间止损
**推荐方案：13:30时间止损（亏损情况下）**

**触发条件**：
- 持仓至次日13:30
- 如果此时处于亏损状态
- 立即卖出

**预期效果**：
- 有效控制日内回撤
- 保留日内上涨空间

**备选时间点**：
- 10:30止损：过于激进，可能错过早盘反弹
- 14:30止损：接近收盘，意义不大

### 3.3 组合止损
**推荐方案：-5%止损 或 13:30时间止损**

**规则说明**：
- 两个条件取先触发者
- 任意一个条件满足立即卖出

**优势**：
- 结合固定止损和动态止损
- 既控制单笔最大亏损，又控制持仓时间
- 适合爆发型策略

## 4. 持仓周期管理

### 最短持仓
**1个交易日（T+1）**

### 最长持仓
**2个交易日（T+2）**

### 推荐方案
**T+1或T+2（涨停持有）**

**规则说明**：
1. 默认T+1收盘卖出
2. 如果T+1涨停，则持有到T+2收盘卖出
3. T+2无论涨跌必须卖出

### 特殊情况处理

#### 涨停处理
- T+1涨停：持有到T+2
- T+2涨停：T+2收盘必须卖出，不继续持有
- 避免过度持仓风险

#### 停牌处理
- 买入后发现停牌：等待复牌后第一个交易日卖出
- 复牌后涨停：按涨停规则处理（持有到次日）
- 复牌后跌停：等待打开跌停后卖出

#### 复牌处理
- 复牌首日按正常交易日处理
- 如遇涨跌停，按涨跌停规则处理

## 5. 熔断规则

### 5.1 个股熔断
**推荐方案：单票最大亏损10%**

**规则说明**：
- 单笔交易最大亏损不超过10%
- 触发后立即卖出
- 同时触发策略熔断（暂停交易1日）

**处理方式**：
- 立即市价卖出
- 记录为异常亏损
- 暂停次日交易，重新评估策略

### 5.2 组合熔断
**推荐方案：组合单日亏损>5%**

**规则说明**：
- 组合当日总亏损超过5%

- 冷静期后恢复正常交易

### 5.3 情绪熔断
**推荐方案：市场情绪骤变暂停交易**

**触发条件**：
- 涨停数从>50降至<30
- 涨跌停比反转（涨停<跌停）
- 大盘指数跌幅>5%

**处理方式**：
- 立即暂停所有交易
- 等待情绪恢复（涨停数重回30以上）
- 重新评估市场环境

### 5.4 连续亏损熔断
**推荐方案：近10日胜率<50%降仓至50%**

**规则说明**：
- 统计最近10个交易日胜率
- 如果胜率低于50%，降低仓位至50%
- 如果近20日收益<-5%，暂停交易

## 6. 卖出执行细节

### 6.1 竞价卖出
**时间**：9:15-9:25
**价格**：集合竞价成交价
**适用场景**：
- 需要规避日内风险
- 开盘预期不佳
- 隔夜重大利空

**注意事项**：
- 竞价卖出可能错过日内上涨
- 需提前设置好卖出指令
- 注意竞价流动性

### 6.2 盘中卖出
**时间**：交易时段（9:30-14:57）
**价格**：市价或限价
**适用场景**：
- 触发止损/止盈条件
- 涨停打开
- 异常波动

**执行方式**：
- 止损触发：市价卖出，确保成交
- 止盈触发：限价卖出，锁定利润
- 时间止损：14:45-14:50卖出

**注意事项**：
- 避免在快速下跌时恐慌卖出
- 注意盘中流动性
- 大单需考虑冲击成本

### 6.3 收盘卖出
**时间**：14:50-15:00
**价格**：收盘价附近
**适用场景**：
- 常规卖出（默认方案）
- 未触发其他卖出条件

**执行方式**：
- 14:50开始下单
- 使用限价单，价格设置为当前价附近
- 确保14:57前成交

**注意事项**：
- 避免在最后几分钟才下单
- 注意收盘集合竞价
- 确保成交，不留隔夜仓

## 8. 完整卖出流程图

```
买入次日
    |
    ├── 竞价阶段（9:15-9:25）
    |   └── 检查重大利空 → 如有，竞价卖出
    |
    ├── 开盘阶段（9:30-9:35）
    |   └── 检查开盘表现
    |
    ├── 盘中阶段（9:35-13:30）
    |   ├── 检查止损条件
    |   |   ├── 触发-5%止损 → 立即卖出
    |   |   └── 未触发 → 继续
    |   ├── 检查涨停状态
    |   |   ├── 涨停 → 持有到T+2
    |   |   └── 未涨停 → 继续
    |   └── 13:30检查
    |       ├── 亏损 → 卖出
    |       └── 盈利 → 继续
    |
    └── 收盘阶段（14:50-15:00）
        └── 卖出（默认方案）
```

## 9. 风险提示

### 已知风险
1. **流动性风险**：涨跌停可能无法成交
2. **滑点风险**：实际成交价与预期价格偏差
3. **策略衰减风险**：历史表现不代表未来
4. **市场环境变化**：极端行情下规则可能失效

### 应对措施
1. 建立熔断机制，控制单笔和组合风险
2. 考虑滑点影响，留有安全边际
3. 持续监控策略表现，及时调整参数
4. 定期回测验证规则有效性

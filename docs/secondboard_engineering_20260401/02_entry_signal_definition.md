# 任务02：二板接力买入信号详细定义

## 任务定位
**阶段**：定义阶段  
**优先级**：P0  
**依赖**：任务01（策略逻辑定义）  
**产出**：买入信号规则文档 V1.0

## 背景
已知二板接力策略有效，但买入信号不够精确。需要明确：
1. 什么时候买（时机）
2. 买什么标的（筛选）
3. 买多少（仓位）
4. 什么条件下不买（过滤）

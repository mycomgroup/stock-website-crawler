# 历史OOS回测验证

## 一、验证方案概述

### 1.1 验证目的

历史OOS回测验证的目的是：
1. **模拟真实OOS验证过程**：用历史数据模拟策略上线后的验证过程
2. **验证验证框架有效性**：验证OOS验证框架能否有效检测策略衰退
3. **评估衰退检测机制**：测试衰退检测机制的准确性和误报率
4. **提供参考基准**：为实际OOS验证提供参考标准和经验积累

### 1.2 验证方法

采用两种验证方法：
- **滚动窗口验证**：每季度模拟一次OOS验证，检测策略衰减趋势
- **累积验证**：模拟策略自上线以来的累积表现，评估长期有效性

### 1.3 数据范围

- **总数据范围**：2021-01-01 至 2024-12-31（4年）
- **IS（样本内）数据**：2021-01-01 至 2023-12-31（3年）
- **OOS（样本外）模拟数据**：2022Q1 至 2024Q4（12个季度）

### 1.4 IS表现基准

基于2021-2023年回测结果，IS表现基准如下：

| 指标类别 | 具体指标 | IS表现基准 |
|---------|---------|-----------|
| 收益指标 | 年化收益率 | 394% |
| 收益指标 | 累计收益率 | ~500%（3年） |
| 质量指标 | 胜率 | 87.95% |
| 质量指标 | 盈亏比 | 21.91 |
| 风险指标 | 最大回撤 | 0.60% |
| 风险指标 | 回撤持续时间 | 2天 |
| 风险调整收益 | 夏普比率 | 20+ |
| 风险调整收益 | 卡玛比率 | 30+ |

### 6.3 验证框架代码示例

```python
class HistoricalOOSValidation:
    def run_rolling_window_validation(self, start_date, end_date):
        """
        滚动窗口验证
        
        参数:
        - start_date: 验证开始日期
        - end_date: 验证结束日期
        
        返回:
        - validation_results: 验证结果列表
        """
        results = []
        
        # 生成季度窗口
        quarters = generate_quarter_windows(start_date, end_date)
        
        for quarter in quarters:
            # 确定IS训练期和OOS验证期
            is_period = quarter['is_period']  # 前期数据
            oos_period = quarter['oos_period']  # 当前季度
            
            # 获取数据
            is_data = fetch_data(is_period)
            oos_data = fetch_data(oos_period)
            
            # 计算指标
            is_performance = calculate_performance(is_data)
            oos_performance = calculate_performance(oos_data)
            
            # 计算衰减
            decay_ratio = (oos_performance['annual_return'] - 
                          is_performance['annual_return']) / \
                          is_performance['annual_return']
            
            # 衰退检测
            alert_level = detect_decay(
                oos_performance,
                is_performance
            )
            
            results.append({
                'quarter': quarter['name'],
                'oos_return': oos_performance['annual_return'],
                'is_return': is_performance['annual_return'],
                'decay_ratio': decay_ratio,
                'alert_level': alert_level
            })
        
        return results
    
    def run_cumulative_validation(self, start_date, end_date):
        """
        累积验证
        
        参数:
        - start_date: 累积开始日期（策略上线日）
        - end_date: 当前日期
        
        返回:
        - validation_result: 累积验证结果
        """
        # 获取累积数据
        cumulative_data = fetch_data(start_date, end_date)
        
        # 计算累积指标
        cumulative_performance = calculate_performance(cumulative_data)
        
        # 获取IS基准
        is_performance = get_is_performance()
        
        # 计算衰减
        decay_ratio = (cumulative_performance['annual_return'] - 
                      is_performance['annual_return']) / \
                      is_performance['annual_return']
        
        # 衰退检测
        alert_level = detect_decay(
            cumulative_performance,
            is_performance
        )
        
        return {
            'cumulative_return': cumulative_performance['cumulative_return'],
            'annual_return': cumulative_performance['annual_return'],
            'sharpe_ratio': cumulative_performance['sharpe'],
            'is_return': is_performance['annual_return'],
            'decay_ratio': decay_ratio,
            'alert_level': alert_level
        }
```

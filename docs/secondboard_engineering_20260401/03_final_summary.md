## 二、核心发现 ⭐⭐⭐

### 最重要的3个发现

#### 1️⃣ **-5%止损效果显著**（优先级：P0）

```
基准：无止损，平均收益 -2.34%，最大亏损 -13.00%
优化：-5%止损，平均收益 +0.17%，最大亏损 -5.00%

改善：收益提升 2.51%，风险降低 8.00%
```

**结论**: 止损规则带来显著改善，**必须采用**

---

#### 2️⃣ **熔断规则有效降低风险**（优先级：P0）

```
基准：无熔断，平均收益 -2.34%，最大亏损 -13.00%
优化：近10日胜率<50%降仓，平均收益 -0.43%，最大亏损 -7.07%

改善：收益提升 1.91%，风险降低 5.93%
```

**结论**: 动态仓位调整有效，**必须采用**

---

#### 3️⃣ **开盘卖出优于收盘卖出**（优先级：P1）

```
基准：收盘卖出，平均收益 -1.89%
优化：开盘卖出，平均收益 -0.39%

改善：收益提升 1.50%
```

**结论**: 规避日内风险有效，**推荐采用**

## 三、推荐交易规则

### 完整规则

```python
# ===== 买入规则 =====
市值范围：流通市值 5-15亿
情绪过滤：涨停数 >= 30
二板定义：连续两日涨停且非一字板
买入时机：次日开盘买入

# ===== 卖出规则（优化后）=====
卖出时机：次日开盘卖出
止损规则：-5%止损（触发立即卖出）⭐ 必须采用
止盈规则：无固定止盈
持仓周期：T+1或T+2（涨停持有）
熔断规则：近10日胜率<50%时，仓位降至50% ⭐ 必须采用
```

### 预期效果

```
基准收益（2024Q1，无优化）：-2.34%
优化收益（预计）：正收益

收益提升来源：
- 止损规则：+2.51%
- 熔断规则：+1.91%
- 开盘卖出：+1.50%
- 组合效果：预计 +5.92%
```

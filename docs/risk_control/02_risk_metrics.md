## 六、指标联动关系

### 6.1 市场风险指标联动

```
沪深300涨跌幅 <-3% 
  → 大盘波动率上升 
  → 跌停家数增加 
  → 涨停家数减少 
  → 情绪开关关闭
```

### 6.2 策略风险指标联动

```
胜率下降 <45% 
  → 连续亏损次数增加 
  → 最大回撤扩大 
  → 卡玛比率下降 
  → 策略熔断
```

### 6.3 尾部风险指标联动

```
单日亏损 >3% 
  → 当前回撤扩大 
  → 连续亏损金额增加 
  → 回撤持续天数增加 
  → 熔断触发
```

### 6.4 综合风险预警

**第一级预警（黄色）：**
- 涨停家数 < 50 或 > 150
- 最大回撤 > 10%
- 连续亏损次数 ≥ 3笔
- 单日亏损 > 3%

**第二级预警（橙色）：**
- 沪深300涨跌幅 < -3%
- 最大回撤 > 12%
- 连续亏损次数 = 4笔

**第三级熔断（红色）：**
- 涨停家数 < 30 或 > 200
- 最大回撤 > 15%
- 连续亏损次数 ≥ 5笔
- 单日亏损 > 5%
- 沪深300涨跌幅 < -5%

## 七、指标计算示例

### 7.1 最大回撤计算

```python
def calculate_max_drawdown(net_values):
    """
    计算最大回撤
    net_values: 净值序列
    """
    peak = net_values[0]
    max_dd = 0
    
    for value in net_values:
        if value > peak:
            peak = value
        dd = (peak - value) / peak
        if dd > max_dd:
            max_dd = dd
    
    return max_dd
```

### 7.2 夏普比率计算

```python
def calculate_sharpe_ratio(daily_returns, risk_free_rate=0.03):
    """
    计算夏普比率
    daily_returns: 日收益率序列
    risk_free_rate: 无风险利率（年化）
    """
    excess_returns = daily_returns - risk_free_rate / 252
    sharpe = excess_returns.mean() / excess_returns.std() * np.sqrt(252)
    return sharpe
```

### 7.3 连续亏损次数计算

```python
def calculate_consecutive_losses(trades):
    """
    计算连续亏损次数
    trades: 交易记录
    """
    max_consecutive = 0
    current_consecutive = 0
    
    for trade in trades:
        if trade['profit'] < 0:
            current_consecutive += 1
            max_consecutive = max(max_consecutive, current_consecutive)
        else:
            current_consecutive = 0
    
    return max_consecutive
```

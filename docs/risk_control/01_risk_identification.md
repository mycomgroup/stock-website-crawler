## 二、风险等级排序

### 2.1 高优先级风险（必须控制）

1. **情绪骤变**（发生概率中 × 影响程度高 = 高风险）
   - 风险描述：情绪开关阈值失效，市场情绪骤降
   - 控制措施：双阈值（50-150），情绪冰点预警

2. **策略衰减**（发生概率高 × 影响程度高 = 高风险）
   - 风险描述：市场环境变化，策略失效
   - 控制措施：样本外验证、策略监控、定期评估

3. **连续亏损**（发生概率中 × 影响程度高 = 高风险）
   - 风险描述：策略连续亏损，心理和资金压力
   - 控制措施：连续亏损熔断、暂停交易

4. **大盘暴跌**（发生概率低 × 影响程度高 = 中高风险）
   - 风险描述：系统性风险，策略无法规避
   - 控制措施：组合熔断、降低仓位

### 2.2 中优先级风险（建议控制）

5. **一字涨停无法买入**（发生概率高 × 影响程度中 = 中风险）
   - 风险描述：涨停板封单大，无法成交
   - 控制措施：放弃信号、不计入统计

6. **滑点**（发生概率高 × 影响程度中 = 中风险）
   - 风险描述：实际成交价与预期偏差
   - 控制措施：容量限制、分批建仓

7. **板块轮动**（发生概率高 × 影响程度中 = 中风险）
   - 风险描述：题材切换，龙头熄火
   - 控制措施：情绪监控、板块热度跟踪

### 2.3 低优先级风险（关注监控）

8. **一字跌停无法卖出**（发生概率低 × 影响程度高 = 中高风险，但概率低）
   - 风险描述：闪崩、利空跌停
   - 控制措施：止损、预警

9. **技术故障**（发生概率低 × 影响程度高 = 中高风险，但概率低）
   - 风险描述：系统故障、网络中断
   - 控制措施：备份方案、人工干预

## 三、风险关联性分析

### 3.1 同时发生的风险组合

**组合1：极端行情组合**
- 大盘暴跌 + 情绪骤变 + 一字跌停无法卖出
- 发生场景：股灾、熔断
- 应对：组合熔断、空仓观望

**组合2：策略失效组合**
- 策略衰减 + 样本外失效 + 连续亏损
- 发生场景：市场环境变化、风格切换
- 应对：暂停交易、重新评估

**组合3：操作风险组合**
- 滑点 + 延迟 + 低成交额
- 发生场景：流动性差的股票
- 应对：流动性过滤、放弃信号

### 3.2 风险传导路径

```
大盘暴跌 → 情绪骤变 → 涨停数下降 → 情绪开关关闭 → 无法开仓
   ↓
流动性枯竭 → 一字跌停 → 无法止损 → 回撤失控
   ↓
连续亏损 → 心理压力 → 策略暂停
```

## 六、总结

### 6.1 核心风险

**前三大必须控制的风险：**
1. 情绪骤变（双阈值控制）
2. 策略衰减（样本外验证）
3. 连续亏损（熔断机制）

### 6.2 风险关联性

**最危险的风险组合：**
- 大盘暴跌 + 情绪骤变 + 一字跌停无法卖出
- 发生概率低，但影响程度极高

### 6.3 控制重点

**风控措施优先级：**
1. 情绪开关（双阈值50-150）
2. 熔断机制（连续亏损、单日亏损、组合回撤）
3. 流动性过滤（成交额、换手率）
4. 仓位管理（单票、总仓位、行业集中度）

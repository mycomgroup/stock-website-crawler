# 打板核心逻辑总结

> 核心结论：打板 = 情绪 × 个股结构 × 题材

---

## 一、三个核心维度

### 维度权重分配

| 维度 | 权重 | 核心作用 |
|------|------|---------|
| **情绪** | 40-50% | 决定打板能否成功、能否溢价 |
| **个股结构** | 30-40% | 决定哪些板值得打、封单质量 |
| **题材** | 20-30% | 决定板的持续性、能否走出连板 |

---

## 二、情绪维度（权重最高）

### 2.1 情绪决定了什么

| 指标 | 强情绪 | 弱情绪 | 影响结果 |
|------|--------|--------|---------|
| **打板成功率** | 高 (>70%) | 低 (<50%) | 涨停能否封住 |
| **次日溢价** | 高 (平均+3%以上) | 低 (平均0%以下) | 打板是否赚钱 |
| **炸板概率** | 低 (<20%) | 高 (>40%) | 是否会吃面 |
| **容错空间** | 大（烂板也能回封） | 小（好板也容易炸） | 失误成本 |

### 2.2 情绪核心指标

**量化指标**：
- 涨停家数：>100为强情绪，<50为弱情绪
- 晋级率：>30%为强情绪，<20%为弱情绪
- 最高连板数：>=5板为强情绪，<=2板为弱情绪
- 涨跌停比：>2为强情绪，<1为弱情绪

**情绪周期**：
```
启动期 → 发酵期 → 高潮期 → 退潮期 → 冰点期 → 启动期
```

### 2.3 情绪与策略匹配

| 策略 | 最优情绪环境 | 最差情绪环境 | 情绪依赖度 |
|------|------------|------------|-----------|
| 首板低开 | 中等情绪 | 冰点期 | 中 |
| 弱转强竞价 | **强情绪（发酵-高潮）** | 退潮-冰点 | **高** |
| 234板接力 | **强情绪（发酵-高潮）** | 退潮-冰点 | **高** |
| 龙头底分型 | 弱情绪（冰点期） | 高潮期 | 低 |

---

## 三、个股结构维度

### 3.1 个股结构决定了什么

| 指标 | 说明 | 关键阈值 |
|------|------|---------|
| **位置** | 相对高低位 | 低位(<30%)优于高位 |
| **形态** | K线结构 | 首板、二板、底分型等 |
| **量能** | 成交量、换手率 | 缩量板优于放量板 |
| **市值** | 流通市值 | 小市值(20-50亿)弹性大 |
| **筹码** | 筹码集中度 | 筹码稳定优于分散 |

### 3.2 板位结构对比

| 板位 | 成功率 | 收益预期 | 风险 | 适合情绪 |
|------|--------|---------|------|---------|
| **首板** | 中等 | 低(次日+2-5%) | 中 | 中等情绪 |
| **二板** | 较高 | 中(次日+5-10%) | 中 | 中-强情绪 |
| **三板** | 较低 | 高(+10-20%) | 高 | 强情绪 |
| **四板+** | 低 | 很高(+20%+) | 很高 | 极强情绪 |

### 3.3 个股筛选优先级

```
1. 位置过滤：相对位置<=30%
2. 市值过滤：流通市值20-100亿
3. 量能过滤：换手率5-20%
4. 形态识别：符合特定形态（首板、底分型等）
5. 题材加分：属于当前主线题材
```

---

## 四、题材维度

### 4.1 题材决定了什么

| 指标 | 主线题材 | 支线题材 | 冷门题材 |
|------|---------|---------|---------|
| **持续性** | 强 (3-7天) | 中 (1-3天) | 弱 (1天) |
| **连板概率** | 高 | 中 | 低 |
| **板块效应** | 有（多只涨停） | 弱 | 无 |
| **资金关注度** | 高 | 中 | 低 |

### 4.2 题材判断标准

**主线题材特征**：
1. 板块涨停家数>=3只
2. 有明确催化剂（政策、事件、业绩）
3. 市场讨论度高
4. 龙头股走出连板

**题材轮动规律**：
```
新题材爆发 → 龙头连板 → 板块扩散 → 高潮分歧 → 题材退潮
```

### 4.3 题材与打板策略

| 策略 | 题材要求 | 操作要点 |
|------|---------|---------|
| 首板低开 | 题材加分但不必须 | 更看重位置和形态 |
| 弱转强竞价 | **最好是主线题材** | 题材+情绪共振 |
| 234板接力 | **必须是主线题材** | 做题材龙头接力 |
| 龙头底分型 | 题材不重要 | 做龙头股本身 |

---

## 五、三维共振模型

### 5.1 最佳出手时机

**三者共振** = 情绪好 + 个股结构优 + 题材强

```
情绪（强） ─────┐
                ├──→ 最佳出手点
个股（优） ─────┤
                │
题材（强） ─────┘
```

### 5.2 三维组合决策矩阵

| 情绪 | 个股 | 题材 | 决策 | 预期收益 |
|------|------|------|------|---------|
| 强 | 优 | 强 | **重仓出击** | 高 |
| 强 | 优 | 弱 | 可出手 | 中 |
| 强 | 差 | 强 | 轻仓试错 | 中低 |
| 弱 | 优 | 强 | 可出手 | 中 |
| 弱 | 优 | 弱 | **不出手** | - |
| 弱 | 差 | 强 | 不出手 | - |

### 5.3 不同维度的容错空间

| 维度组合 | 容错空间 | 说明 |
|---------|---------|------|
| 情绪好+个股差 | 大 | 烂板也能回封，容错高 |
| 情绪差+个股好 | 小 | 好板也容易炸，容错低 |
| 题材强+情绪差 | 中 | 有题材支撑，炸板率略降 |
| 三者都差 | 无 | **坚决不出手** |

## 六、实战应用框架

### 6.1 打板决策流程

```
Step 1: 判断情绪（看天）
├── 强情绪 → 进入Step 2
├── 中情绪 → 进入Step 2（降低仓位）
└── 弱情绪 → 停止操作或只做低情绪策略

Step 2: 筛选个股（看地）
├── 位置过滤
├── 市值过滤
├── 量能过滤
└── 形态识别

Step 3: 确认题材（看催化剂）
├── 是否主线题材
├── 是否有催化剂
└── 板块效应如何

Step 4: 盘口确认（看人）
├── 封单质量
├── 竞价表现
└── 分时走势

Step 5: 执行交易
├── 确定仓位
├── 确定买入价
└── 设置止损止盈
```

### 6.2 仓位分配建议

| 三维评分 | 仓位上限 | 说明 |
|---------|---------|------|
| 三者全优 | 10% | 最佳机会 |
| 二优一良 | 8% | 较好机会 |
| 一优二良 | 5% | 一般机会 |
| 其他情况 | 0% | 不出手 |

## 七、常见误区

### 误区一：只看个股不看情绪

**错误**：个股形态很好，但情绪差，打板后炸板吃面

**正确**：先判断情绪，情绪不好坚决不出手

### 误区二：只看情绪不看个股

**错误**：情绪好，乱打板，结果选中劣质股

**正确**：情绪好只是放大了个股的收益，不能替代个股筛选

### 误区三：忽视题材

**错误**：打的板没有题材支撑，次日无溢价

**正确**：题材决定了板的持续性，要做有题材支撑的板

### 误区四：三维要求过于苛刻

**错误**：等待三者完美共振，结果错过机会

**正确**：二维达标即可出手，三维全优可重仓

## 八、总结

### 打板心法

```
先看天（情绪），再看地（个股），最后看人（盘口）
情绪是环境，个股是标的，题材是催化剂
三者共振，重仓出击；二维达标，轻仓试错
单一维度，坚决放弃
```

### 核心要点

1. **情绪权重最高**：没有好情绪，好板也难赚钱
2. **个股结构是基础**：决定了哪些板值得打
3. **题材是加分项**：决定了板的持续性
4. **三维共振最优**：但二维达标即可出手
5. **风控始终第一**：设置止损，控制仓位

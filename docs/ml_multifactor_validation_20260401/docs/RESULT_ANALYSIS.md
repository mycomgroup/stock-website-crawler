### 2.2 统计分析

#### 逻辑回归
- **累计收益**: 57.35%
- **年化收益**: 约20.8%
- **季度胜率**: 88.9% (8胜1负)
- **最大单季盈利**: +14.55% (2025-Q1)
- **最大单季亏损**: -5.87% (2023-Q4)
- **收益波动**: 中等
- **夏普比率**: 约2.1

**特点分析**:
- 表现最稳定，9个季度中8个盈利
- 回撤控制最好，仅1个季度亏损超过5%
- 适合震荡市和熊市
- 推荐作为主力策略

#### SVM
- **累计收益**: 50.38%
- **年化收益**: 约18.3%
- **季度胜率**: 66.7% (6胜3负)
- **最大单季盈利**: +29.48% (2025-Q3)
- **最大单季亏损**: -9.45% (2023-Q3)
- **收益波动**: 高
- **夏普比率**: 约1.5

**特点分析**:
- 在牛市爆发力强（2025-Q3 +29.48%）
- 但波动大，回撤较深
- 适合趋势明显的市场
- 建议作为牛市备选

#### 随机森林
- **累计收益**: 25.35%
- **年化收益**: 约9.2%
- **季度胜率**: 66.7% (6胜3负)
- **最大单季盈利**: +19.50% (2025-Q3)
- **最大单季亏损**: -10.13% (2024-Q4)
- **收益波动**: 高
- **夏普比率**: 约0.8

**特点分析**:
- 表现明显落后
- 波动最大，回撤最深
- 容易过拟合
- **建议淘汰**

### 4.2 可信度评估

**本次测试更可信的原因**:

1. ✅ 严格的Walk-Forward验证
2. ✅ 无数据泄露（特征仅用历史数据）
3. ✅ 标准化参数仅从训练集计算
4. ✅ 真实的交易成本模拟（双边0.2%）

**文档数据可能存在的问题**:

1. ❌ 可能使用了未来信息
2. ❌ 可能使用了全部股票（流动性问题）
3. ❌ 可能忽略了交易成本
4. ❌ 标准化可能使用了全样本数据

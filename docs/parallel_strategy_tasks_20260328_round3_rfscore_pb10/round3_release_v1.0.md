## 四、风险提示

### 4.1 已知风险

| 风险 | 严重程度 | 应对措施 |
|------|----------|----------|
| 候选稀疏 | 🔴 高 | 启用PB20%备用池 |
| 行业集中 | 🔴 高 | 设置单行业上限30% |
| ROA偏低 | 🟡 中 | 增加ROA>2%硬过滤 |
| 周期暴露 | 🟡 中 | 监控周期走势 |

### 4.2 使用限制

- 策略历史表现不代表未来收益
- 需配合市场环境调整仓位
- 建议先用模拟盘验证
- 定期监控质量评分

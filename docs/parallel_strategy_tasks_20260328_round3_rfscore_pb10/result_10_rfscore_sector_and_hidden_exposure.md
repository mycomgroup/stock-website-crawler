## 四、策略本质分析

### 4.1 策略在押什么？

**核心逻辑：**
```
RFScore7 PB10 = 质量因子 × 价值因子
              = 高财务质量 × 深度低估
              = 央国企周期龙头的估值修复
```

**策略本质：**
1. **不是**纯粹的低PB价值策略（因为银行、煤炭等纯低PB行业被质量门槛过滤）
2. **不是**纯粹的质量策略（因为需要PB最低10%的估值保护）
3. **是** "质量低估"策略：寻找财务稳健但被市场低估的大盘周期股

### 4.2 期望暴露 vs 意外副产物

| 暴露类型 | 是否是期望的 | 说明 |
|----------|--------------|------|
| 钢铁 | ⚠️ 半预期 | 知道会选周期，但没想到占比这么高 |
| 房地产 | ⚠️ 半预期 | 知道会选低PB，但没想到有房地产 |
| 央国企 | ✅ 是预期 | 质量因子自然筛选结果 |
| 大盘股 | ✅ 是预期 | 质量因子自然筛选结果 |
| 银行暴露缺失 | ❌ 意外副产物 | 质量门槛过滤了低ROA行业 |
| 煤炭暴露缺失 | ❌ 意外副产物 | 煤炭周期上行，PB未压缩 |

## 六、建议：是否需要行业约束？

### 6.1 当前策略问题

| 问题 | 严重程度 | 说明 |
|------|----------|------|
| 候选数量过少 | 🔴 严重 | 多数月份无候选或仅1-2只 |
| 行业过度集中 | 🔴 严重 | 钢铁占比43%，前3行业占比79% |
| 周期暴露过高 | 🟡 中等 | 57%暴露于强周期行业 |
| 缺少银行分散 | 🟡 中等 | 错失低PB行业的分散效果 |

### 6.2 行业约束方案对比

| 方案 | 具体做法 | 优点 | 缺点 |
|------|----------|------|------|
| **A: 不加约束** | 保持现状 | 简单，质量门槛已过滤 | 周期暴露过高，候选过少 |
| **B: 单行业上限** | 单行业≤30% | 分散风险 | 可能进一步减少候选数量 |
| **C: 加入银行** | 降低质量门槛或单独选银行 | 增加候选，降低波动 | 银行ROA低，可能拖累收益 |
| **D: 扩大PB范围** | PB10%→PB20% | 候选增多，参考报告#11 | 收益可能下降（但仍优于原版） |

### 6.3 推荐方案

**🎯 推荐：方案D（扩大PB范围至20%）**

理由：
1. 报告#11已验证PB20%年化收益15.03%，优于原版
2. PB10%过于严格，导致候选过少，实操困难
3. PB20%可以增加银行、多元金融等行业的暴露
4. 在候选数量和质量之间取得平衡

**备选：方案B（单行业上限30%）**

如果坚持PB10%，建议：
1. 设置单行业上限30%（钢铁行业不超过1只）
2. 当候选不足时，放宽至RFScore6或扩大PB至20%
3. 增加现金仓位作为缓冲

## 七、结论与行动建议

### 7.1 隐藏暴露总结

| 暴露 | 占比 | 是期望的吗？ |
|------|------|--------------|
| 钢铁 | 43% | ⚠️ 部分意外，周期暴露过高 |
| 房地产 | 14% | ⚠️ 部分意外，杠杆行业风险 |
| 央国企 | 100% | ✅ 是期望的，质量筛选结果 |
| 银行 | 0% | ❌ 意外缺失，质量门槛过滤 |

### 7.2 核心结论

1. **策略本质**：押注"高质量周期龙头的估值修复"，不是纯价值，也不是纯质量
2. **隐藏风险**：过度集中于钢铁等强周期行业，经济下行期风险大
3. **实操困难**：PB10%过于严格，多数月份无候选或候选过少

### 7.3 行动建议

**立即执行：**
- [ ] 将PB10%调整为PB20%（参考报告#11结论）
- [ ] 或设置单行业上限30%，防止过度集中

**中期优化：**
- [ ] 研究是否放宽质量门槛以纳入银行（如RFScore6+银行）
- [ ] 增加现金缓冲机制，应对候选不足的情况

**风险监控：**
- [ ] 持续监控钢铁行业景气度
- [ ] 当单一行业占比>50%时，发出风险警告

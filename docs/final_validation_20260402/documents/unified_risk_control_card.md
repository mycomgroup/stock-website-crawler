## 五、应用顺序

```
信号 → 情绪过滤 → 买入 → 卖出执行 → 仓位检查 → 循环
```

1. **信号过滤**：假弱高开定义（+0.5%~+1.5%，市值50-150亿，相对位置≤30%）
2. **情绪门槛**：涨停家数≥30只（不达标不交易）
3. **买入执行**：当日买入
4. **卖出执行**：次日冲高+3%卖 / 14:50尾盘
5. **止损执行**：低开>4%开盘即止损
6. **仓位管理**：每笔后检查近10笔收益状态，自动切半仓/正常仓

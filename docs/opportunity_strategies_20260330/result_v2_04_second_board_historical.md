## 七、跨周期稳定性判定

### 7.1 稳定性判定

**判定：稳定**

**判定依据：**

| 门槛 | 要求 | 状态 | 说明 |
|------|------|------|------|
| 门槛1 | 2021-2023至少2年收益为正 | ✓ 通过 | 3年均估算为正收益 |
| 门槛2 | 各年胜率差异<15% | ✓ 通过 | 差异约10% |
| 门槛3 | 能给出稳定性判定 | ✓ 通过 | 基于估算+实测可判定 |

### 7.2 不稳定年份分析

**无显著不稳定年份**

- 2022年信号偏少（市场情绪低迷），但收益仍为正
- 2024年表现异常优异，需警惕是否为特殊年份

### 7.3 失效条件识别

**已识别的失效场景：**

1. **情绪持续低迷**（涨停家数连续<10）
   - 触发条件：市场进入深度熊市
   - 影响：交易机会减少，收益下降
   - 对策：严格执行情绪开关

2. **二板后次日热点切换**
   - 触发条件：板块轮动加速
   - 影响：次日收益低于预期
   - 对策：选择热点延续个股

3. **小市值流动性问题**
   - 触发条件：市值过小
   - 影响：滑点增大
   - 对策：设置市值下限（5亿）

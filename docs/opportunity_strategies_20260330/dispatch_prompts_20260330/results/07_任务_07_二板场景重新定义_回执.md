## 一、最终保留的二板定义

### 观察版规则 V1.0

```
选股条件：
  1. 昨日二板（非一字板）
  2. 换手率 < 30%
  3. 缩量：昨日量 ≤ 前日 × 1.875
  4. 情绪开关：涨停家数 ≥ 10（可选）
  5. 市值：按流通市值从小到大排序

买入条件：
  - 非涨停开盘（开盘价 < 涨停价）
  - 市价单买入
  - 滑点：+0.5%

卖出条件：
  - 次日卖出

仓位控制：
  - 单票上限：300万元
  - 总仓位：情绪动态调整
```

### 两类场景

| 场景 | 定义 | 优先级 |
|------|------|--------|
| **场景A**：非涨停开盘后的承接转强 | 开盘价<涨停价，观察盘中承接 | 优先 |
| **场景B**：小幅高开但可成交 | 开盘价略高于昨收但未涨停 | 补充 |

## 二、被删除的规则清单

### 2.1 删除的板位

| 删除项 | 原因 |
|--------|------|
| **三板** | 交易机会过少（20次），样本不足 |
| **四板** | 逻辑复杂，风险高，回撤大（0.92%） |

### 2.2 删除的成交假设

| 删除项 | 原因 |
|--------|------|
| **涨停排板30%成交率** | 无法验证，纯假设 |
| **涨停排板10%成交率** | 成交概率过低，不具可执行性 |

### 2.3 删除理由

1. **聚焦二板**：样本最多、胜率最高、收益最高、回撤可控
2. **删除不可执行场景**：涨停排板无法验证真实成交率
3. **保证可复盘**：只保留100%可成交的场景

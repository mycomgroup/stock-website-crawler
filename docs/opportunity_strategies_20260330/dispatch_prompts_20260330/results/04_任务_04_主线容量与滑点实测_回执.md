## 关键问题回答

### 1. 资金从哪一档开始明显恶化？

**答案：1000万之后**

数据支撑：
- 500万 → 1000万：年化衰减5%（轻微）
- 1000万 → 3000万：年化衰减10%（明显），回撤增加2%（明显）
- 3000万 → 5000万：年化衰减15%（严重），回撤增加5%（严重）

**恶化阈值**：
- 年化衰减超过60%
- 回撤增加超过4%
- 3000万资金档同时触发两个阈值

### 2. 小资金试盘建议上限是多少？

**答案：500万**

理由：
- 年化收益55.6%（可观正收益）
- 卡玛比率4.3（风险收益平衡良好）
- 成交额占比2.5-5%（轻松成交）
- 回撤13%（可控，低于警戒线15%）

### 3. 是否支持后续模拟盘转小实盘？

**答案：支持（需满足5项条件）**

条件：
1. 资金规模 ≤ 500万
2. 单票资金 ≤ 50万
3. 滑点控制 ≤ 0.2%
4. 成交额占比 ≤ 10%
5. 模拟盘验证 ≥ 3个月

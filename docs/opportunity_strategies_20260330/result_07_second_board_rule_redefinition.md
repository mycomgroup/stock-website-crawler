## 二、最终保留的二板定义

### 2.1 唯一观察版规则

**【二板观察版 V1.0】**

```
选股条件：
  1. 昨日二板（昨日收盘为第二个涨停板）
  2. 非一字板（排除一字涨停）
  3. 换手率 < 30%
  4. 缩量条件：昨日成交量 ≤ 前日 × 1.875
  5. 情绪开关：市场涨停家数 ≥ 10（可选）
  6. 排序：按流通市值从小到大，取最小

买入条件：
  - 开盘价 < 涨停价（非涨停开盘）
  - 买入方式：市价单
  - 滑点假设：+0.5%

卖出条件：
  - 持仓周期：1天
  - 卖出方式：次日开盘卖出（或最高价卖出）

仓位控制：
  - 单票上限：300万元
  - 总仓位：根据情绪动态调整
  - 情绪好（涨停≥20）：可满仓
  - 情绪差（涨停<10）：空仓
```

### 2.2 二板的两类场景对比

| 场景类型 | 定义 | 可执行性 | 样本数 | 胜率 | 累计收益 |
|----------|------|----------|--------|------|----------|
| **场景A：非涨停开盘后的承接转强** | 开盘价<涨停价，观察盘中承接强度 | 高（市价可成交） | 多数 | 87.95% | 394.61% |
| **场景B：小幅高开但可成交** | 开盘价略高于昨收但未涨停 | 中（滑点可能更大） | 少数 | 待验证 | 待验证 |

**对比结论**：
- **场景A优先**：样本多、可执行性高、历史表现好
- **场景B作为补充**：可少量观察，但不作为主要场景

### 2.3 规则简化版本（便于执行）

**核心三要素**：
1. 昨日二板 + 非一字板 + 换手率<30% + 缩量
2. 今日非涨停开盘
3. 市值最小优先

**情绪开关（可选）**：
- 涨停家数≥10：开仓
- 涨停家数<10：观望

## 四、推荐观察仓位上限

### 4.1 单票仓位上限

**推荐：300万元**

**理由**：
1. 策略选股为小市值（5-20亿流通市值）
2. 小市值股票流动性有限
3. 单票300万可避免明显滑点
4. 符合result_03中的建议（300-500万）

### 4.2 总仓位上限

| 情绪状态 | 涨停家数 | 总仓位上限 | 说明 |
|----------|----------|------------|------|
| 高情绪 | ≥20 | 100% | 积极开仓 |
| 中等情绪 | 10-20 | 50% | 谨慎开仓 |
| 低情绪 | <10 | 0% | 空仓观望 |

### 4.3 仓位管理建议

```
单票仓位 = min(300万, 总仓位 × 10%)
最大持仓数 = 总仓位 / 300万
```

**示例**：
- 总资金1000万，情绪≥20
- 单票上限：min(300万, 1000万×10%) = 100万
- 最大持仓：1000万 / 300万 = 3票

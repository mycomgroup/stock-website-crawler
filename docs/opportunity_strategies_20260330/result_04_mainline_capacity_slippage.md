## 三、实测设计

### 3.1 测试参数

**资金档位**：
| 档位 | 单票资金 | 说明 |
|------|----------|------|
| 500万 | 单票≤50万 | 小资金试盘上限 |
| 1000万 | 单票≤100万 | 小实盘上限 |
| 3000万 | 单票≤300万 | 中资金上限 |
| 5000万 | 单票≤500万 | 大资金上限 |

**滑点档位**：
| 档位 | 滑点值 | 场景说明 |
|------|--------|----------|
| 0.2% | 买卖各0.2% | 保守估计（低流动性） |
| 0.5% | 买卖各0.5% | 正常估计（中流动性） |
| 1.0% | 买卖各1.0% | 悲观估计（高流动性冲击） |

**实际限制**：
1. **手续费**：买入0.03%，卖出0.03% + 印花税0.1% = 总成本0.16%
2. **成交额占比上限**：单票成交额≤10%当日成交额
3. **无法无限成交**：实际成交受流动性限制

### 3.2 理论计算方法

#### 收益衰减公式

```
实际单日收益 = 基准单日收益 - 手续费损耗 - 滑点损耗 - 流动性损耗

手续费损耗 = 买手续费0.03% + 卖手续费0.03% + 印花税0.1% = 0.16%
滑点损耗 = 滑点×2（买+卖）
流动性损耗 = 根据成交额占比估算
```

#### 年化收益估算

```
实际年化 = 基准年化 × (实际单日收益 / 基准单日收益)
```

#### 回撤估算

```
实际回撤 = 基准回撤 + 流动性冲击增回撤
流动性冲击增回撤 = 单票资金占比 × 滑点档位系数
```

### 8.2 滑点控制执行要点

**买入执行**：
- 集合竞价观察开盘价，开盘立即买入
- 使用限价单，价格=开盘价×1.002（预留0.2%滑点）
- 如果开盘价偏离预期>0.5%，放弃买入

**卖出执行**：
- 次日冲高时卖出，设置动态止盈线
- 使用限价单，价格=当前价×0.998（预留0.2%滑点）
- 如果冲高不足3%，收盘前卖出

## 九、风险提示

### 9.1 主要风险

1. **滑点超标风险**：实际执行滑点可能超过0.2%，导致收益转负
2. **流动性风险**：目标股票成交额不足，无法按计划买入
3. **市场风格风险**：首板低开策略依赖市场情绪，情绪低迷时失效
4. **回撤风险**：虽然理论回撤13%，但实际可能超过15%

### 9.2 应对措施

1. **严格执行滑点控制**：开盘价买入，预留0.2%滑点，超标放弃
2. **成交额占比检查**：单票不超过10%，超标分批或放弃
3. **情绪过滤**：涨停家数≥50时执行，低于50暂停
4. **止损机制**：单笔亏损≥4%止损，连续亏损暂停策略

**推荐版本**：放宽C（市值40-200亿 + 位置≤50% + 假弱高开）

**完整规则**：

```python
筛选条件：
1. 昨日涨停（收盘价=涨停价，误差≤1%）
2. 今日假弱高开（开盘涨幅+0.5%~+1.5%）
3. 流通市值40-200亿
4. 相对位置≤50%（15日内）
5. 近1日无涨停（避免连板接力）

买入时间：开盘价买入（9:30）
卖出规则：次日冲高+3%即卖，否则尾盘卖出
```

- 全年信号数：约968个（日均4.00个）

- 平均最大收益：+4.56%

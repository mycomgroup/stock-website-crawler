# 二板观察线固化结果

> 固化日期：2026-04-03
> 研究范围：仅二板，不涉及三板/四板扩展
> 输出目的：直接指导"小仓观察线"使用

---

## 一、三个版本对比

| 版本 | 情绪阈值 | 交易次数 | 胜率 | 累计收益 | 年化收益 | 最大回撤 | 盈亏比 |
|------|---------|---------|------|---------|---------|---------|--------|
| **A：无情绪** | 无 | 139 | 86.33% | 671.90% | 694.12% | 0.58% | - |
| **B：情绪≥10** | ≥10 | 83 | 87.95% | 394.61% | 407.65% | 0.60% | 21.91 |
| **C：情绪≥15+缩量** | ≥15+缩量 | 47 | 91.49% | 240.44% | 248.39% | 0.32% | 28.61 |

---

## 二、推荐版本

### 【二板观察线 V1.0 推荐版】

| 维度 | 内容 |
|------|------|
| **核心条件** | 昨日二板 + 非一字板 + 换手率<30% + 缩量（量≤前日×1.875） |
| **情绪开关** | 涨停家数≥10 |
| **买入执行** | 非涨停开盘，市价单，滑点+0.5% |
| **卖出执行** | 次日开盘价/最高价卖出，持仓1天 |
| **排序逻辑** | 流通市值从小到大，取最小 |

### 为何选B而非A或C

- **vs A（无情绪）**：交易次数接近（139 vs 83），但B版胜率更高（87.95% vs 86.33%），盈亏比明确（21.91），情绪开关可规避弱市
- **vs C（情绪≥15+缩量）**：C版胜率更高但交易机会少一半（47次），年化收益低约40%，容量有限不适合观察线
- **B版平衡点**：83次交易机会 + 87.95%胜率 + 0.60%回撤 + 情绪过滤

## 四、适用边界

| 边界条件 | 说明 |
|---------|------|
| **市值范围** | 小市值（流通市值5-20亿最优），市值太小流动性不足 |
| **情绪下限** | 涨停<10家 → 空仓观望，不开仓 |
| **情绪上限** | 涨停≥20 → 可积极开仓，总仓位可提高 |
| **板位确认** | 仅限二板，三板/四板已删除 |
| **成交确认** | 必须非涨停开盘，一字板/涨停开盘放弃 |

## 五、容量与执行风险

| 风险类型 | 描述 | 应对 |
|---------|------|------|
| **单票容量** | 小市值股票流动性有限 | 单票上限300万元 |
| **滑点风险** | 回测+0.5%，实盘可能更高 | 预留至+1%空间 |
| **成交率** | 非涨停开盘=100%成交（已验证） | 严格执行非涨停开盘 |
| **策略容量** | 估计5000万-1亿 | 大资金需分散多个观察线 |
| **情绪周期** | 弱市时交易机会锐减 | 情绪<10时空仓 |

---

## 六、为什么只能做观察线而非主仓

1. **容量硬伤**：小市值选股（流通市值最小优先），单票上限300万，总容量有限
2. **机会依赖情绪**：83次/年（情绪≥10），弱市年份可能仅20-30次，容量无法撑满
3. **持仓周期短**：仅1天持仓，无法承载大资金进出
4. **回撤虽低但收益集中**：年化400%收益依赖高胜率复利，大资金滑点会显著拉低收益
5. **观察线优势**：仓位轻（单票300万）+ 情绪开关可空仓 = 适合作为主仓的补充验证

**结论**：二板观察线是"小资金、高胜率、情绪驱动"的精确制导武器，而非大资金主战场。

## 七、仓位管理

### 单票仓位上限
- 推荐：300万元
- 理由：小市值流动性有限，300万可避免明显滑点

### 总仓位动态调整

| 情绪状态 | 涨停家数 | 总仓位上限 |
|----------|----------|------------|
| 高情绪 | ≥20 | 100% |
| 中等情绪 | 10-20 | 50% |
| 低情绪 | <10 | 0%（空仓） |

### 仓位计算公式
```
单票仓位 = min(300万, 总仓位 × 10%)
最大持仓数 = 总仓位 / 300万
```

## 八、核心过滤条件（执行清单）

```
1. 昨日二板（收盘确认）
2. 非一字板（盘中观察）
3. 换手率 < 30%（收盘计算）
4. 缩量：昨日量 ≤ 前日量 × 1.875（收盘计算）
5. 情绪：涨停家数 ≥ 10（收盘确认）
6. 市值排序：流通市值从小到大，取最小
7. 买入：非涨停开盘，市价单
8. 卖出：次日开盘/最高价，持仓1天
```

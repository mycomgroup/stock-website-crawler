## 五、停机方案效果分析

### 5.1 为什么连亏停机反而增加回撤？

**原因分析**：
1. **错过反弹**：连亏后往往是反弹机会，停机反而错过
2. **停机时机不对**：连亏3笔时可能已经接近底部
3. **样本量不足**：最大连亏仅5笔，停机触发频繁但效果不佳

### 5.2 为什么近10笔半仓效果最好？

**原因分析**：
1. **不完全停机**：保持市场参与，不错过反弹
2. **动态调整**：根据近10笔收益自动切换
3. **降低风险**：半仓运行降低风险敞口

### 7.2 实施要点

```
近10笔半仓规则：
  1. 记录最近10笔交易收益
  2. 累计收益转负 → 半仓运行
  3. 累计收益转正 → 恢复正常仓位
  4. 盘中可执行：每日收盘后计算
  
执行逻辑：
  1. 每日收盘后计算近10笔收益
  2. 如转负，次日半仓运行
  3. 如转正，次日恢复正常仓位
```

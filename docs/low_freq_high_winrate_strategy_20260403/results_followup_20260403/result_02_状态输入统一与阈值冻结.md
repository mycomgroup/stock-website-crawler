## 一、四类状态输入定义表

### 1. 市场宽度（Breadth）

| 维度 | 冻结口径 |
|------|---------|
| **定义** | 沪深300成分股中，收盘价 > 20日移动平均线的股票占比 |
| **计算公式** | 宽度 = count(close > MA20) / total(沪深300成分股) × 100% |
| **样本池** | 沪深300指数成分股（排除ST、新股、科创板、北交所） |
| **更新频率** | 日频（收盘后计算）；状态路由器每周运行一次 |
| **数据来源** | 聚宽/米筐：`get_index_stocks('000300.XSHG')` + 行情数据 |
| **用途** | 状态路由器主判定指标；择时集成中的广度开关 |

### 2. FED指标（估值）

| 维度 | 冻结口径 |
|------|---------|
| **定义** | 股市盈利收益率与无风险利率之差，衡量股票相对债券的估值吸引力 |
| **计算公式** | FED = (100 / 沪深300 PE中位数) - 10年期国债收益率 |
| **样本池** | 沪深300成分股PE中位数；10年期国债收益率（中国债券信息网/聚宽宏观数据） |
| **更新频率** | 日频（PE每日更新，国债收益率每日更新） |
| **数据来源** | 聚宽：`get valuation` 获取PE，宏观数据获取国债收益率 |
| **用途** | 辅助估值判断（低估/中性/高估），配合宽度做状态判定 |

### 3. RSRS（趋势）

| 维度 | 冻结口径 |
|------|---------|
| **定义** | 阻力支撑相对强度（Resistance Support Relative Strength），通过滚动OLS回归斜率衡量趋势方向 |
| **计算公式** | 对沪深300指数进行N日滚动OLS回归（最高价 vs 最低价），取斜率序列，再计算Z-Score标准化 |
| **样本池** | 沪深300指数（000300.XSHG）日K线最高价、最低价 |
| **更新频率** | 日频（收盘后计算） |
| **数据来源** | 聚宽：`get_price('000300.XSHG', fields=['high', 'low'])` |
| **用途** | 趋势方向判定（向上/震荡/向下），配合宽度做状态判定 |

### 4. 情绪（Sentiment）

| 维度 | 冻结口径 |
|------|---------|
| **主指标** | 当日涨停家数（涨幅 >= 9.5%） |
| **辅助指标** | 跌停家数、最高连板数、晋级率（二板数/首板数）、龙头状态 |
| **样本池** | 全市场A股（排除ST、新股、科创板68开头、北交所/创业板3开头、4/8开头） |
| **更新频率** | 日频（盘前9:00用T-1日数据计算，避免未来函数） |
| **数据来源** | 聚宽/米筐：全市场行情涨停/跌停统计 |
| **用途** | 短线策略择时开关（开仓/关仓/仓位调节） |

## 二、冲突清单

以下按严重程度排列，标注来源文档及冲突内容。

### 冲突 1：宽度阈值体系不同（严重）

| 维度 | 状态路由器V2 (Doc 09/11) | 择时集成文档V1.0 |
|------|------------------------|-----------------|
| **关闭** | 无此状态 | <15% |
| **底部试错/防守** | <30% | 15-25% |
| **震荡轮动/正常** | 30-50% | 25-35% |
| **趋势进攻/进攻** | >50% + 趋势向上 | >=35% + 情绪>=80 |
| **高估防守** | >70% | 无此状态 |

**冲突原因**: 两套体系服务于不同场景。状态路由器V2用于**中低频资产配置**（周频），择时集成V1.0用于**短线二板接力**（日频）。阈值差异反映的是策略频率和风险偏好的不同。

**统一方案**: **保留两套阈值，明确用途边界**。状态路由器阈值用于资产配置决策，择时集成阈值用于短线交易开关。两者不混用。

### 冲突 2：情绪阈值定义不同（严重）

| 维度 | 择时集成文档V1.0 | 情绪退潮指南 |
|------|----------------|-------------|
| **冰点/退潮** | <30只 | <20只（退潮），<10只（冰点） |
| **启动** | 30-50只 | 10-30只 |
| **发酵/正常** | 50-80只 | 30-50只 |
| **高潮/进攻** | >80只 | 50-80只 |
| **退潮期** | 无独立定义 | 涨停<20 + 跌停>30 + 连板<2 |

**冲突原因**: 择时集成文档的"冰点"对应情绪退潮指南的"启动期"，两者口径错位。退潮指南更细粒度，区分了冰点和退潮两个阶段。

**统一方案**: **以择时集成文档V1.0的>=30只为短线开仓开关阈值**（经提示词8.1验证，IS/OOS一致）。退潮指南的五阶段模型作为**辅助诊断工具**，不直接用于开仓决策。

### 冲突 3：FED计算中PE口径（中等）

| 维度 | Doc 09 | Doc 11 |
|------|--------|--------|
| **PE取值** | 未明确标注"中位数" | 明确标注"沪深300 PE中位数: 21.2" |
| **公式** | 100/PE - 1.8% | 100/PE中位数 - 国债收益率 |
| **国债收益率** | 固定1.8% | 动态取值（未明确期限） |

**冲突原因**: Doc 09使用固定国债收益率1.8%，Doc 11使用动态值但期限不明。PE口径Doc 11明确为中位数，Doc 09未标注。

**统一方案**: **PE使用沪深300成分股PE中位数**；**国债收益率使用10年期国债收益率（动态取值）**。不再使用固定1.8%。

### 冲突 4：RSRS用法差异（中等）

| 维度 | Doc 09 | Doc 11 | Doc 15 |
|------|--------|--------|--------|
| **状态** | 成功运行，Z-Score=0.24 | 计算失败，默认震荡 | 未使用RSRS |
| **用途** | 趋势判定（向上/震荡/向下） | 趋势判定 | 使用风险溢价和牛熊指标替代 |
| **阈值** | 震荡区间(-0.5, 0.5) | 无 | 无 |

**冲突原因**: RSRS在Doc 11中因技术问题计算失败，Doc 15完全未使用RSRS而改用其他择时指标。

**统一方案**: **RSRS保留为趋势信号指标**，阈值：Z-Score > 0.5 为趋势向上，<-0.5 为趋势向下，中间为震荡。需确保计算稳定性（Doc 11失败问题需修复）。

### 冲突 5：更新频率不一致（轻微）

| 指标 | Doc 09 | Doc 11 | 择时集成V1.0 |
|------|--------|--------|-------------|
| **宽度** | 每周运行 | 每日更新即可 | 日频（收盘后） |
| **FED** | 未明确 | 未明确 | 未涉及 |
| **RSRS** | 未明确 | 未明确 | 未涉及 |
| **情绪** | 未涉及 | 未涉及 | 日频（9:00用T-1日数据） |

**统一方案**: 宽度/FED/RSRS **日频计算、周频决策**；情绪 **日频计算、日频决策**（短线策略需要）。

### 冲突 6：格雷厄姆指数是否纳入（轻微）

| 维度 | Doc 11 | 其他文档 |
|------|--------|---------|
| **格雷厄姆指数** | 作为估值辅助指标（>1.5表示低估） | 未提及 |

**统一方案**: **格雷厄姆指数作为FED的辅助验证指标，不单独作为状态输入**。当FED与格雷厄姆指数信号一致时增强置信度，不一致时以FED为准。

| 指标 | 状态路由器V2 | 择时集成V1.0 | 情绪退潮指南 | 用途类型 |
|------|------------|-------------|-------------|---------|
| **宽度** | ✓ 主判定 | ✓ 广度开关 | - | 确认 + 仓位映射 |

- **仓位映射**: 宽度决定资产配置的基准仓位，情绪决定短线策略的仓位上限

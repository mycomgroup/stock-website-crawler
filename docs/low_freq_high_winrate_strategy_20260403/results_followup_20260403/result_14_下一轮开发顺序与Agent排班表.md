## 下一轮开发顺序

### P0 立即进入开发（已完成验证，具备开发条件）

#### 1. 主仓动态路由V1代码实现

- **优先级**: P0
- **前置**: result_04（V1规格书已完成）
- **开发内容**: 
  - 四状态判定逻辑实现（宽度/FED/RSRS/情绪）

- **预计工时**: 1-2天

#### 3. 稳健型ETF策略（#85）→ 主仓补充线

## 暂停推进清单

### 明确停止的方向

| 方向 | 来源 | 停止理由 |
|------|------|---------|
| FFScore叠加RFScore | result_11（No-Go） | 指标高度重叠，2个完全相同，2个高度相关；预期改善<0.5%不显著；候选稀缺放大风险 |
| #18 20行代码躲大跌 | result_13（暂不推进） | 技术分析信号与ETF动量基线重叠；信号频率偏高；逻辑不透明难转译 |
| #53 PE标准差定投模式 | result_13（暂不推进） | 定投模式与主仓选股模式不兼容；与PB10%估值逻辑重叠 |
| 微盘股400每日轮动 | result_13 | 高换手（日频），与低频稳健约束不符 |
| 小市值/板效应策略 | result_13 | 高风险A股个股，容量有限，与低频稳健主仓定位不符 |

### 暂缓验证的RSRS版本

| 版本 | 暂缓理由 |
|------|---------|
| V1斜率指标 | 阈值固定，已被标准分全面替代 |
| V2标准分 | 已被V3/V4替代 |
| V3修正标准分 | 净值反而低于标准分 |
| V5价格优化右偏 | 引入MA(20)外部过滤，偏离纯RSRS定位 |
| V6量价相关优化 | MaxDD风险失控 |
| V7/V8线性/指数加权 | 2019报告明确效果不佳 |
| V9成交额加权（不含钝化） | 提升有限，已并入V11 |

### ML方向暂停条件

若以下条件出现，ML方向全面暂停：
1. 修复幸存者偏差后，季频+10只胜率<70%
2. 扩维因子IC均值<-5%且IR<0.3
3. 连续2个季度跑输RFScore7基线5%以上

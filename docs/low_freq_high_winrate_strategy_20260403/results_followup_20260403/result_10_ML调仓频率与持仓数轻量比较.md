## 主推荐

### ✅ **A: 季频 + 10只**

#### 推荐理由

1. **4档中综合评分最高**（15.66分），排名一致第一
2. **回撤最小**（-26.86%），仅为月频配置的一半
3. **胜率最高**（52.6%，10/19季盈利），与季频+15只并列但回撤更小
4. **年化亏损最小**（-1.74%），在4因子基线不足的情况下损失最少
5. **执行最简单**：每年4次调仓，每次10只股票
6. **最符合"低频主仓"定位**：
   - 主仓策略的核心是"稳"而非"快"
   - 低换手=低犯错概率=高可执行性
   - 在不利市场环境下回撤控制最好

#### 操作规格

```
模型: LogisticRegression(C=100, max_iter=500, random_state=42)
因子: 4因子基线（pe_ratio, pb_ratio, roe, roa）
     → 待升级为13因子短名单（result_09）
股票池: 中证500 (000905.XSHG)
调仓频率: 季度（每年1/4/7/10月第一个交易日）
持仓数量: 10只（等权配置）
训练窗口: 4个季度（12个月）
交易成本: 双边0.2%
```

#### ⚠️ 重要前提

**当前4因子基线在完整周期下收益为负**，因此"季频+10只"作为主推荐的前提是：
- 必须配合13因子短名单（result_09）升级
- 必须配合RSRS过滤层（任务07）做弱市空仓
- 必须在因子升级后重新验证

## 不推荐项

### ❌ D: 月频 + 15只（最不推荐）

**淘汰理由**：
1. **累计收益最差**：-29.72%，4档中亏损最多
2. **回撤最大**：-58.29%，超出任何主仓策略可接受范围
3. **胜率最低**：46.4%（26/56月），低于随机水平
4. **双重劣势叠加**：月频信号噪声 + 15只尾部拖累
5. **与主仓定位完全冲突**

### ❌ C: 月频 + 10只

**不推荐理由**：
1. **回撤过大**：-55.07%，为季频的2倍
2. **胜率不足**：48.2%（27/56月），低于50%
3. **年化亏损**：-3.63%，为季频的2倍
4. **夏普虽非最差**（-0.012），但这是月频期数更多带来的统计偏差，实际风险调整后收益并不优
5. **适用场景错位**：月频更适合"卫星策略"而非主仓

### ⚠️ B: 季频 + 15只（备选）

**不推荐理由**：
1. **累计收益更差**：-15.44% vs -8.01%（A档）
2. **回撤更大**：-28.26% vs -26.86%
3. **胜率相同但收益更低**：同为52.6%，但15只引入尾部拖累
4. **夏普更差**：-0.388 vs -0.129

**适用场景**：仅在资金规模>500万、10只集中度导致冲击成本过高时考虑。

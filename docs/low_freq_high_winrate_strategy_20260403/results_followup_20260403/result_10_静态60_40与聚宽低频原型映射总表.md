## 一、静态 60/40 结论

### 1.1 组合结构

| 组件 | 权重 | 选股逻辑 | 持仓数 | 调仓频率 |
|------|------|----------|--------|----------|
| **RFScore7 进攻仓** | 60% | PB低20% + ROA/ROE排序，沪深300+中证500 | 15只 | 月频 |
| **红利小盘防守仓** | 40% | PE<30 + 净利润增速>5% + 市值10-100亿 | 10只 | 月频 |

**总持仓**: 25只股票，固定60/40比例，不做动态权重调整

### 1.2 实测表现（2022-2025）

| 指标 | 数值 | 备注 |
|------|------|------|
| 年化收益 | +8.6% | |
| 最大回撤 | **-15.0%** | 所有版本中最小 |
| 夏普比率 | 0.55 | |
| 月胜率 | 55% | |

### 1.3 核心优点

1. **回撤最小**：-15.0% 优于 V1 纯进攻（-16.0%）和其他动态版本
2. **攻守均衡**：60% 进攻仓获取牛市弹性，40% 防守仓提供防御
3. **逻辑简洁**：固定比例无需市场状态判断，避免动态调仓的时机失误
4. **月频低换手**：减少交易成本和滑点损耗

### 1.4 现存不足

1. 无宏观视角：组合只看估值/质量，不考虑经济周期位置
2. 无跨资产分散：纯 A 股股票，无债券/黄金/纳指等外部资产
3. 无估值仓位调节：PE/PB 仅用于选股，不参与仓位决策
4. 无风险开关机制：熊市期间全程持仓，下跌全程承受

## 四、可复用要点

### 4.1 #85 稳健型ETF → 主仓补充线（优先级：⭐⭐⭐⭐⭐）

**复用价值**：
- 月频 MACD 多资产轮动（沪深300/纳指/红利/黄金/国债），低频稳健
- 与现有 RFScore7 形成股/债/商品全覆盖，弥补无外盘/黄金的缺陷
- 回撤仅 5.86%，可作为主仓的"防守增强层"

**复用方式**：
- 提取 MACD 月线判断函数，改写为月频调仓模块
- 接入仓位框架，占比建议 10-15%，替代现有部分红利小盘或国债固收+
- 当前市场（国债动量最强），MACD 月线可自动切换到防守资产

**与 60/40 的互补点**：

| 60/40 现有层 | 互补缺陷 | #85 可补充 |
|-------------|----------|-----------|
| RFScore7 进攻（60%） | 只有 A 股，无外盘 | + 纳指 ETF 分散 |
| 红利小盘防守（40%） | 只有 A 股小盘，无避险资产 | + 黄金 ETF 对冲 |
| 国债固收+（隐含） | 纯固定收益，无增长 | + 红利 ETF 收益增强 |

### 4.2 #48/#21 宏观PMI → 主仓补充线（优先级：⭐⭐⭐⭐）

**复用价值**：
- PMI 斜率判断经济周期位置，填补 60/40 完全无宏观视角的空白
- mode=0（PMI 低迷期）纯债券，与当前（2026-03）市场防守阶段高度适配
- 覆盖完整牛熊周期（2013-2024），回撤仅 9%，夏普更高

**复用方式**：
- 接入月频 PMI 数据，维护 `calc_provide_and_need` 和 `calc_save` 函数
- mode=0 时保持债券/红利仓位，替代部分红利小盘仓
- 复苏确认后（mode=1/2）增加沪深300ETF/定投股比

**与 60/40 的互补点**：

| 60/40 现有层 | 互补缺陷 | #48/#21 可补充 |
|-------------|----------|----------------|
| RFScore7 + 红利小盘 | 无经济周期判断 | + PMI 驱动的股债切换 |
| 固定 60/40 比例 | 无应对经济衰退的仓位调节 | + mode=0 纯债保护 |

### 4.3 #53 PE标准差 → 仓位器（优先级：⭐⭐⭐）

**复用价值**：
- `QuantTiming.std()` 输出的 `stdpeH/stdpeL` 高低估线，可直接作为 RFScore7 仓位上限调节器
- 逻辑简单：PE > 高估线 → 压缩 RFScore7 仓位 10-15%；PE < 低估线 → 可超额配置
- 与现有 PB10% 选股逻辑协同，但承担"仓位调节"而非"选股"职责

**复用方式**：
- 将 stdpeH/stdpeL 作为过滤器门槛，不改变 60/40 固定比例结构
- 高 PE 区间：RFScore7 实际仓位 = 60% × (1 - 压缩比例)
- 低 PE 区间：RFScore7 实际仓位 = 60% × (1 + 超配比例)

**与 60/40 的互补点**：

| 60/40 现有层 | 互补缺陷 | #53 可补充 |
|-------------|----------|-----------|
| RFScore7（PE/PB选股） | 选股用PE，PE不参与仓位决策 | + PE仓位调节功能 |
| 固定60/40比例 | 无估值弹性 | + 高PE压缩仓位，低PE加仓 |

## 五、不建议复用点

### 5.1 #18 极简股债切换 — 暂不推进

| 评估维度 | 结论 |
|----------|------|
| **功能重叠** | 32日价量信号与现有 ETF 动量基线（Mom20d_Hold10d）功能高度重叠 |
| **信号频率** | 日频信号，切换频率可能过高，不符合"低频稳健"约束 |
| **逻辑不透明** | `argmax/argmin` 结构难以转译为稳健规则 |
| **实盘噪音** | 日频价量判断对市场噪音敏感，实盘表现存疑 |
| **建议** | 暂不推进；如需风险开关，优先考虑宏观PMI的mode判断 |

### 5.2 #53 定投模式 — 不适合主仓

| 评估维度 | 结论 |
|----------|------|
| **模式不兼容** | 每周/每月固定买入模式与 60/40 的月频选股模式不兼容 |
| **逻辑重叠** | PE/PB 估值判断与现有 RFScore7 的 PB10% 选股逻辑重叠 |
| **定投定位** | 更适合作为个人投资者的"工资定投补充线"，而非主仓仓位管理 |
| **建议** | 作为独立定投补充线处理，不并入主仓框架 |

### 5.3 高频/高风险原型 — 不符合主仓定位

以下聚宽原型虽然存在，但不适合作为低频稳健主仓补充：

| 原型类别 | 原因 |
|----------|------|
| 微盘股400每日轮动 | 日频换手，与低频约束不符 |
| 小市值策略（多个） | 高风险 A 股个股，不适合稳健主仓 |
| 板效应策略（首板/连板/龙头） | 事件驱动，容量有限 |
| 日频 RSRS 择时 | 信号频率偏高，与宏观/低频定位不符 |

## 六、综合结论

### 6.1 映射总表

| 聚宽原型 | 推荐角色 | 对接模块 | 优先级 |
|----------|----------|----------|--------|
| #85 稳健型ETF | **主仓底座** | 多资产轮动层（替代/增强国债固收+） | 🥇 第一 |
| #48/#21 宏观PMI | **主仓底座** | 经济周期择时层（弥补无宏观视角） | 🥈 第二 |
| #53 PE标准差 | **仓位器** | RFScore7 仓位调节过滤器 | 🥉 第三 |
| #18 极简股债切换 | **过滤器** | 风险信号（暂缓，与ETF动量重叠） | 暂缓 |

### 6.2 60/40 升级路径建议

```
当前 60/40 结构：
  RFScore7 (60%) + 红利小盘 (40%)

升级后 60/40+ 结构：
  RFScore7 (50-55%) + 红利小盘 (25-30%) + #85 ETF轮动 (10-15%) + #53 PE仓位器(调节)
  
宏观增强（独立模块）：
  #48/#21 PMI周期判断 → 在 mode=0 时提升固收比例，复苏后提升股比
```

### 6.3 关键区分原则

| 角色 | 定义 | 包含原型 |
|------|------|----------|
| **主仓底座** | 可独立承载仓位、与现有组合互补的策略思想 | #85, #48/#21 |
| **仓位器** | 不独立提供仓位，但可调节主仓比例 | #53 PE标准差 |
| **过滤器** | 仅提供信号，不参与仓位决策 | #18 极简切换（暂缓） |
| **原型参考** | 思想有价值但实盘整合难度大 | 小市值/板效应等高频策略 |

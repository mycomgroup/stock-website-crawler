# 低频高胜率策略研究总纲

日期：2026-04-03

## 1. 研究目标

基于以下三类材料：
- `聚宽有价值策略558`
- `docs`
- `strategies`

统一梳理“信号比较稀疏，但成功率/确定性较高”的现有成果，并明确：
1. 哪些方向已经有较充分结论；
2. 哪些方向适合作为主仓；
3. 哪些方向只适合作为机会仓；
4. 哪些模块最适合被复用成统一过滤器或二次进场机制。

## 2. 核心结论

### 2.1 现成最像“稀疏 + 高胜率”的

这类方向主要集中在 `docs/opportunity_strategies_20260330` 的机会策略研究中：
- `主线假弱高开`：严格版信号极少，但质量最高，适合作为极小仓机会线；
- `真低开B + 放宽C`：比严格版更平衡，适合持续跟踪；
- `二板（情绪>=15+缩量）`：胜率很高，但本质仍是事件驱动短线，不适合作为大资金主仓。

### 2.2 最像“低频高确定性主仓”的

这类方向主要来自 `strategies` 与 `聚宽有价值策略558`：
- `strategies/rfscore_defensive_dynamic_hedge.py`
- `strategies/enhanced_v2/rfscore7_pb10_optimized.py`
- `strategies/dividend_value_quality_v1/v3_combo_static_60_40.py`
- `聚宽有价值策略558/85 稳健型ETF策略.txt`
- `聚宽有价值策略558/48 一种宏观数据的中长线策略，年化15%，最大回撤9%.txt`
- `聚宽有价值策略558/18 20行代码8年高胜率躲过了牛年第一场大跌.txt`
- `聚宽有价值策略558/53 基于大盘PE标准差偏离度的聪明基金定投策略.txt`

这些方向的共同点是：
- 调仓更低频；
- 强过滤、多确认；
- 更重视防守层、空仓、债券或现金缓冲；
- 接近“只在可交易环境里出手”。

### 2.3 最值得复用的统一过滤器

后续如果要把策略改造成“低频高胜率”，最值得复用的是：
- `情绪开关阈值优化` 与 `双侧过滤`；
- `四状态路由器`（连续2日确认）；
- `卖出规则` 与 `停手机制`；
- `防守底仓 / 国债ETF / 黄金ETF / 现金缓冲`。

## 3. 分层框架

### 3.1 机会仓

只做高质量、低频的精选事件驱动：
- 主线假弱高开严格版；
- 主线放宽版（真低开B / 放宽C）；
- 二板情绪过滤版。

定位：
- 小仓；
- 高选择性；
- 不承担主账户收益责任；
- 需要显式停手和市场状态过滤。

### 3.2 主仓

适合承担中低频资金管理任务：
- `rfscore_defensive_dynamic_hedge.py`
- `rfscore7_pb10_optimized.py`
- `v3_combo_static_60_40.py`
- 聚宽稳健 ETF / 宏观慢变量 / 股债切换类原型。

定位：
- 低频；
- 强风控；
- 允许放弃部分收益换取更高确定性。

### 3.3 过滤器

过滤器不负责创造 Alpha，负责把策略变得更稳：
- 情绪阈值；
- 市场广度；
- 四状态路由；
- 空仓/半仓/防守仓切换；
- 停手规则。

### 3.4 原型库

聚宽目录中的低频原型更适合作为“思想来源”，包括：
- 稳健 ETF 轮动；
- 极简股债切换；
- PMI 宏观慢变量策略；
- PE 标准差偏离度仓位控制。

## 核心结论

| 问题 | 答案 |
|------|------|
| 主仓中 RSRS 角色 | **confirm（V4/V10）+ veto（V11）** |
| 机会仓中 RSRS 角色 | **veto（V4/V11），只做风险过滤** |
| 默认角色 | V4=confirm（主仓）/veto（机会仓），V10=confirm（主仓）/neutral（机会仓），V11=veto（双仓） |
| 主仓调节方式 | 加减法，基准仓位 ±5%，受状态上限约束 |
| 机会仓调节方式 | 折扣法，基准 × 0/0.5/1.0 |
| 更新频率 | 日频计算，主仓周频决策，机会仓日频决策 |
| 默认阈值 | V4: ±0.8, V10: ±0.7, V11: ±0.8 |

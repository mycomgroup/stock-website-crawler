## 关键冲突决议

| 冲突 | 决议 |
|------|------|
| 宽度阈值两套体系 | 保留两套，明确边界（周频资产配置 vs 日频短线交易） |
| 情绪阈值定义不同 | 以择时集成V1.0的>=30只为开仓开关阈值 |
| FED中PE口径 | PE中位数 + 10Y国债动态值（不再用固定1.8%） |
| RSRS计算失败 | 保留为趋势信号，需修复计算稳定性 |

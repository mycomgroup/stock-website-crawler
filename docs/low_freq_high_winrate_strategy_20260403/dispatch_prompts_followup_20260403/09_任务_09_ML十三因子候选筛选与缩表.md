你正在做“ML 因子池收敛”任务，不是在无约束地往模型里堆因子。

硬约束：
1. 只能从仓库已有材料中选因子。
2. 必须先形成候选池，再缩到可测试短名单。
3. 必须考虑数据可得性、更新频率、滞后要求、与现有 4 因子的重叠度。
4. 不允许一次性把所有看起来有意思的因子都放进去。

- 方法：

  2. 扩出一个“十三因子候选池”；
  3. 对每个因子评估：可得性、滞后、解释性、与基线重叠、预期增益；
  4. 缩到一个可测试短名单，并说明删减原因。
- 验证标准：

  2. 必须形成短名单；
  3. 必须解释每个删减原因。

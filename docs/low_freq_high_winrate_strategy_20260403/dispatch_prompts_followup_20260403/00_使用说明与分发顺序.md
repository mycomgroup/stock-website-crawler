# 20260403 后续探索提示词包

适用目录：`/Users/fengzhi/Downloads/git/testlixingren/docs/low_freq_high_winrate_strategy_20260403`

本轮不是重新开题，而是把“下一步应该深入哪些策略方向”拆成一批可并行、可验证、可收口的小任务。

## 使用方式

- 每个任务文件都是可单独复制的完整提示词。
- 每个任务都要求“双写回”：
  1. 主结果写入 `docs/low_freq_high_winrate_strategy_20260403/results_followup_20260403/result_XX_*.md`
  2. 简版回执写入 `docs/low_freq_high_winrate_strategy_20260403/dispatch_prompts_followup_20260403/results/XX_*.md`
- 如果任务没有完成实测或没有形成明确结论，必须写 `未完成`，不要写模糊推断。

## 统一硬约束

1. 优先基于已有 `docs / strategies / QuantsPlaybook / 聚宽有价值策略558` 材料整理，不允许脱离现有材料重新发明策略。
2. 除非任务明确要求轻量验证，否则不要重复做长周期全量回测。
3. 允许做小范围、目的明确的增量验证，但不允许大规模参数搜索。
4. 如需新增验证脚本，优先写到 `docs/low_freq_high_winrate_strategy_20260403/results_followup_20260403/scripts/`。
5. 不要修改正式线上策略代码；若确有必要，只能新增临时验证脚本或说明性配置。
6. 每个任务都必须区分：
   - 旧结论复述
   - 新增验证
   - 最终判断
7. 每个任务都必须在主结果和回执里写清：
   - 状态：`Go / Watch / No-Go / 未完成`
   - 关键结论
   - 关键证据 / 数据
   - 产出脚本 / 文件
   - 未完成项
   - 下一步建议
8. 如果不同文档之间存在冲突，必须解释冲突来源；不能跳过，也不能私自统一成一个未经说明的新口径。

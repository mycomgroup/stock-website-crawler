# 任务 05：二板规则重定义与推荐版本固化

你正在做二板研究成果的固化整理，不是在重新定义什么叫二板。

硬约束：
1. 只能基于已有二板规则重定义结果。
2. 不允许重新做三板、四板扩展题。
3. 必须明确推荐版本及其边界条件。
4. 输出必须能直接指导“小仓观察线”使用。

技术方案：
- 输入材料：
  - `docs/opportunity_strategies_20260330/result_07_second_board_rule_redefinition.md`
  - `docs/opportunity_strategies_20260329/result_03_234_board.md`
- 方法：
  1. 固化二板定义；
  2. 对比无情绪、情绪>=10、情绪>=15+缩量；
  3. 给出推荐版本与次优版本；
  4. 说明其为何只能做观察线而非主仓。
- 输出结构：
  - 推荐版本
  - 备选版本
  - 适用场景
  - 容量/执行风险
  - 明确边界

任务提示词：
请只整编已有二板研究成果，明确回答：二板观察线到底应该固化为哪个版本，为什么不是别的版本。你需要给出推荐版本、备选版本、核心过滤条件、适用边界和容量风险，并说明它为什么更适合作为小仓观察线，而不是大资金主仓。

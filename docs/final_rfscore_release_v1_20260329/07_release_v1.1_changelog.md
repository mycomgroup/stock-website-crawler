# RFScore PB10 Release V1.1 更新日志

## 更新时间
2026-03-31

## 更新内容

### 1. 调仓频率调整
- **原设置**：每月第一个交易日 09:35 调仓
- **新设置**：每周第一个交易日 09:35 调仓
- **原因**：用户反馈大盘股策略月度调仓周期较长，希望更及时调整仓位，但不需要每日调仓

### 2. 持仓数量调整
- **原设置**：双档持仓（20/10）
- **新设置**：四档持仓（10/8/6/0）

| 市场状态 | 原 V1 | 新 V1.1 |
|---------|-------|---------|
| `breadth < 0.15` | 10只（趋势走弱时空仓） | 0只（直接空仓） |
| `0.15 <= breadth < 0.25` | 10只 | 6只 |
| `0.25 <= breadth < 0.35 且趋势走弱` | 10只 | 8只 |
| 其他情况 | 20只 | 10只 |

### 3. breadth 阈值调整
- 新增 `breadth_normal = 0.35` 阈值，用于区分正常持仓和中度减仓
- 保留 `breadth_reduce = 0.25`
- 保留 `breadth_stop = 0.15`

### 4. 持仓决策逻辑优化
- **V1 版本**：
  - `breadth < 0.15 且趋势走弱` -> 空仓
  - `breadth < 0.25 且趋势走弱` -> 减仓至10只
  - 其他 -> 正常持仓20只

- **V1.1 版本**：
  - `breadth < 0.15` -> 直接空仓（不再要求趋势走弱）
  - `0.15 <= breadth < 0.25` -> 减仓至6只
  - `0.25 <= breadth < 0.35 且趋势走弱` -> 减仓至8只
  - 其他 -> 正常持仓10只

## 风险与收益预期

### 持仓数量减少的影响
1. **单只股票权重增加**：从 ~6.7% 提升到 ~10%
2. **集中度风险上升**：持仓更集中，个股波动对组合影响更大
3. **换手率可能提升**：周调仓频率高于月度，交易成本增加
4. **市场响应更及时**：能更快响应市场变化

### 预期效果
- 牛市中可能跑赢 V1（集中持仓优势）
- 震荡市中波动可能加大
- 熊市中止损更及时

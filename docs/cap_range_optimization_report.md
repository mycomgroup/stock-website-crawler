## 一、问题背景

### 1.1 v1版本问题

**当前配置**:
- 市值区间: 15-60亿
- 选股逻辑: 市值排名后30% + PB<1.5 + PE<20
- 持仓数量: 15只

**实测问题**:
- 候选池稀疏: 多个调仓日候选不足10只
- 极端情况: 部分月份候选<5只，无法有效分散
- 流动性风险: 市值过小导致冲击成本高

### 1.2 优化方向

**候选方案**:
1. 10-50亿 (保守扩展)
2. 10-80亿 (适度扩展)
3. 10-100亿 (充分扩展)
4. 20-80亿 (上移重心)
5. 20-100亿 (最大范围)

## 八、风险提示

### 8.1 主要风险

1. **流动性风险**
   - 10-20亿市值股票流动性较差
   - 大规模资金可能面临冲击成本
   - 建议资金规模<1亿

2. **市值漂移风险**
   - 牛市中股票市值上涨，超出区间
   - 熊市中股票市值下跌，低于区间
   - 需要定期调仓

3. **风格切换风险**
   - 大盘股行情时表现不佳
   - 需要配合风格判断

### 8.2 应对措施

1. **规模控制**
   - 单只股票持仓<500万
   - 总规模<1亿

2. **流动性管理**
   - 分批建仓
   - 避开涨跌停
   - 设置换手率下限

3. **风格监控**
   - 监控大小盘相对强弱
   - 适时调整仓位

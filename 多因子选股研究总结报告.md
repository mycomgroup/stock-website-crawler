1. ROE > 0
2. ROA > 0
3. 毛利率 > 30%
4. 净利润同比增长 > 0
5. 营收同比增长 > 0

### 4.2 因子有效性检验方法

#### IC值检验

```python
def calc_ic(factor_df, forward_returns):
    """计算因子IC值"""
    return factor_df.corrwith(forward_returns, axis=1)

def calc_rank_ic(factor_df, forward_returns):
    """计算因子Rank IC值"""
    return factor_df.rank(axis=1).corrwith(forward_returns.rank(axis=1), axis=1)
```

#### IC统计指标

| 指标 | 说明 |
|------|------|
| IC均值 | 因子预测能力 |
| IC标准差 | 预测稳定性 |
| IC_IR | IC均值/IC标准差 |
| IC胜率 | IC > 0的比例 |

#### 分层回测

```python
def quantile_returns(factor_df, forward_returns, n_quantiles=5):
    """计算分层收益"""
    quantiles = factor_df.rank(axis=1, pct=True)
    group_returns = {}
    for q in range(1, n_quantiles + 1):
        lower = (q - 1) / n_quantiles
        upper = q / n_quantiles
        mask = (quantiles >= lower) & (quantiles < upper)
        group_returns[f'Q{q}'] = (forward_returns * mask).mean(axis=1)
    return pd.DataFrame(group_returns)
```

## 7. 未来工作方法论

### 7.1 六条工作主线

#### 主线1：因子治理

**现状**：散落的因子研究，缺乏统一标准

**目标**：建立因子字典、因子分类体系、因子有效性档案

**具体做法**：
- 整理260个聚宽因子
- 建立因子卡片：名称、类别、计算方式、方向、适用场景、历史IC
- 按因子簇分类：估值、质量、成长、动量、情绪、流动性

#### 主线2：因子状态切换

**现状**：固定权重或固定排序

**目标**：根据市场状态动态调整因子权重

**具体做法**：
- 定义市场状态变量：波动率、市场宽度、情绪指标、拥挤度
- 研究不同状态下因子有效性的差异
- 构建因子权重的动态调整机制

#### 主线3：严格样本外验证

**现状**：简单训练/测试集切分

**目标**：滚动窗口、walk-forward验证

**具体做法**：
- 滚动窗口回测（训练2年→预测3个月→滚动）
- 跨周期验证（不同牛熊市分别测试）
- 加入真实交易约束（滑点、涨跌停、停牌、成交额限制）

#### 主线4：机器学习升级

**现状**：预测涨跌二分类

**目标**：学习股票相对排序

**具体做法**：
- 尝试pairwise ranking、listwise ranking
- 使用LambdaMART、RankNet等排序学习算法
- 评估指标从准确率改为NDCG、MAP

#### 主线5：因子自动挖掘

**现状**：人工设计因子，效率低

**目标**：自动化因子搜索+有效性验证

**具体做法**：
- 基于遗传算法挖掘因子
- 建立因子候选池→初筛→精筛→组合测试的流水线
- 设置过拟合检测机制

#### 主线6：工程化

**现状**：分散的notebook，难以维护

**目标**：模块化、配置化、可复现

**具体做法**：
```
├── data/           # 数据获取模块
├── factors/        # 因子计算模块
├── models/         # 模型训练模块
├── backtest/       # 回测引擎
├── evaluation/     # 评估指标
├── config/         # 策略配置
└── reports/        # 结果输出
```

```python
def calc_fscore_factors(stocks, date):
    """计算FScore相关因子"""
    q = query(
        valuation.code,
        indicator.roe,
        indicator.roa,
        indicator.gross_profit_margin,
        indicator.inc_net_profit_year_on_year,
        indicator.inc_revenue_year_on_year,
        valuation.pe_ratio,
        valuation.pb_ratio
    ).filter(valuation.code.in_(stocks))
    
    df = get_fundamentals(q, date=date)
    df['fscore'] = 0
    
    # 7因子评分
    df.loc[df['roe'] > 0, 'fscore'] += 1
    df.loc[df['roa'] > 0, 'fscore'] += 1
    df.loc[df['gross_profit_margin'] > 30, 'fscore'] += 1
    df.loc[df['inc_net_profit_year_on_year'] > 0, 'fscore'] += 1
    df.loc[df['inc_revenue_year_on_year'] > 0, 'fscore'] += 1
    df.loc[(df['pe_ratio'] > 0) & (df['pe_ratio'] < 30), 'fscore'] += 1
    df.loc[df['pb_ratio'] < 5, 'fscore'] += 1
    
    return df
```

#### 因子预处理代码

```python
def preprocess_factor(factor_df):
    """因子预处理：去极值、标准化"""
    # 去极值 (MAD方法)
    median = factor_df.median()
    mad = ((factor_df - median).abs()).median()
    factor_df = factor_df.clip(median - 3*1.4826*mad, median + 3*1.4826*mad)
    
    # 标准化
    factor_df = (factor_df - factor_df.mean()) / factor_df.std()
    
    return factor_df
```

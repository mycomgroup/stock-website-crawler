什么是代码解释器 MCP 服务？

代码解释器是一个可以执行代码的组件，阿里云百炼提供部署在云上的沙盒（sandbox）环境，用于安全地执行代码，防止恶意操作影响系统。

当智能体需要一个执行 Python 代码的环境时，可以使用代码解释器 MCP 服务，安全、隔离的运行代码，用于数学计算、数据分析与可视化、数据处理等任务。当前代码解释器的 Python 运行版本为 v3.10。

如何在阿里云百炼上使用代码解释器 MCP 服务？

阿里云百炼已为您部署好了云端的代码解释器 MCP 服务，您仅需在阿里云百炼确认开通后使用。代码解释器运行在公共的函数计算实例上，按照会话隔离，所有临时文件都不会保留，请勿进行违法操作。

目前 MCP 服务已支持添到智能体和工作流中。

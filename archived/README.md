# 归档文件说明

这个文件夹包含了项目早期开发过程中的测试代码、调试脚本和临时文件。

## 文件分类

### 早期测试脚本
- `check-missing.js` - 检查缺失链接的脚本
- `crawl-doc-to-md.js` - 早期的文档爬取脚本
- `debug-extract-examples.js` - 调试示例提取
- `debug-find-all-links.js` - 调试链接查找
- `debug-measures-page.js` - 调试措施页面
- `fetch-api-links.js` - 获取API链接
- `login.js` - 早期登录测试
- `test-api.js` - API测试
- `test-single-api.js` - 单个API测试

### 调试数据文件
- `api-links.json` - API链接数据
- `captured-responses.json` - 捕获的响应数据
- `debug-links.json` - 调试链接数据
- `test-links.txt` - 测试链接列表
- `test-measures.txt` - 测试措施列表
- `links.txtbak` - 链接备份

### 日志文件
- `crawl-log.txt` - 爬取日志
- `crawl-output.log` - 爬取输出日志

### 调试截图
- `after-login.png` - 登录后截图
- `debug-doc-page.png` - 文档页面调试截图
- `debug-measures.png` - 措施页面调试截图
- `debug-page.png` - 页面调试截图

### 临时文件
- `table-debug.html` - 表格调试HTML

## 注意

这些文件已经不再使用，保留仅供参考。当前项目的主要代码在 `stock-crawler/` 目录中。

## 实测数据（result_06）

### 情绪择时效果

| 策略 | 年化收益 | 最大回撤 | 夏普比率 |

| 无择时(A组) | -28.91%（样本外） | -40.34% | N/A |

| 特性 | 原始版 | 增强版 |

## 核心模块说明

### 1. SentimentSwitch（情绪开关）

```python
开仓条件：涨停>=15 + 连板>=2
仓位映射：冰点0% / 低迷30% / 中性60% / 活跃80% / 狂热100%
```

### 2. FourTierPosition（四档仓位）

```python
正常(>=40%): 15只
防守(25-40%+trend_off): 12只
底部(15-25%): 10只
极端(<15%): 0只
```

### 3. RiskControl（风控模块）

```python
10:30时间止损（未盈利即走）
跳空止损（低开4%）
组合熔断（周亏8%/月亏15%）
```

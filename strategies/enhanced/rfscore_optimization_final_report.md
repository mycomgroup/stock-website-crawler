### 4.2 方案B：分级筛选策略

```python
# 分级筛选（渐进式）
def tiered_selection(stocks, date, target=10):
    """
    分级筛选策略
    """
    # 第一级：基础池（PB<2.0）
    pool_1 = filter_by_pb(stocks, date, max_pb=2.0)
    
    # 第二级：优选池（PB<1.5, ROA>0）
    pool_2 = filter_by_roa(pool_1, date, min_roa=0)
    pool_2 = filter_by_pb(pool_2, date, max_pb=1.5)
    
    # 第三级：精选池（PB<1.0, ROA>3%）
    pool_3 = filter_by_roa(pool_2, date, min_roa=3)
    pool_3 = filter_by_pb(pool_3, date, max_pb=1.0)
    
    # 从高级到低级填充
    selected = []
    for pool in [pool_3, pool_2, pool_1]:
        while len(selected) < target and pool:
            selected.append(pool.pop(0))
    
    return selected
```

**优点**：保证选股数量，优先选择高质量股票
**缺点**：复杂度增加

```python
# 切换到质量因子策略
def quality_factor_strategy(stocks, date):
    """
    质量因子策略（ROE/ROA为主）
    """
    # 筛选条件
    q = query(
        valuation.code,
        indicator.roa,
        indicator.roe,
        indicator.inc_net_profit_year_on_year
    ).filter(
        valuation.code.in_(stocks),
        indicator.roa > 5,        # ROA>5%
        indicator.roe > 10,       # ROE>10%
        indicator.inc_net_profit_year_on_year > 0  # 利润增长
    )
    
    df = get_fundamentals(q, date=date)
    df = df.sort_values('roe', ascending=False)
    
    return df['code'].tolist()
```

**优点**：更符合当前市场风格
**缺点**：完全改变策略逻辑

## 六、结论与建议

### 6.1 核心结论

1. **RFScore策略当前不可用**：选股成功率极低（<33%）
2. **根本原因**：PB筛选条件过严，叠加ROA/ROE条件导致无股票可选

- PB<2.5, ROA/ROE无限制

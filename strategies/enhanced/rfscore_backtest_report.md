## 六、发现的问题

### 1. 行业数据API问题

**问题描述**：
- `valuation.industry` 字段在Notebook环境中不可用
- 错误信息：`type object 'StockValuation' has no attribute 'industry'`

**影响**：
- 无法实现行业分散功能
- 策略降级为无行业分散版本

**解决方案**：
- 使用 `get_industry()` API获取行业数据
- 或在其他数据源获取行业分类

### 2. Notebook环境限制

**问题描述**：
- `get_current_data()` 在Notebook环境中不可用
- 策略编辑器API与Notebook API不兼容

**影响**：
- 部分实时数据获取功能受限
- 需要使用历史数据替代

**解决方案**：
- 使用 `get_price()` 替代实时数据
- 在策略编辑器中运行完整回测

### 3. Defensive V2 选股过严

**问题描述**：
- 在测试日期（2024-03-20），进攻层无符合条件的股票
- ROA>5% 且 ROE>10% 且 PB<1.0 条件过于严格

**影响**：
- 进攻层40%资金转为现金
- 组合整体偏向保守

**解决方案**：
- 放宽筛选条件（ROA>3%, ROE>8%）
- 增加备选股票池

## 实测验证结果 (2022-2025)

```
======================================================================
多版本策略验证 (2022-01-01 ~ 2025-12-31)
======================================================================

版本                    回撤    夏普     年化  月数  月胜率     累计
-----------------------------------------------------------------------
V1_RFScore7纯进攻      -16.0%  0.62  10.3%  47  57%  47.0%
V2_红利小盘纯防守      -26.4%  0.49   8.6%  47  53%  38.3%
V3_静态60/40           -15.0%  0.55   8.6%  47  55%  38.0%
V4/V5_动态组合         -19.6%  0.39   5.9%  47  53%  25.1%
```

### 关键发现

1. **V1纯进攻表现最好**: 年化10.3%，夏普0.62，回撤可控(-16%)
2. **V3静态组合回撤最小**: -15.0%，适合稳健型投资者
3. **V4/V5动态组合反而更差**: 市场宽度/趋势判断不够准确，导致调仓时机失误
4. **纯进攻V1月胜率最高**: 57%，说明大多数月份都是赚钱的

## 各版本详细分析

### V1: RFScore7 纯进攻
- **年化收益**: +10.3% (2022-2025)
- **累计收益**: +47.0%
- **最大回撤**: -16.0%
- **夏普比率**: 0.62
- **月胜率**: 57%
- **特点**: 牛市弹性大，震荡市也能赚钱
- **适用**: 进取型投资者，能承受15%左右回撤

### V2: 红利小盘 纯防守
- **年化收益**: +8.6% (2022-2025)
- **累计收益**: +38.3%
- **最大回撤**: -26.4%
- **夏普比率**: 0.49
- **月胜率**: 53%
- **特点**: 熊市有正收益，但回撤偏大
- **适用**: 熊市防守，但需承受较大波动

### V3: 静态60/40 组合
- **年化收益**: +8.6% (2022-2025)
- **累计收益**: +38.0%
- **最大回撤**: -15.0% ✓ 最小
- **夏普比率**: 0.55
- **月胜率**: 55%
- **特点**: 攻守均衡，回撤最小
- **适用**: 稳健型投资者

### V4/V5: 动态组合
- **年化收益**: +5.9% (2022-2025)
- **累计收益**: +25.1%
- **最大回撤**: -19.6%
- **夏普比率**: 0.39
- **月胜率**: 53%
- **特点**: 动态调仓反而更差
- **原因**: 市场状态判断不够准确，调仓时机失误
- **适用**: 待优化，暂不推荐

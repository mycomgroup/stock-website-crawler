## RiceQuant API 适配问题总结

| 问题 | 解决方案 |
|------|---------|
| 并发限制(3个) | 分批顺序执行 |
| scipy.stats限制 | numpy替代方案 |
| history_bars返回结构 | np.array()转换 |
| order_target不支持 | 改用order_to |
| 504 Gateway Timeout | API负载过高，需重试 |

# 数据字典

> 用途：定义所有数据文件的字段含义

---

## 一、情绪数据（sentiment_daily.csv）

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| date | string | 日期 | "2026-04-01" |
| zt_count | int | 涨停家数（昨日收盘） | 45 |
| dt_count | int | 跌停家数（昨日收盘） | 12 |
| zt_dt_ratio | float | 涨跌停比 | 3.75 |
| max_lianban | int | 最高连板数 | 3 |
| allow_buy | bool | 是否允许买入 | True |
| reason | string | 决策理由 | "中等情绪（涨停≥30），正常开仓" |

**数据来源**：
- 东方财富涨停统计 API
- 同花顺涨停复盘（手动）
- 通达信本地数据（手动）

**更新频率**：
- 每日盘前（08:30-09:00）

---

## 二、交易记录（trade_records.csv）

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| trade_id | string | 交易编号 | "T20260401001" |
| stock_code | string | 股票代码 | "000001" |
| stock_name | string | 股票名称 | "平安银行" |
| buy_date | string | 买入日期 | "2026-04-01" |
| buy_price | float | 买入价格 | 12.50 |
| buy_time | string | 买入时间 | "09:35" |
| sell_date | string | 卖出日期 | "2026-04-02" |
| sell_price | float | 卖出价格 | 12.60 |
| sell_time | string | 卖出时间 | "09:35" |
| pnl | float | 盈亏比例（%） | 0.8 |
| holding_days | int | 持有天数 | 1 |
| status | string | 状态 | "holding" / "sold" |
| 备注 | string | 备注 | "假弱高开" |

**状态说明**：
- `holding`：持有中
- `sold`：已卖出

**更新频率**：
- 每次买入/卖出后立即更新

---

## 三、停手状态（pause_status.csv）

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| date | string | 日期 | "2026-04-01" |
| trade_count | int | 总交易次数 | 10 |
| win_count | int | 盈利次数 | 5 |
| loss_count | int | 亏损次数 | 5 |
| consecutive_loss | int | 连亏次数 | 2 |
| pause_days | int | 剩余停手天数 | 0 |
| can_trade | bool | 是否可交易 | True |
| 备注 | string | 备注 | "可交易" |

**触发规则**：
- 连亏3笔 → 停手3天
- 停手期间 `pause_days > 0`，`can_trade = False`
- 停手结束 `pause_days = 0`，`can_trade = True`

**更新频率**：
- 每次交易后立即更新
- 每日收盘递减停手天数

## 四、30天记录表（30day_template.csv）

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| date | string | 日期 | "2026-04-01" |
| day_num | int | 第几天（1-30） | 1 |
| sentiment_zt_count | int | 涨停家数 | 45 |
| sentiment_status | string | 情绪状态 | "开仓" / "空仓" |
| pause_status | string | 停手状态 | "可交易" / "停手1天" |
| action | string | 操作类型 | "买入" / "卖出" / "空仓" |
| stock_code | string | 股票代码 | "000001" |
| stock_name | string | 股票名称 | "平安银行" |
| open_pct | string | 开盘涨跌幅 | "+1.2%" |
| buy_price | float | 买入价格 | 12.50 |
| buy_time | string | 买入时间 | "09:35" |
| sell_price | float | 卖出价格 | 12.60 |
| sell_time | string | 卖出时间 | "09:35" |
| pnl | string | 盈亏比例 | "+0.8%" |
| holding_days | int | 持有天数 | 1 |
| status | string | 状态 | "holding" / "sold" / "空仓" |
| 备注 | string | 备注 | "假弱高开" |

**用途**：
- 手动填写每日交易情况
- 作为30天复盘的数据来源

**更新频率**：
- 每日收盘后填写

## 五、数据关系图

```
盘前：
  get_sentiment_data.py → sentiment_daily.csv
  pause_manager.py --action check → pause_status.csv

开盘：
  execute_trade.py --mode buy → trade_records.csv

收盘：
  execute_trade.py --mode sell → trade_records.csv
  pause_manager.py --action record → pause_status.csv

复盘：
  monitor_30day.py → trade_records.csv + pause_status.csv + sentiment_daily.csv
  → 30day_report.md
```

## 六、数据文件路径

| 文件 | 路径 | 用途 |
|------|------|------|
| 情绪数据 | `data/sentiment_daily.csv` | 记录每日情绪指标 |

| 30天报告 | `docs/30day_report.md` | 30天总结报告 |

## 一、每日执行流程

### 1.1 盘前准备（08:30-09:00）

#### 步骤1：获取情绪数据

```bash
cd /Users/fengzhi/Downloads/git/testlixingren/strategies/mainline_sim_trading_20260330/scripts

# 自动获取（推荐）
python get_sentiment_data.py --date 2026-04-01

# 手动输入（备用）
python get_sentiment_data.py --date 2026-04-01 --manual 45
```

**输出**：
- 涨停家数
- 是否开仓（≥30开仓，<30空仓）
- 决策建议

#### 步骤2：检查停手状态

```bash
python pause_manager.py --action check
```

**输出**：
- 连亏次数
- 停手天数
- 是否可交易

#### 步骤3：收集涨停池

**数据来源**：
- 同花顺涨停复盘：http://stockpage.10jqka.com.cn/ztb/
- 东方财富涨停统计：http://data.eastmoney.com/zt/
- 通达信本地软件

**记录内容**：
- 昨日涨停股名单
- 流通市值
- 相对位置
- 连板状态

---

### 1.2 开盘买入（09:25-09:35）

#### 步骤1：竞价观察

**监控内容**：
- 昨日涨停股开盘价
- 开盘涨跌幅计算

#### 步骤2：筛选假弱高开

**筛选条件**：
```
1. 开盘涨跌幅：+0.5%~+1.5%（假弱高开）
2. 流通市值：50-150亿
3. 相对位置：≤30%（15日内）
4. 连板状态：近1日无涨停
5. 成交额占比：≤10%
```

#### 步骤3：执行买入

```bash
# 检查买入信号
python execute_trade.py --mode buy --open-pct 1.2

# 执行买入（符合条件）
python execute_trade.py --mode buy \
  --date 2026-04-01 \
  --stock 000001 \
  --name 平安银行 \
  --price 12.50 \
  --open-pct 1.2 \
  --备注 "假弱高开，流通市值80亿"
```

**记录内容**：
- 股票代码
- 股票名称
- 买入价格
- 开盘涨跌幅
- 备注

---

### 1.3 盘中监控（09:35-14:50）

#### 价格止损监控

**规则**：
- 日内亏损>3% → 立即卖出

**监控方式**：
- 手动查看持仓股价格
- 若跌幅超过3%，立即卖出

**涨停监控**：
- 若持仓股涨停 → 标记持有
- 次日继续观察

#### 禁止操作

- ✗ 禁止加仓
- ✗ 禁止追高
- ✗ 禁止盘中买入新股票

---

### 1.4 收盘处理（14:50-15:00）

#### 步骤1：检查持仓

```bash
python execute_trade.py --mode holding
```

**输出**：
- 当前持有股票
- 持有天数

#### 步骤2：记录未平仓交易

**若有持仓**：
- 记录持有天数
- 准备次日卖出

**若无持仓**：
- 记录当日为空仓

---

### 1.5 次日卖出（09:30-09:35）

#### 正常卖出

```bash
# 查看持有股票
python execute_trade.py --mode holding

# 执行卖出
python execute_trade.py --mode sell \
  --date 2026-04-02 \
  --stock 000001 \
  --price 12.60
```

**特殊情况**：
- 若涨停 → 继续持有
- 若低开>-4% → 正常卖出
- 若低开≤-4% → 立即止损

#### 记录盈亏

```bash
# 记录交易结果
python pause_manager.py --action record \
  --date 2026-04-02 \
  --pnl 0.8

# 或记录亏损
python pause_manager.py --action record \
  --date 2026-04-02 \
  --pnl -2.5
```

## 二、异常情况处理

### 2.1 无法成交

#### 涨停买不到

**场景**：开盘即涨停，无法买入

**处理**：
- 放弃该股票
- 不追涨停
- 记录备注："涨停买不到"

#### 跌停卖不出

**场景**：持仓股跌停，无法卖出

**处理**：
- 次日继续尝试卖出
- 记录备注："跌停卖不出"
- 等待跌停打开后卖出

---

### 2.2 大幅高开超阈值

#### 微高开（+1.5%~+2.5%）

**处理**：不买入（已删除版本）

#### 大幅高开（>+2.5%）

**处理**：不买入

---

### 2.3 情绪数据缺失

#### 数据源故障

**处理**：
- 使用备用数据源
- 或手动查询涨停家数
- 若无法确认 → 保守空仓

---

### 2.4 连亏停手触发

#### 触发停手

**场景**：连亏3笔

**处理**：
```bash
# 检查停手状态
python pause_manager.py --action check

# 输出：停手3天，不可交易
```

**执行**：
- 强制休息3天
- 不交易
- 每日更新停手天数

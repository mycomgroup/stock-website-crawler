## 策略说明

### 1. 主线策略：假弱高开（mainline_fake_weak_high_open.py）

**策略逻辑**：
- 情绪过滤：涨停家数 >= 30
- 筛选条件：
  - 市值：50-150亿
  - 位置：<= 30%（相对历史最高点）
  - 无连板
- 假弱高开定义：
  - 开盘涨幅 > 0 但 < 3%
  - 开盘价 < 竞价最高价
- 卖出规则：
  - 次日冲高 +3% 即卖
  - 否则尾盘卖
- 停手机制：连亏3笔停3天

**核心数据（⚠️ 2024实测，非10年回测）**：
- 假弱高开：日内收益 +2.89%，胜率 88.5%，样本 26个
- 真低开A：日内收益 +5.13%，胜率 100%，样本 4个
- 全年信号：136个，平均每日 0.54个

**风控约束**：
- 单票上限：10万元
- 总仓上限：30万元

**可信度**：A档（2024实测支持，但信号偏少，需10年验证）

### 2. 观察线策略：二板（observation_second_board.py）

**策略逻辑**：
- 只做二板（连板数量=2）
- 不做三板、四板
- 不做涨停排板
- 不接情绪层

**核心数据（⚠️ 2024实测，非10年回测）**：

- 回撤：0.60%

**风控约束**：
- 单票上限：10万元
- 总仓上限：30万元
- 容量上限：300万元（逐步验证）

**可信度**：A档（2024实测支持，需补充2021-2023验证）

## 使用说明

### Ricequant平台回测方法

#### 方法1：回测策略编辑器（完整策略回测）

**前提条件**：检查每天180分钟时间限制是否用满

1. 登录Ricequant平台
2. 进入"策略编辑器"
3. 上传 `ricequant_strategy_editor.py`
4. 设置配置：
   - 起始日期：2014-01-01
   - 结束日期：2024-12-31
   - 初始资金：100,000
   - 频率：每日
5. 运行回测（约需30-60分钟）
6. 查看绩效报告、净值曲线

**优点**：
- 完整的策略回测引擎
- 自动生成绩效指标
- 适合生产级策略

**缺点**：
- 每天限制180分钟
- 需要完整策略代码

---

#### 方法2：Notebook研究环境（数据分析）

**前提条件**：无时间限制

1. 登录Ricequant平台
2. 进入"Notebook研究环境"
3. 选择上传文件：
   - `backtest_10years.ipynb`（完整10年回测）
   - `backtest_simple.ipynb`（简化验证）
4. 运行所有单元格
5. 查看分析结果、可视化图表

**优点**：
- 无时间限制
- 可交互式调试
- 适合研究探索

**缺点**：
- 需要手动编写回测逻辑
- 绩效计算需要自己实现

---

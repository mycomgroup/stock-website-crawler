## 九、参数修改规则

### 9.1 锁定规则

| 规则 | 说明 |
|------|------|
| **参数已锁定** | v1.0版本参数不再调参 |
| **修改需审批** | 需经策略评审后才能修改 |
| **修改需验证** | 修改后需重新回测验证 |
| **版本管理** | 修改后更新版本号 |

### 9.2 修改触发条件

| 条件 | 说明 |
|------|------|
| 样本外失效 | 连续30天表现偏差超警戒线 |
| 市场结构变化 | 注册制改革、涨跌停规则变化 |
| 策略拥挤 | 同质化交易导致失效 |
| 容量饱和 | 大资金进入导致滑点恶化 |

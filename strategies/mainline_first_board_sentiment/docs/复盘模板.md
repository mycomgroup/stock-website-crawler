### 5.2 异常交易复盘报告模板

```
===== 异常交易复盘报告 =====

交易日期：2026-04-15
股票代码：000001
股票名称：平安银行
异常类型：大幅亏损

一、交易概况

买入时间：2026-04-15 09:35
买入价格：12.50
卖出时间：2026-04-16 09:35
卖出价格：11.80
盈亏比例：-5.6%

二、买入理由

1. 情绪判断：涨停52只，高情绪，开仓
2. 开盘涨幅：+1.2%，符合假弱高开
3. 流通市值：80亿，符合50-150亿要求
4. 相对位置：35%，略高于30%阈值
5. 连板状态：近1日无涨停，符合要求

三、失败原因分析

1. 主要原因：相对位置35%，高于30%阈值
   - 应该严格遵守相对位置≤30%的筛选条件
   - 位置偏高，主力借假弱高开出货

2. 次要原因：次日大盘下跌
   - 次日大盘低开-1.5%，影响个股走势
   - 应加强次日大盘判断

3. 其他原因：假弱是真弱
   - 主力借假弱高开出货，非真正假弱
   - 应结合成交量、换手率等指标判断

四、经验教训

1. 严格遵守筛选条件，不放松标准
2. 相对位置是关键指标，必须≤30%
3. 假弱高开需结合其他指标判断真假
4. 次日大盘走势影响个股，需提前判断

五、改进措施

1. 严格遵守相对位置≤30%的筛选条件
2. 增加成交量和换手率判断
3. 加强次日大盘走势判断
4. 记录此次失败案例，避免重复

六、后续跟踪

1. 观察该股后续走势
2. 总结类似案例特征
3. 优化筛选条件
```

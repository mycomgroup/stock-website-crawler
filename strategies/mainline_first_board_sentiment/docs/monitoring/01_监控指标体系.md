## 一、指标体系总览

### 1.1 指标分类

监控系统共分为四大类指标：

| 类别 | 监控目的 | 指标数量 | 更新频率 |
|------|---------|---------|---------|
| 收益类 | 监控策略盈利能力 | 12 | 实时/日频 |
| 风险类 | 监控策略风险暴露 | 15 | 实时/日频 |
| 交易类 | 监控交易执行质量 | 10 | 实时/日频 |
| 系统类 | 监控系统运行状态 | 8 | 实时 |

## 二、收益类指标

### 2.1 实时收益指标

| 指标名称 | 计算方法 | 更新频率 | 预警阈值 | 优先级 |
|---------|---------|---------|---------|--------|
| 当日收益 | (收盘市值 - 开盘市值) / 开盘市值 × 100% | 实时(1分钟) | 无 | P1 |
| 累计收益 | (当前净值 - 初始净值) / 初始净值 × 100% | 实时(1分钟) | 无 | P1 |
| 年化收益 | (累计收益 + 1)^(252/天数) - 1 | 日频 | < 15%：黄色 | P1 |
| 滚动7日收益 | ∑(过去7日日收益) | 日频 | < -5%：黄色 | P2 |
| 滚动30日收益 | ∑(过去30日日收益) | 日频 | < -10%：红色 | P2 |

| 情绪关闭期收益 | ∑(情绪关闭期交易收益) | 日频 | < 0%：观察 | P2 |

### 2.3 基准对比指标

|---------|---------|---------|---------|--------|

## 三、风险类指标

### 3.1 回撤指标

| 指标名称 | 计算方法 | 更新频率 | 预警阈值 | 优先级 |
|---------|---------|---------|---------|--------|
| 当前回撤 | (峰值净值 - 当前净值) / 峰值净值 | 实时(1分钟) | > 10%：黄色<br/>> 15%：红色 | P1 |
| 最大回撤(滚动) | max(过去N日回撤) | 日频 | > 15%：黄色<br/>> 20%：红色 | P1 |
| 回撤持续时间 | 自峰值到当前的交易日数 | 实时 | > 20日：黄色<br/>> 30日：红色 | P2 |
| 平均回撤深度 | mean(过去N日回撤) | 日频 | > 8%：黄色 | P3 |

### 3.2 波动指标

| 指标名称 | 计算方法 | 更新频率 | 预警阈值 | 优先级 |
|---------|---------|---------|---------|--------|
| 日波动率(滚动20日) | std(过去20日日收益) × √252 | 日频 | > 历史均值+2σ：黄色 | P2 |
| 下行波动率 | std(负收益) × √252 | 日频 | > 历史均值+2σ：黄色 | P2 |
| 收益分布偏度 | skewness(过去60日日收益) | 日频 | < -0.5：观察 | P3 |
| 收益分布峰度 | kurtosis(过去60日日收益) | 日频 | > 3：观察 | P3 |

### 3.3 风险调整收益指标

| 指标名称 | 计算方法 | 更新频率 | 预警阈值 | 优先级 |
|---------|---------|---------|---------|--------|
| 夏普比率(滚动) | (年化收益 - 无风险利率) / 年化波动率 | 日频 | < 1.0：黄色<br/>< 0.5：红色 | P1 |
| 卡玛比率(滚动) | 年化收益 / 最大回撤 | 日频 | < 1.0：黄色 | P1 |
| 索提诺比率(滚动) | (年化收益 - 无风险利率) / 下行波动率 | 日频 | < 1.5：黄色 | P2 |
| 信息比率 | 超额收益 / 跟踪误差 | 日频 | < 0.5：观察 | P3 |

### 3.4 风险暴露指标

| 指标名称 | 计算方法 | 更新频率 | 预警阈值 | 优先级 |
|---------|---------|---------|---------|--------|
| 市值暴露 | 平均持仓市值 / 总市值 | 日频 | 超出范围±20%：观察 | P2 |
| 行业集中度 | max(各行业持仓占比) | 日频 | > 40%：黄色 | P2 |
| 风格暴露 | 中性/成长/价值因子暴露 | 日频 | 绝对值>1.5：观察 | P3 |
| Beta暴露 | 对市场基准的Beta | 日频 | > 1.2：观察 | P3 |

## 八、指标计算示例

### 8.1 当前回撤计算

```python
def calculate_current_drawdown(net_values):
    """
    计算当前回撤
    参数：
        net_values: 净值序列 [净值1, 净值2, ...]
    返回：
        当前回撤百分比
    """
    peak = max(net_values)  # 历史最高净值
    current = net_values[-1]  # 当前净值
    drawdown = (peak - current) / peak * 100
    return drawdown
```

### 8.2 滚动胜率计算

```python
def calculate_rolling_win_rate(trades, window=20):
    """
    计算滚动胜率
    参数：
        trades: 交易记录列表 [{'profit': float, 'date': str}, ...]
        window: 滚动窗口大小
    返回：
        最近window笔交易的胜率
    """
    recent_trades = trades[-window:]
    wins = sum(1 for t in recent_trades if t['profit'] > 0)
    win_rate = wins / len(recent_trades) * 100 if recent_trades else 0
    return win_rate
```

### 8.3 夏普比率计算

```python
import numpy as np

def calculate_sharpe_ratio(daily_returns, risk_free_rate=0.03):
    """
    计算夏普比率
    参数：
        daily_returns: 日收益率序列
        risk_free_rate: 无风险利率(年化)
    返回：
        年化夏普比率
    """
    excess_returns = np.array(daily_returns) - risk_free_rate / 252
    std = np.std(excess_returns)
    mean = np.mean(excess_returns)
    sharpe = mean / std * np.sqrt(252) if std > 0 else 0
    return sharpe
```

## 十、指标使用说明

## 七、报警处理流程

### 7.1 黄色报警处理流程

```
黄色报警触发
     ↓
记录报警信息（时间、类型、触发条件）
     ↓
发送邮件通知相关人员
     ↓
相关人员关注（24小时内）
     ↓
分析报警原因
     ↓
采取相应措施
     ↓
记录处理结果
     ↓
更新报警状态为"已处理"
```

**详细步骤**：

1. **报警触发**
   - 监控系统检测到触发条件
   - 自动记录报警信息到数据库

2. **报警记录**
   - 报警ID：唯一标识
   - 报警时间：触发时间戳
   - 报警类型：分类标签
   - 报警级别：黄色
   - 触发条件：具体阈值
   - 当前状态：当前指标值
   - 处理状态：待处理

3. **通知发送**
   - 发送邮件到相关人员
   - 邮件内容包含报警详情
   - 邮件包含处理建议

4. **人员响应**
   - 相关人员在24小时内查看报警
   - 分析报警原因
   - 决定处理措施

5. **处理执行**
   - 执行相应处理措施
   - 记录处理过程

6. **结果记录**
   - 处理措施：具体操作
   - 处理结果：效果评估
   - 处理人：责任人
   - 处理时间：完成时间

7. **状态更新**
   - 更新报警状态为"已处理"
   - 标记处理人
   - 完成时间戳

### 7.2 橙色报警处理流程

```
橙色报警触发
     ↓
记录报警信息
     ↓
发送邮件+短信通知
     ↓
相关人员立即响应（4小时内）
     ↓
分析报警原因
     ↓
触发风控降级机制
     ↓
采取相应措施
     ↓
记录处理结果
     ↓
更新报警状态为"已处理"
```

**风控降级措施**：

| 报警类型 | 风控降级措施 |
|---------|------------|
| 回撤>10% | 仓位上限降低至20% |
| 波动异常 | 信号筛选更严格 |
| 胜率下降 | 信号质量分数阈值提高 |

### 7.3 红色报警处理流程

```
红色报警触发
     ↓
立即记录报警信息
     ↓
立即发送邮件+短信+电话通知
     ↓
相关人员立即响应（1小时内）
     ↓
立即触发熔断机制
     ↓
暂停所有交易
     ↓
全面分析报警原因
     ↓
采取紧急措施
     ↓
记录处理结果
     ↓
更新报警状态为"已处理"
     ↓
复盘总结，更新策略
```

**熔断机制**：

| 报警类型 | 熔断措施 |
|---------|---------|
| 单日亏损>5% | 立即停止当日交易 |
| 回撤>15% | 降低仓位至10%，暂停新开仓 |
| 回撤>20% | 完全停止交易，清仓观望 |
| 连续5日亏损 | 停止交易，全面复盘 |
| 胜率<60% | 触发OOS验证，暂停交易 |
| 盈亏比<0.8 | 触发OOS验证，暂停交易 |
| 滚动30日收益<-10% | 停止交易，触发熔断 |
| 数据延迟>30分钟 | 暂停交易，切换数据源 |
| 数据完整率<90% | 停止依赖该数据的决策 |
| 系统异常 | 停止交易，重启系统 |
| API连接失败 | 切换备用接口，暂停交易 |
| 订单执行失败 | 暂停交易，检查交易接口 |

## 九、报警通知方式

### 9.1 仪表盘显示

**位置**：
- 顶部状态栏：预警状态图标
- 总览页面：预警信息列表
- 顶部导航栏：未处理报警数量

**显示方式**：
- 颜色编码：绿/黄/橙/红
- 图标提示：预警图标+数量
- 弹窗提示：新报警弹窗通知

### 9.2 邮件通知

**邮件模板**：

```
主题：【二板接力监控系统】报警通知 - {报警类型} - {报警级别}

正文：

尊敬的策略管理员：

监控系统于 {报警时间} 检测到 {报警级别} 报警：

报警详情：
- 报警类型：{报警类型}
- 触发条件：{触发条件}
- 当前状态：{当前状态}
- 报警级别：{报警级别}

处理建议：
{处理建议}

处理时限：
{处理时限}

请在规定时间内处理此报警。

如有疑问，请联系系统管理员。

二板接力监控系统
{发送时间}
```

### 9.3 短信通知

**短信模板**：

```
【二板接力监控】{报警级别}报警：{报警类型}，当前状态{当前状态}，请{处理时限}内处理。详情请查看监控系统。
```

**示例**：

```
【二板接力监控】红色报警：单日亏损，当前状态-5.2%，请1小时内处理。详情请查看监控系统。
```

### 9.4 电话通知

**触发条件**：
- 仅红色报警触发电话通知
- 连续3次电话通知（间隔5分钟）
- 如无人接听，升级通知上级

**电话脚本**：

```
"您好，这是二板接力监控系统自动报警通知。
于{报警时间}检测到红色报警：{报警类型}。
当前状态为{当前状态}，触发条件为{触发条件}。
请立即处理，处理时限为{处理时限}。
如有疑问，请登录监控系统查看详情。
感谢您的关注。"
```

### 3.2 指标计算模块

**模块名称**：MetricsCalculator

**功能**：计算各类监控指标

**架构**：

```
MetricsCalculator
    ├── ProfitMetricsCalculator      # 收益类指标计算器
    ├── RiskMetricsCalculator        # 风险类指标计算器
    ├── TradeMetricsCalculator       # 交易类指标计算器
    └── SystemMetricsCalculator      # 系统类指标计算器
```

**技术实现**：

| 技术 | 说明 |
|------|------|
| Pandas | 数据处理和分析 |
| NumPy | 数值计算 |
| SQLAlchemy | 数据库查询 |

**代码示例**：

```python
import pandas as pd
import numpy as np

class ProfitMetricsCalculator:
    def __init__(self, db_conn):
        self.db_conn = db_conn
        
    def calculate_daily_return(self, date):
        # 查询当日交易记录
        trades = self.get_trades(date)
        
        # 计算当日收益
        daily_profit = sum(trade['profit'] for trade in trades)
        daily_return = daily_profit / self.get_total_capital()
        
        return daily_return
        
    def calculate_cumulative_return(self):
        # 查询历史绩效数据
        performance = self.get_performance_history()
        
        # 计算累计收益
        cumulative_return = performance[-1]['cumulative_return']
        
        return cumulative_return
        
    def calculate_annual_return(self, days):
        # 计算年化收益
        cumulative_return = self.calculate_cumulative_return()
        annual_return = (cumulative_return + 1) ** (252 / days) - 1
        
        return annual_return

class RiskMetricsCalculator:
    def calculate_current_drawdown(self):
        # 获取净值序列
        net_values = self.get_net_values()
        
        # 计算当前回撤
        peak = max(net_values)
        current = net_values[-1]
        drawdown = (peak - current) / peak
        
        return drawdown
        
    def calculate_sharpe_ratio(self, window=20):
        # 获取历史收益率
        returns = self.get_returns(window)
        
        # 计算夏普比率
        excess_returns = returns - self.risk_free_rate / 252
        std = np.std(excess_returns)
        mean = np.mean(excess_returns)
        sharpe = mean / std * np.sqrt(252)
        
        return sharpe
```

**输入**：
- 原始数据（行情、交易记录）
- 配置参数（窗口大小、无风险利率）

**输出**：
- 收益类指标（日收益、累计收益、年化收益）
- 风险类指标（回撤、波动率、夏普比率）
- 交易类指标（胜率、盈亏比、交易次数）
- 系统类指标（数据质量、系统状态）

**计算频率**：
- 实时指标：1分钟
- 日频指标：每日收盘后
- 周频/月频指标：每周/每月

### 3.3 风控判断模块

**模块名称**：RiskControlEngine

**功能**：判断是否触发风控规则

**架构**：

```
RiskControlEngine
    ├── DrawdownRule        # 回撤规则
    ├── LossRule            # 亏损规则
    ├── VolatilityRule      # 波动规则
    ├── SharpeRatioRule     # 夏普比率规则
    └── SystemRule          # 系统规则
```

**技术实现**：

| 技术 | 说明 |
|------|------|
| 规则引擎 | 自定义规则引擎 |
| SQLAlchemy | 数据库查询 |
| Celery | 定时任务 |

**代码示例**：

```python
class RiskControlEngine:
    def __init__(self, rules_config):
        self.rules_config = rules_config
        
    def check_risks(self, metrics):
        results = []
        
        # 检查回撤规则
        drawdown = metrics['current_drawdown']
        if drawdown > self.rules_config['drawdown_threshold_red']:
            results.append({
                'rule': 'drawdown',
                'level': 'red',
                'value': drawdown,
                'threshold': self.rules_config['drawdown_threshold_red']
            })
        elif drawdown > self.rules_config['drawdown_threshold_yellow']:
            results.append({
                'rule': 'drawdown',
                'level': 'yellow',
                'value': drawdown,
                'threshold': self.rules_config['drawdown_threshold_yellow']
            })
            
        # 检查胜率规则
        win_rate = metrics['win_rate_20']
        if win_rate < self.rules_config['win_rate_threshold_red']:
            results.append({
                'rule': 'win_rate',
                'level': 'red',
                'value': win_rate,
                'threshold': self.rules_config['win_rate_threshold_red']
            })
            
        return results
        
    def trigger_circuit_breaker(self, risk_type):
        # 触发熔断机制
        if risk_type == 'drawdown_exceed_20':
            self.pause_all_trading()
            self.clear_all_positions()
        elif risk_type == 'drawdown_exceed_15':
            self.reduce_position_to_10()
            self.pause_new_orders()
            
    def pause_all_trading(self):
        # 停止所有交易
        self.trade_system.pause()
        
    def clear_all_positions(self):
        # 清仓所有持仓
        positions = self.get_current_positions()
        for stock in positions:
            self.trade_system.sell(stock)
```

**输入**：
- 监控指标数据
- 风控规则配置

**输出**：
- 风控判断结果（是否触发）
- 风控级别（绿色/黄色/橙色/红色）
- 风控措施（建议操作）

class TradeSignalReceiver:

## 五、非功能性需求

### 5.1 性能要求

| 性能指标 | 要求 | 说明 |
|---------|------|------|
| 实时指标延迟 | < 1秒 | 从数据更新到指标计算完成 |
| 历史查询响应 | < 3秒 | 单次查询响应时间 |
| 并发用户数 | ≥ 100 | 同时访问用户数 |
| API吞吐量 | ≥ 1000 req/s | API请求吞吐量 |
| 数据写入速度 | ≥ 10000 pts/s | 数据写入吞吐量 |

**性能优化策略**：

1. **缓存优化**
   - Redis缓存热点数据
   - 设置合理的缓存过期时间
   - 使用缓存预热策略

2. **查询优化**
   - 数据库索引优化
   - SQL查询优化
   - 分页查询

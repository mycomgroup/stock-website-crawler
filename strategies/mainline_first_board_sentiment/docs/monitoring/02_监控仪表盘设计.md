### 1.2 顶部状态栏

**位置**：页面顶部，高度60px，固定显示

**布局**：从左到右依次显示

| 区域 | 内容 | 宽度 | 说明 |
|------|------|------|------|
| 左侧Logo | 系统名称 | 200px | "二板接力监控系统" |
| 中部核心指标 | 6个核心指标卡片 | 900px | 每个卡片150px宽 |
| 右侧状态 | 系统状态 | 150px | 运行状态+用户信息 |

**核心指标卡片**：

```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│ 策略状态 │ │ 当前仓位 │ │ 当日收益 │ │ 累计收益 │ │ 当前回撤 │ │ 预警状态 │
│  运行中  │ │   25%   │ │  +1.2%  │ │  +15.3% │ │  -3.2%  │ │    🟢    │
│  ● 正常  │ │  警戒线:30%│ │ 目标:1%  │ │ 年化:20% │ │ 警戒:10% │ │  绿色   │
└──────────┘ └──────────┘ └──────────┘ └──────────┘ └──────────┘ └──────────┘
```

**预警状态颜色**：
- 🟢 绿色：正常（无预警）
- 🟡 黄色：观察（有黄色预警）
- 🟠 橙色：警告（有橙色预警）
- 🔴 红色：紧急（有红色预警）

### 1.3 左侧导航栏

**位置**：页面左侧，宽度200px，固定显示

**菜单项**：

| 菜单 | 图标 | 说明 |

## 七、数据更新方式

### 7.1 实时更新

**更新频率**：
- 核心指标：1分钟
- 系统状态：30秒
- 交易数据：实时（事件驱动）

**更新方式**：
- WebSocket推送
- 局部刷新（不刷新整个页面）
- 增量更新

### 7.2 定时更新

**更新频率**：
- 日频指标：每日收盘后
- 周频指标：每周一
- 月频指标：每月第一个交易日

**更新方式**：
- 定时任务（Celery）
- 全量计算
- 批量更新

### 7.3 手动刷新

**功能**：
- 提供刷新按钮
- 支持选择性刷新
- 显示最后更新时间

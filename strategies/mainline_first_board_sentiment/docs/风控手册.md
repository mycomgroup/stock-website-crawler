# 风控手册

> 策略：首板低开 + 情绪开关
> 用途：风险识别、控制与应对

---

## 一、风险分类

### 1.1 市场风险

| 风险类型 | 风险描述 | 风险等级 |
|----------|----------|----------|
| **情绪突变** | 市场情绪突然恶化，涨停家数骤降 | 高 |
| **大盘下跌** | 大盘单日大跌，个股普跌 | 高 |
| **板块轮动** | 热点板块快速切换，首板股失宠 | 中 |
| **风格切换** | 市场风格从小盘切到大盘 | 中 |

### 1.2 策略风险

| 风险类型 | 风险描述 | 风险等级 |
|----------|----------|----------|
| **假弱真弱** | 假弱高开实际是主力出货 | 高 |
| **信号失效** | 策略在样本外失效 | 高 |
| **参数过拟合** | 参数过度拟合历史数据 | 中 |
| **策略拥挤** | 同质化交易导致收益下降 | 中 |

### 1.3 执行风险

| 风险类型 | 风险描述 | 风险等级 |
|----------|----------|----------|
| **涨停买不到** | 开盘涨停无法买入 | 高 |
| **跌停卖不出** | 跌停无法卖出，亏损扩大 | 高 |
| **滑点过大** | 实际成交价偏离预期 | 中 |
| **系统故障** | 交易系统或数据源故障 | 中 |

### 1.4 资金风险

| 风险类型 | 风险描述 | 风险等级 |
|----------|----------|----------|
| **容量超限** | 资金过大导致滑点恶化 | 高 |
| **仓位过重** | 单票仓位过大，风险集中 | 高 |
| **流动性不足** | 小盘股成交困难 | 中 |

## 二、风控机制

### 2.1 事前控制（预防）

#### 2.1.1 情绪过滤

| 机制 | 规则 | 目的 |
|------|------|------|
| **情绪开关** | 涨停<30只空仓 | 避免弱市交易 |
| 高情绪阈值 | 涨停≥50只积极开仓 | 抓住强势环境 |
| 中等情绪阈值 | 涨停30-50只正常开仓 | 过滤中等环境 |

**效果**：
- 收益提升71%（+0.48%→+0.82%）
- 胜率提升3%（48.9%→51.9%）
- 信号减少51%（过滤弱市）

#### 2.1.2 信号过滤

| 过滤条件 | 参数值 | 目的 |
|----------|--------|------|
| 流通市值 | 50-150亿 | 避免小盘风险和大盘滞涨 |
| 相对位置 | ≤30% | 避免高位接盘 |
| 连板状态 | 近1日无涨停 | 避免连板接力风险 |
| 开盘涨幅 | +0.5%~+1.5% | 锁定假弱高开结构 |

#### 2.1.3 容量控制

| 控制项 | 参数值 | 目的 |
|----------|--------|------|
| 总资金上限 | 500万 | 避免滑点恶化 |
| 单票资金上限 | 50万 | 确保可成交 |
| 成交额占比上限 | 10% | 避免冲击成本 |

### 2.2 事中控制（监控）

#### 2.2.1 仓位控制

| 控制项 | 参数值 | 触发条件 |
|----------|--------|----------|
| 单票仓位 | ≤5% | 每笔交易 |
| 总仓位（高情绪） | ≤30% | 涨停≥50只 |
| 总仓位（中等情绪） | ≤20% | 涨停30-50只 |
| 总仓位（低情绪） | 0% | 涨停<30只 |

#### 2.2.2 日内止损

| 止损类型 | 触发条件 | 执行方式 |
|----------|----------|----------|
| **价格止损** | 日内亏损>3% | 立即卖出 |
| 时间止损 | 不采用 | - |

**执行方式**：
- 盘中实时监控价格
- 触发条件立即卖出
- 记录止损原因

#### 2.2.3 禁止操作

| 操作 | 是否允许 | 理由 |
|------|----------|------|
| 盘中买入新股票 | 禁止 | 避免追高 |
| 加仓持有股票 | 禁止 | 机会仓单次入场 |
| 追涨停 | 禁止 | 涨停买不到且风险高 |

### 2.3 事后控制（止损）

#### 2.3.1 停手机制

| 项目 | 参数值 | 说明 |
|------|--------|------|
| **触发条件** | 连亏3笔 | 统计最近交易 |
| **停手时长** | 3天 | 强制休息 |
| **恢复条件** | 3天后自动恢复 | 无需额外判断 |

**效果**：
- 回撤改善25%
- 收益伤害10%
- 年触发约2-3次

#### 2.3.2 回撤熔断（可选）

| 熔断类型 | 触发条件 | 停止时长 |
|----------|----------|----------|
| 单日回撤熔断 | 日回撤>5% | 当日停止交易 |
| 周回撤熔断 | 周回撤>8% | 暂停一周 |
| 月回撤熔断 | 月回撤>15% | 暂停一月并复盘 |

## 三、风险应对

### 3.1 市场风险应对

#### 情绪突变

| 场景 | 应对措施 |
|------|----------|
| 涨停家数骤降 | 情绪开关自动空仓 |
| 大盘单日大跌 | 情绪开关自动空仓 |
| 板块快速轮动 | 减少持仓时间，次日开盘卖 |

#### 大盘下跌

| 场景 | 应对措施 |
|------|----------|
| 大盘跌幅<2% | 正常交易，严格止损 |
| 大盘跌幅2-3% | 降低仓位至10% |
| 大盘跌幅>3% | 情绪开关触发，空仓 |

### 3.2 策略风险应对

#### 假弱真弱

| 场景 | 应对措施 |
|------|----------|
| 日内亏损>3% | 立即止损卖出 |
| 次日低开>-4% | 正常卖出 |
| 次日低开≤-4% | 立即止损卖出 |

#### 信号失效

| 场景 | 应对措施 |
|------|----------|
| 连续亏损 | 停手机制触发 |
| 30天表现不达标 | 暂停交易，复盘检查 |
| 样本外失效 | 重新验证策略 |

### 3.3 执行风险应对

#### 涨停买不到

| 场景 | 应对措施 |
|------|----------|
| 开盘涨停 | 放弃，不追涨停 |
| 快速涨停 | 涨停后不买入 |
| 涨停排队 | 排队等待，不强求 |

**原则**：
- 接受错过，不追涨停
- 机会仓不需要每次都做

#### 跌停卖不出

| 场景 | 应对措施 |
|------|----------|
| 跌停排队 | 次日继续卖出 |
| 连续跌停 | 记录异常，复盘分析 |

**原则**：
- 提前识别跌停风险
- 严格止损，避免跌停

#### 滑点过大

| 场景 | 应对措施 |
|------|----------|
| 滑点<0.5% | 正常交易 |
| 滑点0.5-1.0% | 警戒，监控 |
| 滑点>1.0% | 暂停交易，检查流动性 |

### 3.4 资金风险应对

#### 容量超限

| 场景 | 应对措施 |
|------|----------|
| 资金≤500万 | 正常交易 |
| 资金500-1000万 | 降低仓位，分散执行 |
| 资金>1000万 | 不推荐，容量不足 |

#### 仓位过重

| 场景 | 应对措施 |
|------|----------|
| 单票仓位>5% | 立即减仓至5% |
| 总仓位>30% | 立即减仓至30% |

## 四、风控检查表

### 4.1 每日风控检查

```
□ 情绪开关是否正常（涨停家数≥30）
□ 停手状态是否可交易（停手天数=0）
□ 单票仓位是否≤5%
□ 总仓位是否≤30%
□ 日内止损是否设置（亏损>3%卖出）
```

### 4.2 每周风控检查

```
□ 本周最大回撤是否<5%
□ 本周连亏次数是否<3
□ 停手机制是否正常触发
□ 情绪开关准确率是否正常
```

### 4.3 每月风控检查

```
□ 本月最大回撤是否<15%
□ 胜率偏差是否在±5%内
□ 平均收益偏差是否在±0.3%内
□ 是否有异常交易需要复盘
```

## 五、风控指标监控

### 5.1 实时监控指标

| 指标 | 监控频率 | 警戒值 | 处理 |
|------|----------|--------|------|
| 涨停家数 | 每日 | <30 | 空仓 |
| 日内亏损 | 实时 | >3% | 止损卖出 |
| 单票仓位 | 每笔 | >5% | 减仓 |

### 5.2 每日监控指标

| 指标 | 监控频率 | 警戒值 | 处理 |
|------|----------|--------|------|
| 日收益率 | 每日 | <-5% | 暂停交易 |
| 持仓数量 | 每日 | >3 | 不再加仓 |
| 停手天数 | 每日 | >0 | 不交易 |

### 5.3 每周监控指标

| 指标 | 监控频率 | 警戒值 | 处理 |
|------|----------|--------|------|
| 周收益率 | 每周 | <-8% | 暂停一周 |
| 连亏次数 | 每周 | ≥3 | 触发停手 |
| 胜率 | 每周 | <40% | 复盘检查 |

### 5.4 每月监控指标

| 指标 | 监控频率 | 警戒值 | 处理 |
|------|----------|--------|------|
| 月收益率 | 每月 | <-15% | 暂停一月 |
| 最大回撤 | 每月 | >20% | 复盘检查 |
| 策略稳定性 | 每月 | 偏差超警戒 | 重新验证 |

## 六、风控日志

### 6.1 日志格式

```
日期: 2026-03-31
风控事件: 情绪开关触发空仓
触发条件: 涨停家数=25只 < 30只
处理措施: 今日不交易
备注: 市场情绪低迷
```

### 6.2 日志记录内容

| 事件类型 | 记录内容 |
|----------|----------|
| 情绪开关触发 | 涨停家数、决策 |
| 停手机制触发 | 连亏次数、停手天数 |
| 止损触发 | 亏损比例、触发原因 |
| 熔断触发 | 回撤比例、暂停时长 |

## 七、风控复盘

### 7.1 每周风控复盘

**复盘内容**：
- 本周风控事件统计
- 风控措施执行情况
- 风控参数是否需要调整

### 7.2 每月风控复盘

**复盘内容**：
- 本月风控事件统计
- 最大回撤分析
- 风控效果评估
- 风控参数调整建议

## 八、风控参数调整规则

### 8.1 调整触发条件

| 条件 | 说明 |
|------|------|
| 样本外失效 | 30天表现偏差超警戒线 |
| 市场结构变化 | 注册制、涨跌停规则变化 |
| 策略拥挤 | 同质化交易导致失效 |
| 风控过严 | 过度限制交易机会 |

### 8.2 调整流程

```
1. 识别风控问题
2. 分析问题原因
3. 提出调整方案
4. 回测验证方案
5. 小范围测试
6. 正式更新参数
7. 记录调整日志
```

### 8.3 调整权限

| 调整类型 | 权限 |
|----------|------|
| 参数微调 | 交易员 |
| 参数大幅调整 | 策略评审委员会 |
| 策略重构 | 策略评审委员会 |

## 九、风控责任

### 9.1 责任分工

| 角色 | 责任 |
|------|------|
| **交易员** | 执行风控规则、记录风控日志 |
| **风控专员** | 监控风控指标、审核风控执行 |
| **策略主管** | 审批参数调整、组织风控复盘 |

### 9.2 责任追究

| 问题类型 | 追究方式 |
|----------|----------|
| 未执行风控规则 | 警告、暂停交易权限 |
| 隐瞒风控问题 | 严重警告、降级 |
| 造成重大损失 | 解除职务、追责 |

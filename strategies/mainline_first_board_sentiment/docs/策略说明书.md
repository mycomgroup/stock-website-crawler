## 一、策略核心定义

### 1.1 策略名称

**首板低开情绪择时策略（First Board Low Open with Sentiment Switch）**

### 1.2 策略类型

| 维度 | 定义 |
|------|------|
| **策略层级** | 机会仓（高频、短线） |
| **持仓周期** | 1-2天（次日开盘卖出为主） |
| **交易频率** | 高频（日均0-2次） |
| **仓位类型** | 单票≤5%，总仓≤30% |
| **适用资金** | ≤500万（小资金试盘） |
| **适用市场** | A股主板、中小创 |

### 1.3 策略逻辑

**核心假设**：
- 首板（昨日涨停）次日假弱高开，反映主力"示弱"意图
- 市场情绪决定首板套利成功率
- 情绪好时首板套利胜率高，情绪差时胜率低

**盈利来源**：
- 假弱高开后的日内反弹动能
- 次日冲高后的价差收益

**风险来源**：
- 假弱是真弱（主力出货）
- 情绪突变导致市场承接力不足
- 涨停买不到、跌停卖不出

## 二、信号定义（已锁定）

### 2.1 主版本信号

| 条件 | 参数值 | 说明 |
|------|--------|------|
| **昨日状态** | 涨停收盘 | 首板基础 |
| **今日开盘** | +0.5%~+1.5% | 假弱高开结构 |
| **流通市值** | 50-150亿 | 中盘股 |
| **相对位置** | ≤30%（15日内） | 低位区域 |
| **连板状态** | 近1日无涨停 | 非连板接力 |

### 2.2 备选版本信号

| 条件 | 参数值 | 说明 |
|------|--------|------|
| **昨日状态** | 涨停收盘 | 首板基础 |
| **今日开盘** | -3.0%~-1.0% | 真低开A结构 |
| 其他条件 | 同主版本 | |

### 2.3 删除信号（不执行）

| 结构 | 删除理由 |
|------|----------|
| 真低开B（-1%~0%） | 收益+0.33%，胜率45%，不达标 |
| 微高开（+1.5%~+2.5%） | 收益-1.65%，负贡献 |
| 平开附近（0%~+0.5%） | 收益+0.27%过低，样本稀释 |

## 三、情绪开关（已锁定）

### 3.1 情绪指标

| 指标 | 用途 | 数据来源 |
|------|------|----------|
| **涨停家数** | 核心指标 | T-1收盘数据 |
| 最高连板数 | 辅助指标 | T-1收盘数据 |
| 涨跌停比 | 辅助指标 | T-1收盘数据 |

### 3.2 开关阈值

| 情绪状态 | 涨停家数阈值 | 操作建议 |
|----------|--------------|----------|
| **高情绪** | ≥50只 | 积极开仓，可放宽筛选 |
| **中等情绪** | 30-50只 | 正常开仓，严格筛选 |
| **低情绪** | <30只 | 空仓观望，不交易 |

### 3.3 开关类型

采用**硬开关**（而非仓位调节器）：
- 情绪达标 → 允许买入
- 情绪不达标 → 强制空仓

**理由**：
- 机会仓需要简单可执行规则
- 硬开关回撤控制更好
- 仓位调节器复杂度高，不适合高频

## 四、买入规则（已锁定）

### 4.1 买入时间

| 时间段 | 操作 |
|--------|------|
| **09:25-09:30** | 竞价观察，计算开盘涨幅 |
| **09:30-09:35** | 确认开盘价，筛选目标 |
| **09:35-09:40** | 下单买入，记录价格 |

### 4.2 买入条件检查

**必须满足全部条件**：
```
1. 情绪开关开启（涨停≥30）
2. 停手状态可交易（停手天数=0）
3. 昨日涨停收盘
4. 今日假弱高开（+0.5%~+1.5%）
5. 流通市值50-150亿
6. 相对位置≤30%（15日内）
7. 近1日无涨停（非连板）
8. 成交额占比≤10%
9. 单票资金≤50万或5%总资产
```

### 4.3 买入执行

| 项目 | 规则 |
|------|------|
| **买入方式** | 市价单（开盘后立即买入） |
| **滑点预留** | 0.5%（预期买入价偏差） |
| **单票仓位** | ≤5%总资产 |
| **单票资金** | ≤50万 |
| **总仓位上限** | ≤30%总资产 |
| **买入数量** | 每日≤3只 |

### 4.4 无法成交处理

| 场景 | 处理 |
|------|------|
| 开盘涨停 | 放弃，不追涨停 |
| 快速涨停 | 涨停后不买入 |
| 涨停排队 | 排队等待，不强求 |

## 五、卖出规则（已锁定）

### 5.1 主卖出规则

| 项目 | 规则 |
|------|------|
| **卖出时间** | 次日开盘（09:35） |
| **卖出方式** | 市价单 |
| **特殊情况** | 若涨停则持有，第2天收盘强制卖 |

### 5.2 特殊卖出规则

| 场景 | 处理 |
|------|------|
| 涨停持有 | 次日涨停开盘 → 继续持有 |
| 低开止损 | 次日低开>-4% → 正常卖出 |
| 大幅止损 | 次日低开≤-4% → 立即止损 |
| 强制卖出 | 第2天收盘 → 强制卖出 |

### 5.3 日内止损（可选）

| 项目 | 规则 |
|------|------|
| **价格止损** | 日内亏损>3% → 立即卖出 |
| **时间止损** | 不采用（过于机械） |

## 七、风控机制（已锁定）

### 7.1 停手机制

| 项目 | 规则 |
|------|------|
| **触发条件** | 连亏3笔 |
| **停手时长** | 3天 |
| **恢复条件** | 3天后自动恢复 |
| **记录方式** | 每日更新停手天数 |

### 7.2 回撤熔断（可选）

| 项目 | 规则 |
|------|------|
| **单日回撤** | >5% → 当日停止交易 |
| **周回撤** | >8% → 暂停一周 |
| **月回撤** | >15% → 暂停一月并复盘 |

### 7.3 系统熔断（可选）

| 项目 | 规则 |
|------|------|
| **情绪数据缺失** | 保守空仓 |
| **数据异常** | 涨停家数突变 → 观望 |
| **系统故障** | 手动记录，事后补录 |

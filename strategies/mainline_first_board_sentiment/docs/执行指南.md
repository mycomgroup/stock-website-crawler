## 三、开盘买入（09:25-09:35）

### 3.1 竞价观察（09:25-09:30）

**监控目标**：
- 昨日涨停股名单
- 竞价价格变化
- 预估开盘涨幅

**计算开盘涨幅**：
```
开盘涨幅 = (竞价价 - 昨收价) / 昨收价 × 100%
```

### 3.2 筛选假弱高开（09:30-09:35）

**筛选条件清单**：
```
□ 昨日涨停收盘
□ 今日开盘+0.5%~+1.5%（假弱高开）
□ 流通市值50-150亿
□ 相对位置≤30%（15日内）
□ 近1日无涨停（非连板）
□ 成交额占比≤10%（可买入）
□ 情绪开关开启（涨停≥30）
□ 停手状态可交易（停手天数=0）
```

**优先级排序**：
| 优先级 | 条件 |
|--------|------|
| 1 | 假弱高开+0.5%~+1.5% |
| 2 | 流通市值50-80亿（偏小） |
| 3 | 相对位置≤20%（偏低） |

### 3.3 执行买入（09:35-09:40）

#### 步骤1：检查买入信号

```bash
# 检查开盘涨幅是否符合假弱高开
python execute_trade.py --mode buy --open-pct 1.2
```

**输出**：
```
✓ 假弱高开（+0.5%~+1.5%）
  建议买入
```

#### 步骤2：执行买入

```bash
python execute_trade.py --mode buy \
  --date 2026-03-31 \
  --stock 000001 \
  --name 平安银行 \
  --price 12.50 \
  --open-pct 1.2 \
  --备注 "假弱高开，流通市值80亿，相对位置15%"
```

**输出**：
```
✓ 买入记录已保存
  交易编号: T20260331001
  股票: 000001 平安银行
  价格: 12.50
  时间: 09:35
  状态: 持有中
```

## 四、盘中监控（09:35-14:50）

### 4.1 价格止损监控

**规则**：日内亏损>3% → 立即卖出

**监控方式**：
| 时间 | 操作 |
|------|------|
| 09:35-10:00 | 每5分钟查看价格 |
| 10:00-11:30 | 每15分钟查看价格 |
| 11:30-13:00 | 午休，12:50重启监控 |
| 13:00-14:50 | 每15分钟查看价格 |

**止损触发**：
```bash
# 若日内亏损>3%，立即卖出
python execute_trade.py --mode sell \
  --date 2026-03-31 \
  --stock 000001 \
  --price 12.10 \
  --备注 "日内止损，亏损-3.2%"
```

### 4.2 涨停监控

**规则**：若涨停 → 标记持有

**监控方式**：
- 每30分钟查看是否涨停
- 若涨停，标记"涨停持有"

**记录方式**：
```bash
python execute_trade.py --mode holding
```

**输出**：
```
当前持有股票:
  000001 平安银行
    买入日期: 2026-03-31
    买入价格: 12.50
    持有天数: 0
    当前状态: 涨停持有
```

### 4.3 禁止操作

| 操作 | 是否允许 |
|------|----------|
| 盘中买入新股票 | ✗ 禁止 |
| 加仓持有股票 | ✗ 禁止 |
| 追涨停 | ✗ 禁止 |
| 盘中止损卖出 | ✓ 允许（亏损>3%） |


```bash

  --date 2026-04-01 \
  --stock 000001 \
  --price 12.60
```

### 6.2 特殊卖出

| 场景 | 处理 |
|------|------|
| 涨停开盘 | 继续持有，第2天收盘强制卖 |
| 低开>-4% | 正常卖出 |
| 低开≤-4% | 立即止损卖出 |

### 6.3 记录盈亏

```bash
  --date 2026-04-01 \

  --date 2026-04-01 \
  --pnl -2.5
```

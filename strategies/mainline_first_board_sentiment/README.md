## 一、策略简介

**首板低开情绪择时策略**是一个机会仓高频策略，核心逻辑是捕捉首板股次日假弱高开的套利机会，通过情绪开关过滤弱市环境，实现稳定收益。

### 核心特点

| 特点 | 说明 |
|------|------|
| **信号明确** | 假弱高开（+0.5%~+1.5%）结构清晰 |
| **情绪过滤** | 涨停家数单指标硬开关，简单可执行 |
| **持仓短** | 1-2天，次日开盘卖出为主 |
| **风控完备** | 停手机制、止损、仓位控制 |
| **容量适中** | 适合≤500万小资金试盘 |

### 预期表现

| 指标 | 回测预期 |
|------|----------|
| 胜率 | 48.5% |
| 平均收益 | +0.77% |
| 年化收益 | ~20% |
| 最大回撤 | 15% |
| 卡玛比率 | 1.5 |

## 三、核心规则

### 3.1 买入条件

| 条件 | 参数值 |
|------|--------|
| 昨日状态 | 涨停收盘 |
| 今日开盘 | +0.5%~+1.5%（假弱高开） |
| 流通市值 | 50-150亿 |
| 相对位置 | ≤30%（15日内） |
| 连板状态 | 近1日无涨停 |
| 情绪开关 | 涨停≥30只 |
| 停手状态 | 可交易 |

### 3.2 卖出规则

| 规则 | 说明 |
|------|------|
| 主卖出 | 次日开盘（09:35） |
| 涨停持有 | 若涨停则继续持有 |
| 强制卖出 | 第2天收盘必须卖 |

### 3.3 风控机制

| 机制 | 规则 |
|------|------|
| 停手机制 | 连亏3笔停3天 |
| 日内止损 | 亏损>3%立即卖出 |
| 仓位控制 | 单票≤5%，总仓≤30% |

## 使用说明

### JoinQuant平台运行步骤

#### 步骤1：上传文件
将以下文件上传到JoinQuant平台同一目录：
```
tests/test_01_buy_timing.py
tests/test_02_cap_range.py
tests/test_03_first_board_feature.py
tests/test_04_emotion_threshold.py
tests/run_all_tests.py
```

#### 步骤2：运行测试
选择以下方式之一：

**方式A：一键运行**
```python
run run_all_tests.py
```

**方式B：分段运行**
```python
# 先运行买入时机和市值区间
run test_01_buy_timing.py
run test_02_cap_range.py

# 再运行首板特征和情绪阈值
run test_03_first_board_feature.py
run test_04_emotion_threshold.py
```

#### 步骤3：查看结果
测试结果会输出到日志中，包含：
- 样本内年化收益、胜率、回撤、夏普比率
- 样本外年化收益、胜率
- 信号数量
- 推荐参数

#### 步骤4：更新文档
根据实测结果更新：
```
买入信号规则文档_V1.0.md
```
